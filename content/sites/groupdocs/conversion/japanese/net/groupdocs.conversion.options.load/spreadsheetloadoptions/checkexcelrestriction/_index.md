## SpreadsheetLoadOptions.CheckExcelRestriction property

ユーザーがセル関連のオブジェクトを変更するときに、Excel ファイルの制限をチェックするかどうか。たとえば、Excel では 32K を超える文字列値を入力できません。 32K を超える値を入力すると、このプロパティが true の場合、例外が発生します。このプロパティが false の場合、入力文字列値がセルの値として受け入れられるため、後で CSV などの他のファイル形式の完全な文字列値を出力できます。ただし、Excel ファイル形式に無効な値を設定した場合は、後でワークブックを Excel ファイル形式で保存しないでください。そうしないと、生成された Excel ファイルに予期しないエラーが発生する可能性があります。

```csharp
public bool CheckExcelRestriction { get; set; }
```
