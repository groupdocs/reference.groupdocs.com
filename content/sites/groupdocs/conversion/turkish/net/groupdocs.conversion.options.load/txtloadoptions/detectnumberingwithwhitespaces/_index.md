## TxtLoadOptions.DetectNumberingWithWhitespaces property

Düz metin belgesi dönüştürüldüğünde numaralı liste öğelerinin nasıl tanınacağını belirlemeye izin verir. Varsayılan değer doğrudur.

```csharp
public bool DetectNumberingWithWhitespaces { get; set; }
```

### Notlar

Bu seçenek false olarak ayarlanırsa, liste numaraları nokta, sağ ayraç veya madde işareti sembolleriyle ("•", "*", "-" veya "o" gibi) ile bittiğinde, liste tanıma algoritması liste paragraflarını algılar.

Bu seçenek true olarak ayarlanırsa, boşluklar liste numarası sınırlayıcıları olarak da kullanılır: Arap stili numaralandırma için liste tanıma algoritması (1., 1.1.2.) hem boşlukları hem de nokta (".") sembollerini kullanır.
