## PresentationFileType class

Defines Presentation file formats that store collection of records to accommodate presentation data such as slides, shapes, text, animations, video, audio and embedded objects.
Includes the following file types:
[`PresentationFileType.Odp`](/conversion/python-net/groupdocs.conversion.filetypes/presentationfiletype),
[`PresentationFileType.Otp`](/conversion/python-net/groupdocs.conversion.filetypes/presentationfiletype),
[`PresentationFileType.Pot`](/conversion/python-net/groupdocs.conversion.filetypes/presentationfiletype),
[`PresentationFileType.Potm`](/conversion/python-net/groupdocs.conversion.filetypes/presentationfiletype),
[`PresentationFileType.Potx`](/conversion/python-net/groupdocs.conversion.filetypes/presentationfiletype),
[`PresentationFileType.Pps`](/conversion/python-net/groupdocs.conversion.filetypes/presentationfiletype),
[`PresentationFileType.Ppsm`](/conversion/python-net/groupdocs.conversion.filetypes/presentationfiletype),
[`PresentationFileType.Ppsx`](/conversion/python-net/groupdocs.conversion.filetypes/presentationfiletype),
[`PresentationFileType.Ppt`](/conversion/python-net/groupdocs.conversion.filetypes/presentationfiletype),
[`PresentationFileType.Pptm`](/conversion/python-net/groupdocs.conversion.filetypes/presentationfiletype),
[`PresentationFileType.Pptx`](/conversion/python-net/groupdocs.conversion.filetypes/presentationfiletype).
Learn more about Presentation formats [here](https://wiki.fileformat.com/presentation).
