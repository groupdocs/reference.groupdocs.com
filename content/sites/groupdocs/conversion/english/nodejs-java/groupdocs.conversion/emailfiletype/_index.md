## EmailFileType class

 Defines Email file formats that are used by email applications to store their various data including email messages, attachments, folders, address books etc.
 Includes the following file types:
  EmailFileType#Eml,
  EmailFileType#Emlx,
  EmailFileType#Mht,
  EmailFileType#Msg,
  EmailFileType#Vcf.
