## CadFileType class

 Defines CAD documents (Computer Aided Design) that are used for a 3D graphics file formats and may contain 2D or 3D designs.
 Includes the following types:
  CadFileType#Dgn,
  CadFileType#Dwf,
  CadFileType#Dwg,
  CadFileType#Dwt,
  CadFileType#Dxf,
  CadFileType#Ifc,
  CadFileType#Igs,
  CadFileType#Plt,
  CadFileType#Stl.
