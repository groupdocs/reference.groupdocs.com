## IConversionHandlersStage interface

Flattened conversion handlers stage. Allows setting `OnConversionCompleted` or `OnConversionFailed` in any order and any number of times, before proceeding to `Convert` / `Compress`. Events should be registered at the early stage via [`WithEvents`](../iconversionsettings/withevents) instead of in this stage.

```csharp
public interface IConversionHandlersStage : IConversionConvertOrCompress
```
