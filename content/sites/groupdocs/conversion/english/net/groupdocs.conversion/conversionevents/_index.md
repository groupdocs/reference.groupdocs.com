## ConversionEvents class

Aggregates conversion lifecycle event handlers. Pass an instance to the [`Converter`](../converter) constructor's `events` parameter or to the fluent `WithEvents` method. Prefer this over the individual [`ConverterSettings`](../convertersettings) handler properties, which are obsolete.
