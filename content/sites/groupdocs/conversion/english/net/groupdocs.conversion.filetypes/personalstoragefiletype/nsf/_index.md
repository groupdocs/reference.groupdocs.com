## PersonalStorageFileType.Nsf field

A file with .nsf (Notes Storage Facility) extension is a database file format used by the IBM Notes software, which was previously known as Lotus Notes. It defines the schema to store different kind of objects such like emails, appointments, documents, forms and views. Learn more about this file format [here](https://docs.fileformat.com/database/nsf).
