## CompressionFileType.Bz2 field

BZ2 are compressed files generated using the BZIP2 open source compression method, mostly on UNIX or Linux system. It is used for compression of a single file and is not meant for archiving of multiple files. Learn more about this file format [here](https://docs.fileformat.com/compression/bz2/).
