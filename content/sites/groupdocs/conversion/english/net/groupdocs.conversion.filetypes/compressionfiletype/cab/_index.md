## CompressionFileType.Cab field

A file with a .cab extension belongs to a windows cabinet file that belongs to the category of system files. It is a file that is saved in the archive file format in the versions of Microsoft Windows that support compressed data algorithms, such as the LZX, Quantum, and ZIP. Learn more about this file format [here](https://docs.fileformat.com/system/cab/).
