## FontFileType.Woff2 field

A file with .woff extension is a web font file based on the Web Open Font Format (WOFF). It has format-specific compressed container based on either TrueType (.TTF) or OpenType (.OTT) font types. Learn more about this file format [here](https://docs.fileformat.com/font/woff/).
