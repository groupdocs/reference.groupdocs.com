## ImageFileType.Heic field

An HEIC file is a High-Efficiency Container Image file format that can store multiple images as a collection in a single file. The format was adopted by Apple as variant of the HEIF with the launch of iOS 11. Learn more about this file format [here](https://docs.fileformat.com/image/heic/).
