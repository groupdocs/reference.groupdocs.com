## VideoFileType.Mp4 field

MP4(short for MPEG-4 Part 14) is a file format based on ISO/IEC 14496-12:2004 that is based on QuickTime File Format but formally specifies support for Initial Object Descriptors (IOD) and other MPEG features. Learn more about this file format [here](https://docs.fileformat.com/video/mp4/).
