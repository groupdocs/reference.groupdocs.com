## GisFileType.GeoJson field

GeoJSON is a JSON based format designed to represent the geographical features with their non-spatial attributes. This format defines different JSON (JavaScript Object Notation) objects and their joining fashion. JSON format represents a collective information about the Geographical features, their spatial extents, and properties. Learn more about this file format [here](https://docs.fileformat.com/gis/geojson/).
