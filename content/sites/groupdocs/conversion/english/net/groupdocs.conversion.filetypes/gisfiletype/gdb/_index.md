## GisFileType.Gdb field

ESRI file Geodatabase (FileGDB) is a collection of files in a folder on disc that hold related geospatial data such as feature datasets, feature classes and associated tables. It requires certain other files to be kept alongside the .gdb file in the same directory for it to work. Learn more about this file format [here](https://docs.fileformat.com/database/gdb/).
