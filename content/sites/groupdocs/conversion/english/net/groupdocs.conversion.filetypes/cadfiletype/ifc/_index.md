## CadFileType.Ifc field

Files with IFC extension refer to Industry Foundation Classes (IFC) file format that establishes international standards to import and export building objects and their properties. This file format provides interoperability between different software applications. Learn more about this file format [here](https://wiki.fileformat.com/cad/ifc).
