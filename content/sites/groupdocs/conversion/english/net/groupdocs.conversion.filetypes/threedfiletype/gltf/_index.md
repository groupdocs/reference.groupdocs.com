## ThreeDFileType.Gltf field

glTF (GL Transmission Format) is a 3D file format that stores 3D model information in JSON format. The use of JSON minimizes both the size of 3D assets and the runtime processing needed to unpack and use those assets. Learn more about this file format [here](https://docs.fileformat.com/3d/gltf).
