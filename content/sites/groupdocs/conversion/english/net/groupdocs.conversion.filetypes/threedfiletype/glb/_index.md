## ThreeDFileType.Glb field

GLB is the binary file format representation of 3D models saved in the GL Transmission Format (glTF). This binary format stores the glTF asset (JSON, .bin and images) in a binary blob. Learn more about this file format [here](https://docs.fileformat.com/3d/glb).
