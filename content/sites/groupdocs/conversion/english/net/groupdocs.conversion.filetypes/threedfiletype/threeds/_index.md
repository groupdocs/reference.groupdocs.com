## ThreeDFileType.ThreeDS field

A file with .3ds extension represents 3D Sudio (DOS) mesh file format used by Autodesk 3D Studio. Autodesk 3D Studio has been in 3D file format market since 1990s and has now evolved to 3D Studio MAX for working with 3D modeling, animation and rendering. Learn more about this file format [here](https://docs.fileformat.com/3d/3ds).
