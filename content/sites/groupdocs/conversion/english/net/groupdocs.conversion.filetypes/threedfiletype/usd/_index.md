## ThreeDFileType.Usd field

A file with .usd extension is a Universal Scene Description file format that encodes data for the purpose of data interchanging and augmenting between digital content creation applications. Developed by Pixar, USD provides the ability to interchange elemental assets (such as models) or animation. Learn more about this file format [here](https://docs.fileformat.com/3d/usd).
