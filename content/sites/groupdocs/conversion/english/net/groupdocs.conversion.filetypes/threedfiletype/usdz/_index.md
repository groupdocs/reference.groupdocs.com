## ThreeDFileType.Usdz field

A file with .usdz is an uncompressed and unencrypted ZIP archive for the USD (Universal Scene Description) file format that contains and proxies for files of other formats (such as textures, and animations) embedded within the archive and runs them directly with the USD run-time without any need of unpacking. Learn more about this file format [here](https://docs.fileformat.com/3d/usdz).
