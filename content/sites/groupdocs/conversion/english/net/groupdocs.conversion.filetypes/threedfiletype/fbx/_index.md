## ThreeDFileType.Fbx field

FBX, FilmBox, is a popular 3D file format that was originally developed by Kaydara for MotionBuilder. It was acquired by Autodesk Inc in 2006 and is now one of the main 3D exchange formats as used by many 3D tools. FBX is available in both binary and ASCII file format. Learn more about this file format [here](https://docs.fileformat.com/3d/fbx).
