## SpreadsheetFileType.Fods field

A file with .fods extension is a type of OpenDocument Spreadsheet document format that stores data in rows and columns. The format is specified as part of ODF 1.2 specifications published and maintained by OASIS. Learn more about this file format [here](https://wiki.fileformat.com/spreadsheet/fods).
