## ConvertOptionsProvider delegate

Describes delegate to provide convert options for specific source document. The delegate will be called before each conversion and provide a chance to provide specific convert options for desired target conversion. The decision could be made based on provided source file name and source file type.

```csharp
public delegate ConvertOptions ConvertOptionsProvider(string sourceDocumentName, 
    FileType sourceType);
```

| Parameter | Type | Description |
| --- | --- | --- |
| sourceDocumentName | String | Source file name |
| sourceType | FileType | Source file type |

### Return Value

Must return [`ConvertOptions`](../../groupdocs.conversion.options.convert/convertoptions) to be used for conversion of [`sourceType`](../../groupdocs.conversion.filetypes/filetype) document
