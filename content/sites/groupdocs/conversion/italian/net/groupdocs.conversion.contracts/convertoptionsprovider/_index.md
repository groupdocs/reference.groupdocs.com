## ConvertOptionsProvider delegate

Descrive il delegato per fornire opzioni di conversione per un documento di origine specifico. Il delegato verrà chiamato prima di ogni conversione e offrirà la possibilità di fornire opzioni di conversione specifiche per la conversione di destinazione desiderata. La decisione potrebbe essere presa in base al nome del file di origine e al file di origine forniti tipo.

```csharp
public delegate ConvertOptions ConvertOptionsProvider(string sourceDocumentName, 
    FileType sourceType);
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| sourceDocumentName | String | Nome del file di origine |
| sourceType | FileType | Tipo di file di origine |

### Valore di ritorno

Deve tornare[`ConvertOptions`](../../groupdocs.conversion.options.convert/convertoptions) da utilizzare per la conversione di[`sourceType`](../../groupdocs.conversion.filetypes/filetype) documento
