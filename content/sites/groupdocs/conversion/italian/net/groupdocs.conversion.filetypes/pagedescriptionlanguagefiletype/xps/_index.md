## PageDescriptionLanguageFileType.Xps field

Un file XPS rappresenta i file di layout di pagina basati su XML Paper Specifications creati da Microsoft. Questo formato è stato sviluppato da Microsoft in sostituzione del formato di file EMF ed è simile al formato di file PDF, ma utilizza XML nel layout, nell'aspetto e nelle informazioni di stampa di un documento.
