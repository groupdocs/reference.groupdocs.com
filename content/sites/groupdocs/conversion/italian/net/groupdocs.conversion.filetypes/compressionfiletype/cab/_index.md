## CompressionFileType.Cab field

Un file con estensione .cab appartiene a un file CAB di Windows che appartiene alla categoria dei file di sistema. È un file che viene salvato nel formato file di archivio nelle versioni di Microsoft Windows che supportano algoritmi di dati compressi, come LZX, Quantum e ZIP.
