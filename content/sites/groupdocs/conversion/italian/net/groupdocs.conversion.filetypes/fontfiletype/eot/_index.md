## FontFileType.Eot field

Un file con estensione .eot è un font OpenType incorporato in un documento. Questi sono utilizzati principalmente in file Web come una pagina Web. È stato creato da Microsoft ed è supportato dai prodotti Microsoft, incluso il file .pps di presentazione PowerPoint.
