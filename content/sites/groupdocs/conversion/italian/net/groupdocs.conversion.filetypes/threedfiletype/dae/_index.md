## ThreeDFileType.Dae field

Un file DAE è un formato di file Digital Asset Exchange utilizzato per lo scambio di dati tra applicazioni 3D interattive. Questo formato di file si basa sullo schema XML COLLADA (COLLAborative Design Activity), che è uno schema XML standard aperto per lo scambio di risorse digitali tra applicazioni software di grafica.
