## ImageFileType.Svgz field

Un file SVGZ è in realtà una versione compressa di un file SVG. Ciò consente una distribuzione più semplice del file online. Quando un file SVG viene compresso utilizzando l'algoritmo di compressione .GZIP, gli viene assegnata l'estensione del file .svgz.
