## ImageFileType.Psb field

Adobe Photoshop salva i file in due formati. I file con dimensioni di 30.000 x 30.000 pixel vengono salvati con l'estensione PSD e i file più grandi di PSD fino a 300.000 x 300.000 pixel vengono salvati con l'estensione PSB nota come "Photoshop Big".
