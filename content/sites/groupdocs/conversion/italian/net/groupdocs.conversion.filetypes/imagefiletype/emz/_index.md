## ImageFileType.Emz field

Un file EMZ è in realtà una versione compressa di un file Microsoft EMF. Ciò consente una più facile distribuzione del file online. Quando un file EMF viene compresso utilizzando l'algoritmo di compressione .GZIP, gli viene assegnata l'estensione file .emz.
