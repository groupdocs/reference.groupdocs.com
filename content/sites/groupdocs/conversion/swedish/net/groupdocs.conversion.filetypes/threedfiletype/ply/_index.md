## ThreeDFileType.Ply field

PLY, Polygon File Format, representerar 3D-filformat som lagrar grafiska objekt som beskrivs som en samling polygoner. Syftet med detta filformat var att skapa en enkel och enkel filtyp som är tillräckligt allmän för att vara användbar för ett brett utbud av modeller.
