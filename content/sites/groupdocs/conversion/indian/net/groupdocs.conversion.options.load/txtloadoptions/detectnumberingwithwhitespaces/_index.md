## TxtLoadOptions.DetectNumberingWithWhitespaces property

यह निर्दिष्ट करने की अनुमति देता है कि सादा पाठ दस्तावेज़ परिवर्तित होने पर क्रमांकित सूची आइटम कैसे पहचाने जाते हैं। डिफ़ॉल्ट मान सत्य है।

```csharp
public bool DetectNumberingWithWhitespaces { get; set; }
```

### टिप्पणियों

यदि यह विकल्प गलत पर सेट है, तो सूची पहचान एल्गोरिथ्म सूची पैराग्राफ का पता लगाता है, जब सूची संख्या के साथ समाप्त होती है या तो डॉट, राइट ब्रैकेट या बुलेट प्रतीक (जैसे "•", "*", "-" या "o")।

यदि यह विकल्प सही पर सेट है, तो व्हाइटस्पेस का उपयोग सूची संख्या सीमांकक के रूप में भी किया जाता है: अरबी शैली नंबरिंग (1., 1.1.2.) के लिए सूची पहचान एल्गोरिदम व्हाइटस्पेस और डॉट ("।") दोनों प्रतीकों का उपयोग करता है।
