## ThreeDFileType.Ply field

PLY, Polygon File Format, representa un formato de archivo 3D que almacena objetos gráficos descritos como una colección de polígonos. El propósito de este formato de archivo fue establecer un tipo de archivo simple y fácil que sea lo suficientemente general como para ser útil para una amplia gama de modelos.
