## MarkupFileType.Mht field

Los archivos con extensión MHTML representan un formato de archivo de página web que pueden crear varias aplicaciones diferentes. El formato se conoce como formato de archivo porque guarda el código HTML web y los recursos asociados en un solo archivo. Más información sobre este formato de archivo[aquí](https://wiki.fileformat.com/web/mhtml) .
