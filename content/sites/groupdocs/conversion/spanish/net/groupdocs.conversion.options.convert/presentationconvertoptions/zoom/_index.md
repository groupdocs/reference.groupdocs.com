## PresentationConvertOptions.Zoom property

Especifica el nivel de zoom en porcentaje. El valor predeterminado es 100. El zoom predeterminado es compatible hasta Microsoft Powerpoint 2010. A partir de Microsoft Powerpoint 2013, el zoom predeterminado ya no se establece en el documento, sino que parece usar el factor de zoom del último documento que se abrió.
