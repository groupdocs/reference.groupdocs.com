## WordProcessingConvertOptions.Zoom property

Especifica el nivel de zoom en porcentaje. El valor predeterminado es 100. El zoom predeterminado es compatible hasta Microsoft Word 2010. A partir de Microsoft Word 2013, el zoom predeterminado ya no está configurado para documento, sino que parece usar el factor de zoom del último documento que se abrió.
