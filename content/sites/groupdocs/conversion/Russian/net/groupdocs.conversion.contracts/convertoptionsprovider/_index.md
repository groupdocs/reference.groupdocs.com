## ConvertOptionsProvider delegate

Описывает делегата для предоставления параметров преобразования для определенного исходного документа. Делегат будет вызываться перед каждым преобразованием и предоставлять возможность предоставить конкретные параметры преобразования для желаемого целевого преобразования. Решение может быть принято на основе предоставленного имени исходного файла и исходного файла. тип.

```csharp
public delegate ConvertOptions ConvertOptionsProvider(string sourceDocumentName, 
    FileType sourceType);
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| sourceDocumentName | String | Имя исходного файла |
| sourceType | FileType | Тип исходного файла |
