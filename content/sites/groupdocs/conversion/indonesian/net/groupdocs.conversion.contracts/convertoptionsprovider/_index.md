## ConvertOptionsProvider delegate

Menjelaskan delegasi untuk memberikan opsi konversi untuk dokumen sumber tertentu. Delegasi akan dipanggil sebelum setiap konversi dan memberikan kesempatan untuk memberikan opsi konversi khusus untuk konversi target yang diinginkan. Keputusan dapat dibuat berdasarkan nama file sumber dan file sumber yang diberikan ketik.

```csharp
public delegate ConvertOptions ConvertOptionsProvider(string sourceDocumentName, 
    FileType sourceType);
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| sourceDocumentName | String | Nama file sumber |
| sourceType | FileType | Jenis file sumber |
