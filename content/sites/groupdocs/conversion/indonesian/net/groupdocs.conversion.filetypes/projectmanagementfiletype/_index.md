## ProjectManagementFileType class

Menentukan format file Proyek yang dibuat oleh perangkat lunak Manajemen Proyek seperti Microsoft Project, Primavera P6 dll. File proyek adalah kumpulan tugas, sumber daya, dan penjadwalannya untuk mendapatkan keluaran terukur dalam bentuk atau produk atau layanan. Dokumen manajemen proyek. Termasuk jenis file berikut: [`Mpp`](./mpp) , [`Mpt`](./mpt) , [`Mpx`](./mpx) . Pelajari lebih lanjut tentang format Manajemen Proyek[Di Sini](https://wiki.fileformat.com/project-management) .
