## CompressionFileType.Lzma field

File dengan ekstensi .lzma adalah file arsip terkompresi yang dibuat menggunakan metode kompresi LZMA (Lempel-Ziv-Markov chain Algorithm). Ini terutama ditemukan/digunakan pada sistem operasi Unix dan mirip dengan algoritma kompresi lainnya seperti ZIP untuk meminimalkan ukuran file.
