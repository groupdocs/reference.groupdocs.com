## CompressionFileType.Lz field

File dengan ekstensi .lz adalah file arsip terkompresi yang dibuat dengan Lzip, yang merupakan alat baris perintah gratis untuk kompresi. Ini mendukung penggabungan untuk mengompres file dukungan. File LZ memiliki aplikasi jenis media/lzip dan mendukung rasio kompresi yang lebih tinggi daripada BZ2.
