## EBookFileType.Azw3 field

AZW3, juga dikenal sebagai Kindle Format 8 (KF8), adalah versi modifikasi dari format file digital ebook AZW yang dikembangkan untuk perangkat Amazon Kindle. Format ini merupakan penyempurnaan dari file AZW lama dan digunakan pada perangkat Kindle Fire hanya dengan kompatibilitas mundur untuk format file leluhur yaitu MOBI dan AZW.
