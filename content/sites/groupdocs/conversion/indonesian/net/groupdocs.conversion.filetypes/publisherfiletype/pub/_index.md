## PublisherFileType.Pub field

File PUB adalah format file dokumen Microsoft Publisher. Ini digunakan untuk membuat beberapa jenis dokumen tata letak desain seperti buletin, selebaran, brosur, kartu pos, dll. File PUB dapat berisi teks, raster, dan gambar vektor. Pelajari lebih lanjut tentang format file ini[Di Sini](https://docs.fileformat.com/publisher/pub/) .
