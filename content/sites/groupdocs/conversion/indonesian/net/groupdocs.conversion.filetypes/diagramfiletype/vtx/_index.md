## DiagramFileType.Vtx field

File dengan ekstensi VTX adalah template gambar Microsoft Visio yang disimpan ke disk dalam format file XML. Template ditujukan untuk menyediakan file dengan pengaturan dasar yang dapat digunakan untuk membuat beberapa file Visio dengan pengaturan yang sama.
