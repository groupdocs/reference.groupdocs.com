## ThreeDFileType.Usd field

File dengan ekstensi .usd adalah format file Deskripsi Pemandangan Universal yang menyandikan data untuk tujuan pertukaran dan penambahan data di antara aplikasi pembuatan konten digital. Dikembangkan oleh Pixar, USD memberikan kemampuan untuk menukar aset elemen (seperti model) atau animasi. Pelajari lebih lanjut tentang format file ini[Di Sini](https://docs.fileformat.com/3d/usd) .
