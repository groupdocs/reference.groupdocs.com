## ImageFileType.Svgz field

File SVGZ sebenarnya adalah versi terkompresi dari file SVG. Ini memungkinkan distribusi file secara online dengan lebih mudah. Saat file SVG dikompres menggunakan algoritme kompresi .GZIP, file tersebut kemudian diberi ekstensi file .svgz.
