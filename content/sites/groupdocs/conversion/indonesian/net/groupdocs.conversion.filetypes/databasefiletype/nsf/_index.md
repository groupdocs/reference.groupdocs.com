## DatabaseFileType.Nsf field

File dengan ekstensi .nsf (Fasilitas Penyimpanan Catatan) adalah format file database yang digunakan oleh perangkat lunak IBM Notes, yang sebelumnya dikenal sebagai Lotus Notes. Ini mendefinisikan skema untuk menyimpan berbagai jenis objek seperti email, janji temu, dokumen, formulir, dan tampilan.
