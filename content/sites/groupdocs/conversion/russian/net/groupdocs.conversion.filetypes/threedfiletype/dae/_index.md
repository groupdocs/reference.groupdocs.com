## ThreeDFileType.Dae field

Файл DAE представляет собой формат файла обмена цифровыми активами, который используется для обмена данными между интерактивными 3D-приложениями. Этот формат файла основан на XML-схеме COLLADA (COLLAborative Design Activity), которая представляет собой открытую стандартную XML-схему для обмена цифровыми активами между графическими приложениями.
