## DiagramFileType.Vtx field

Файл с расширением VTX представляет собой шаблон чертежа Microsoft Visio, который сохраняется на диск в формате файла XML. Шаблон предназначен для предоставления файла с основными настройками, который можно использовать для создания нескольких файлов Visio с одинаковыми настройками.
