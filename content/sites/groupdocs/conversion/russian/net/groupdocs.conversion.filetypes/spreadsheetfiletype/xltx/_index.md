## SpreadsheetFileType.Xltx field

Файл XLTX представляет собой шаблон Microsoft Excel, основанный на спецификациях формата файлов Office OpenXML. Он используется для создания стандартного файла шаблона, который можно использовать для создания файлов XLSX с теми же настройками, что и в файле XLTX.
