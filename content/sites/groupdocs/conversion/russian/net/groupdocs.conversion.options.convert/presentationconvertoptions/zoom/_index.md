## PresentationConvertOptions.Zoom property

Определяет уровень масштабирования в процентах. Значение по умолчанию: 100. Масштаб по умолчанию поддерживается до Microsoft Powerpoint 2010. Начиная с Microsoft Powerpoint 2013 масштаб по умолчанию больше не устанавливается на документ, вместо этого используется коэффициент масштабирования последнего открытого документа.

public int Zoom { get; set; }
