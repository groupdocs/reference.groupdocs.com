## ImageFileType.Svgz field

Eine SVGZ-Datei ist eigentlich eine komprimierte Version einer SVG-Datei. Dies ermöglicht eine einfachere Verteilung der Datei online. Wenn eine SVG-Datei mit dem .GZIP-Komprimierungsalgorithmus komprimiert wird, erhält sie die Dateierweiterung .svgz.
