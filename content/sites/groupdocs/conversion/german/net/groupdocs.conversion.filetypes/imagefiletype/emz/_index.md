## ImageFileType.Emz field

Eine EMZ-Datei ist eigentlich eine komprimierte Version einer Microsoft EMF-Datei. Dies ermöglicht eine einfachere Verteilung der Datei online. Wenn eine EMF-Datei mit dem .GZIP-Komprimierungsalgorithmus komprimiert wird, erhält sie die Dateierweiterung .emz.
