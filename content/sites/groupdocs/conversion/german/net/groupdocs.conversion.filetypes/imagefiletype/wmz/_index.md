## ImageFileType.Wmz field

Eine WMZ-Datei ist eigentlich eine komprimierte Version einer Microsoft WMF-Datei. Dies ermöglicht eine einfachere Verteilung der Datei online. Wenn eine EWMFMF-Datei mit dem .GZIP-Komprimierungsalgorithmus komprimiert wird, erhält sie die Dateierweiterung .wmz.
