## DatabaseFileType.Sql field

Eine Datei mit der Erweiterung .sql ist eine SQL-Datei (Structured Query Language), die Code für die Arbeit mit relationalen Datenbanken enthält. Es wird verwendet, um SQL-Anweisungen für CRUD-Operationen (Create, Read, Update, and Delete) in Datenbanken zu schreiben.
