## TxtLoadOptions.DetectNumberingWithWhitespaces property

일반 텍스트 문서가 변환될 때 번호가 매겨진 목록 항목을 인식하는 방법을 지정할 수 있습니다. 기본값은 true입니다.

```csharp
public bool DetectNumberingWithWhitespaces { get; set; }
```

### 비고

이 옵션을 false로 설정하면 목록 번호가 점, 오른쪽 대괄호 또는 글머리 기호(예: "•", "*", "-" 또는 "o")로 끝나는 경우 목록 인식 알고리즘이 목록 단락을 감지합니다.

이 옵션을 true로 설정하면 공백이 목록 번호 구분 기호로도 사용됩니다. 아랍어 스타일 번호 매기기(1., 1.1.2.)에 대한 목록 인식 알고리즘은 공백과 점(".") 기호를 모두 사용합니다.
