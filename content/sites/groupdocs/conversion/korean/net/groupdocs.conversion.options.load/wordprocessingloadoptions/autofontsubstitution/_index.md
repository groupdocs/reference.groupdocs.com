## WordProcessingLoadOptions.AutoFontSubstitution property

AutoFontSubstitution이 비활성화된 경우 GroupDocs.Conversion은 누락된 글꼴을 대체하기 위해 DefaultFont를 사용합니다. AutoFontSubstitution이 활성화된 경우 GroupDocs.Conversion은 누락된 글꼴에 대한 FontInfo의 모든 관련 필드(Panose, Sig 등)를 평가하고 사용 가능한 글꼴 소스 중에서 가장 근접한 일치 항목을 찾습니다. 글꼴 대체 메커니즘은 누락된 글꼴에 대한 FontInfo는 문서에서 사용할 수 있습니다. 기본값은 True입니다.

```csharp
public bool AutoFontSubstitution { get; set; }
```
