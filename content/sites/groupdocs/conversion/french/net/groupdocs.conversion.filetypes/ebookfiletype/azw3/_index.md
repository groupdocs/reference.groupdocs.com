## EBookFileType.Azw3 field

AZW3, également connu sous le nom de Kindle Format 8 (KF8), est la version modifiée du format de fichier numérique AZW ebook développé pour les appareils Amazon Kindle. Le format est une amélioration des anciens fichiers AZW et est utilisé sur les appareils Kindle Fire uniquement avec une rétrocompatibilité pour le format de fichier ancêtre, c'est-à-dire MOBI et AZW. En savoir plus sur ce format de fichier[ici](https://docs.fileformat.com/ebook/azw3/) .
