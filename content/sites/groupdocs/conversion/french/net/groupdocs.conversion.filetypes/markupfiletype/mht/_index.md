## MarkupFileType.Mht field

Les fichiers avec l'extension MHTML représentent un format d'archive de page Web qui peut être créé par un certain nombre d'applications différentes. Le format est connu sous le nom de format d'archive car il enregistre le code HTML Web et les ressources associées dans un seul fichier.
