## ThreeDFileType.Amf field

Un fichier AMF est constitué de lignes directrices pour la description des objets afin d'être utilisés par les processus de fabrication additive. Il contient une balise XML d'ouverture et se termine par un élément. Elle est précédée d'une ligne de déclaration XML spécifiant la version XML et l'encodage du fichier.
