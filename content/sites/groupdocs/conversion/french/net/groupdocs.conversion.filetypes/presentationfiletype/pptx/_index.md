## PresentationFileType.Pptx field

Les fichiers avec l'extension PPTX sont des fichiers de présentation créés avec l'application Microsoft PowerPoint populaire. Contrairement à la version précédente du format de fichier de présentation PPT qui était binaire, le format PPTX est basé sur le format de fichier de présentation XML ouvert de Microsoft PowerPoint.
