## DatabaseFileType.Nsf field

Un fichier avec l'extension .nsf (Notes Storage Facility) est un format de fichier de base de données utilisé par le logiciel IBM Notes, qui était auparavant connu sous le nom de Lotus Notes. Il définit le schéma pour stocker différents types d'objets tels que les e-mails, les rendez-vous, les documents, les formulaires et les vues.
