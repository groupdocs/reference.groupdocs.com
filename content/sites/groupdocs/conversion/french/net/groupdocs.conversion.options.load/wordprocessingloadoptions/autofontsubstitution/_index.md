## WordProcessingLoadOptions.AutoFontSubstitution property

Si AutoFontSubstitution est désactivé, GroupDocs.Conversion utilise DefaultFont pour la substitution des polices manquantes. Si AutoFontSubstitution est activé, GroupDocs.Conversion évalue tous les champs associés dans FontInfo (Panose, Sig, etc.) pour la police manquante et trouve la correspondance la plus proche parmi les sources de polices disponibles. Notez que le mécanisme de substitution de police remplacera DefaultFont dans les cas où FontInfo pour la police manquante est disponible dans le document. La valeur par défaut est True.
