## WordProcessingBookmarksOptions.ExpandedOutlineLevels property

Spécifie le nombre de niveaux dans le plan du document à afficher développés lorsque le fichier est affiché. La valeur par défaut est 0. La plage valide est de 0 à 9. Notez que cette option ne fonctionnera pas lors de l'enregistrement dans XPS.
