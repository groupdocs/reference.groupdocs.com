## TxtLoadOptions.DetectNumberingWithWhitespaces property

Επιτρέπει τον καθορισμό του τρόπου με τον οποίο αναγνωρίζονται τα αριθμημένα στοιχεία λίστας κατά τη μετατροπή εγγράφου απλού κειμένου. Η προεπιλεγμένη τιμή είναι αληθής.

```csharp
public bool DetectNumberingWithWhitespaces { get; set; }
```

### Παρατηρήσεις

Εάν αυτή η επιλογή έχει οριστεί σε false, ο αλγόριθμος αναγνώρισης παραθέτει τις παραγράφους της λίστας, όταν οι αριθμοί λίστας τελειώνουν με σύμβολα είτε κουκκίδας είτε δεξιάς αγκύλης ή κουκκίδας (όπως "•", "*", "-" ή "o").

Εάν αυτή η επιλογή οριστεί σε true, τα κενά διαστήματα χρησιμοποιούνται επίσης ως διαχωριστές αριθμών λίστας: αλγόριθμος αναγνώρισης λίστας για αρίθμηση σε αραβικό στυλ (1., 1.1.2.) χρησιμοποιεί σύμβολα και κενά και κουκκίδες (".").
