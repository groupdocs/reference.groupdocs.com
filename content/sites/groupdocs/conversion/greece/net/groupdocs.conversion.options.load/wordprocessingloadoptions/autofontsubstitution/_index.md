## WordProcessingLoadOptions.AutoFontSubstitution property

Εάν η AutoFontSubstitution είναι απενεργοποιημένη, το GroupDocs.Conversion χρησιμοποιεί την DefaultFont για την αντικατάσταση των γραμματοσειρών που λείπουν. Εάν το AutoFontSubstitution είναι ενεργοποιημένο, το GroupDocs.Conversion αξιολογεί όλα τα σχετικά πεδία στο FontInfo (Panose, Sig κ.λπ.) για τη γραμματοσειρά που λείπει και βρίσκει την πλησιέστερη αντιστοίχιση μεταξύ των διαθέσιμων πηγών γραμματοσειράς. Σημειώστε ότι ο μηχανισμός αντικατάστασης γραμματοσειράς θα παρακάμψει τις προεπιλεγμένες περιπτώσεις Το FontInfo για τη γραμματοσειρά που λείπει είναι διαθέσιμο στο έγγραφο. Η προεπιλεγμένη τιμή είναι True.

```csharp
public bool AutoFontSubstitution { get; set; }
```
