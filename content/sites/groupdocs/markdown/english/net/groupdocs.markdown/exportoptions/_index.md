### Remarks

Use this class with [`FromMarkdown`](../markdownconverter/frommarkdown) to control how the Markdown content is converted to a document format such as DOCX or PDF.

### Examples

```csharp
var options = new ExportOptions
{
    Format = FileFormat.Pdf
};
MarkdownConverter.FromMarkdown("input.md", "output.pdf", options);
```
