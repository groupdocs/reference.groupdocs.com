## ExportImagesToFileSystemStrategy.ImagesRelativePath property

Gets or sets the path used in the Markdown image references. When `null` or empty, the full [`ImagesFolder`](../imagesfolder) path is used (which may be absolute). Set this to a relative path (e.g. `"images"`) so the Markdown output contains portable references.

```csharp
public string ImagesRelativePath { get; set; }
```

### Property Value

Default is `null` (uses [`ImagesFolder`](../imagesfolder) as-is).
