### Remarks

During document conversion to Markdown, the library creates an instance of this class for each external resource. You can modify the [`ResourceFileName`](./resourcefilename) or [`ResourceFileUri`](./resourcefileuri) properties to customize the resulting link that will be written to Markdown output.
