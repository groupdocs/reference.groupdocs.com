### Remarks

The MarkdownConverter class provides functionality to convert documents from various formats to Markdown. It supports conversion from popular document formats like Word (DOCX, DOC) and Excel (XLSX, XLS). The conversion process can be customized using [`DocumentConverterOptions`](../documentconverteroptions).

### Examples

The following example shows how to convert a Word document to Markdown:

```csharp
using (var converter = new MarkdownConverter("document.docx"))
{
    var result = converter.Convert();
    if (result.IsSuccess)
    {
        string markdown = result.Content;
        Console.WriteLine(markdown);
    }
}
```

The following example shows how to convert a password-protected document:

```csharp
var streamInfo = new StreamInfo
{
    Extension = ".docx",
    Password = "password123"
};

using (var converter = new MarkdownConverter("protected.docx", streamInfo))
{
    var result = converter.Convert();
    if (result.IsSuccess)
    {
        string markdown = result.Content;
        Console.WriteLine(markdown);
    }
}
```
