## UriSavingArgs class

Provides information and controls for customizing how a resource URI is written during document-to-Markdown conversion.

Call [`UriSavingArgs.set_resource_file_uri`](/markdown/python-net/groupdocs.markdown/urisavingargs/set_resource_file_uri/) to change the URI that appears in the Markdown output.
If no override is set, the library uses the default [`UriSavingArgs.resource_file_uri`](/markdown/python-net/groupdocs.markdown/urisavingargs/resource_file_uri/).

The UriSavingArgs type exposes the following members:

### Methods
| Method | Description |
| :- | :- |
| [set_resource_file_uri](/markdown/python-net/groupdocs.markdown/urisavingargs/set_resource_file_uri/#resource_file_uri) | Overrides the URI that will be written to the Markdown output for this resource. |

### Properties
| Property | Description |
| :- | :- |
| [resource_file_name](/markdown/python-net/groupdocs.markdown/urisavingargs/resource_file_name/) | The default file name (without path) generated by the library for this resource. |
| [resource_file_name_output](/markdown/python-net/groupdocs.markdown/urisavingargs/resource_file_name_output/) | The overridden file name set by the caller, or None if no override was specified. |
| [resource_file_uri](/markdown/python-net/groupdocs.markdown/urisavingargs/resource_file_uri/) | The default resource URI that the library would write into the Markdown output. |
| [resource_file_uri_output](/markdown/python-net/groupdocs.markdown/urisavingargs/resource_file_uri_output/) | The overridden URI set by [`UriSavingArgs.set_resource_file_uri`](/markdown/python-net/groupdocs.markdown/urisavingargs/set_resource_file_uri/), or `None` if no override was specified. |
