## IImageSavingHandler class

Callback interface invoked for each image encountered during conversion when using [`CustomImagesStrategy`](/markdown/python-net/groupdocs.markdown/customimagesstrategy/).

Implement this interface to rename images, redirect output to a custom stream, or apply other custom logic.
