## UriExportContext class

Contains information about a resource URI being written during conversion. Passed to `IUriExportStrategy.update_resource_uri`.

Read the default values from [`UriExportContext.resource_file_name`](/python-net/groupdocs.markdown/uriexportcontext/resource_file_name/) and [`UriExportContext.resource_file_uri`](/python-net/groupdocs.markdown/uriexportcontext/resource_file_uri/), then call [`UriExportContext.set_resource_file_name`](/python-net/groupdocs.markdown/uriexportcontext/set_resource_file_name/) or [`UriExportContext.set_resource_file_uri`](/python-net/groupdocs.markdown/uriexportcontext/set_resource_file_uri/) to override them.

The UriExportContext type exposes the following members:

### Methods
| Method | Description |
| :- | :- |
| [set_resource_file_name](/python-net/groupdocs.markdown/uriexportcontext/set_resource_file_name/#resource_file_name) | Overrides the resource file name in the Markdown output. |
| [set_resource_file_uri](/python-net/groupdocs.markdown/uriexportcontext/set_resource_file_uri/#resource_file_uri) | Overrides the URI that will appear in the Markdown output. |

### Properties
| Property | Description |
| :- | :- |
| [resource_file_name](/python-net/groupdocs.markdown/uriexportcontext/resource_file_name/) | The default resource file name generated by the library. |
| [resource_file_name_output](/python-net/groupdocs.markdown/uriexportcontext/resource_file_name_output/) | The overridden file name, or None if no override was set. |
| [resource_file_uri](/python-net/groupdocs.markdown/uriexportcontext/resource_file_uri/) | The default URI that the library would write into the Markdown output. |
| [resource_file_uri_output](/python-net/groupdocs.markdown/uriexportcontext/resource_file_uri_output/) | The overridden URI, or None if no override was set. |
