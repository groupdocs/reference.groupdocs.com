## images_relative_path property

The path used in the Markdown image references.

When `None` or empty, the full [`CustomImagesStrategy.images_folder`](/python-net/groupdocs.markdown/customimagesstrategy/images_folder/) path is used.
Set this to a relative path (e.g. `"images"`) for portable Markdown output.
