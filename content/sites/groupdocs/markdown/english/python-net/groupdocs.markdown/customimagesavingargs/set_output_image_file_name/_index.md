## set_output_image_file_name {#file_name}

Overrides the default file name for this image. The image will be saved under the specified name instead of the library-generated [`CustomImageSavingArgs.image_file_name`](/python-net/groupdocs.markdown/customimagesavingargs/image_file_name/).

```python
def set_output_image_file_name(self, file_name):
    ...
```

| Parameter | Type | Description |
| :- | :- | :- |
| file_name | `str` | The new file name (without path) to use for the image. |
