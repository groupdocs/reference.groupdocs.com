## CustomUriExportStrategy class

Implements a URI export strategy that lets you customize how resource URIs are written to Markdown.

Supply a [`IUriSavingHandler`](/markdown/python-net/groupdocs.markdown/iurisavinghandler/) implementation to rewrite resource URIs (for example, to prepend a CDN base URL).
