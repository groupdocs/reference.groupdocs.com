名前空間は、さまざまな形式のドキュメント内のテキスト データのインデックス作成と検索のためのクラスを提供します。 この名前空間の主なクラスは次のとおりです:

## クラス

| クラス | 説明 |
| --- | --- |
| [Extractor](./extractor) | 後続の高速索引付けの段階を分離するために、文書から事前にデータを抽出するためのツールを表します。 |
| [Index](./index) | ドキュメントのインデックス作成と検索を行うためのメイン クラスを表します。 |
| [IndexRepository](./indexrepository) | 複数のインデックスを結合し、それらに対して共通の操作を実行するためのコンテナーを表します。 |
| [IndexSettings](./indexsettings) | インデックス作成操作をカスタマイズできるインデックス設定を表します。 |
| [IndexUpdater](./indexupdater) | インデックス アップデーターを表します。 このクラスは、古いバージョンのインデックスでドキュメントの再インデックスを実行します。 |
| [License](./license) | ライセンスを適用する方法を提供します。 |
| [Metered](./metered) | 従量制ライセンスで製品をアクティベートし、処理された MB の量を取得できるメソッドを提供します。 従量制ライセンスの詳細[ここ](https://purchase.groupdocs.com/faqs/licensing/metered). |
| [SearchQuery](./searchquery) | 検索クエリをオブジェクト形式で表します。 |
