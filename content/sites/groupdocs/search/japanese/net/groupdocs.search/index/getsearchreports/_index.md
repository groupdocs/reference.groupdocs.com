## Index.GetSearchReports method

検索操作に関するレポートを取得します。

```csharp
public SearchReport[] GetSearchReports()
```

### 戻り値

検索レポート。

### 例

この例は、検索レポートを取得する方法を示しています.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
string query1 = "Einstein";
string query2 = "Newton";

Index index = new Index(indexFolder); // 指定したフォルダにインデックスを作成
index.Add(documentsFolder); // 指定されたフォルダからのドキュメントのインデックス作成

SearchResult result1 = index.Search(query1); // 検索中
SearchResult result2 = index.Search(query2);
SearchResult result3 = index.Search(query1 + " & " + query2);

SearchReport[] reports = index.GetSearchReports(); // 検索レポートを取得する
```
