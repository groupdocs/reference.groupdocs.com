## IndexSettings.TextStorageSettings property

テキスト ストレージ設定を取得または設定します。 デフォルト値は`ヌル`、つまり、ドキュメントのテキストは保存されません.

```csharp
public TextStorageSettings TextStorageSettings { get; set; }
```

### プロパティ値

テキストの保存設定。

### 例

この例は、テキスト ストレージ設定の設定方法を示しています。

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

// インデックス設定インスタンスの作成
IndexSettings settings = new IndexSettings();
settings.TextStorageSettings = new TextStorageSettings(Compression.High); // インデックス テキスト ストレージの圧縮レベルを高く設定する

// 指定したフォルダにインデックスを作成
Index index = new Index(indexFolder, settings);

// ドキュメントのインデックス作成
index.Add(documentsFolder);

// 検索中
SearchResult result = index.Search("Einstein");
```
