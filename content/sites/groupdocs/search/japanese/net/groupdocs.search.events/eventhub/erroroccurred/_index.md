## EventHub.ErrorOccurred event

インデックス操作中にエラーが発生した場合に発生します。

### 例

この例は、イベントの使用方法を示しています。

```csharp

string query = "Einstein";

// インデックスを作成する

// イベントへのサブスクライブ
index.Events.ErrorOccurred += (sender, args) =>

};

// 指定されたフォルダからのドキュメントのインデックス作成

// インデックスで検索
```
