## EventHub.StatusChanged event

Se produit lorsque l'état de l'index change.

```csharp
public event EventHandler<BaseIndexEventArgs> StatusChanged;
```

### Exemples

L'exemple montre comment utiliser l'événement.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

// Création d'un index
Index index = new Index(indexFolder);

// S'inscrire à l'événement
index.Events.StatusChanged += (sender, args) =>
{
    if (args.Status != IndexStatus.InProgress)
    {
        // Une notification de l'achèvement de l'opération devrait être ici
    }
};

// Définition du drapeau pour l'indexation asynchrone
IndexingOptions options = new IndexingOptions();
options.IsAsync = true;

// Documents d'indexation asynchrone à partir du dossier spécifié
// La méthode se termine avant la fin de l'opération
index.Add(documentsFolder, options);
```
