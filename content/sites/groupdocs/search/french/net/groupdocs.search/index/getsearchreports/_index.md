## Index.GetSearchReports method

Obtient les rapports sur les opérations de recherche.

```csharp
public SearchReport[] GetSearchReports()
```

### Return_Value

Les rapports de recherche.

### Exemples

L'exemple montre comment obtenir des rapports de recherche.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
string query1 = "Einstein";
string query2 = "Newton";

Index index = new Index(indexFolder); // Création d'un index dans le dossier spécifié
index.Add(documentsFolder); // Indexation des documents du dossier spécifié

SearchResult result1 = index.Search(query1); // Recherche
SearchResult result2 = index.Search(query2);
SearchResult result3 = index.Search(query1 + " & " + query2);

SearchReport[] reports = index.GetSearchReports(); // Obtenir des rapports de recherche
```
