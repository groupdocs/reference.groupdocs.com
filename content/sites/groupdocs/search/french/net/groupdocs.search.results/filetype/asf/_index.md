## FileType.ASF field

Le format Advanced Systems Format (ASF) est un conteneur multimédia numérique conçu principalement pour stocker et transmettre des flux multimédias. Microsoft Windows Media Video (WMV) est le format vidéo compressé et Microsoft Windows Media Audio (WMA) est le format audio compressé le long avec des métadonnées supplémentaires dans le conteneur ASF développé par Microsoft. En savoir plus sur ce format de fichier [ici](https://wiki.fileformat.com/video/wmv/) .
