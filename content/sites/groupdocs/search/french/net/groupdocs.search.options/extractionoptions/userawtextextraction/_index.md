## ExtractionOptions.UseRawTextExtraction property

Obtient ou définit une valeur indiquant si le mode brut est utilisé pour l'extraction de texte si possible. La valeur par défaut est`vrai` . Le mode brut peut augmenter considérablement la vitesse d'indexation, mais le mode normal améliore la mise en forme du texte extrait.

public bool UseRawTextExtraction { get; set; }

### Valeur de la propriété
