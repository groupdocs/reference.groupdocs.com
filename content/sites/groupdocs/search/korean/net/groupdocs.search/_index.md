네임스페이스는 다양한 형식의 문서에서 텍스트 데이터를 인덱싱하고 검색하기 위한 클래스를 제공합니다. 이 네임스페이스의 기본 클래스는 다음과 같습니다.

## 클래스

| 수업 | 설명 |
| --- | --- |
| [Extractor](./extractor) | 후속 빠른 인덱싱 단계를 분리하기 위해 문서에서 예비 데이터 추출을 위한 도구를 나타냅니다. |
| [Index](./index) | 문서를 인덱싱하고 검색하는 기본 클래스를 나타냅니다. |
| [IndexRepository](./indexrepository) | 여러 인덱스를 결합하고 공통 작업을 수행하기 위한 컨테이너를 나타냅니다. |
| [IndexSettings](./indexsettings) | 인덱싱 작업을 사용자 지정할 수 있는 인덱스 설정을 나타냅니다. |
| [IndexUpdater](./indexupdater) | 인덱스 업데이트 프로그램을 나타냅니다. 이 클래스는 이전 버전의 인덱스에서 문서를 다시 인덱싱합니다. |
| [License](./license) | 라이선스 적용 방법을 제공합니다. |
| [Metered](./metered) | 측정된 라이선스로 제품을 활성화하고 처리된 MB 양을 검색할 수 있는 방법을 제공합니다. 측정된 라이선스에 대해 자세히 알아보기[여기](https://purchase.groupdocs.com/faqs/licensing/metered) . |
| [SearchQuery](./searchquery) | 개체 형태의 검색 쿼리를 나타냅니다. |
