## Update(UpdateOptions) {#update_1}

마지막 업데이트 이후 변경되거나 삭제된 문서를 다시 인덱싱합니다. 인덱싱된 폴더에 추가된 새 파일을 추가합니다.

```csharp
public void Update(UpdateOptions options)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| options | UpdateOptions | 업데이트 옵션. |

### 예

이 예는 특정 업데이트 옵션으로 인덱스를 업데이트하는 방법을 보여줍니다.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

Index index = new Index(indexFolder); // 지정된 폴더에 인덱스 생성
index.Add(documentsFolder); // 지정된 폴더에서 문서 인덱싱

// 문서 폴더에서 문서를 삭제하거나 수정하거나 폴더에 새 문서를 추가합니다.

UpdateOptions options = new UpdateOptions();
options.Threads = 2; // 인덱싱 스레드 수 설정
index.Update(options); // 인덱스 업데이트
```
