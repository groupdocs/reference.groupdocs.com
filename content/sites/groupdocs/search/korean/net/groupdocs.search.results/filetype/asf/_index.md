## FileType.ASF field

ASF(Advanced Systems Format)는 주로 미디어 스트림을 저장하고 전송하기 위해 설계된 디지털 멀티미디어 컨테이너입니다. Microsoft Windows Media Video(WMV)는 압축된 비디오 형식이고 Microsoft Windows Media Audio(WMA) 는 압축된 오디오 형식입니다. Microsoft에서 개발한 ASF 컨테이너의 추가 메타데이터 포함. 이 파일 형식에 대해 자세히 알아보기 [여기](https://wiki.fileformat.com/video/wmv/) .
