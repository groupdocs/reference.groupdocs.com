## IndexSettings.TextStorageSettings property

टेक्स्ट स्टोरेज सेटिंग्स प्राप्त या सेट करता है। डिफ़ॉल्ट मान है`व्यर्थ` , जिसका अर्थ है कि दस्तावेज़ टेक्स्ट संग्रहीत नहीं हैं.

```csharp
public TextStorageSettings TextStorageSettings { get; set; }
```

### संपत्ति मूल्य

पाठ भंडारण सेटिंग्स।

### उदाहरण

उदाहरण दर्शाता है कि टेक्स्ट स्टोरेज सेटिंग्स कैसे सेट करें।

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

// इंडेक्स सेटिंग इंस्टेंस बनाना
IndexSettings settings = new IndexSettings();
settings.TextStorageSettings = new TextStorageSettings(Compression.High); // इंडेक्स टेक्स्ट स्टोरेज के लिए हाई कंप्रेशन लेवल सेट करना

// निर्दिष्ट फ़ोल्डर में एक इंडेक्स बनाना
Index index = new Index(indexFolder, settings);

// इंडेक्सिंग दस्तावेज़
index.Add(documentsFolder);

// खोज कर
SearchResult result = index.Search("Einstein");
```
