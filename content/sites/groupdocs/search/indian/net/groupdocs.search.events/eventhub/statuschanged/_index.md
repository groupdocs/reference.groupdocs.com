## EventHub.StatusChanged event

तब होता है जब सूचकांक स्थिति बदलती है।

```csharp
public event EventHandler<BaseIndexEventArgs> StatusChanged;
```

### उदाहरण

उदाहरण दर्शाता है कि कैसे घटना का उपयोग करना है।

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

// इंडेक्स बनाना
Index index = new Index(indexFolder);

// घटना की सदस्यता लेना
index.Events.StatusChanged += (sender, args) =>
{
    if (args.Status != IndexStatus.InProgress)
    {
        // ऑपरेशन पूरा होने की सूचना यहां होनी चाहिए
    }
};

// अतुल्यकालिक अनुक्रमण के लिए ध्वज सेट करना
IndexingOptions options = new IndexingOptions();
options.IsAsync = true;

// निर्दिष्ट फ़ोल्डर से अतुल्यकालिक अनुक्रमण दस्तावेज़
// ऑपरेशन पूरा होने से पहले विधि समाप्त हो जाती है
index.Add(documentsFolder, options);
```
