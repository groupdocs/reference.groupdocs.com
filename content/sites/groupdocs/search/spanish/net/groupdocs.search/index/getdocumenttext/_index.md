## GetDocumentText(DocumentInfo, OutputAdapter) {#getdocumenttext}

Genera texto con formato HTML para el documento indexado y lo transfiere a través del adaptador de salida.

```csharp
public void GetDocumentText(DocumentInfo documentInfo, OutputAdapter adapter)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| documentInfo | DocumentInfo | La información del documento indexado. |
| adapter | OutputAdapter | El adaptador de salida. |

### Ejemplos

El ejemplo muestra cómo obtener el texto de un documento indexado de un índice.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
 
// Creando un índice en la carpeta especificada
Index index = new Index(indexFolder);
 
// Indexación de documentos de la carpeta especificada
index.Add(documentsFolder);
 
// Obtener la lista de documentos indexados
DocumentInfo[] documents = index.GetIndexedDocuments();
 
// Obtener el texto de un documento
if (documents.Length > 0)
{
    FileOutputAdapter outputAdapter = new FileOutputAdapter(@"C:\Text.html");
    index.GetDocumentText(documents[0], outputAdapter);
}
```
