## Index.GetSearchReports method

Obtiene los informes sobre las operaciones de búsqueda.

```csharp
public SearchReport[] GetSearchReports()
```

### Valor_devuelto

Los informes de búsqueda.

### Ejemplos

El ejemplo muestra cómo obtener informes de búsqueda.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
string query1 = "Einstein";
string query2 = "Newton";

Index index = new Index(indexFolder); // Creando índice en la carpeta especificada
index.Add(documentsFolder); // Indexación de documentos de la carpeta especificada

SearchResult result1 = index.Search(query1); // Buscando
SearchResult result2 = index.Search(query2);
SearchResult result3 = index.Search(query1 + " & " + query2);

SearchReport[] reports = index.GetSearchReports(); // Obtener informes de búsqueda
```
