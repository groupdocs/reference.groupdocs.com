## IndexSettings.TextStorageSettings property

Obtiene o establece la configuración de almacenamiento de texto. El valor predeterminado es`nulo` , lo que significa que los textos de los documentos no se almacenan.

```csharp
public TextStorageSettings TextStorageSettings { get; set; }
```

### El valor de la propiedad

La configuración de almacenamiento de texto.

### Ejemplos

El ejemplo muestra cómo establecer la configuración de almacenamiento de texto.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

// Creando instancia de configuración de índice
IndexSettings settings = new IndexSettings();
settings.TextStorageSettings = new TextStorageSettings(Compression.High); // Establecer un alto nivel de compresión para el almacenamiento de texto de índice

// Creando un índice en la carpeta especificada
Index index = new Index(indexFolder, settings);

// Indexación de documentos
index.Add(documentsFolder);

// Buscando
SearchResult result = index.Search("Einstein");
```
