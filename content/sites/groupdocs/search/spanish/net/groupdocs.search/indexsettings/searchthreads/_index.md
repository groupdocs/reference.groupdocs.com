## IndexSettings.SearchThreads property

Obtiene o establece el número de subprocesos utilizados para la búsqueda. El valor predeterminado esDefault , lo que significa que la búsqueda se realizará utilizando el número de subprocesos igual al número de núcleos del procesador.

```csharp
public NumberOfThreads SearchThreads { get; set; }
```

### El valor de la propiedad

El número de subprocesos utilizados para la búsqueda.
