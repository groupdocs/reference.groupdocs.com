## EventHub.ErrorOccurred event

Ocurre cuando ocurre un error durante una operación de índice.

```csharp
public event EventHandler<IndexErrorEventArgs> ErrorOccurred;
```

### Ejemplos

El ejemplo demuestra cómo usar el evento.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
string query = "Einstein";

// Creando un índice
Index index = new Index(indexFolder);

// Suscribiendose al evento
index.Events.ErrorOccurred += (sender, args) =>
{
    Console.WriteLine(args.Message);
};

// Indexación de documentos de la carpeta especificada
index.Add(documentsFolder);

// Buscando en el índice
SearchResult result = index.Search(query);
```
