## ExtractionOptions.UseRawTextExtraction property

Obtiene o establece un valor que indica si el modo sin formato se usa para la extracción de texto, si es posible. El valor predeterminado es`verdadero` . El modo sin procesar puede aumentar significativamente la velocidad de indexación, pero el modo normal mejora el formato del texto extraído.

```csharp
public bool UseRawTextExtraction { get; set; }
```

### El valor de la propiedad

Un valor que indica si el modo sin procesar se usa para la extracción de texto, si es posible.
