命名空间提供了对不同格式文档中的文本数据进行索引和搜索的类。 这个命名空间中的主要类有：

## 课程

| 班级 | 描述 |
| --- | --- |
| [Extractor](./extractor) | 表示从文档中进行初步数据提取的工具，用于分离后续快速索引的阶段。 |
| [Index](./index) | 表示用于索引文档并通过它们进行搜索的主类。 |
| [IndexRepository](./indexrepository) | 表示一个容器，用于组合多个索引并对它们执行通用操作。 |
| [IndexSettings](./indexsettings) | 表示允许自定义索引操作的索引设置。 |
| [IndexUpdater](./indexupdater) | 表示索引更新器。 该类在旧版本的索引中执行重新索引文档。 |
| [License](./license) | 提供申请许可证的方法。 |
| [Metered](./metered) | 提供允许使用计量许可证激活产品并检索处理的 MB 数量的方法。 了解有关计量许可证的更多信息[这里](https://purchase.groupdocs.com/faqs/licensing/metered). |
| [SearchQuery](./searchquery) | 表示对象形式的搜索查询。 |
