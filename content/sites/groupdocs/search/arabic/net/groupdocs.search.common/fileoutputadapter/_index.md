### أمثلة

يوضح المثال استخدامًا نموذجيًا للفئة.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

Index index = new Index(indexFolder); // إنشاء فهرس في المجلد المحدد

index.Add(documentsFolder); // فهرسة المستندات من المجلد المحدد

DocumentInfo[] documents = index.GetIndexedDocuments(); // الحصول على معلومات عن المستندات المفهرسة

FileOutputAdapter adapter = new FileOutputAdapter(@"c:\DocumentText.htm"); // إنشاء محول إخراج ملف
index.GetDocumentText(documents[0], adapter); // إنشاء نص مستند في ملف الإخراج
```
