## FileType.ASF field

Het Advanced Systems Format (ASF) is een digitale multimediacontainer die in de eerste plaats is ontworpen voor het opslaan en verzenden van mediastreams. Microsoft Windows Media Video (WMV) is het gecomprimeerde videoformaat en Microsoft Windows Media Audio (WMA) is het gecomprimeerde audioformaat naast met aanvullende metadata in de door Microsoft ontwikkelde ASF-container.
