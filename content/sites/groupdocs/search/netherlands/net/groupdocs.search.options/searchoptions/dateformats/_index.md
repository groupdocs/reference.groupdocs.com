## SearchOptions.DateFormats property

Haalt de verzameling datumnotaties op voor zoeken in datumbereiken. De standaard datumnotaties zijn 'dd.MM.jjjj', 'MM/dd/jjjj' en 'jjjj-MM-dd'.

```csharp
public DateFormatCollection DateFormats { get; }
```

### Eigendoms-waarde

De verzameling datumnotaties voor zoeken in datumbereiken.

### Voorbeelden

Het voorbeeld laat zien hoe u de datumnotatie voor de zoekopdracht instelt.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
string query = "Einstein";

Index index = new Index(indexFolder); // Een index maken in de opgegeven map
index.Add(documentsFolder); // Documenten uit de opgegeven map indexeren

SearchOptions options = new SearchOptions();
options.DateFormats.Clear(); // Standaard datumnotaties verwijderen
DateFormatElement[] elements = new DateFormatElement[]
{
    DateFormatElement.MonthTwoDigits,
    DateFormatElement.DayOfMonthTwoDigits,
    DateFormatElement.YearFourDigits,
};
// Een datumnotatiepatroon 'MM/dd/jjjj' maken
DateFormat dateFormat = new DateFormat(elements, "/");
options.DateFormats.Add(dateFormat);

SearchResult result = index.Search(query, options); // Zoeken in index
```
