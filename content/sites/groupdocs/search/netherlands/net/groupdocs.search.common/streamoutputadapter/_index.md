### Voorbeelden

Het voorbeeld demonstreert een typisch gebruik van de klasse.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

Index index = new Index(indexFolder); // Een index maken in de opgegeven map

index.Add(documentsFolder); // Documenten uit de opgegeven map indexeren

DocumentInfo[] documents = index.GetIndexedDocuments(); // Informatie krijgen over geïndexeerde documenten

using (Stream stream = new MemoryStream()) // Een stream maken
{
    StreamOutputAdapter adapter = new StreamOutputAdapter(stream); // Een stream maken output adapter
    index.GetDocumentText(documents[0], adapter); // Een documenttekst genereren in de stream
}
```
