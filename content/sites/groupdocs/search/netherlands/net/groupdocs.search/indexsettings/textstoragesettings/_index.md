## IndexSettings.TextStorageSettings property

Haalt de instellingen voor tekstopslag op of stelt deze in. De standaardwaarde is`nul` , waardoor documentteksten niet worden opgeslagen.

```csharp
public TextStorageSettings TextStorageSettings { get; set; }
```

### Eigendoms-waarde

De instellingen voor tekstopslag.

### Voorbeelden

Het voorbeeld laat zien hoe u de instellingen voor tekstopslag instelt.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

// Instantie voor indexinstellingen maken
IndexSettings settings = new IndexSettings();
settings.TextStorageSettings = new TextStorageSettings(Compression.High); // Een hoog compressieniveau instellen voor de opslag van indextekst

// Een index maken in de opgegeven map
Index index = new Index(indexFolder, settings);

// Documenten indexeren
index.Add(documentsFolder);

// Zoeken
SearchResult result = index.Search("Einstein");
```
