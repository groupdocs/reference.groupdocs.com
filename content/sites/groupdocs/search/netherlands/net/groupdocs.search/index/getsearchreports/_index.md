Krijgt de rapporten over zoekacties.

### Winstwaarde

### Voorbeelden

Het voorbeeld laat zien hoe u zoekrapporten kunt krijgen.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
string query1 = "Einstein";
string query2 = "Newton";

Index index = new Index(indexFolder); // Index maken in de opgegeven map
index.Add(documentsFolder); // Documenten uit de opgegeven map indexeren

SearchResult result1 = index.Search(query1); // Zoeken
SearchResult result2 = index.Search(query2);
SearchResult result3 = index.Search(query1 + " & " + query2);

SearchReport[] reports = index.GetSearchReports(); // Zoekrapporten ophalen
```
