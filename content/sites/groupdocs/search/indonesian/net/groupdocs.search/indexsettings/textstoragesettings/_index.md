## IndexSettings.TextStorageSettings property

Mendapat atau mengatur pengaturan penyimpanan teks. Nilai defaultnya adalah`batal` , artinya teks dokumen tidak disimpan.

```csharp
public TextStorageSettings TextStorageSettings { get; set; }
```

### Nilai properti

Pengaturan penyimpanan teks.

### Contoh

Contoh menunjukkan cara mengatur pengaturan penyimpanan teks.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

// Membuat instance pengaturan indeks
IndexSettings settings = new IndexSettings();
settings.TextStorageSettings = new TextStorageSettings(Compression.High); // Mengatur tingkat kompresi tinggi untuk penyimpanan teks indeks

// Membuat indeks di folder yang ditentukan
Index index = new Index(indexFolder, settings);

// Pengindeksan dokumen
index.Add(documentsFolder);

// Mencari
SearchResult result = index.Search("Einstein");
```
