## GetDocumentText(DocumentInfo, OutputAdapter) {#getdocumenttext}

Menghasilkan teks berformat HTML untuk dokumen yang diindeks dan mentransfernya melalui adaptor keluaran.

```csharp
public void GetDocumentText(DocumentInfo documentInfo, OutputAdapter adapter)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| documentInfo | DocumentInfo | Info dokumen yang diindeks. |
| adapter | OutputAdapter | Adaptor keluaran. |

### Contoh

Contoh menunjukkan cara mendapatkan teks dokumen yang diindeks dari indeks.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
 
// Membuat indeks di folder yang ditentukan
Index index = new Index(indexFolder);
 
// Pengindeksan dokumen dari folder yang ditentukan
index.Add(documentsFolder);
 
// Mendapatkan daftar dokumen yang diindeks
DocumentInfo[] documents = index.GetIndexedDocuments();
 
// Mendapatkan teks dokumen
if (documents.Length > 0)
{
    FileOutputAdapter outputAdapter = new FileOutputAdapter(@"C:\Text.html");
    index.GetDocumentText(documents[0], outputAdapter);
}
```
