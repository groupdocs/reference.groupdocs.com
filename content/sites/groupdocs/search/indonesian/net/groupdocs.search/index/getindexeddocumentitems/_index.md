## Index.GetIndexedDocumentItems method

Mendapat array item bersarang dari dokumen yang ditentukan (untuk dokumen kontainer seperti ZIP, OST, PST).

```csharp
public DocumentInfo[] GetIndexedDocumentItems(DocumentInfo documentInfo)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| documentInfo | DocumentInfo | Informasi dokumen. |

### Nilai Pengembalian

Array item dokumen.

### Contoh

Contoh menunjukkan cara mendapatkan daftar item dokumen yang diindeks dari indeks.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
 
// Membuat indeks di folder yang ditentukan
Index index = new Index(indexFolder);
 
// Pengindeksan dokumen dari folder yang ditentukan
index.Add(documentsFolder);
 
// Mendapatkan daftar dokumen yang diindeks
DocumentInfo[] documents = index.GetIndexedDocuments();
for (int i = 0; i < documents.Length; i++)
{
    DocumentInfo document = documents[i];
    Console.WriteLine(document.FilePath);
    DocumentInfo[] items = index.GetIndexedDocumentItems(document); // Mendapatkan daftar item dokumen
    for (int j = 0; j < items.Length; j++)
    {
        DocumentInfo item = items[j];
        Console.WriteLine("\t" + item.InnerPath);
    }
}
```
