Namespace menyediakan kelas untuk mengindeks dan mencari data teks dalam dokumen dengan format berbeda. Kelas utama dalam namespace ini adalah:

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [Extractor](./extractor) | Merupakan alat untuk ekstraksi data awal dari dokumen untuk memisahkan tahap pengindeksan cepat berikutnya. |
| [Index](./index) | Mewakili kelas utama untuk mengindeks dokumen dan menelusurinya. |
| [IndexRepository](./indexrepository) | Merupakan wadah untuk menggabungkan beberapa indeks dan melakukan operasi umum padanya. |
| [IndexSettings](./indexsettings) | Merupakan pengaturan indeks yang memungkinkan untuk menyesuaikan operasi pengindeksan. |
| [IndexUpdater](./indexupdater) | Mewakili pembaru indeks. Kelas ini melakukan pengindeksan ulang dokumen dalam indeks versi lama. |
| [License](./license) | Menyediakan metode untuk menerapkan lisensi. |
| [Metered](./metered) | Menyediakan metode yang memungkinkan untuk mengaktifkan produk dengan lisensi Terukur dan mengambil jumlah MB yang diproses. Pelajari lebih lanjut tentang Lisensi Terukur[Di Sini](https://purchase.groupdocs.com/faqs/licensing/metered) . |
| [SearchQuery](./searchquery) | Mewakili permintaan pencarian dalam bentuk objek. |
