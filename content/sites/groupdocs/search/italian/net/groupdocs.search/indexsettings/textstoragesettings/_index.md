## IndexSettings.TextStorageSettings property

Ottiene o imposta le impostazioni di archiviazione del testo. Il valore predefinito è`nullo` , il che significa che i testi del documento non vengono memorizzati.

```csharp
public TextStorageSettings TextStorageSettings { get; set; }
```

### Valore della proprietà

Le impostazioni di archiviazione del testo.

### Esempi

L'esempio mostra come configurare le impostazioni di archiviazione del testo.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

// Creazione dell'istanza delle impostazioni dell'indice
IndexSettings settings = new IndexSettings();
settings.TextStorageSettings = new TextStorageSettings(Compression.High); // Impostazione di un livello di compressione elevato per l'archiviazione del testo dell'indice

// Creazione di un indice nella cartella specificata
Index index = new Index(indexFolder, settings);

// Indicizzazione dei documenti
index.Add(documentsFolder);

// Ricerca
SearchResult result = index.Search("Einstein");
```
