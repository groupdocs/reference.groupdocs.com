## IndexSettings.UseRawTextExtraction property

Ottiene o imposta un valore che indica se la modalità raw viene utilizzata per l'estrazione del testo, se possibile. Il valore predefinito è`VERO` . La modalità raw può aumentare significativamente la velocità di indicizzazione, ma la modalità normale migliora la formattazione del testo estratto.

```csharp
public bool UseRawTextExtraction { get; set; }
```
