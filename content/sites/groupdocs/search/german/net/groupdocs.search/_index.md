Der Namensraum stellt Klassen zum Indizieren und Suchen von Textdaten in Dokumenten unterschiedlicher Formate bereit. Die Hauptklassen in diesem Namensraum sind:

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [Extractor](./extractor) | Stellt ein Tool zur vorläufigen Datenextraktion aus Dokumenten dar, um die Phase der anschließenden schnellen Indizierung zu trennen. |
| [Index](./index) | Stellt die Hauptklasse dar, um Dokumente zu indizieren und zu durchsuchen. |
| [IndexRepository](./indexrepository) | Repräsentiert einen Container zum Kombinieren mehrerer Indizes und zum Ausführen gemeinsamer Operationen an ihnen. |
| [IndexSettings](./indexsettings) | Stellt die Indexeinstellungen dar, mit denen die Indizierungsvorgänge angepasst werden können. |
| [IndexUpdater](./indexupdater) | Repräsentiert einen Index-Updater. Diese Klasse führt eine Neuindizierung von Dokumenten in einem Index einer alten Version durch. |
| [License](./license) | Bietet Methoden zum Anwenden von Lizenzen. |
| [Metered](./metered) | Bietet Methoden, die es ermöglichen, ein Produkt mit einer gebührenpflichtigen Lizenz zu aktivieren und die verarbeitete MB-Menge abzurufen. Erfahren Sie mehr über gebührenpflichtige Lizenzen[hier](https://purchase.groupdocs.com/faqs/licensing/metered) . |
| [SearchQuery](./searchquery) | Stellt eine Suchanfrage in Objektform dar. |
