## IndexSettings.Logger property

Ruft einen Logger ab oder legt ihn fest, der zum Protokollieren von Ereignissen und Fehlern im Index verwendet wird. Beachten Sie, dass der Logger nicht gespeichert wird und jedes Mal erstellt und zugewiesen werden muss, wenn der Index erstellt oder geladen wird.

public ILogger Logger { get; set; }
