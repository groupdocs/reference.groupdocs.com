## IndexSettings.TextStorageSettings property

Ruft die Textspeichereinstellungen ab oder legt sie fest. Der Standardwert ist`Null` , was bedeutet, dass Dokumenttexte nicht gespeichert werden.

```csharp
public TextStorageSettings TextStorageSettings { get; set; }
```

### Eigentumswert

Die Textspeichereinstellungen.

### Beispiele

Das Beispiel zeigt, wie die Textspeichereinstellungen festgelegt werden.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

// Indexeinstellungsinstanz erstellen
IndexSettings settings = new IndexSettings();
settings.TextStorageSettings = new TextStorageSettings(Compression.High); // Festlegen einer hohen Komprimierungsstufe für den Indextextspeicher

// Index im angegebenen Ordner erstellen
Index index = new Index(indexFolder, settings);

// Indizierung von Dokumenten
index.Add(documentsFolder);

// Suchen
SearchResult result = index.Search("Einstein");
```
