## GetDocumentText(DocumentInfo, OutputAdapter) {#getdocumenttext}

Generiert HTML-formatierten Text für das indizierte Dokument und überträgt ihn über den Ausgabeadapter.

```csharp
public void GetDocumentText(DocumentInfo documentInfo, OutputAdapter adapter)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| documentInfo | DocumentInfo | Die indizierten Dokumentinformationen. |
| adapter | OutputAdapter | Der Ausgangsadapter. |

### Beispiele

Das Beispiel zeigt, wie man den Text eines indizierten Dokuments aus einem Index erhält.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
 
// Index im angegebenen Ordner erstellen
Index index = new Index(indexFolder);
 
// Indizierung von Dokumenten aus dem angegebenen Ordner
index.Add(documentsFolder);
 
// Liste der indizierten Dokumente abrufen
DocumentInfo[] documents = index.GetIndexedDocuments();
 
// Dokumenttext abrufen
if (documents.Length > 0)
{
    FileOutputAdapter outputAdapter = new FileOutputAdapter(@"C:\Text.html");
    index.GetDocumentText(documents[0], outputAdapter);
}
```
