## Index.GetIndexedDocuments method

Ruft ein Array aller indizierten Dokumente ab.

```csharp
public DocumentInfo[] GetIndexedDocuments()
```

### Rückgabewert

Ein Array aller indizierten Dokumente.

### Beispiele

Das Beispiel zeigt, wie man eine Liste von indizierten Dokumenten aus einem Index erhält.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
 
// Index im angegebenen Ordner erstellen
Index index = new Index(indexFolder);
 
// Indizierung von Dokumenten aus dem angegebenen Ordner
index.Add(documentsFolder);
 
// Liste der indizierten Dokumente abrufen
DocumentInfo[] documents = index.GetIndexedDocuments();
```
