## IndexSettings.TextStorageSettings property

Metin depolama ayarlarını alır veya ayarlar. Varsayılan değer:`hükümsüz` , yani belge metinleri saklanmaz.

```csharp
public TextStorageSettings TextStorageSettings { get; set; }
```

### Mülk değeri

Metin depolama ayarları.

### Örnekler

Örnek, metin depolama ayarlarının nasıl ayarlanacağını gösterir.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

// Dizin ayarları örneği oluşturuluyor
IndexSettings settings = new IndexSettings();
settings.TextStorageSettings = new TextStorageSettings(Compression.High); // İndeks metin depolaması için yüksek sıkıştırma seviyesi ayarlanıyor

// Belirtilen klasörde bir dizin oluşturma
Index index = new Index(indexFolder, settings);

// Belgeleri indeksleme
index.Add(documentsFolder);

// Aranıyor
SearchResult result = index.Search("Einstein");
```
