## FileType.OTT field

Files with OTT extension represent template documents generated by applications in compliance with the OASIS' OpenDocument standard format. These are created with word processor applications such as free OpenOffice Writer and can hold settings that can be used to generate new documents from these template files. Learn more about this file format [here](https://wiki.fileformat.com/word-processing/ott/).
