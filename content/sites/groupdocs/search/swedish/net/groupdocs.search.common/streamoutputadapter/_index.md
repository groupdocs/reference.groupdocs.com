### Exempel

Exemplet visar en typisk användning av klassen.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

Index index = new Index(indexFolder); // Skapar ett index i den angivna mappen

index.Add(documentsFolder); // Indexering av dokument från den angivna mappen

DocumentInfo[] documents = index.GetIndexedDocuments(); // Få information om indexerade dokument

using (Stream stream = new MemoryStream()) // Skapa en ström
{
    StreamOutputAdapter adapter = new StreamOutputAdapter(stream); // Skapa en ström output adapter
    index.GetDocumentText(documents[0], adapter); // Generera en dokumenttext i strömmen
}
```
