## IndexSettings.TextStorageSettings property

Hämtar eller ställer in textlagringsinställningarna. Standardvärdet är`null` , vilket innebär att dokumenttexter inte lagras.

```csharp
public TextStorageSettings TextStorageSettings { get; set; }
```

### Fastighetsvärde

Inställningarna för textlagring.

### Exempel

Exemplet visar hur man ställer in textlagringsinställningarna.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

// Skapar instans av indexinställningar
IndexSettings settings = new IndexSettings();
settings.TextStorageSettings = new TextStorageSettings(Compression.High); // Ställer in hög komprimeringsnivå för lagring av indextext

// Skapar ett index i den angivna mappen
Index index = new Index(indexFolder, settings);

// Indexering av dokument
index.Add(documentsFolder);

// Söker
SearchResult result = index.Search("Einstein");
```
