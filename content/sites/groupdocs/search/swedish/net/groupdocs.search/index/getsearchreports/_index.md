## Index.GetSearchReports method

Hämtar rapporter om sökoperationer.

```csharp
public SearchReport[] GetSearchReports()
```

### Returvärde

Sökningen rapporterar.

### Exempel

Exemplet visar hur du får sökrapporter.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
string query1 = "Einstein";
string query2 = "Newton";

Index index = new Index(indexFolder); // Skapar index i den angivna mappen
index.Add(documentsFolder); // Indexering av dokument från den angivna mappen

SearchResult result1 = index.Search(query1); // Söker
SearchResult result2 = index.Search(query2);
SearchResult result3 = index.Search(query1 + " & " + query2);

SearchReport[] reports = index.GetSearchReports(); // Få sökrapporter
```
