## Index.GetIndexedDocumentItems method

Hämtar en array av kapslade objekt i det angivna dokumentet (för containerdokument som ZIP, OST, PST).

```csharp
public DocumentInfo[] GetIndexedDocumentItems(DocumentInfo documentInfo)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| documentInfo | DocumentInfo | Dokumentinformationen. |

### Returvärde

En uppsättning av dokumentobjekt.

### Exempel

Exemplet visar hur man får en lista över objekt i ett indexerat dokument från ett index.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
 
// Skapar ett index i den angivna mappen
Index index = new Index(indexFolder);
 
// Indexering av dokument från den angivna mappen
index.Add(documentsFolder);
 
// Hämta lista över indexerade dokument
DocumentInfo[] documents = index.GetIndexedDocuments();
for (int i = 0; i < documents.Length; i++)
{
    DocumentInfo document = documents[i];
    Console.WriteLine(document.FilePath);
    DocumentInfo[] items = index.GetIndexedDocumentItems(document); // Hämtar lista över dokumentobjekt
    for (int j = 0; j < items.Length; j++)
    {
        DocumentInfo item = items[j];
        Console.WriteLine("\t" + item.InnerPath);
    }
}
```
