Namnutrymmet tillhandahåller klasser för indexering och sökning av textdata i dokument av olika format. Huvudklasserna i detta namnutrymme är:

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [Extractor](./extractor) | Representerar ett verktyg för preliminär dataextraktion från dokument för att separera stadiet för efterföljande snabb indexering. |
| [Index](./index) | Representerar huvudklassen för att indexera dokument och söka igenom dem. |
| [IndexRepository](./indexrepository) | Representerar en behållare för att kombinera flera index och utföra vanliga operationer på dem. |
| [IndexSettings](./indexsettings) | Representerar indexinställningarna som gör det möjligt att anpassa indexeringsoperationerna. |
| [IndexUpdater](./indexupdater) | Representerar en indexuppdatering. Denna klass utför omindexering av dokument i ett index av en gammal version. |
| [License](./license) | Tillhandahåller metoder för att ansöka licens. |
| [Metered](./metered) | Tillhandahåller metoder som gör det möjligt att aktivera produkt med Metered-licens och hämta mängden bearbetade MB. Läs mer om Metered-licenser[här](https://purchase.groupdocs.com/faqs/licensing/metered) . |
| [SearchQuery](./searchquery) | Representerar en sökfråga i objektform. |
