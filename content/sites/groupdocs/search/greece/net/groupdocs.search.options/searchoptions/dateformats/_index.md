## SearchOptions.DateFormats property

Λαμβάνει τη συλλογή μορφών ημερομηνιών για αναζήτηση εύρους ημερομηνιών. Οι προεπιλεγμένες μορφές ημερομηνίας είναι «ηη.ΜΜ.εεεε», «ΜΜ/ηη/εεεε» και «εεεε-ΜΜ-ηη».

```csharp
public DateFormatCollection DateFormats { get; }
```

### Αξία περιουσίας

Η συλλογή μορφών ημερομηνιών για αναζήτηση εύρους ημερομηνιών.

### Παραδείγματα

Το παράδειγμα δείχνει πώς να ορίσετε τις μορφές ημερομηνίας για την αναζήτηση.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
string query = "Einstein";

Index index = new Index(indexFolder); // Δημιουργία ευρετηρίου στον καθορισμένο φάκελο
index.Add(documentsFolder); // Δημιουργία ευρετηρίου εγγράφων από τον καθορισμένο φάκελο

SearchOptions options = new SearchOptions();
options.DateFormats.Clear(); // Αφαίρεση προεπιλεγμένων μορφών ημερομηνίας
DateFormatElement[] elements = new DateFormatElement[]
{
    DateFormatElement.MonthTwoDigits,
    DateFormatElement.DayOfMonthTwoDigits,
    DateFormatElement.YearFourDigits,
};
// Δημιουργία μοτίβου μορφής ημερομηνίας «ΜΜ/ηη/εεεε»
DateFormat dateFormat = new DateFormat(elements, "/");
options.DateFormats.Add(dateFormat);

SearchResult result = index.Search(query, options); // Αναζήτηση στο ευρετήριο
```
