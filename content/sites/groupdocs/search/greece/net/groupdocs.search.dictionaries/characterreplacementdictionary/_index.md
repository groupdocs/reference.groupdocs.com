## CharacterReplacementDictionary class

Αντιπροσωπεύει ένα λεξικό αντικατάστασης χαρακτήρων που χρησιμοποιείται κατά τη διαδικασία δημιουργίας ευρετηρίου. Η αντικατάσταση χαρακτήρων μπορεί να χρησιμοποιηθεί, για παράδειγμα, για την αφαίρεση τονισμού από τονισμένους χαρακτήρες ή για τη δημιουργία ευρετηρίου χωρίς διάκριση πεζών-κεφαλαίων.
