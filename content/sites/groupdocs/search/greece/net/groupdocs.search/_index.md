Ο χώρος ονομάτων παρέχει κλάσεις για ευρετηρίαση και αναζήτηση δεδομένων κειμένου σε έγγραφα διαφορετικών μορφών. Οι κύριες κλάσεις σε αυτόν τον χώρο ονομάτων είναι:

## Τάξεις

| Τάξη | Περιγραφή |
| --- | --- |
| [Extractor](./extractor) | Αντιπροσωπεύει ένα εργαλείο για την προκαταρκτική εξαγωγή δεδομένων από έγγραφα για τον διαχωρισμό του σταδίου της επόμενης γρήγορης ευρετηρίασης. |
| [Index](./index) | Αντιπροσωπεύει την κύρια τάξη για την ευρετηρίαση εγγράφων και την αναζήτηση μέσω αυτών. |
| [IndexRepository](./indexrepository) | Αντιπροσωπεύει ένα κοντέινερ για το συνδυασμό πολλών ευρετηρίων και την εκτέλεση κοινών λειτουργιών σε αυτούς. |
| [IndexSettings](./indexsettings) | Αντιπροσωπεύει τις ρυθμίσεις ευρετηρίου που επιτρέπουν την προσαρμογή των λειτουργιών ευρετηρίασης. |
| [IndexUpdater](./indexupdater) | Αντιπροσωπεύει ένα πρόγραμμα ενημέρωσης ευρετηρίου. Αυτή η κλάση εκτελεί εκ νέου ευρετηρίαση εγγράφων σε ευρετήριο παλιάς έκδοσης. |
| [License](./license) | Παρέχει μεθόδους για την εφαρμογή άδειας. |
| [Metered](./metered) | Παρέχει μεθόδους που επιτρέπουν την ενεργοποίηση προϊόντος με άδεια Metered και την ανάκτηση του όγκου των MB που έχουν υποστεί επεξεργασία. Μάθετε περισσότερα για τις άδειες Metered[εδώ](https://purchase.groupdocs.com/faqs/licensing/metered) . |
| [SearchQuery](./searchquery) | Αντιπροσωπεύει ένα ερώτημα αναζήτησης σε μορφή αντικειμένου. |
