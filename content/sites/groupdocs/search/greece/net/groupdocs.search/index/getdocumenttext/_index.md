## GetDocumentText(DocumentInfo, OutputAdapter) {#getdocumenttext}

Δημιουργεί κείμενο μορφοποιημένου HTML για έγγραφο με ευρετήριο και το μεταφέρει μέσω του προσαρμογέα εξόδου.

```csharp
public void GetDocumentText(DocumentInfo documentInfo, OutputAdapter adapter)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| documentInfo | DocumentInfo | Οι πληροφορίες του εγγράφου με ευρετήριο. |
| adapter | OutputAdapter | Ο προσαρμογέας εξόδου. |

### Παραδείγματα

Το παράδειγμα δείχνει πώς να λάβετε το κείμενο ενός εγγράφου με ευρετήριο από ένα ευρετήριο.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
 
// Δημιουργία ευρετηρίου στον καθορισμένο φάκελο
Index index = new Index(indexFolder);
 
// Δημιουργία ευρετηρίου εγγράφων από τον καθορισμένο φάκελο
index.Add(documentsFolder);
 
// Λήψη λίστας εγγράφων με ευρετήριο
DocumentInfo[] documents = index.GetIndexedDocuments();
 
// Λήψη κειμένου εγγράφου
if (documents.Length > 0)
{
    FileOutputAdapter outputAdapter = new FileOutputAdapter(@"C:\Text.html");
    index.GetDocumentText(documents[0], outputAdapter);
}
```
