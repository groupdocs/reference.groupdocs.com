## FileType.PPTX field

Τα αρχεία με επέκταση PPTX είναι αρχεία παρουσίασης που δημιουργήθηκαν με τη δημοφιλή εφαρμογή Microsoft PowerPoint. Σε αντίθεση με την προηγούμενη έκδοση της μορφής αρχείου παρουσίασης PPT που ήταν δυαδική, η μορφή PPTX βασίζεται στη μορφή αρχείου παρουσίασης ανοιχτής XML του Microsoft PowerPoint. Μάθετε περισσότερα για αυτό το αρχείο format [εδώ](https://wiki.fileformat.com/presentation/pptx/) .
