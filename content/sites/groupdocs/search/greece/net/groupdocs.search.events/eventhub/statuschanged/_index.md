## EventHub.StatusChanged event

Εμφανίζεται όταν αλλάζει η κατάσταση του ευρετηρίου.

```csharp
public event EventHandler<BaseIndexEventArgs> StatusChanged;
```

### Παραδείγματα

Το παράδειγμα δείχνει πώς να χρησιμοποιήσετε το συμβάν.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

// Δημιουργία ευρετηρίου
Index index = new Index(indexFolder);

// Εγγραφή στην εκδήλωση
index.Events.StatusChanged += (sender, args) =>
{
    if (args.Status != IndexStatus.InProgress)
    {
        // Θα πρέπει να υπάρχει μια ειδοποίηση για την ολοκλήρωση της λειτουργίας
    }
};

// Ρύθμιση της σημαίας για ασύγχρονη ευρετηρίαση
IndexingOptions options = new IndexingOptions();
options.IsAsync = true;

// Ασύγχρονη δημιουργία ευρετηρίου εγγράφων από τον καθορισμένο φάκελο
// Η μέθοδος τερματίζεται πριν ολοκληρωθεί η λειτουργία
index.Add(documentsFolder, options);
```
