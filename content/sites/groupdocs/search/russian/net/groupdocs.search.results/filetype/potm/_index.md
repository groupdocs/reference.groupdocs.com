## FileType.POTM field

Файлы с расширением POTM представляют собой файлы шаблонов Microsoft PowerPoint с поддержкой макросов. Файлы POTM создаются с помощью PowerPoint 2007 или более поздней версии и содержат настройки по умолчанию, которые можно использовать для создания дополнительных файлов презентаций. Узнайте больше об этом файле format [здесь](https://wiki.fileformat.com/presentation/potm/) .
