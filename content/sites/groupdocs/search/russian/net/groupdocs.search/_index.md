Пространство имен предоставляет классы для индексации и поиска текстовых данных в документах разных форматов. Основными классами в этом пространстве имен являются:

## Классы

| Учебный класс | Описание |
| --- | --- |
| [Extractor](./extractor) | Представляет собой инструмент предварительного извлечения данных из документов для выделения этапа последующего быстрого индексирования. |
| [Index](./index) | Представляет основной класс для индексации документов и поиска по ним. |
| [IndexRepository](./indexrepository) | Представляет собой контейнер для объединения нескольких индексов и выполнения над ними общих операций. |
| [IndexSettings](./indexsettings) | Представляет параметры индекса, которые позволяют настраивать операции индексирования. |
| [IndexUpdater](./indexupdater) | Представляет средство обновления индекса. Этот класс выполняет переиндексацию документов в индексе старой версии. |
| [License](./license) | Предоставляет методы применения лицензии. |
| [Metered](./metered) | Предоставляет методы, которые позволяют активировать продукт с лицензией Metered и получать количество обработанных МБ. Подробнее о лицензиях Metered[здесь](https://purchase.groupdocs.com/faqs/licensing/metered) . |
| [SearchQuery](./searchquery) | Представляет поисковый запрос в форме объекта. |
