## IndexSettings.UseRawTextExtraction property

Получает или задает значение, указывающее, используется ли необработанный режим для извлечения текста, если это возможно. Значение по умолчанию:`истинный` . Необработанный режим может значительно увеличить скорость индексации, но обычный режим улучшает форматирование извлеченного текста.

```csharp
public bool UseRawTextExtraction { get; set; }
```

### Стоимость имущества
