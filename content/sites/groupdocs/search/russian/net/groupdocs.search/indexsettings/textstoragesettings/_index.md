## IndexSettings.TextStorageSettings property

Получает или задает параметры хранения текста. Значение по умолчанию:`нулевой` , что означает, что тексты документов не сохраняются.

```csharp
public TextStorageSettings TextStorageSettings { get; set; }
```

### Стоимость имущества

Настройки хранения текста.

### Примеры

В примере показано, как установить параметры хранения текста.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";

// Создание экземпляра настроек индекса
IndexSettings settings = new IndexSettings();
settings.TextStorageSettings = new TextStorageSettings(Compression.High); // Установка высокого уровня сжатия для хранения индексного текста

// Создание индекса в указанной папке
Index index = new Index(indexFolder, settings);

// Индексация документов
index.Add(documentsFolder);

// Идет поиск
SearchResult result = index.Search("Einstein");
```
