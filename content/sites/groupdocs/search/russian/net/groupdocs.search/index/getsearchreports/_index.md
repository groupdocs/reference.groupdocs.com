## Index.GetSearchReports method

Получает отчеты о поисковых операциях.

```csharp
public SearchReport[] GetSearchReports()
```

### Возвращаемое значение

Об этом сообщает поиск.

### Примеры

В примере показано, как получить отчеты о поиске.

```csharp
string indexFolder = @"c:\MyIndex\";
string documentsFolder = @"c:\MyDocuments\";
string query1 = "Einstein";
string query2 = "Newton";

Index index = new Index(indexFolder); // Создание индекса в указанной папке
index.Add(documentsFolder); // Индексация документов из указанной папки

SearchResult result1 = index.Search(query1); // Идет поиск
SearchResult result2 = index.Search(query2);
SearchResult result3 = index.Search(query1 + " & " + query2);

SearchReport[] reports = index.GetSearchReports(); // Получение отчетов о поиске
```
