Получает отчеты об операциях индексирования.


### Примеры

Пример демонстрирует, как получить отчеты об индексации.

```csharp

string documentsFolder = @"c:\MyDocuments\";

Index index = new Index(indexFolder); // Создание индекса в указанной папке
```
