# GroupDocs.Comparison — API Reference Source Files

This repository holds the Markdown source files used to generate the **GroupDocs.Comparison**
API reference, published at <https://reference.groupdocs.com/comparison/>.

## About GroupDocs.Comparison

[GroupDocs.Comparison](https://products.groupdocs.com/comparison/) is a developer API for document
processing: detect and merge changes in text, style, and formatting across documents. It is available for the following platforms, each with its own
API reference:

- **.NET** — <https://reference.groupdocs.com/comparison/net/>
- **Java** — <https://reference.groupdocs.com/comparison/java/>
- **Python via .NET** — <https://reference.groupdocs.com/comparison/python-net/>

## Repository purpose

The files in this repository are used to:

- Build and maintain the official **GroupDocs.Comparison** API reference.
- Keep the reference in sync with the latest product release.
- Provide a consistent structure for automated Hugo builds and publication.

Content is English-only and compiled into a static site with [Hugo](https://gohugo.io/).
