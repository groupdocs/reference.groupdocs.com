## WordCompareOptions.ShowRevisions property

Gets or sets whether the result document keeps revision markup visible. If false, all revisions are accepted and the result appears as final text. This setting is meaningful only when [`DisplayMode`](../displaymode) is set to Highlight. The default value is true.
