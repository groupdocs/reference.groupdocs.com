Provides classes for working with component styles in Cells format files.

The classes in this package allow you to work with various styles and formatting options specific to Cells format files (e.g., Excel files).

Key classes in this package include:

 *  [MergeType](../../com.groupdocs.comparison.cells.style/mergetype) - Represents the enumeration the type of cell merge.

For more details on working with revisions and tracked changes in Word documents using GroupDocs.Comparison for Java, please refer to the [GroupDocs.Comparison Documentation][].
