## CompareOptions.SensitivityOfComparison property

Получает или задает чувствительность сравнения.

```csharp
public int SensitivityOfComparison { get; set; }
```

### Стоимость имущества

Процент удаленных и вставленных элементов двух сравниваемых объектов по отношению ко всем элементам этих объектов. при превышении этого процента объекты не сравниваются, а считаются полностью вставленными и удаленными. Минимальное значение - 0% = &gt; Сравнение не происходит ни при какой длине общей подпоследовательности двух сравниваемых объектов. Значение по умолчанию - 75% =&gt; Сравнение происходит , если процент удаленных и вставленных элементов двух сравниваемых объектов относительно всех элементов этих объектов не более 75. Максимальное значение - 100% =&gt; Сравнение происходит при любой длине общей подпоследовательности двух сравниваемых объектов.
