## PdfOptions.FixLinkIssue property

Tries to fix the issue when whole HTML page content is a link. Works only when input format is PDF and output format is HTML (with embedded or external resources). By default is disabled (`false`). Turn it on only when you know what and why you're doing. Enabling this option increases the document processing time.
