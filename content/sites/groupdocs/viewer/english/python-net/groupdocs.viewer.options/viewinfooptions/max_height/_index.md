## max_height property


Sets the maximum height of an output image (in pixels, for rendering to PNG/JPG only).

### Remarks 


Use this property to set the maximum output image height (in pixels). For code example, see the [documentation](https://docs.groupdocs.com/viewer/net/set-image-size-limits-when-rendering-to-png-jpg/).




If you set the [`ViewInfoOptions.height`](/viewer/python-net/groupdocs.viewer.options/viewinfooptions#height) property, this property is ignored.
