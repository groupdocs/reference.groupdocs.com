## image_quality property


Sets the image quality in the output PDF file (in percent).

### Remarks 


To change the image quality, first set the [`PdfOptimizationOptions.compress_images`](/viewer/python-net/groupdocs.viewer.options/pdfoptimizationoptions#compress_images) property to `true`. For code example, see this [documentation](https://docs.groupdocs.com/viewer/net/optimization-pdf-reduce-image-quality/).
