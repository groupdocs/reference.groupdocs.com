## watermark property


The text watermark to be applied to each page.

### Remarks 


If both [`ViewOptions.watermark`](/viewer/python-net/groupdocs.viewer.options/viewoptions#watermark) and [`ViewOptions.PageRotations`](/viewer/python-net/groupdocs.viewer.options/viewoptions) are specified, the watermark is applied to the rotated pages.




For details, see the [documentation](https://docs.groupdocs.com/viewer/net/add-text-watermark/).
