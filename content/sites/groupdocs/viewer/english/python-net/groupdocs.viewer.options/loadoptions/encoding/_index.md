## encoding property


Sets the encoding used when opening text-based files or email messages such as
[`FileType.CSV`](/viewer/python-net/groupdocs.viewer/filetype),
[`FileType.TXT`](/viewer/python-net/groupdocs.viewer/filetype),
and [`FileType.MSG`](/viewer/python-net/groupdocs.viewer/filetype).
Default value is UTF8.

### Remarks 


For code example, see the [documentation](https://docs.groupdocs.com/viewer/net/specify-encoding-when-loading-documents/).
