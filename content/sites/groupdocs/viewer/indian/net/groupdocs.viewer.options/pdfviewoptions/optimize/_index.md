## PdfViewOptions.Optimize property

टाइम्स न्यू रोमन और एरियल जैसे सामान्य फोंट को हटाकर और अन्य अनुकूलन तकनीकों को लागू करके आउटपुट फ़ाइल आकार को कम करें।

```csharp
public bool Optimize { get; set; }
```

### टिप्पणियों

संस्करण 23.1 में यह विकल्प समर्थित है[स्प्रेडशीट फ़ाइल स्वरूप](https://docs.groupdocs.com/viewer/net/supported-document-formats/#spreadsheet-file-formats) केवल.

एक्सेल और ऐप्पल नंबर स्प्रेडशीट को PDF में निर्यात करने का तरीका जानें

* [स्प्रेडशीट को PDF के रूप में रेंडर करें](https://docs.groupdocs.com/viewer/net/render-excel-and-apple-numbers-spreadsheets/#render-spreadsheets-as-pdf)

### उदाहरण

उदाहरण इस विकल्प के सामान्य उपयोग को प्रदर्शित करता है।

```csharp
using (var viewer = new Viewer("employees.xlsx"))
{
    PdfViewOptions viewOptions = new PdfViewOptions();
    viewOptions.Optimize = true;

    viewer.View(viewOptions);
}
```
