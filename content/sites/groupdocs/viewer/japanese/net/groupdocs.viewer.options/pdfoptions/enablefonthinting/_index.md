## PdfOptions.EnableFontHinting property

フォント ヒンティングを有効にします。フォント ヒンティングは、アウトライン フォントの表示を調整します。 ソース ドキュメントでこれらのフォントが使用されている場合、TTF フォントに対してのみサポートされます。

```csharp
public bool EnableFontHinting { get; set; }
```

### 備考

このオプションは、PNG または JPG 形式にレンダリングする場合にサポートされます。
