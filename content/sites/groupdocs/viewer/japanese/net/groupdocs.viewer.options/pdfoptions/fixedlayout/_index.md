## PdfOptions.FixedLayout property

PDF は固定形式であるため、すべての要素がページ上の特定の場所にあります. 出力 HTML がソース PDF と同じように見えるようにするために、ドキュメントはデフォルトで固定レイアウトの HTML にレンダリングされます. 固定レイアウト 出力 HTML は固定サイズであるため、要素はウィンドウ サイズに関係なく同じ場所に留まります。 流動レイアウトで HTML にレンダリングするには、このプロパティを`間違い`.

```csharp
public bool FixedLayout { get; set; }
```

### 備考

デフォルト値は`真実`.

このオプションは、HTML へのレンダリング時にサポートされます。

Fluid Layout イメージへのレンダリングがスキップされる場合。 テキスト コンテンツを含む PDF ドキュメントをレンダリングする場合は Fluid Layout を使用します。

### 例

この例は、流動的なレイアウトで PDF ドキュメントを HTML にレンダリングする方法を示しています.

```csharp
using GroupDocs.Viewer;
using GroupDocs.Viewer.Options;
//..

using (var viewer = new Viewer("resume.pdf"))
{
   var viewOptions = HtmlViewOptions.ForEmbeddedResources("page_{0}.html");
   viewOptions.FixedLayout = false;
   viewer.View(viewOptions);
}        
```
