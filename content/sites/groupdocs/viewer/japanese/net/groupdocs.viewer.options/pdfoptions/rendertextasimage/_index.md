## PdfOptions.RenderTextAsImage property

このオプションが真実、テキストは出力 HTML でイメージとしてレンダリングされます。 このオプションを有効にすると、テキストを選択できないようにするか、文字のレンダリングを修正して、HTML を PDF のように見せることができます。 デフォルト値は間違い.

public bool RenderTextAsImage { get; set; }
