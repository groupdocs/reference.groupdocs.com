## MailStorageOptions.MaxItems property

レンダリングするメッセージまたはアイテムの最大数。

```csharp
public int MaxItems { get; set; }
```

### 備考

Lotus Notes データ ファイルは大きくなる可能性があり、すべてのメッセージを取得するにはかなりの時間がかかる可能性があります. この設定は、表示されるメッセージまたはアイテムの最大数を制限します. デフォルト値は 0 です - すべてのメッセージが表示されます
