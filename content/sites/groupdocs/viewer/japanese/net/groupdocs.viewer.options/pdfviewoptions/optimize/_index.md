## PdfViewOptions.Optimize property

Times New Roman や Arial などの一般的なフォントを除外し、他の最適化手法を適用して、出力ファイル サイズを縮小します。

```csharp
public bool Optimize { get; set; }
```

### 備考

バージョン 23.1 では、このオプションは[スプレッドシートのファイル形式](https://docs.groupdocs.com/viewer/net/supported-document-formats/#spreadsheet-file-formats)のみ.

Excel および Apple Numbers スプレッドシートを PDF にエクスポートする方法の詳細

* [スプレッドシートを PDF としてレンダリング](https://docs.groupdocs.com/viewer/net/render-excel-and-apple-numbers-spreadsheets/#render-spreadsheets-as-pdf)

### 例

この例は、このオプションの典型的な使用法を示しています.

```csharp
using (var viewer = new Viewer("employees.xlsx"))
{
    PdfViewOptions viewOptions = new PdfViewOptions();
    viewOptions.Optimize = true;

    viewer.View(viewOptions);
}
```
