## HtmlViewOptions.FontsToExclude property

HTML ドキュメントから除外するフォント名のリスト。

```csharp
public List<string> FontsToExclude { get; set; }
```

### 備考

このオプションは、プレゼンテーションでのみサポートされています。 HTML ドキュメントに追加されたフォントは、出力ビューの安定性を向上させます, 同時にレンダリング結果のサイズを大きくします.このオプション、lets は、安定性と出力サイズの間の妥協点を見つけます。一般的でほとんどのシステムにインストールされているフォント名 を含めます。 このプロパティは、[`ExcludeFonts`](../excludefonts)オプションは無効です.
