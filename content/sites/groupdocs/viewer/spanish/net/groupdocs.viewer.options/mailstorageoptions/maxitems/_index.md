## MailStorageOptions.MaxItems property

El número máximo de mensajes o elementos para renderizar.

```csharp
public int MaxItems { get; set; }
```

### Observaciones

Los archivos de datos de Lotus Notes pueden ser grandes y la recuperación de todos los mensajes puede llevar mucho tiempo. Esta configuración limita la cantidad máxima de mensajes o elementos que se procesan. El valor predeterminado es 0: todos los mensajes se procesarán
