## PdfOptions.RenderOriginalPageSize property

Cuando esta opción está habilitada, las páginas de salida tendrán el mismo tamaño en píxeles que el tamaño de página en un documento PDF de origen. De manera predeterminada, GroupDocs.Viewer calcula el tamaño de página de la imagen de salida para una mejor calidad de representación.

```csharp
public bool RenderOriginalPageSize { get; set; }
```

### Observaciones

Esta opción es compatible cuando se renderiza en formatos PNG o JPG.
