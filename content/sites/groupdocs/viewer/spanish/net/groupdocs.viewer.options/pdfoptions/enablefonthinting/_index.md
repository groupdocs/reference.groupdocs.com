## PdfOptions.EnableFontHinting property

Habilita la sugerencia de fuente. La sugerencia de fuente ajusta la visualización de una fuente de contorno. Compatible solo con fuentes TTF cuando estas fuentes se usan en el documento de origen.

```csharp
public bool EnableFontHinting { get; set; }
```

### Observaciones

Esta opción es compatible cuando se renderiza en formatos PNG o JPG.
