## PdfOptions.FixedLayout property

PDF es un formato fijo, por lo que todos los elementos tienen un lugar específico en una página. Para garantizar que el HTML de salida tenga el mismo aspecto que el PDF de origen, los documentos se procesan en HTML con un diseño fijo de forma predeterminada. Al procesar con diseño fijo, el HTML de salida tiene un tamaño fijo, por lo que los elementos permanecerán en el mismo lugar independientemente del tamaño de la ventana. Para representar a HTML con un diseño fluido, establezca esta propiedad en`FALSO` .

```csharp
public bool FixedLayout { get; set; }
```

### Observaciones

El valor predeterminado es`verdadero` .

Esta opción es compatible cuando se procesa en HTML.

Al omitir la representación en imágenes de diseño fluido. Utilice el diseño fluido al representar documentos PDF con contenido de texto.

### Ejemplos

Este ejemplo muestra cómo convertir un documento PDF a HTML con un diseño fluido.

```csharp
using GroupDocs.Viewer;
using GroupDocs.Viewer.Options;
//..

using (var viewer = new Viewer("resume.pdf"))
{
   var viewOptions = HtmlViewOptions.ForEmbeddedResources("page_{0}.html");
   viewOptions.FixedLayout = false;
   viewer.View(viewOptions);
}        
```
