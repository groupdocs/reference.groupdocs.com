## LoadOptions.DetectEncoding property

Esta opción habilita[`TXT`](../../../groupdocs.viewer/filetype/txt) ,[`TSV`](../../../groupdocs.viewer/filetype/tsv) , y[`CSV`](../../../groupdocs.viewer/filetype/csv)detección de codificación de archivos. En caso de que la codificación no se pueda detectar, el valor predeterminado[`Encoding`](../encoding) se usa.

```csharp
public bool DetectEncoding { get; set; }
```

### Ejemplos

El ejemplo demuestra un uso típico de esta opción.

```csharp
LoadOptions loadOptions = new LoadOptions();
loadOptions.DetectEncoding = true; // Habilitar la detección de codificación

using (Viewer viewer = new Viewer("employees.csv", loadOptions))
{
    HtmlViewOptions viewOptions = 
        HtmlViewOptions.ForEmbeddedResources();

    viewer.View(viewOptions);
}
```
