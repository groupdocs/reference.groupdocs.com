## OutlookOptions.MaxItemsInFolder property

El número máximo de mensajes o elementos que se pueden representar desde una carpeta.

```csharp
public int MaxItemsInFolder { get; set; }
```

### Observaciones

Los archivos de datos de Outlook pueden ser grandes y la recuperación de todos los mensajes puede llevar mucho tiempo. Esta configuración limita la cantidad máxima de mensajes o elementos (como contactos y tareas) que se procesan. El valor predeterminado es 50. Para procesar todos los mensajes, configure el valor a 0.
