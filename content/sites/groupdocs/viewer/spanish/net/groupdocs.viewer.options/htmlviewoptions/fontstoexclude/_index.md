## HtmlViewOptions.FontsToExclude property

La lista de nombres de fuentes, para excluir del documento HTML.

```csharp
public List<string> FontsToExclude { get; set; }
```

### Observaciones

Esta opción solo se admite para presentaciones. Las fuentes que se agregan al documento HTML mejoran la estabilidad de la vista de salida, al mismo tiempo que aumentan el tamaño del resultado de representación. Esta opción, permite encontrar un compromiso entre la estabilidad y el tamaño de salida. Incluya los nombres de fuente que son populares e instalados en la mayoría de los sistemas. Tenga en cuenta que esta propiedad está activa solo cuando[`ExcludeFonts`](../excludefonts) las opciones están deshabilitadas.
