## MailStorageOptions.MaxItems property

Die maximale Anzahl von Nachrichten oder Elementen zum Rendern.

```csharp
public int MaxItems { get; set; }
```

### Bemerkungen

Lotus Notes-Datendateien können groß sein und das Abrufen aller Nachrichten kann viel Zeit in Anspruch nehmen. Diese Einstellung begrenzt die maximale Anzahl von Nachrichten oder Elementen, die gerendert werden. Der Standardwert ist 0 – alle Nachrichten werden gerendert
