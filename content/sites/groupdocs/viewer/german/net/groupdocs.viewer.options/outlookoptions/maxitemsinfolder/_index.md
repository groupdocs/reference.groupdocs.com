## OutlookOptions.MaxItemsInFolder property

Die maximale Anzahl von Nachrichten oder Elementen, die von einem Ordner gerendert werden können.

```csharp
public int MaxItemsInFolder { get; set; }
```

### Bemerkungen

Outlook-Datendateien können groß sein und das Abrufen aller Nachrichten kann viel Zeit in Anspruch nehmen. Diese Einstellung begrenzt die maximale Anzahl von Nachrichten oder Elementen (wie Kontakte und Aufgaben), die gerendert werden. Der Standardwert ist 50. Um alle Nachrichten zu rendern, einstellen den Wert auf 0.
