## PdfOptions.RenderOriginalPageSize property

Wenn diese Option aktiviert ist, haben die Ausgabeseiten dieselbe Größe in Pixel wie die Seitengröße in einem PDF-Quelldokument. Standardmäßig berechnet GroupDocs.Viewer die Seitengröße des Ausgabebilds für eine bessere Wiedergabequalität.

```csharp
public bool RenderOriginalPageSize { get; set; }
```

### Bemerkungen

Diese Option wird beim Rendern im PNG- oder JPG-Format unterstützt.
