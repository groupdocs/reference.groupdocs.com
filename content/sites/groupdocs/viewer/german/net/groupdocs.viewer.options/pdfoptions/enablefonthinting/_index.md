## PdfOptions.EnableFontHinting property

Aktiviert Schrifthinweise. Das Font-Hinting passt die Anzeige eines Outline-Fonts an. Wird nur für TTF-Fonts unterstützt, wenn diese Fonts im Quelldokument verwendet werden.

```csharp
public bool EnableFontHinting { get; set; }
```

### Bemerkungen

Diese Option wird beim Rendern im PNG- oder JPG-Format unterstützt.
