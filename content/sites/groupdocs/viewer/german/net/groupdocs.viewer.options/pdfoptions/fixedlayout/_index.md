## PdfOptions.FixedLayout property

PDF ist ein festes Format, sodass alle Elemente einen bestimmten Platz auf einer Seite haben. Um sicherzustellen, dass das Ausgabe-HTML genauso aussieht wie das Quell-PDF, werden die Dokumente standardmäßig mit einem festen Layout in HTML gerendert. Beim Rendern mit Festes Layout Das Ausgabe-HTML hat eine feste Größe, sodass die Elemente unabhängig von der Fenstergröße an der gleichen Stelle bleiben. Um HTML mit flüssigem Layout zu rendern, setzen Sie diese Eigenschaft auf`FALSCH` .

```csharp
public bool FixedLayout { get; set; }
```

### Bemerkungen

Der Standardwert ist`WAHR` .

Diese Option wird beim Rendern in HTML unterstützt.

Beim Rendern in flüssiges Layout werden Bilder übersprungen. Verwenden Sie flüssiges Layout beim Rendern von PDF-Dokumenten mit Textinhalt.

### Beispiele

Dieses Beispiel zeigt, wie ein PDF-Dokument mit fließendem Layout in HTML gerendert wird.

```csharp
using GroupDocs.Viewer;
using GroupDocs.Viewer.Options;
//..

using (var viewer = new Viewer("resume.pdf"))
{
   var viewOptions = HtmlViewOptions.ForEmbeddedResources("page_{0}.html");
   viewOptions.FixedLayout = false;
   viewer.View(viewOptions);
}        
```
