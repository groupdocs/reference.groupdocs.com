## PdfViewOptions.Optimize property

Reduzieren Sie die Größe der Ausgabedatei, indem Sie gängige Schriftarten wie Times New Roman und Arial ausschließen und andere Optimierungstechniken anwenden.

```csharp
public bool Optimize { get; set; }
```

### Bemerkungen

In Version 23.1 wird diese Option unterstützt[Dateiformate für Tabellenkalkulationen](https://docs.groupdocs.com/viewer/net/supported-document-formats/#spreadsheet-file-formats) nur.

Weitere Informationen zum Exportieren von Excel- und Apple Numbers-Tabellen in PDF

* [Rendern Sie Tabellenkalkulationen als PDF](https://docs.groupdocs.com/viewer/net/render-excel-and-apple-numbers-spreadsheets/#render-spreadsheets-as-pdf)

### Beispiele

Das Beispiel zeigt eine typische Verwendung dieser Option.

```csharp
using (var viewer = new Viewer("employees.xlsx"))
{
    PdfViewOptions viewOptions = new PdfViewOptions();
    viewOptions.Optimize = true;

    viewer.View(viewOptions);
}
```
