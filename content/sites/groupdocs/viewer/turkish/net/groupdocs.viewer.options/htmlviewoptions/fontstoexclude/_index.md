## HtmlViewOptions.FontsToExclude property

HTML belgesinden hariç tutulacak yazı tipi adlarının listesi.

```csharp
public List<string> FontsToExclude { get; set; }
```

### Notlar

Bu seçenek yalnızca sunumlar için desteklenir. HTML belgesine eklenen yazı tipleri çıktı görünümünün kararlılığını artırır, aynı zamanda işleme sonucunun boyutunu artırır. Bu seçenek, kararlılık ve çıktı boyutu arasında uzlaşma bulmanızı sağlar. Popüler olan ve çoğu sisteme yüklenmiş olan name yazı tipini dahil edin. Lütfen bu özelliğin yalnızca şu durumlarda etkin olduğunu unutmayın:[`ExcludeFonts`](../excludefonts) seçenekler devre dışı bırakıldı.
