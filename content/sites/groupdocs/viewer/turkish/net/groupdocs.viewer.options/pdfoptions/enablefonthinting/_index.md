## PdfOptions.EnableFontHinting property

Yazı tipi ipuçlarını etkinleştirir. Yazı tipi ipucu, anahat yazı tipinin görüntüsünü ayarlar. Yalnızca TTF yazı tipleri için, bu yazı tipleri kaynak belgede kullanıldığında desteklenir.

```csharp
public bool EnableFontHinting { get; set; }
```

### Notlar

Bu seçenek, PNG veya JPG biçimlerine dönüştürülürken desteklenir.
