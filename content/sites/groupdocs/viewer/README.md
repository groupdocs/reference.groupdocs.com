# GroupDocs.Viewer — API Reference Source Files

This repository holds the Markdown source files used to generate the **GroupDocs.Viewer**
API reference, published at <https://reference.groupdocs.com/viewer/>.

## About GroupDocs.Viewer

[GroupDocs.Viewer](https://products.groupdocs.com/viewer/) is a developer API for document
processing: render documents and images to HTML, image, or PDF across 170+ formats. It is available for the following platforms, each with its own
API reference:

- **.NET** — <https://reference.groupdocs.com/viewer/net/>
- **Java** — <https://reference.groupdocs.com/viewer/java/>
- **Node.js via Java** — <https://reference.groupdocs.com/viewer/nodejs-java/>
- **Python via .NET** — <https://reference.groupdocs.com/viewer/python-net/>

## Repository purpose

The files in this repository are used to:

- Build and maintain the official **GroupDocs.Viewer** API reference.
- Keep the reference in sync with the latest product release.
- Provide a consistent structure for automated Hugo builds and publication.

Content is English-only and compiled into a static site with [Hugo](https://gohugo.io/).
