## PdfViewOptions.Optimize property

通过排除 Times New Roman 和 Arial 等常见字体并应用其他优化技术来减小输出文件大小。

```csharp
public bool Optimize { get; set; }
```

### 评论

在版本 23.1 中支持此选项[电子表格文件格式](https://docs.groupdocs.com/viewer/net/supported-document-formats/#spreadsheet-file-formats)只有.

### 例子

该示例演示了此选项的典型用法。

```csharp
using (var viewer = new Viewer("employees.xlsx"))
{
    PdfViewOptions viewOptions = new PdfViewOptions();
    viewOptions.Optimize = true;

    viewer.View(viewOptions);
}
```
