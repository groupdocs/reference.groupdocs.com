## PdfOptions.RenderTextAsImage property

当这个选项设置为真的，文本在输出 HTML 中呈现为图像。 启用此选项可使文本不可选择或修复字符呈现并使 HTML 看起来像 PDF。 默认值为错误的.

```csharp
public bool RenderTextAsImage { get; set; }
```
