## LoadOptions.DetectEncoding property

此选项启用[`TXT`](../../../groupdocs.viewer/filetype/txt),[`TSV`](../../../groupdocs.viewer/filetype/tsv) ， 和[`CSV`](../../../groupdocs.viewer/filetype/csv)文件编码检测. 以防无法检测到默认编码[`Encoding`](../encoding)使用.

```csharp
public bool DetectEncoding { get; set; }
```

### 例子

该示例演示了此选项的典型用法。

```csharp
LoadOptions loadOptions = new LoadOptions();
loadOptions.DetectEncoding = true; // 启用编码检测

using (Viewer viewer = new Viewer("employees.csv", loadOptions))
{
    HtmlViewOptions viewOptions = 
        HtmlViewOptions.ForEmbeddedResources();

    viewer.View(viewOptions);
}
```
