## PdfOptions.EnableFontHinting property

Schakelt hints naar lettertypen in. De lettertypehint past de weergave van een contourlettertype aan. Alleen ondersteund voor TTF-lettertypen wanneer deze lettertypen in het brondocument worden gebruikt.

```csharp
public bool EnableFontHinting { get; set; }
```

### Opmerkingen

Deze optie wordt ondersteund bij weergave in PNG- of JPG-indeling.
