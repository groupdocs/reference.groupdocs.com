## MailStorageOptions.MaxItems property

Het maximale aantal berichten of items voor weergave.

```csharp
public int MaxItems { get; set; }
```

### Opmerkingen

Lotus Notes-gegevensbestanden kunnen groot zijn en het ophalen van alle berichten kan veel tijd in beslag nemen. Deze instelling beperkt het maximale aantal berichten of items dat wordt weergegeven. Standaardwaarde is 0 - alle berichten worden weergegeven
