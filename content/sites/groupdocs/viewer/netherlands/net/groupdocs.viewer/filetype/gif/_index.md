## FileType.GIF field

Graphical Interchange Format File (.gif) is een type sterk gecomprimeerde afbeelding. Voor elke afbeelding staat GIF doorgaans maximaal 8 bits per pixel toe en maximaal 256 kleuren zijn toegestaan in de afbeelding.
