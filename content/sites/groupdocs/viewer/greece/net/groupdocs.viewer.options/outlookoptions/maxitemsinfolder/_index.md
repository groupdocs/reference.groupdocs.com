## OutlookOptions.MaxItemsInFolder property

Ο μέγιστος αριθμός μηνυμάτων ή στοιχείων που μπορούν να αποδοθούν από έναν φάκελο.

```csharp
public int MaxItemsInFolder { get; set; }
```

### Παρατηρήσεις

Τα αρχεία δεδομένων του Outlook μπορεί να είναι μεγάλα και η ανάκτηση όλων των μηνυμάτων μπορεί να διαρκέσει σημαντικό χρόνο. Αυτή η ρύθμιση περιορίζει τον μέγιστο αριθμό μηνυμάτων ή στοιχείων (όπως επαφές και εργασίες) που αποδίδονται. Η προεπιλεγμένη τιμή είναι 50. Για την απόδοση όλων των μηνυμάτων, ορίστε η τιμή σε 0.
