## PdfViewOptions.Optimize property

Μειώστε το μέγεθος του αρχείου εξόδου εξαιρώντας κοινές γραμματοσειρές όπως το Times New Roman και το Arial και εφαρμόζοντας άλλες τεχνικές βελτιστοποίησης.

```csharp
public bool Optimize { get; set; }
```

### Παρατηρήσεις

Στην έκδοση 23.1 αυτή η επιλογή υποστηρίζεται[Μορφές αρχείων υπολογιστικών φύλλων](https://docs.groupdocs.com/viewer/net/supported-document-formats/#spreadsheet-file-formats) μόνο.

Μάθετε περισσότερα πώς να εξάγετε υπολογιστικά φύλλα Excel και Apple Numbers σε PDF

* [Απόδοση υπολογιστικών φύλλων ως PDF](https://docs.groupdocs.com/viewer/net/render-excel-and-apple-numbers-spreadsheets/#render-spreadsheets-as-pdf)

### Παραδείγματα

Το παράδειγμα δείχνει μια τυπική χρήση αυτής της επιλογής.

```csharp
using (var viewer = new Viewer("employees.xlsx"))
{
    PdfViewOptions viewOptions = new PdfViewOptions();
    viewOptions.Optimize = true;

    viewer.View(viewOptions);
}
```
