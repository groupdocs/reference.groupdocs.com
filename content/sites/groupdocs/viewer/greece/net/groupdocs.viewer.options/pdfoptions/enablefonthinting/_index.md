## PdfOptions.EnableFontHinting property

Ενεργοποιεί την υπόδειξη γραμματοσειράς. Η υπόδειξη γραμματοσειράς προσαρμόζει την εμφάνιση μιας γραμματοσειράς περίγραμμα. Υποστηρίζεται μόνο για γραμματοσειρές TTF όταν αυτές οι γραμματοσειρές χρησιμοποιούνται στο έγγραφο προέλευσης.

```csharp
public bool EnableFontHinting { get; set; }
```

### Παρατηρήσεις

Αυτή η επιλογή υποστηρίζεται κατά την απόδοση σε μορφές PNG ή JPG.
