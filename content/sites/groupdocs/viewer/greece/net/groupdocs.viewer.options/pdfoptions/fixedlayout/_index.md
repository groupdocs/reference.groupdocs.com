## PdfOptions.FixedLayout property

Το Το PDF είναι μια σταθερή μορφή, έτσι όλα τα στοιχεία έχουν μια συγκεκριμένη θέση σε μια σελίδα. Για να διασφαλιστεί ότι το HTML εξόδου μοιάζει με το αρχείο προέλευσης PDF, τα έγγραφα αποδίδονται σε HTML με σταθερή διάταξη από προεπιλογή. Κατά την απόδοση με σταθερή διάταξη το HTML εξόδου έχει σταθερό μέγεθος, επομένως τα στοιχεία θα παραμένουν στην ίδια θέση ανεξάρτητα από το μέγεθος του παραθύρου. Για απόδοση σε HTML με ρευστή διάταξη ορίστε αυτήν την ιδιότητα σε`ψευδής` .

```csharp
public bool FixedLayout { get; set; }
```

### Παρατηρήσεις

Η προεπιλεγμένη τιμή είναι`αληθής` .

Αυτή η επιλογή υποστηρίζεται κατά την απόδοση σε HTML.

Κατά την απόδοση σε ρευστή διάταξη, οι εικόνες παραλείπονται. Χρησιμοποιήστε ρευστή διάταξη κατά την απόδοση εγγράφων PDF με περιεχόμενο κειμένου.

### Παραδείγματα

Αυτό το παράδειγμα δείχνει πώς να αποδώσετε έγγραφο PDF σε HTML με ρευστή διάταξη.

```csharp
using GroupDocs.Viewer;
using GroupDocs.Viewer.Options;
//..

using (var viewer = new Viewer("resume.pdf"))
{
   var viewOptions = HtmlViewOptions.ForEmbeddedResources("page_{0}.html");
   viewOptions.FixedLayout = false;
   viewer.View(viewOptions);
}        
```
