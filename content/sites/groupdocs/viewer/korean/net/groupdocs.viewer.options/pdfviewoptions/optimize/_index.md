## PdfViewOptions.Optimize property

Times New Roman 및 Arial과 같은 일반적인 글꼴을 제외하고 다른 최적화 기술을 적용하여 출력 파일 크기를 줄입니다.

```csharp
public bool Optimize { get; set; }
```

### 비고

버전 23.1에서 이 옵션은 다음에 대해 지원됩니다.[스프레드시트 파일 형식](https://docs.groupdocs.com/viewer/net/supported-document-formats/#spreadsheet-file-formats) 만.

Excel 및 Apple Numbers 스프레드시트를 PDF 로 내보내는 방법 자세히 알아보기

* [스프레드시트를 PDF로 렌더링](https://docs.groupdocs.com/viewer/net/render-excel-and-apple-numbers-spreadsheets/#render-spreadsheets-as-pdf)

### 예

이 예는 이 옵션의 일반적인 사용법을 보여줍니다.

```csharp
using (var viewer = new Viewer("employees.xlsx"))
{
    PdfViewOptions viewOptions = new PdfViewOptions();
    viewOptions.Optimize = true;

    viewer.View(viewOptions);
}
```
