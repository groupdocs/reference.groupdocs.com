## LoadOptions.DetectEncoding property

이 옵션은[`TXT`](../../../groupdocs.viewer/filetype/txt) ,[`TSV`](../../../groupdocs.viewer/filetype/tsv) , 그리고[`CSV`](../../../groupdocs.viewer/filetype/csv)파일 인코딩 감지. 인코딩을 감지할 수 없는 경우 기본값[`Encoding`](../encoding) 사용중입니다.

```csharp
public bool DetectEncoding { get; set; }
```

### 예

이 예는 이 옵션의 일반적인 사용법을 보여줍니다.

```csharp
LoadOptions loadOptions = new LoadOptions();
loadOptions.DetectEncoding = true; // 인코딩 감지 활성화

using (Viewer viewer = new Viewer("employees.csv", loadOptions))
{
    HtmlViewOptions viewOptions = 
        HtmlViewOptions.ForEmbeddedResources();

    viewer.View(viewOptions);
}
```
