## PdfOptions.FixedLayout property

PDF는 고정 형식이므로 모든 요소가 페이지의 특정 위치에 있습니다. 출력 HTML이 소스 PDF와 동일하게 보이도록 문서는 기본적으로 고정 레이아웃의 HTML로 렌더링됩니다. 렌더링 시 고정 레이아웃 출력 HTML은 고정 크기이므로 요소는 창 크기에 관계없이 같은 위치에 유지됩니다. 유동 레이아웃을 사용하여 HTML로 렌더링하려면 이 속성을`거짓` .

```csharp
public bool FixedLayout { get; set; }
```

### 비고

기본값은`진실` .

이 옵션은 HTML로 렌더링할 때 지원됩니다.

유동 레이아웃으로 렌더링할 때 이미지를 건너뜁니다. 텍스트 콘텐츠가 있는 PDF 문서를 렌더링할 때 유동 레이아웃을 사용합니다.

### 예

이 예는 PDF 문서를 유동 레이아웃을 사용하여 HTML로 렌더링하는 방법을 보여줍니다.

```csharp
using GroupDocs.Viewer;
using GroupDocs.Viewer.Options;
//..

using (var viewer = new Viewer("resume.pdf"))
{
   var viewOptions = HtmlViewOptions.ForEmbeddedResources("page_{0}.html");
   viewOptions.FixedLayout = false;
   viewer.View(viewOptions);
}        
```
