## WordProcessingOptions.PageSize property

출력 페이지의 크기입니다. 기본값은Unspecified 이는 페이지 설정(Page Setup)에서 설정한 페이지 크기를 의미합니다.  HTM 및 HTML 파일을 렌더링할 때 기본 페이지 크기는 Letter 792 x 612포인트로 설정됩니다. 결과적으로 일부 콘텐츠가 페이지 프레임에 맞지 않을 수 있습니다. 더 큰 페이지 크기를 설정하십시오.A3 내용에 맞게.
