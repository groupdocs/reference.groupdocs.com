## PdfViewOptions.Optimize property

تقليل حجم ملف الإخراج عن طريق استبعاد الخطوط الشائعة مثل Times New Roman و Arial ، وتطبيق تقنيات التحسين الأخرى.

```csharp
public bool Optimize { get; set; }
```

### ملاحظات

في الإصدار 23.1 ، يتم دعم هذا الخيار لـ[تنسيقات ملفات جدول البيانات](https://docs.groupdocs.com/viewer/net/supported-document-formats/#spreadsheet-file-formats) فقط .

تعرف على المزيد حول كيفية تصدير جداول بيانات Excel و Apple Numbers إلى PDF

* [اعرض جداول البيانات بصيغة PDF](https://docs.groupdocs.com/viewer/net/render-excel-and-apple-numbers-spreadsheets/#render-spreadsheets-as-pdf)

### أمثلة

يوضح المثال استخدامًا نموذجيًا لهذا الخيار.

```csharp
using (var viewer = new Viewer("employees.xlsx"))
{
    PdfViewOptions viewOptions = new PdfViewOptions();
    viewOptions.Optimize = true;

    viewer.View(viewOptions);
}
```
