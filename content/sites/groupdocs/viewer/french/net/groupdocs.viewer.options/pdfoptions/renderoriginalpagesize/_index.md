## PdfOptions.RenderOriginalPageSize property

Lorsque cette option est activée, les pages de sortie auront la même taille en pixels que la taille de la page dans un document PDF source. Par défaut, GroupDocs.Viewer calcule la taille de la page de l'image de sortie pour une meilleure qualité de rendu.

```csharp
public bool RenderOriginalPageSize { get; set; }
```

### Remarques

Cette option est prise en charge lors du rendu aux formats PNG ou JPG.
