## PdfOptions.FixedLayout property

PDF est un format fixe, de sorte que tous les éléments ont une place spécifique sur une page. Pour garantir que le HTML de sortie ressemble au PDF source, les documents sont rendus au format HTML avec une mise en page fixe par défaut. Lors du rendu avec mise en page fixe le HTML de sortie a une taille fixe afin que les éléments restent au même endroit quelle que soit la taille de la fenêtre. Pour rendre au format HTML avec une mise en page fluide, définissez cette propriété sur`FAUX` .

```csharp
public bool FixedLayout { get; set; }
```

### Remarques

La valeur par défaut est`vrai` .

Cette option est prise en charge lors du rendu en HTML.

Lors du rendu en mise en page fluide, les images sont ignorées. Utilisez la mise en page fluide lors du rendu de documents PDF avec un contenu textuel.

### Exemples

Cet exemple montre comment convertir un document PDF en HTML avec une mise en page fluide.

```csharp
using GroupDocs.Viewer;
using GroupDocs.Viewer.Options;
//..

using (var viewer = new Viewer("resume.pdf"))
{
   var viewOptions = HtmlViewOptions.ForEmbeddedResources("page_{0}.html");
   viewOptions.FixedLayout = false;
   viewer.View(viewOptions);
}        
```
