## PdfViewOptions.Optimize property

Réduisez la taille du fichier de sortie en excluant les polices courantes telles que Times New Roman et Arial, et en appliquant d'autres techniques d'optimisation.

```csharp
public bool Optimize { get; set; }
```

### Remarques

Dans la version 23.1, cette option est prise en charge pour[Formats de fichier de feuille de calcul](https://docs.groupdocs.com/viewer/net/supported-document-formats/#spreadsheet-file-formats) seulement.

En savoir plus sur l'exportation de feuilles de calcul Excel et Apple Numbers au format PDF

* [Rendu des feuilles de calcul au format PDF](https://docs.groupdocs.com/viewer/net/render-excel-and-apple-numbers-spreadsheets/#render-spreadsheets-as-pdf)

### Exemples

L'exemple illustre une utilisation typique de cette option.

```csharp
using (var viewer = new Viewer("employees.xlsx"))
{
    PdfViewOptions viewOptions = new PdfViewOptions();
    viewOptions.Optimize = true;

    viewer.View(viewOptions);
}
```
