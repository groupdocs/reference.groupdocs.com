## MailStorageOptions.MaxItems property

Le nombre maximal de messages ou d'éléments à afficher.

```csharp
public int MaxItems { get; set; }
```

### Remarques

Les fichiers de données Lotus Notes peuvent être volumineux et la récupération de tous les messages peut prendre beaucoup de temps. Ce paramètre limite le nombre maximal de messages ou d'éléments rendus. La valeur par défaut est 0 - tous les messages seront rendus
