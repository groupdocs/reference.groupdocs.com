## OutlookOptions.MaxItemsInFolder property

Le nombre maximum de messages ou d'éléments pouvant être rendus à partir d'un dossier.

```csharp
public int MaxItemsInFolder { get; set; }
```

### Remarques

Les fichiers de données Outlook peuvent être volumineux et la récupération de tous les messages peut prendre beaucoup de temps. Ce paramètre limite le nombre maximal de messages ou d'éléments (comme les contacts et les tâches) qui sont affichés. La valeur par défaut est 50. Pour afficher tous les messages, définissez la valeur à 0.
