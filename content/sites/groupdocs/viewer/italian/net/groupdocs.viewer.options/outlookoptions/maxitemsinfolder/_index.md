## OutlookOptions.MaxItemsInFolder property

Il numero massimo di messaggi o elementi che possono essere visualizzati da una cartella.

```csharp
public int MaxItemsInFolder { get; set; }
```

### Osservazioni

I file di dati di Outlook possono essere di grandi dimensioni e il recupero di tutti i messaggi può richiedere molto tempo. Questa impostazione limita il numero massimo di messaggi o elementi (come contatti e attività) che vengono visualizzati. Il valore predefinito è 50. Per eseguire il rendering di tutti i messaggi, impostare il valore a 0.
