## PdfOptions.RenderOriginalPageSize property

Quando questa opzione è abilitata, le pagine di output avranno le stesse dimensioni in pixel delle dimensioni della pagina in un documento PDF di origine. Per impostazione predefinita, GroupDocs.Viewer calcola le dimensioni della pagina dell'immagine di output per una migliore qualità di rendering.

```csharp
public bool RenderOriginalPageSize { get; set; }
```

### Osservazioni

Questa opzione è supportata durante il rendering nei formati PNG o JPG.
