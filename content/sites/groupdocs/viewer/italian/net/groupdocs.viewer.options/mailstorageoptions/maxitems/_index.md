## MailStorageOptions.MaxItems property

Il numero massimo di messaggi o elementi per il rendering.

```csharp
public int MaxItems { get; set; }
```

### Osservazioni

I file di dati di Lotus Notes possono essere di grandi dimensioni e il recupero di tutti i messaggi può richiedere molto tempo. Questa impostazione limita il numero massimo di messaggi o elementi di cui viene eseguito il rendering. Il valore predefinito è 0: verrà eseguito il rendering di tutti i messaggi
