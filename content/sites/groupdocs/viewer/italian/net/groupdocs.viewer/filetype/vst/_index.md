## FileType.VST field

Visio Drawing Template (.vst) sono file di immagini vettoriali creati con Microsoft Visio e fungono da modello per la creazione di ulteriori file. Questi file modello sono in formato di file binario e contengono il layout e le impostazioni predefiniti utilizzati per la creazione di nuovi disegni di Visio.
