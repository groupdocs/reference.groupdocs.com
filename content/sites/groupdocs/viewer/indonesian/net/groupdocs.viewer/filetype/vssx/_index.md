## FileType.VSSX field

Visio Stencil File (.vssx) adalah stensil gambar yang dibuat dengan Microsoft Visio 2013 ke atas. Format file VSSX dapat dibuka dengan Visio 2013 ke atas. File Visio dikenal untuk representasi berbagai elemen gambar seperti kumpulan bentuk, konektor, bagan alur, tata letak jaringan, diagram UML, Pelajari lebih lanjut tentang format file ini[Di Sini](https://wiki.fileformat.com/image/vssx) .
