## FileType.STL field

Stereolithography File (.stl) adalah format file yang dapat dipertukarkan yang merepresentasikan geometri permukaan 3 dimensi. Format file menemukan penggunaannya di beberapa bidang seperti pembuatan prototipe cepat, pencetakan 3D, dan pembuatan dengan bantuan komputer.
