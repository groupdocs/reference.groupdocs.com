## FileType.PLT field

PLT (HPGL) (.plt) adalah file plotter berbasis vektor yang diperkenalkan oleh Autodesk, Inc. dan berisi informasi untuk file CAD tertentu. Detail plot membutuhkan akurasi dan presisi dalam produksi, dan penggunaan file PLT menjamin ini karena semua gambar dicetak menggunakan garis, bukan titik.
