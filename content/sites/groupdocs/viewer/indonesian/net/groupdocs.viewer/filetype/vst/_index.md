## FileType.VST field

Visio Drawing Template (.vst) adalah file gambar vektor yang dibuat dengan Microsoft Visio dan berfungsi sebagai template untuk membuat file selanjutnya. File template ini dalam format file biner dan berisi tata letak dan pengaturan default yang digunakan untuk pembuatan gambar Visio baru.
