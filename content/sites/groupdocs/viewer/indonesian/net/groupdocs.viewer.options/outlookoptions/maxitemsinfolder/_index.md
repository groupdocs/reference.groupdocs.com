## OutlookOptions.MaxItemsInFolder property

Jumlah maksimum pesan atau item, yang dapat dirender dari satu folder.

```csharp
public int MaxItemsInFolder { get; set; }
```

### Perkataan

File data Outlook bisa berukuran besar dan mengambil semua pesan bisa memakan waktu lama. Pengaturan ini membatasi jumlah maksimum pesan atau item (seperti kontak dan tugas) yang ditampilkan. Nilai default adalah 50. Untuk merender semua pesan, tetapkan nilai ke 0.
