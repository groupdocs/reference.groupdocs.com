## PdfOptions.RenderOriginalPageSize property

Ketika opsi ini diaktifkan, halaman keluaran akan memiliki ukuran yang sama dalam piksel seperti ukuran halaman dalam dokumen PDF sumber. Secara default GroupDocs.Viewer menghitung ukuran halaman gambar keluaran untuk kualitas perenderan yang lebih baik.

```csharp
public bool RenderOriginalPageSize { get; set; }
```

### Perkataan

Opsi ini didukung saat merender ke dalam format PNG atau JPG.
