## HtmlViewOptions.FontsToExclude property

Daftar nama font, untuk dikecualikan dari dokumen HTML.

```csharp
public List<string> FontsToExclude { get; set; }
```

### Perkataan

Opsi ini hanya didukung untuk presentasi. Font yang ditambahkan ke dalam dokumen HTML meningkatkan stabilitas tampilan output, sekaligus meningkatkan ukuran hasil rendering. Opsi ini, memungkinkan Anda menemukan kompromi, antara stabilitas dan ukuran keluaran. Sertakan nama font yang populer dan dipasang di sebagian besar sistem. Perlu diketahui, properti ini hanya aktif jika[`ExcludeFonts`](../excludefonts) opsi dinonaktifkan.
