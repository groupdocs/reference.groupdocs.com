## LoadOptions.DetectEncoding property

Opsi ini diaktifkan[`TXT`](../../../groupdocs.viewer/filetype/txt) ,[`TSV`](../../../groupdocs.viewer/filetype/tsv) , Dan[`CSV`](../../../groupdocs.viewer/filetype/csv)deteksi pengkodean file. Jika pengkodean tidak dapat dideteksi secara default[`Encoding`](../encoding) digunakan.

```csharp
public bool DetectEncoding { get; set; }
```

* [Render dokumen teks sebagai file HTML, PDF, dan gambar](https://docs.groupdocs.com/viewer/net/render-text-files/)
* [Render spreadsheet Excel dan Apple Numbers sebagai file HTML, PDF, dan gambar](https://docs.groupdocs.com/viewer/net/render-excel-and-apple-numbers-spreadsheets/)

### Contoh

Contoh menunjukkan penggunaan umum opsi ini.

```csharp
LoadOptions loadOptions = new LoadOptions();
loadOptions.DetectEncoding = true; // Aktifkan deteksi penyandian

using (Viewer viewer = new Viewer("employees.csv", loadOptions))
{
    HtmlViewOptions viewOptions = 
        HtmlViewOptions.ForEmbeddedResources();

    viewer.View(viewOptions);
}
```
