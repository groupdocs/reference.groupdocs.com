## FileType.GZIP field

Gnu Zipped File (.gzip) introducerades som ett gratis verktyg för att ersätta Compress-programmet som används i Unix-system. Sådana filer kan öppnas och extraheras med ett flertal applikationer som WinZip som är tillgängligt på både Windows och MacOS.
