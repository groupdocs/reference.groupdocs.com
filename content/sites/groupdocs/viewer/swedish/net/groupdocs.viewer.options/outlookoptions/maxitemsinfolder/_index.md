## OutlookOptions.MaxItemsInFolder property

Det maximala antalet meddelanden eller objekt som kan renderas från en mapp.

```csharp
public int MaxItemsInFolder { get; set; }
```

### Anmärkningar

Outlook-datafiler kan vara stora och det kan ta lång tid att hämta alla meddelanden. Den här inställningen begränsar det maximala antalet meddelanden eller objekt (som kontakter och uppgifter) som renderas. Standardvärdet är 50. För att rendera alla meddelanden, ställ in värdet till 0.
