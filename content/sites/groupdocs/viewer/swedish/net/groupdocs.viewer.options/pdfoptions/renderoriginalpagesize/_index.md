## PdfOptions.RenderOriginalPageSize property

När det här alternativet är aktiverat kommer utgående sidor att ha samma storlek i pixlar som sidstorleken i ett PDF-källdokument. Som standard beräknar GroupDocs.Viewer utdatabildens sidstorlek för bättre renderingskvalitet.

```csharp
public bool RenderOriginalPageSize { get; set; }
```

### Anmärkningar

Det här alternativet stöds vid rendering till PNG- eller JPG-format.
