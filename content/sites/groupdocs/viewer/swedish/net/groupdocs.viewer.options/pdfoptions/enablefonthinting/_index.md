## PdfOptions.EnableFontHinting property

Aktiverar teckensnittstips. Teckensnittstipset justerar visningen av ett konturteckensnitt. Stöds endast för TTF-teckensnitt när dessa teckensnitt används i källdokumentet.

```csharp
public bool EnableFontHinting { get; set; }
```

### Anmärkningar

Det här alternativet stöds vid rendering till PNG- eller JPG-format.
