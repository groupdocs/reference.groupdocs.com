## PdfViewOptions.Optimize property

Уменьшите размер выходного файла, исключив распространенные шрифты, такие как Times New Roman и Arial, и применив другие методы оптимизации.

```csharp
public bool Optimize { get; set; }
```

### Примечания

В версии 23.1 эта опция поддерживается для[Форматы файлов электронных таблиц](https://docs.groupdocs.com/viewer/net/supported-document-formats/#spreadsheet-file-formats) только.

Узнайте больше, как экспортировать электронные таблицы Excel и Apple Numbers в формат PDF

* [Рендеринг электронных таблиц в формате PDF](https://docs.groupdocs.com/viewer/net/render-excel-and-apple-numbers-spreadsheets/#render-spreadsheets-as-pdf)

### Примеры

Пример демонстрирует типичное использование этой опции.

```csharp
using (var viewer = new Viewer("employees.xlsx"))
{
    PdfViewOptions viewOptions = new PdfViewOptions();
    viewOptions.Optimize = true;

    viewer.View(viewOptions);
}
```
