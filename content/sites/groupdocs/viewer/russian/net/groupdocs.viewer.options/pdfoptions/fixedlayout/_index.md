## PdfOptions.FixedLayout property

PDF — это фиксированный формат, поэтому все элементы занимают определенное место на странице. Чтобы гарантировать, что выходной HTML выглядит так же, как исходный PDF, документы по умолчанию преобразуются в HTML с фиксированным макетом. При отображении с помощью фиксированный макет выходной HTML-код имеет фиксированный размер, поэтому элементы останутся на одном месте независимо от размера окна. Для рендеринга в HTML с плавным макетом установите для этого свойства значение`ЛОЖЬ` .

```csharp
public bool FixedLayout { get; set; }
```

### Примечания

Значение по умолчанию:`истинный` .

Этот параметр поддерживается при рендеринге в HTML.

При рендеринге в гибкий макет изображения пропускаются. Использовать гибкий макет при рендеринге PDF-документов с текстовым содержимым.

### Примеры

В этом примере показано, как преобразовать PDF-документ в HTML с помощью гибкого макета.

```csharp
using GroupDocs.Viewer;
using GroupDocs.Viewer.Options;
//..

using (var viewer = new Viewer("resume.pdf"))
{
   var viewOptions = HtmlViewOptions.ForEmbeddedResources("page_{0}.html");
   viewOptions.FixedLayout = false;
   viewer.View(viewOptions);
}        
```
