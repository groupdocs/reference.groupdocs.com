## PdfOptions.RenderOriginalPageSize property

Когда эта опция включена, выходные страницы будут иметь тот же размер в пикселях, что и размер страницы в исходном документе PDF. По умолчанию GroupDocs.Viewer вычисляет размер страницы выходного изображения для лучшего качества рендеринга.

```csharp
public bool RenderOriginalPageSize { get; set; }
```

### Примечания

Этот параметр поддерживается при рендеринге в форматы PNG или JPG.
