## PdfOptions.EnableFontHinting property

Включает подсказку шрифта. Подсказка шрифта настраивает отображение контурного шрифта. Поддерживается только для шрифтов TTF, когда эти шрифты используются в исходном документе.

```csharp
public bool EnableFontHinting { get; set; }
```

### Примечания

Этот параметр поддерживается при рендеринге в форматы PNG или JPG.
