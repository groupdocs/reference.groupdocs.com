## MailStorageOptions.MaxItems property

Максимальное количество сообщений или элементов для рендеринга.

```csharp
public int MaxItems { get; set; }
```

### Примечания

Файлы данных Lotus Notes могут быть большими, и извлечение всех сообщений может занять значительное время. Этот параметр ограничивает максимальное количество отображаемых сообщений или элементов. Значение по умолчанию: 0 — все сообщения будут отображаться
