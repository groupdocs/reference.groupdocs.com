## OutlookOptions.MaxItemsInFolder property

Максимальное количество сообщений или элементов, которые можно отобразить из одной папки.

```csharp
public int MaxItemsInFolder { get; set; }
```

### Примечания

Файлы данных Outlook могут быть большими, и извлечение всех сообщений может занять значительное время. Этот параметр ограничивает максимальное количество отображаемых сообщений или элементов (например, контактов и задач). значение 0.
