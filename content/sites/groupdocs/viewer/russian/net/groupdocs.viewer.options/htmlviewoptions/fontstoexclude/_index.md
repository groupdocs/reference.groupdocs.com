## HtmlViewOptions.FontsToExclude property

Список имен шрифтов, которые необходимо исключить из документа HTML.

```csharp
public List<string> FontsToExclude { get; set; }
```

### Примечания

Эта опция поддерживается только для презентаций. Шрифты, добавляемые в HTML-документ, улучшают стабильность выходного представления, в то же время увеличивают размер результата рендеринга. Эта опция позволяет найти компромисс между стабильностью и размером вывода. Включите имена шрифтов , которые популярны и установлены в большинстве систем. Обратите внимание, что это свойство активно только при[`ExcludeFonts`](../excludefonts) параметры отключены.
