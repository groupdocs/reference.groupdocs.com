## IDocumentInfo.RawPageCount property

Ottiene il numero totale di pagine non elaborate del documento.

```csharp
public int RawPageCount { get; }
```

### Valore della proprietà

Un valore intero che rappresenta un numero totale di pagine non elaborate.

### Osservazioni

Usa`RawPageCount` proprietà invece di[`PageCount`](../pagecount) proprietà per l'estrazione del testo non elaborato. Alcuni documenti hanno numeri di pagina diversi nelle modalità di estrazione del testo accurato e non elaborato. [`PageCount`](../pagecount)la proprietà può eseguire calcoli extra che influiscono sulla velocità di estrazione del testo in modalità raw.
