## FileType.POTM field

I file con estensione POTM sono file modello di Microsoft PowerPoint con supporto per le macro. I file POTM vengono creati con PowerPoint 2007 o versioni successive e contengono impostazioni predefinite che possono essere utilizzate per creare ulteriori file di presentazione.
