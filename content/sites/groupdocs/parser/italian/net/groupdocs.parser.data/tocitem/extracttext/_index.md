## TocItem.ExtractText method

Estrae un testo dal documento in cui[`TocItem`](../../tocitem) l'oggetto fa riferimento.

```csharp
public virtual TextReader ExtractText()
```

### Valore di ritorno

Un'istanza diTextReader classe con il testo estratto.

### Esempi

Il seguente esempio su come estrarre un testo da un elemento del sommario:

```csharp
// Crea un'istanza della classe Parser
using (Parser parser = new Parser(Constants.SampleDocxWithToc))
{
    // Ottieni il sommario
    IEnumerable<TocItem> tocItems = parser.GetToc();
    // Controlla se l'estrazione di toc è supportata
    if (tocItems == null)
    {
        Console.WriteLine("Table of contents extraction isn't supported");
    }
    // Itera sugli elementi
    foreach (TocItem tocItem in tocItems)
    {
        // Stampa il testo del capitolo
        using (TextReader reader = tocItem.ExtractText())
        {
            Console.WriteLine("----");
            Console.WriteLine(reader.ReadToEnd());
        }
    }
}
```
