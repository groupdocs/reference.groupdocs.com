## ContainerItem.Metadata property

Ottiene la raccolta di elementi di metadati.

```csharp
public IEnumerable<MetadataItem> Metadata { get; }
```

### Valore della proprietà

Una collezione di[`MetadataItem`](../../metadataitem) oggetti; vuoto se i metadati non sono impostati.

### Osservazioni

Questi metadati si riferiscono a un elemento contenitore stesso, non a un documento. A seconda del tipo di contenitore i metadati possono contenere i seguenti elementi:

**Allegati e-mail**

**Nome**

**Descrizione**

**tipo di contenuto**

Il tipo MIME del contenuto dell'allegato.

**Archivi ZIP**

**Nome**

**Descrizione**

**data**

L'ora e la data dell'ultima modifica del file indicato dalla Voce Zip.

**Archiviazione di Outlook**

**Nome**

**Descrizione**

**data**

L'ora e la data in cui l'elemento di archiviazione di Outlook è stato modificato l'ultima volta.

**e-mail mittente**

Il valore del campo "mittente".

**invia una email a**

Il valore del campo "a".

**soggetto**

Il valore del campo "oggetto".
