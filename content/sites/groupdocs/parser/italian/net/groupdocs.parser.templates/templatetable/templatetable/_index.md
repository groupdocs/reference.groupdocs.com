| name | String | Il nome della tabella. |
| pageIndex | Nullable`1 | L'indice della pagina in cui si trova la tabella del modello; `nullo` se la tabella del modello si trova su qualsiasi pagina. |

### Esempi

Se una tabella modello è impostata dai parametri del rilevatore, la tabella viene rilevata automaticamente:

```csharp
TemplateTableParameters parameters = new TemplateTableParameters(
    new Rectangle(new Point(175, 350), new Size(400, 200)),
    new double[] { 185, 370, 425, 485, 545 });

TemplateTable table = new TemplateTable(parameters, "Details", 0);

// Crea un modello di documento
Template template = new Template(new TemplateItem[] { table });
```
