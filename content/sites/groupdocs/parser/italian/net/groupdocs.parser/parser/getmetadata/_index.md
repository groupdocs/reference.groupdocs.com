### Esempi

L'esempio seguente mostra come estrarre i metadati da un documento:

```csharp
// Crea un'istanza della classe Parser
using(Parser parser = new Parser(filePath))
{
    // Estrai i metadati dal documento
    IEnumerable<MetadataItem> metadata = parser.GetMetadata();
    // Controlla se l'estrazione dei metadati è supportata
    if(metadata == null)
    {
        Console.WriteLine("Metatada extraction isn't supported");
    }
 
    // Itera sugli elementi dei metadati
    foreach(MetadataItem item in metadata)
    {
        // Stampa il nome e il valore di un elemento
        Console.WriteLine(string.Format("{0}: {1}", item.Name, item.Value));
    }
}
```
