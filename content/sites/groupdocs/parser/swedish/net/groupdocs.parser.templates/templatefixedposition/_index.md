### Exempel

Det här är enklaste sättet att definiera fältpositionen. Det kräver att man anger ett rektangulärt område på sidan som begränsar fältvärdet. All text som finns (även delvis) i det rektangulära området kommer att extraheras som ett värde:

```csharp
// Skapa ett fast mallfält med "Adress" namn som avgränsas av en rektangel vid positionen (35, 160) och med storleken (110, 20)
TemplateField templateField = new TemplateField(
    new TemplateFixedPosition(new Rectangle(new Point(35, 160), new Size(110, 20))),
    "Address");
```
