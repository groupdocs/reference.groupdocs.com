### Exempel

Följande exempel visar koden för situationen om det är känt att fältet med ett fakturanummer är placerat till höger om strängen "Fakturanummer" används följande kod:

```csharp
// Skapa ett regex-mallfält för att hitta texten "Fakturanummer".
TemplateField invoice = new TemplateField(new TemplateRegexPosition("Invoice Number"), "Invoice");

// Skapa ett relaterat mallfält kopplat till "Faktura"-fältet och extrahera värdet till höger om det
TemplateField invoiceNumber = new TemplateField(
    new TemplateLinkedPosition("invoice", new Size(100, 15), new TemplateLinkedPositionEdges(false, false, true, false)),
    "InvoiceNumber");
```
