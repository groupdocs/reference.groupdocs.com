## Parser.GetHighlight method

Extraherar en markering från dokumentet.

```csharp
public HighlightItem GetHighlight(int position, bool isDirect, HighlightOptions options)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| position | Int32 | Höjdpunktens startposition. |
| isDirect | Boolean | Värdet som indikerar om extrahering av högdagrar är direkt. `Sann` om höjdpunkten är utdragen till höger om*position* ; annat,`falsk` . |
| options | HighlightOptions | Extraheringsalternativen för markering. |

### Returvärde

En instans av[`HighlightItem`](../../../groupdocs.parser.data/highlightitem) klass som representerar den extraherade höjdpunkten; `null` om extrahering av högdagrar inte stöds.

### Exempel

Följande exempel visar hur man extraherar en markering som innehåller 3 ord:

```csharp
// Skapa en instans av Parser-klassen
using (Parser parser = new Parser(filePath))
{
    // Extrahera en höjdpunkt:
    HighlightItem hl = parser.GetHighlight(2, true, new HighlightOptions(3));
    
    // Kontrollera om utdragning av markeringar stöds
    if (hl == null)
    {
        Console.WriteLine("Highlight extraction isn't supported");
        return;
    }
    
    // Skriv ut en extraherad höjdpunkt
    Console.WriteLine(string.Format("At {0}: {1}", hl.Position, hl.Text));
}
```
