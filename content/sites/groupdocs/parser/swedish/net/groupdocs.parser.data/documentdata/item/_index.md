Iteration via alla fält:

[`FieldData`](../../fielddata) klass representerar fältdata. Beroende på fält[`PageArea`](../../fielddata/pagearea) property kan innehålla vilken som helst av arvtagarna till[`PageArea`](../../pagearea) klass. Till exempel,[`ParseForm`](../../../groupdocs.parser/parser/parseform) method extraherar endast textfält:

```csharp
for (int i = 0; i < data.Count; i++)
{
    Console.Write(data[i].Name + ": ");
    PageTextArea area = data[i].PageArea as PageTextArea;
    Console.WriteLine(area == null ? "Not a template field" : area.Text);
}
```

```csharp
// Skapa tolken
using (Parser parser = new Parser(filePath))
{
    // Extrahera data från PDF-formulär
    DocumentData data = parser.ParseForm();
    // Iterera över extraherade fält
    for (int i = 0; i < data.Count; i++)
    {
        // Hämta det extraherade fältet
        FieldData field = data[i];
        // Skriv ut fältnamnet
        Console.Write(field.Name + ": ");
        // Kontrollera om fältvärdet är en text och skriv ut det
        PageTextArea area = field.PageArea as PageTextArea;
        Console.WriteLine(area == null ? "Not a template field" : area.Text);
    }
}
```
