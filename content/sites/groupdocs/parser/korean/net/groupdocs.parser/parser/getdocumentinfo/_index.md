### 예

```csharp
// Parser 클래스의 인스턴스 생성
using(Parser parser = new Parser(filePath))

    // 문서 정보 가져오기
    IDocumentInfo info = parser.GetDocumentInfo();

    Console.WriteLine(string.Format("FileType: {0}", info.FileType));

    Console.WriteLine(string.Format("Size: {0}", info.Size));
}
```
