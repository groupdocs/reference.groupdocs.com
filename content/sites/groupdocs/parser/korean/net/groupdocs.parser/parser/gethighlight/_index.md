## Parser.GetHighlight method

문서에서 하이라이트를 추출합니다.

```csharp
public HighlightItem GetHighlight(int position, bool isDirect, HighlightOptions options)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| position | Int32 | 하이라이트의 시작 위치입니다. |
| isDirect | Boolean | 하이라이트 추출이 직접적인지 여부를 나타내는 값. `진실` 하이라이트가 오른쪽으로 추출되면*position* ; 그렇지 않으면,`거짓` . |
| options | HighlightOptions | 하이라이트 추출 옵션. |

### 반환 값

의 인스턴스[`HighlightItem`](../../../groupdocs.parser.data/highlightitem) 추출된 하이라이트를 나타내는 클래스; `없는` 하이라이트 추출이 지원되지 않는 경우.

### 예

다음 예에서는 3단어가 포함된 하이라이트를 추출하는 방법을 보여줍니다.

```csharp
// Parser 클래스의 인스턴스 생성
using (Parser parser = new Parser(filePath))
{
    // 하이라이트 추출:
    HighlightItem hl = parser.GetHighlight(2, true, new HighlightOptions(3));
    
    // 하이라이트 추출 지원 여부 확인
    if (hl == null)
    {
        Console.WriteLine("Highlight extraction isn't supported");
        return;
    }
    
    // 추출된 하이라이트 출력
    Console.WriteLine(string.Format("At {0}: {1}", hl.Position, hl.Text));
}
```
