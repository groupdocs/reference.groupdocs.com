## Parser.GetHighlight method

Извлекает выделение из документа.

```csharp
public HighlightItem GetHighlight(int position, bool isDirect, HighlightOptions options)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| position | Int32 | Начальная позиция выделения. |
| isDirect | Boolean | Значение, указывающее, является ли извлечение выделения прямым. `истинный` если выделение извлекается по праву*position* ; в противном случае,`ЛОЖЬ` . |
| options | HighlightOptions | Параметры выделения выделения. |

### Возвращаемое значение

Экземпляр[`HighlightItem`](../../../groupdocs.parser.data/highlightitem) класс, представляющий извлеченную подсветку; `нулевой` если извлечение выделения не поддерживается.

### Примеры

В следующем примере показано, как извлечь выделение, содержащее 3 слова:

```csharp
// Создаем экземпляр класса Parser
using (Parser parser = new Parser(filePath))
{
    // Извлекаем блик:
    HighlightItem hl = parser.GetHighlight(2, true, new HighlightOptions(3));
    
    // Проверяем, поддерживается ли извлечение выделения
    if (hl == null)
    {
        Console.WriteLine("Highlight extraction isn't supported");
        return;
    }
    
    // Печатаем извлеченную подсветку
    Console.WriteLine(string.Format("At {0}: {1}", hl.Position, hl.Text));
}
```
