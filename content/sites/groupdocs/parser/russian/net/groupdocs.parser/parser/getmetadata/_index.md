## Parser.GetMetadata method

### Примеры

В следующем примере показано, как извлечь метаданные из документа:

```csharp
// Создаем экземпляр класса Parser
using(Parser parser = new Parser(filePath))
{
    // Извлечь метаданные из документа
    IEnumerable<MetadataItem> metadata = parser.GetMetadata();
    // Проверяем, поддерживается ли извлечение метаданных
    if(metadata == null)
    {
        Console.WriteLine("Metatada extraction isn't supported");
    }
 
    // Перебираем элементы метаданных
    foreach(MetadataItem item in metadata)
    {
        // Выводим имя и значение элемента
        Console.WriteLine(string.Format("{0}: {1}", item.Name, item.Value));
    }
}
```
