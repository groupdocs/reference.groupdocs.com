## IDocumentInfo.RawPageCount property

Получает общее количество необработанных страниц документа.

```csharp
public int RawPageCount { get; }
```

### Стоимость имущества

Целочисленное значение, представляющее общее количество необработанных страниц.

### Примечания

Использовать`RawPageCount` собственность вместо[`PageCount`](../pagecount) для извлечения необработанного текста. Некоторые документы имеют разные номера страниц в режимах извлечения точного и необработанного текста. [`PageCount`](../pagecount)свойство может выполнять дополнительные вычисления, которые влияют на скорость извлечения текста в необработанном режиме.
