### Примеры

Это самый простой способ определить позицию поля. Требуется установить прямоугольную область на странице, которая ограничивает значение поля. Весь текст, который содержится (даже частично) в прямоугольной области, будет извлечен как значение:

```csharp
// Создаем фиксированное поле шаблона с именем "Адрес", которое ограничено прямоугольником в позиции (35, 160) и размером (110, 20)
TemplateField templateField = new TemplateField(
    new TemplateFixedPosition(new Rectangle(new Point(35, 160), new Size(110, 20))),
    "Address");
```
