| Параметр | Тип | Описание |
| --- | --- | --- |
| parameters | TemplateTableParameters | Параметры для определения таблицы в автоматическом режиме. |
| name | String | Имя таблицы. |
| pageIndex | Nullable`1 | Индекс страницы, на которой находится таблица шаблонов; `нулевой` если таблица шаблона находится на любой странице. |

### Примеры

Если таблица-шаблон задана параметрами детектора, то таблица определяется автоматически:

```csharp
TemplateTableParameters parameters = new TemplateTableParameters(
    new Rectangle(new Point(175, 350), new Size(400, 200)),
    new double[] { 185, 370, 425, 485, 545 });

TemplateTable table = new TemplateTable(parameters, "Details", 0);

// Создаем шаблон документа
Template template = new Template(new TemplateItem[] { table });
```
