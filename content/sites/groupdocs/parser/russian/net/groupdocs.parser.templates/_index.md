Пространство имен предоставляет классы для создания шаблонов для анализа данных документа.

| [TemplateBarcode](./templatebarcode) | Предоставляет поле штрих-кода шаблона. |
| [TemplateField](./templatefield) | Предоставляет текстовое поле шаблона. |

| [TemplateTableParameters](./templatetableparameters) | Предоставляет параметры для алгоритмов обнаружения таблиц. |
