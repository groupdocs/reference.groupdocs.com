## DocumentData indexer

इंडेक्स द्वारा फ़ील्ड डेटा प्राप्त करता है।

public FieldData this[int index] { get; }

| --- | --- |

### प्रतिलाभ की मात्रा

का एक उदाहरण[`FieldData`](../../fielddata) कक्षा।

### उदाहरण

सभी क्षेत्रों के माध्यम से पुनरावृति:

[`FieldData`](../../fielddata) वर्ग फ़ील्ड डेटा का प्रतिनिधित्व करता है। मैदान पर निर्भर करता है[`PageArea`](../../fielddata/pagearea) property में इनमें से कोई भी विरासत हो सकती है[`PageArea`](../../pagearea) कक्षा। उदाहरण के लिए,[`ParseForm`](../../../groupdocs.parser/parser/parseform) method केवल टेक्स्ट फ़ील्ड निकालता है:

```csharp
for (int i = 0; i < data.Count; i++)
{
    Console.Write(data[i].Name + ": ");
    PageTextArea area = data[i].PageArea as PageTextArea;
    Console.WriteLine(area == null ? "Not a template field" : area.Text);
}
```

```csharp
// पार्सर बनाएं
using (Parser parser = new Parser(filePath))
{
    // पीडीएफ फॉर्म से डेटा निकालें
    DocumentData data = parser.ParseForm();
    // निकाले गए फ़ील्ड पर पुनरावृति करें
    for (int i = 0; i < data.Count; i++)
    {
        // निकाले गए क्षेत्र को प्राप्त करें
        FieldData field = data[i];
        // फ़ील्ड का नाम प्रिंट करें
        Console.Write(field.Name + ": ");
        // जांचें कि क्या फ़ील्ड मान एक टेक्स्ट है और इसे प्रिंट करें
        PageTextArea area = field.PageArea as PageTextArea;
        Console.WriteLine(area == null ? "Not a template field" : area.Text);
    }
}
```
