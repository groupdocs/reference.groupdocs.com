## TocItem.ExtractText method

उस दस्तावेज़ से एक पाठ निकालता है जिसमें[`TocItem`](../../tocitem) वस्तु संदर्भित करता है.

```csharp
public virtual TextReader ExtractText()
```

### प्रतिलाभ की मात्रा

का एक उदाहरणTextReader निकाले गए पाठ के साथ वर्ग।

### उदाहरण

निम्नलिखित उदाहरण सामग्री तालिका के किसी आइटम द्वारा पाठ को कैसे निकाला जाए:

```csharp
// पार्सर वर्ग का एक उदाहरण बनाएँ
using (Parser parser = new Parser(Constants.SampleDocxWithToc))
{
    // सामग्री की तालिका प्राप्त करें
    IEnumerable<TocItem> tocItems = parser.GetToc();
    // जांचें कि क्या toc निष्कर्षण समर्थित है
    if (tocItems == null)
    {
        Console.WriteLine("Table of contents extraction isn't supported");
    }
    // वस्तुओं पर पुनरावृति
    foreach (TocItem tocItem in tocItems)
    {
        // अध्याय का पाठ प्रिंट करें
        using (TextReader reader = tocItem.ExtractText())
        {
            Console.WriteLine("----");
            Console.WriteLine(reader.ReadToEnd());
        }
    }
}
```
