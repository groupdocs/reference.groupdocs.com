## Parser.Features property

### संपत्ति मूल्य

का एक उदाहरण[`Features`](../../../groupdocs.parser.options/features) वर्ग जो समर्थित सुविधाओं का प्रतिनिधित्व करता है।

### टिप्पणियों

* [समर्थित सुविधाएँ प्राप्त करें](https://docs.groupdocs.com/display/parsernet/Get+supported+features)

### उदाहरण

यदि सुविधा समर्थित नहीं है, तो विधि वापस आ जाती है`व्यर्थ` मूल्य के बजाय। कुछ कार्यों में काफी समय लग सकता है। तो सुविधा के लिए समर्थन की जांच करने के लिए विधि को कॉल करना इष्टतम नहीं है। इस उद्देश्य के लिए सुविधाओं की संपत्ति का उपयोग किया जाता है:

```csharp
using(Parser parser = new Parser("doc.zip"))
{
    if(!parser.Features.Text)
    {
        Console.WriteLine("Text extraction isn't supported");
        return;
    }
 
    using(TextReader reader = parser.GetText())
    {
        Console.WriteLine(reader.ReadToEnd());
    }
}
```
