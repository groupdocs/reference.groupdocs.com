### उदाहरण

निम्न उदाहरण दिखाता है कि किसी दस्तावेज़ से मेटाडेटा कैसे निकाला जाए:

```csharp
// पार्सर वर्ग का एक उदाहरण बनाएँ
using(Parser parser = new Parser(filePath))
{
    // दस्तावेज़ से मेटाडेटा निकालें
    IEnumerable<MetadataItem> metadata = parser.GetMetadata();
    // जांचें कि क्या मेटाडेटा निष्कर्षण समर्थित है
    if(metadata == null)
    {
        Console.WriteLine("Metatada extraction isn't supported");
    }
 
    // मेटाडेटा आइटम पर पुनरावृति करें
    foreach(MetadataItem item in metadata)
    {
        // एक आइटम का नाम और मूल्य प्रिंट करें
        Console.WriteLine(string.Format("{0}: {1}", item.Name, item.Value));
    }
}
```
