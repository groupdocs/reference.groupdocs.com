### Örnekler

Özellik desteklenmiyorsa, yöntem döndürür`hükümsüz` değer yerine. Bazı işlemler önemli ölçüde zaman alabilir. Bu nedenle, yalnızca özellik desteğini kontrol etmek için yöntemi çağırmak en uygun değildir. Bu amaçla, Özellikler özelliği kullanılır:

```csharp
using(Parser parser = new Parser("doc.zip"))
{
    if(!parser.Features.Text)
    {
        Console.WriteLine("Text extraction isn't supported");
        return;
    }
 
    using(TextReader reader = parser.GetText())
    {
        Console.WriteLine(reader.ReadToEnd());
    }
}
```
