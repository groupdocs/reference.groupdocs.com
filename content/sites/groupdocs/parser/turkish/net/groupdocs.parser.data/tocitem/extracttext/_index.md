### Örnekler

Aşağıdaki örnek, bir içindekiler tablosu öğesi tarafından bir metnin nasıl çıkarılacağı:

```csharp
// Parser sınıfının bir örneğini oluşturun
using (Parser parser = new Parser(Constants.SampleDocxWithToc))
{
    // içindekiler tablosunu al
    IEnumerable<TocItem> tocItems = parser.GetToc();
    // Toc çıkarmanın desteklenip desteklenmediğini kontrol edin
    if (tocItems == null)
    {
        Console.WriteLine("Table of contents extraction isn't supported");
    }
    // Öğeler üzerinde yineleme
    foreach (TocItem tocItem in tocItems)
    {
        // Bölümün metnini yazdır
        using (TextReader reader = tocItem.ExtractText())
        {
            Console.WriteLine("----");
            Console.WriteLine(reader.ReadToEnd());
        }
    }
}
```
