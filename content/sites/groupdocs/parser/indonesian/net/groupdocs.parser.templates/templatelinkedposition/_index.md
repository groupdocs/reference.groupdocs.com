### Contoh

Contoh berikut menunjukkan kode untuk situasi jika diketahui bahwa bidang dengan nomor faktur ditempatkan di sebelah kanan dari string "Nomor faktur" kode berikut digunakan:

```csharp
// Buat bidang template regex untuk menemukan teks "Nomor Faktur".
TemplateField invoice = new TemplateField(new TemplateRegexPosition("Invoice Number"), "Invoice");

// Buat bidang template terkait yang dikaitkan dengan bidang "Faktur" dan ekstrak nilai di sebelah kanannya
TemplateField invoiceNumber = new TemplateField(
    new TemplateLinkedPosition("invoice", new Size(100, 15), new TemplateLinkedPositionEdges(false, false, true, false)),
    "InvoiceNumber");
```
