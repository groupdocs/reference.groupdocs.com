### Nilai Pengembalian

### Contoh

Iterasi melalui semua bidang:

[`FieldData`](../../fielddata) kelas mewakili data lapangan. Tergantung lapangan[`PageArea`](../../fielddata/pagearea) property dapat berisi salah satu dari pewaris[`PageArea`](../../pagearea) kelas. Misalnya,[`ParseForm`](../../../groupdocs.parser/parser/parseform) method hanya mengekstrak bidang teks:

```csharp
for (int i = 0; i < data.Count; i++)
{
    Console.Write(data[i].Name + ": ");
    PageTextArea area = data[i].PageArea as PageTextArea;
    Console.WriteLine(area == null ? "Not a template field" : area.Text);
}
```

```csharp
// Buat parser
using (Parser parser = new Parser(filePath))
{
    // Ekstrak data dari Formulir PDF
    DocumentData data = parser.ParseForm();
    // Ulangi bidang yang diekstraksi
    for (int i = 0; i < data.Count; i++)
    {
        // Dapatkan bidang yang diekstraksi
        FieldData field = data[i];
        // Cetak nama bidang
        Console.Write(field.Name + ": ");
        // Periksa apakah nilai bidang adalah teks dan cetak
        PageTextArea area = field.PageArea as PageTextArea;
        Console.WriteLine(area == null ? "Not a template field" : area.Text);
    }
}
```
