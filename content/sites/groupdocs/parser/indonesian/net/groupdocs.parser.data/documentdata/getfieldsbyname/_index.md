## DocumentData.GetFieldsByName method

Mengembalikan kumpulan data bidang yang namanya sama dengan*fieldName* .


| Parameter | Jenis | Keterangan |

### Nilai Pengembalian

Sebuah koleksi dari[`FieldData`](../../fielddata) benda; koleksi kosong jika tidak ada data lapangan yang ditemukan.

### Contoh

Temukan bidang dengan nama bidang:

[`FieldData`](../../fielddata) kelas mewakili data lapangan. Tergantung lapangan[`PageArea`](../../fielddata/pagearea) property dapat berisi salah satu dari pewaris[`PageArea`](../../pagearea) kelas. Misalnya,[`ParseForm`](../../../groupdocs.parser/parser/parseform) method hanya mengekstrak bidang teks.

```csharp
// Dapatkan semua bidang dengan nama "Alamat".
IList<FieldData> addressFields = data.GetFieldsByName("Address");
if(addressFields.Count == 0) {
    Console.WriteLine("Address not found");
}
else {
    Console.WriteLine("Address");
    // Ulangi koleksi bidang
    for (int i = 0; i < addressFields.Count; i++) {
        PageTextArea area = addressFields[i].PageArea as PageTextArea;
        Console.WriteLine(area == null ? "Not a template field" : area.Text);       
         
        // Jika itu bidang terkait:
        if(addressFields[i].LinkedField != null) {
            Console.Write("Linked to ");
            PageTextArea linkedArea = addressFields[i].LinkedField.PageArea as PageTextArea;
            Console.WriteLine(area == null ? "Not a template field" : area.Text);           
        }
    }
}
```
