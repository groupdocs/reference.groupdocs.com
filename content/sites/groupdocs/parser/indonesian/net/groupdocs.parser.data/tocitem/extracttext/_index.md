### Nilai Pengembalian

Contoh dariTextReader kelas dengan teks yang diekstraksi.

### Contoh

Berikut contoh cara mengekstrak teks dengan item daftar isi:

```csharp
// Buat instance dari kelas Parser
using (Parser parser = new Parser(Constants.SampleDocxWithToc))
{
    // Dapatkan daftar isi
    IEnumerable<TocItem> tocItems = parser.GetToc();
    // Periksa apakah ekstraksi toc didukung
    if (tocItems == null)
    {
        Console.WriteLine("Table of contents extraction isn't supported");
    }
    // Ulangi item
    foreach (TocItem tocItem in tocItems)
    {
        // Cetak teks bab ini
        using (TextReader reader = tocItem.ExtractText())
        {
            Console.WriteLine("----");
            Console.WriteLine(reader.ReadToEnd());
        }
    }
}
```
