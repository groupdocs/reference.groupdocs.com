## Parser.GetHighlight method

Mengekstrak sorotan dari dokumen.

```csharp
public HighlightItem GetHighlight(int position, bool isDirect, HighlightOptions options)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| position | Int32 | Posisi awal sorotan. |
| isDirect | Boolean | Nilai yang menunjukkan apakah ekstraksi sorot langsung. `BENAR` jika higlight diekstraksi dengan hak*position* ; jika tidak,`PALSU` . |
| options | HighlightOptions | Opsi ekstraksi sorot. |

### Nilai Pengembalian

Contoh dari[`HighlightItem`](../../../groupdocs.parser.data/highlightitem) kelas yang mewakili sorotan yang diekstrak; `batal` jika sorot ekstraksi tidak didukung.

### Contoh

Contoh berikut menunjukkan cara mengekstrak sorotan yang berisi 3 kata:

```csharp
// Buat instance dari kelas Parser
using (Parser parser = new Parser(filePath))
{
    // Ekstrak sorotan:
    HighlightItem hl = parser.GetHighlight(2, true, new HighlightOptions(3));
    
    // Periksa apakah ekstraksi sorot didukung
    if (hl == null)
    {
        Console.WriteLine("Highlight extraction isn't supported");
        return;
    }
    
    // Cetak sorotan yang diekstrak
    Console.WriteLine(string.Format("At {0}: {1}", hl.Position, hl.Text));
}
```
