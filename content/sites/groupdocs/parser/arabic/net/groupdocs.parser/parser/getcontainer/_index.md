## Parser.GetContainer method

استخراج كائن حاوية من المستند للعمل مع التنسيقات التي تحتوي على مرفقات وأرشيفات ZIP وما إلى ذلك.

```csharp
public IEnumerable<ContainerItem> GetContainer()
```

### قيمة الإرجاع

مجموعة من عناصر الحاوية `لا شيء`إذا كان استخراج الحاوية غير مدعوم.

### أمثلة

يوضح المثال التالي كيفية استخراج المرفقات من حاوية:

```csharp
// إنشاء مثيل لفئة المحلل اللغوي
using(Parser parser = new Parser(filePath))
{
    // استخراج المرفقات من الحاوية
    IEnumerable<ContainerItem> attachments = parser.GetContainer();
    // تحقق مما إذا كان استخراج الحاوية مدعومًا
    if(attachments == null)
    {
        Console.WriteLine("Container extraction isn't supported");
    }
 
    // كرر عبر المرفقات
    foreach(ContainerItem item in attachments)
    {
        // طباعة اسم العنصر وحجمه
        Console.WriteLine(string.Format("{0}: {1}", item.Name, item.Size));
    }
}
```
