## Parser.GetHighlight method

لاستخراج تمييز من المستند.

```csharp
public HighlightItem GetHighlight(int position, bool isDirect, HighlightOptions options)
```

| معامل | يكتب | وصف |
| --- | --- | --- |
| position | Int32 | موضع بداية التظليل. |
| isDirect | Boolean | القيمة التي تشير إلى ما إذا كان استخراج التمييز مباشرًا أم لا.`حقيقي` إذا تم استخراج الضوء بواسطة حق*position* ؛ خلاف ذلك،`خاطئة` . |
| options | HighlightOptions | تسليط الضوء على خيارات الاستخراج. |

### قيمة الإرجاع

مثيل[`HighlightItem`](../../../groupdocs.parser.data/highlightitem) الفئة التي تمثل التمييز المستخرج ؛ `لا شيء` إذا لم يتم دعم استخراج التمييز.

* [يبرز استخراج](https://docs.groupdocs.com/display/parsernet/Extract+highlights)

### أمثلة

يوضح المثال التالي كيفية استخراج تمييز يحتوي على 3 كلمات:

```csharp
// إنشاء مثيل لفئة المحلل اللغوي
using (Parser parser = new Parser(filePath))
{
    // استخراج تسليط الضوء:
    HighlightItem hl = parser.GetHighlight(2, true, new HighlightOptions(3));
    
    // تحقق مما إذا كان استخراج التمييز مدعومًا
    if (hl == null)
    {
        Console.WriteLine("Highlight extraction isn't supported");
        return;
    }
    
    // طباعة تمييز مستخرج
    Console.WriteLine(string.Format("At {0}: {1}", hl.Position, hl.Text));
}
```
