### Exemples

L'exemple suivant comment extraire un texte par un élément de table des matières :

```csharp
// Crée une instance de la classe Parser
using (Parser parser = new Parser(Constants.SampleDocxWithToc))
{
    // Récupère la table des matières
    IEnumerable<TocItem> tocItems = parser.GetToc();
    // Vérifie si l'extraction de la table des matières est prise en charge
    if (tocItems == null)
    {
        Console.WriteLine("Table of contents extraction isn't supported");
    }
    // Itérer sur les éléments
    foreach (TocItem tocItem in tocItems)
    {
        // Imprime le texte du chapitre
        using (TextReader reader = tocItem.ExtractText())
        {
            Console.WriteLine("----");
            Console.WriteLine(reader.ReadToEnd());
        }
    }
}
```
