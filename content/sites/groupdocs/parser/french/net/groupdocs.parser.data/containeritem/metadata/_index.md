## ContainerItem.Metadata property

Obtient la collection d'éléments de métadonnées.

```csharp
public IEnumerable<MetadataItem> Metadata { get; }
```

### Valeur de la propriété

Une collection de[`MetadataItem`](../../metadataitem) objets; vide si les métadonnées ne sont pas définies.

### Remarques

Ces métadonnées font référence à un élément de conteneur lui-même, pas à un document. Selon le type de conteneur , les métadonnées peuvent contenir les éléments suivants :

**Pièces jointes aux e-mails**

**Nom**

**La description**

**type de contenu**

Le type MIME du contenu de la pièce jointe.

**Archives ZIP**

**Nom**

**La description**

**Date**

L'heure et la date auxquelles le fichier indiqué par l'entrée Zip a été modifié pour la dernière fois.

**Stockage Outlook**

**Nom**

**La description**

**Date**

L'heure et la date auxquelles l'élément de stockage Outlook a été modifié pour la dernière fois.

**expéditeur de l'e-mail**

La valeur du champ "expéditeur".

**envoyer à**

La valeur du champ "à".

**matière**

La valeur du champ "sujet".
