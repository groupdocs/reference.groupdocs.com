## Parser.GetContainer method

Extrait un objet conteneur du document pour travailler avec des formats contenant des pièces jointes, des archives ZIP, etc.

```csharp
public IEnumerable<ContainerItem> GetContainer()
```

### Return_Value

Une collection d'éléments de conteneur ; `nul`si l'extraction de conteneur n'est pas prise en charge.

### Exemples

L'exemple suivant montre comment extraire des pièces jointes d'un conteneur :

```csharp
// Crée une instance de la classe Parser
using(Parser parser = new Parser(filePath))
{
    // Extraire les pièces jointes du conteneur
    IEnumerable<ContainerItem> attachments = parser.GetContainer();
    // Vérifie si l'extraction du conteneur est prise en charge
    if(attachments == null)
    {
        Console.WriteLine("Container extraction isn't supported");
    }
 
    // Itérer sur les pièces jointes
    foreach(ContainerItem item in attachments)
    {
        // Affiche le nom et la taille d'un élément
        Console.WriteLine(string.Format("{0}: {1}", item.Name, item.Size));
    }
}
```
