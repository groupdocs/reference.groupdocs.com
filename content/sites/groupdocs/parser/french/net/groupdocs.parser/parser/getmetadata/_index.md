### Exemples

L'exemple suivant montre comment extraire les métadonnées d'un document :

```csharp
// Crée une instance de la classe Parser
using(Parser parser = new Parser(filePath))
{
    // Extraire les métadonnées du document
    IEnumerable<MetadataItem> metadata = parser.GetMetadata();
    // Vérifie si l'extraction des métadonnées est prise en charge
    if(metadata == null)
    {
        Console.WriteLine("Metatada extraction isn't supported");
    }
 
    // Itérer sur les éléments de métadonnées
    foreach(MetadataItem item in metadata)
    {
        // Affiche le nom et la valeur d'un élément
        Console.WriteLine(string.Format("{0}: {1}", item.Name, item.Value));
    }
}
```
