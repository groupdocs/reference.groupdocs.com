## Parser.GetHighlight method

Extrait une surbrillance du document.

```csharp
public HighlightItem GetHighlight(int position, bool isDirect, HighlightOptions options)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| position | Int32 | La position de départ de la surbrillance. |
| isDirect | Boolean | La valeur qui indique si l'extraction de la surbrillance est directe. `vrai` si le surlignage est extrait par le droit de*position* ; sinon,`FAUX` . |
| options | HighlightOptions | Les options d'extraction de surbrillance. |

### Return_Value

Une instance de[`HighlightItem`](../../../groupdocs.parser.data/highlightitem) classe qui représente la surbrillance extraite ; `nul` si l'extraction de surbrillance n'est pas prise en charge.

### Exemples

L'exemple suivant montre comment extraire un surlignage contenant 3 mots :

```csharp
// Crée une instance de la classe Parser
using (Parser parser = new Parser(filePath))
{
    // Extraire un surlignage :
    HighlightItem hl = parser.GetHighlight(2, true, new HighlightOptions(3));
    
    // Vérifie si l'extraction de surbrillance est prise en charge
    if (hl == null)
    {
        Console.WriteLine("Highlight extraction isn't supported");
        return;
    }
    
    // Imprimer un surlignage extrait
    Console.WriteLine(string.Format("At {0}: {1}", hl.Position, hl.Text));
}
```
