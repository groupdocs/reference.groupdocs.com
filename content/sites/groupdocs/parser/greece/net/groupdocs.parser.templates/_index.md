Ο χώρος ονομάτων παρέχει κλάσεις για τη δημιουργία προτύπων για την ανάλυση δεδομένων εγγράφων.

| --- | --- |

| [TemplateBarcode](./templatebarcode) | Παρέχει το πεδίο γραμμικού κώδικα προτύπου. |
| [TemplateField](./templatefield) | Παρέχει το πεδίο κειμένου προτύπου. |
| [TemplateFixedPosition](./templatefixedposition) | Παρέχει μια θέση πεδίου προτύπου που ορίζεται από την ορθογώνια περιοχή. |
| [TemplateItem](./templateitem) | Παρέχει μια βασική αφηρημένη κλάση για στοιχεία προτύπων. |
| [TemplateLinkedPosition](./templatelinkedposition) | Παρέχει μια θέση πεδίου προτύπου που χρησιμοποιεί το συνδεδεμένο πεδίο. |
| [TemplateLinkedPositionEdges](./templatelinkedpositionedges) | Παρέχει τις άκρες του συνδεδεμένου πεδίου όπου γίνεται αναζήτηση του πεδίου κειμένου. | [TemplatePosition](./templateposition) | Παρέχει μια βασική αφηρημένη κλάση για θέσεις προτύπων. |

| [TemplateTable](./templatetable) | Παρέχει τον πίνακα προτύπων. |
| [TemplateTableLayout](./templatetablelayout) | Παρέχει τη διάταξη του πίνακα προτύπου που χρησιμοποιείται από[`TemplateTable`](../groupdocs.parser.templates/templatetable) κλάση για τον καθορισμό της θέσης του πίνακα. |
