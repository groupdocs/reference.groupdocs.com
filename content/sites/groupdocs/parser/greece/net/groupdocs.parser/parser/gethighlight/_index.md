## Parser.GetHighlight method

Εξάγει μια επισήμανση από το έγγραφο.

```csharp
public HighlightItem GetHighlight(int position, bool isDirect, HighlightOptions options)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| position | Int32 | Η αρχική θέση της επισήμανσης. |
| isDirect | Boolean | Η τιμή που υποδεικνύει εάν η εξαγωγή επισήμανσης είναι άμεση. `αληθής` αν η επισήμανση εξάγεται από το δικαίωμα του*position* ; σε διαφορετική περίπτωση,`ψευδής` . |
| options | HighlightOptions | Οι επιλογές εξαγωγής επισήμανσης. |

### Επιστρεφόμενη Αξία

Ένα στιγμιότυπο του[`HighlightItem`](../../../groupdocs.parser.data/highlightitem) κλάση που αντιπροσωπεύει το εξαγόμενο highlight; `μηδενικό` εάν δεν υποστηρίζεται η εξαγωγή επισήμανσης.

### Παραδείγματα

Το ακόλουθο παράδειγμα δείχνει πώς να εξαγάγετε μια επισήμανση που περιέχει 3 λέξεις:

```csharp
// Δημιουργία μιας παρουσίας κλάσης Parser
using (Parser parser = new Parser(filePath))
{
    // Εξαγωγή επισήμανσης:
    HighlightItem hl = parser.GetHighlight(2, true, new HighlightOptions(3));
    
    // Ελέγξτε εάν υποστηρίζεται η εξαγωγή επισήμανσης
    if (hl == null)
    {
        Console.WriteLine("Highlight extraction isn't supported");
        return;
    }
    
    // Εκτύπωση μιας εξαγόμενης επισήμανσης
    Console.WriteLine(string.Format("At {0}: {1}", hl.Position, hl.Text));
}
```
