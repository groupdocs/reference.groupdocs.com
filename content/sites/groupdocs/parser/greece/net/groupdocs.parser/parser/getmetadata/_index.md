### Παραδείγματα

Το ακόλουθο παράδειγμα δείχνει πώς να εξαγάγετε μεταδεδομένα από ένα έγγραφο:

```csharp
// Δημιουργία μιας παρουσίας κλάσης Parser
using(Parser parser = new Parser(filePath))
{
    // Εξαγωγή μεταδεδομένων από το έγγραφο
    IEnumerable<MetadataItem> metadata = parser.GetMetadata();
    // Ελέγξτε εάν υποστηρίζεται η εξαγωγή μεταδεδομένων
    if(metadata == null)
    {
        Console.WriteLine("Metatada extraction isn't supported");
    }
 
    // Επανάληψη σε στοιχεία μεταδεδομένων
    foreach(MetadataItem item in metadata)
    {
        // Εκτύπωση ονόματος και τιμής στοιχείου
        Console.WriteLine(string.Format("{0}: {1}", item.Name, item.Value));
    }
}
```
