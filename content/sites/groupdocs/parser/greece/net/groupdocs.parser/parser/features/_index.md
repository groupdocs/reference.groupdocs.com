### Παραδείγματα

Εάν η δυνατότητα δεν υποστηρίζεται, η μέθοδος επιστρέφει`μηδενικό` αντί της αξίας. Ορισμένες λειτουργίες ενδέχεται να απαιτούν σημαντικό χρόνο. Επομένως, δεν είναι βέλτιστο να καλέσετε τη μέθοδο για να ελέγξετε απλώς την υποστήριξη για τη δυνατότητα. Για το σκοπό αυτό χρησιμοποιείται η ιδιότητα Χαρακτηριστικών:

```csharp
using(Parser parser = new Parser("doc.zip"))
{
    if(!parser.Features.Text)
    {
        Console.WriteLine("Text extraction isn't supported");
        return;
    }
 
    using(TextReader reader = parser.GetText())
    {
        Console.WriteLine(reader.ReadToEnd());
    }
}
```
