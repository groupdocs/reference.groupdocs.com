### Επιστρεφόμενη Αξία

### Παραδείγματα

Το ακόλουθο παράδειγμα πώς να εξαγάγετε ένα κείμενο από ένα στοιχείο του πίνακα περιεχομένων:

```csharp
// Δημιουργία μιας παρουσίας κλάσης Parser
using (Parser parser = new Parser(Constants.SampleDocxWithToc))
{
    // Λήψη πίνακα περιεχομένων
    IEnumerable<TocItem> tocItems = parser.GetToc();
    // Ελέγξτε αν υποστηρίζεται η εξαγωγή toc
    if (tocItems == null)
    {
        Console.WriteLine("Table of contents extraction isn't supported");
    }
    // Επανάληψη σε στοιχεία
    foreach (TocItem tocItem in tocItems)
    {
        // Εκτυπώστε το κείμενο του κεφαλαίου
        using (TextReader reader = tocItem.ExtractText())
        {
            Console.WriteLine("----");
            Console.WriteLine(reader.ReadToEnd());
        }
    }
}
```
