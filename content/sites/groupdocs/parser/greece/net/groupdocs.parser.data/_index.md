Ο χώρος ονομάτων παρέχει κλάσεις που αντιπροσωπεύουν αποτελέσματα ανάλυσης.

## Τάξεις

| Τάξη | Περιγραφή |
| --- | --- |
| [ContainerItem](./containeritem) | Αντιπροσωπεύει ένα στοιχείο κοντέινερ όπως οντότητα αρχείου Zip, συνημμένο email, στοιχείο χαρτοφυλακίου PDF και ούτω καθεξής. |
| [DocumentData](./documentdata) | Αντιπροσωπεύει δεδομένα του εγγράφου. Αποτελείται απο[`FieldData`](../groupdocs.parser.data/fielddata)αντικείμενα που περιέχουν δεδομένα πεδίου από έγγραφο. |
| [FieldData](./fielddata) | Αντιπροσωπεύει δεδομένα πεδίου όπως όνομα, ευρετήριο σελίδας, τιμή πεδίου και ούτω καθεξής. Ανάλογα με το πεδίο, η τιμή μπορεί να είναι κείμενο, εικόνα, πίνακας και ούτω καθεξής. |
| [HighlightItem](./highlightitem) | Αντιπροσωπεύει μια επισήμανση, ένα μέρος του κειμένου που χρησιμοποιείται συνήθως για να εξηγήσει το πλαίσιο του κειμένου που βρέθηκε στη λειτουργία αναζήτησης. |
| [MetadataItem](./metadataitem) | Αντιπροσωπεύει ένα στοιχείο μεταδεδομένων που χρησιμοποιείται σε στοιχεία κοντέινερ και λειτουργία εξαγωγής μεταδεδομένων. |
| [Page](./page) | Αντιπροσωπεύει τις πληροφορίες της σελίδας του εγγράφου, όπως ευρετήριο σελίδας και μέγεθος σελίδας. Χρησιμοποιείται για την αναπαράσταση της σελίδας που περιέχει κληρονόμους του[`PageArea`](../groupdocs.parser.data/pagearea)class στη λειτουργικότητα ανάλυσης κατά πρότυπο. |
| [PageArea](./pagearea) | Αντιπροσωπεύει μια αφηρημένη βασική κλάση για περιοχές σελίδων που χρησιμοποιούνται για την αναπαράσταση μπλοκ στη σελίδα του εγγράφου στη λειτουργικότητα ανάλυσης κατά πρότυπο. |
| [PageBarcodeArea](./pagebarcodearea) | Αντιπροσωπεύει μια περιοχή γραμμικού κώδικα σελίδας που χρησιμοποιείται για να αναπαραστήσει μια τιμή γραμμικού κώδικα στη λειτουργικότητα ανάλυσης κατά πρότυπο. |
| [PageGroupArea](./pagegrouparea) | Αντιπροσωπεύει μια ομάδα περιοχών σελίδας που χρησιμοποιείται για την ομαδοποίηση διαφορετικών τύπων μπλοκ της σελίδας εγγράφου στη λειτουργικότητα ανάλυσης κατά πρότυπο. |
| [PageHyperlinkArea](./pagehyperlinkarea) | Αντιπροσωπεύει μια περιοχή σελίδας που χρησιμοποιείται για να αναπαραστήσει έναν υπερσύνδεσμο στη σελίδα. |
| [PageImageArea](./pageimagearea) | Αντιπροσωπεύει μια περιοχή εικόνας σελίδας που χρησιμοποιείται για την αναπαράσταση μιας εικόνας στη σελίδα στην ανάλυση κατά πρότυπο functionality ή ένα συνημμένο εικόνας εάν οι εικόνες εξάγονται από μηνύματα ηλεκτρονικού ταχυδρομείου ή από αρχεία Zip. |
| [PageTableArea](./pagetablearea) | Αντιπροσωπεύει μια περιοχή σελίδας πίνακα που χρησιμοποιείται για την αναπαράσταση ενός πίνακα στη λειτουργικότητα ανάλυσης κατά πρότυπο. |
| [PageTableAreaCell](./pagetableareacell) | Αντιπροσωπεύει ένα κελί πίνακα που χρησιμοποιείται σε[`PageTableArea`](../groupdocs.parser.data/pagetablearea) τάξη. |
| [PageTextArea](./pagetextarea) | Αντιπροσωπεύει μια περιοχή κειμένου σελίδας που χρησιμοποιείται για την αναπαράσταση μιας τιμής κειμένου στη λειτουργία ανάλυσης κατά πρότυπο ή φόρμας ανάλυσης. |
| [Point](./point) | Αντιπροσωπεύει ένα σημείο. |
| [Rectangle](./rectangle) | Αντιπροσωπεύει μια ορθογώνια περιοχή. |
| [SearchResult](./searchresult) | Αντιπροσωπεύει το αποτέλεσμα αναζήτησης στη λειτουργία αναζήτησης. |
| [Size](./size) | Αντιπροσωπεύει μέγεθος. |
| [TextStyle](./textstyle) | Αντιπροσωπεύει το στυλ του κειμένου, όπως ένα όνομα γραμματοσειράς, ένα μέγεθος γραμματοσειράς και ούτω καθεξής. |
| [TocItem](./tocitem) | Αντιπροσωπεύει το στοιχείο που χρησιμοποιείται στη λειτουργία εξαγωγής του πίνακα περιεχομένων. |
