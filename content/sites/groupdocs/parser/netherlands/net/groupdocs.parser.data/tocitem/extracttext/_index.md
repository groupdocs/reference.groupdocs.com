### Voorbeelden

Het volgende voorbeeld hoe een tekst te extraheren door middel van een inhoudsopgave:

```csharp
// Maak een instantie van de Parser-klasse
using (Parser parser = new Parser(Constants.SampleDocxWithToc))
{
    // Haal de inhoudsopgave op
    IEnumerable<TocItem> tocItems = parser.GetToc();
    // Controleer of toc-extractie wordt ondersteund
    if (tocItems == null)
    {
        Console.WriteLine("Table of contents extraction isn't supported");
    }
    // Items herhalen
    foreach (TocItem tocItem in tocItems)
    {
        // Druk de tekst van het hoofdstuk af
        using (TextReader reader = tocItem.ExtractText())
        {
            Console.WriteLine("----");
            Console.WriteLine(reader.ReadToEnd());
        }
    }
}
```
