### Voorbeelden

Als de functie niet wordt ondersteund, keert de methode terug`nul` in plaats van de waarde. Sommige bewerkingen kunnen veel tijd in beslag nemen. Het is dus niet optimaal om de methode aan te roepen om alleen de ondersteuning voor de functie te controleren. Hiervoor wordt de eigenschap Features gebruikt:

```csharp
using(Parser parser = new Parser("doc.zip"))
{
    if(!parser.Features.Text)
    {
        Console.WriteLine("Text extraction isn't supported");
        return;
    }
 
    using(TextReader reader = parser.GetText())
    {
        Console.WriteLine(reader.ReadToEnd());
    }
}
```
