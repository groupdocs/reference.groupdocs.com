### Voorbeelden

Het volgende voorbeeld laat zien hoe u een markering kunt extraheren die 3 woorden bevat:

```csharp
// Maak een instantie van de Parser-klasse
using (Parser parser = new Parser(filePath))
{
    // Extraheer een hoogtepunt:
    HighlightItem hl = parser.GetHighlight(2, true, new HighlightOptions(3));
    
    // Controleer of extractie van highlights wordt ondersteund
    if (hl == null)
    {
        Console.WriteLine("Highlight extraction isn't supported");
        return;
    }
    
    // Druk een geëxtraheerd hoogtepunt af
    Console.WriteLine(string.Format("At {0}: {1}", hl.Position, hl.Text));
}
```
