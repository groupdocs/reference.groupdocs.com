### Voorbeelden

Het volgende voorbeeld toont de code voor de situatie als bekend is dat het veld met een factuurnummer rechts van de "Factuurnummer"-string staat, wordt de volgende code gebruikt:

```csharp
// Maak een regex-sjabloonveld om de tekst "Factuurnummer" te vinden
TemplateField invoice = new TemplateField(new TemplateRegexPosition("Invoice Number"), "Invoice");

// Maak een gerelateerd sjabloonveld dat is gekoppeld aan het veld "Factuur" en extraheer de waarde rechts ervan
TemplateField invoiceNumber = new TemplateField(
    new TemplateLinkedPosition("invoice", new Size(100, 15), new TemplateLinkedPositionEdges(false, false, true, false)),
    "InvoiceNumber");
```
