## ContainerItem.Metadata property

Obtiene la colección de elementos de metadatos.

```csharp
public IEnumerable<MetadataItem> Metadata { get; }
```

### El valor de la propiedad

Una coleccion de[`MetadataItem`](../../metadataitem) objetos; vacío si los metadatos no están configurados.

### Observaciones

Estos metadatos se refieren a un elemento contenedor en sí mismo, no a un documento. Según el tipo de contenedor los metadatos pueden contener los siguientes elementos:

**Adjuntos de correo electrónico**

**Nombre**

**Descripción**

**tipo de contenido**

El tipo MIME del contenido del archivo adjunto.

**Archivos ZIP**

**Nombre**

**Descripción**

**fecha**

La hora y la fecha en que se modificó por última vez el archivo indicado por Zip Entry.

**Almacenamiento de Outlook**

**Nombre**

**Descripción**

**fecha**

La hora y la fecha en que se modificó por última vez el elemento de almacenamiento de Outlook.

**remitente de correo electrónico**

El valor del campo "remitente".

**Email para**

El valor del campo "a".

**sujeto**

El valor del campo "asunto".
