| loadOptions | LoadOptions | Las opciones para abrir el archivo. |

### Valor_devuelto

una instancia de[`FileInfo`](../../../groupdocs.parser.options/fileinfo) clase.

### Ejemplos

El siguiente código muestra cómo verificar un tipo de archivo del documento protegido por contraseña:

```csharp
// Obtener información de un archivo
Options.FileInfo info = Parser.GetFileInfo(filePath, new LoadOptions("password"));
// Comprobar la propiedad IsEncrypted

// Imprime el tipo de archivo
Console.WriteLine(info.FileType.ToString());
```
