### Observaciones

Una instancia de[`TextOptions`](../textoptions) la clase se usa como parámetro en[`GetText`](../../groupdocs.parser/parser/gettext) y[`GetText`](../../groupdocs.parser/parser/gettext) métodos. Vea los ejemplos de uso allí.

Se utiliza para especificar el modo sin formato de extracción de texto. Un texto en este modo se extrae de forma imprecisa pero más rápido que en el modo estándar. Si el modo sin formato no es compatible con el formato del documento, se ignora este parámetro y se utiliza el modo estándar.
