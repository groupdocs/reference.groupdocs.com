## Parser.GetHighlight method

ドキュメントからハイライトを抽出します。

```csharp
public HighlightItem GetHighlight(int position, bool isDirect, HighlightOptions options)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| position | Int32 | ハイライトの開始位置。 |
| isDirect | Boolean | ハイライト抽出が直接かどうかを示す値. `真実`ハイライトが次の権利によって抽出される場合*position*;さもないと、`間違い`. |
| options | HighlightOptions | ハイライト抽出オプション。 |

### 戻り値

のインスタンス[`HighlightItem`](../../../groupdocs.parser.data/highlightitem)抽出されたハイライトを表すクラス; `ヌル`ハイライト抽出がサポートされていない場合.

### 例

次の例は、3 つの単語を含むハイライトを抽出する方法を示しています。

```csharp
// Parser クラスのインスタンスを作成します
using (Parser parser = new Parser(filePath))
{
    // ハイライトを抽出:
    HighlightItem hl = parser.GetHighlight(2, true, new HighlightOptions(3));
    
    // ハイライト抽出に対応しているかチェック
    if (hl == null)
    {
        Console.WriteLine("Highlight extraction isn't supported");
        return;
    }
    
    // 抽出されたハイライトを出力します
    Console.WriteLine(string.Format("At {0}: {1}", hl.Position, hl.Text));
}
```
