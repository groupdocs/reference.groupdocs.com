The namespace provides classes for creating templates for parsing document data.

### Classes
| Class | Description |
| :- | :- |
| [`Template`](/parser/python-net/groupdocs.parser.templates/template) | Provides the document page template. It consists of [`TemplateItem`](/parser/python-net/groupdocs.parser.templates/templateitem) objects<br/>which represent the items of the template such as text field and table definitions. |
| [`TemplateBarcode`](/parser/python-net/groupdocs.parser.templates/templatebarcode) | Provides the template barcode field. |
| [`TemplateCollection`](/parser/python-net/groupdocs.parser.templates/templatecollection) | Represents a collection of document parsing templates. |
| [`TemplateField`](/parser/python-net/groupdocs.parser.templates/templatefield) | Provides the template text field. |
| [`TemplateFixedPosition`](/parser/python-net/groupdocs.parser.templates/templatefixedposition) | Provides a template field position which is defined by the rectangular area. |
| [`TemplateItem`](/parser/python-net/groupdocs.parser.templates/templateitem) | Provides a base abstract class for template items. |
| [`TemplateLinkedPosition`](/parser/python-net/groupdocs.parser.templates/templatelinkedposition) | Provides a template field position which uses the linked field. |
| [`TemplateLinkedPositionEdges`](/parser/python-net/groupdocs.parser.templates/templatelinkedpositionedges) | Provides the edges of the linked field where the text field is seached. <br/>[`TemplateLinkedPositionEdges`](/parser/python-net/groupdocs.parser.templates/templatelinkedpositionedges) is used in [`TemplateLinkedPosition`](/parser/python-net/groupdocs.parser.templates/templatelinkedposition) class. |
| [`TemplateOptions`](/parser/python-net/groupdocs.parser.templates/templateoptions) | Provides the options which are used for parsing by template functionality. |
| [`TemplatePosition`](/parser/python-net/groupdocs.parser.templates/templateposition) | Provides a base abstract class for template positions. |
| [`TemplateRegexPosition`](/parser/python-net/groupdocs.parser.templates/templateregexposition) | Provides a template field position which uses the regular expression. |
| [`TemplateTable`](/parser/python-net/groupdocs.parser.templates/templatetable) | Provides the template table. |
| [`TemplateTableLayout`](/parser/python-net/groupdocs.parser.templates/templatetablelayout) | Provides the template table layout which is used by [`TemplateTable`](/parser/python-net/groupdocs.parser.templates/templatetable) class to define table position. |
| [`TemplateTableParameters`](/parser/python-net/groupdocs.parser.templates/templatetableparameters) | Provides parameters for the table detection algorithms. |
