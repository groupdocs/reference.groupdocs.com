## SearchOptions.SearchByPages property

Gets the value that indicates whether the search is performed by pages.

```csharp
public bool SearchByPages { get; set; }
```

### Property Value

`true` if the search is performed by pages; otherwise, `false` and the search is performed on the whole document without including [`PageIndex`](../../../groupdocs.parser.data/searchresult/pageindex) property value in [`SearchResult`](../../../groupdocs.parser.data/searchresult) class.
