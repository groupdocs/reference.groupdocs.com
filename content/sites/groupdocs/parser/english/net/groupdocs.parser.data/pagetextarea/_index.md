### Remarks

An instance of [`PageTextArea`](../pagetextarea) class is used as return value of the following methods:

* [`GetTextAreas`](../../groupdocs.parser/parser/gettextareas)
* [`GetTextAreas`](../../groupdocs.parser/parser/gettextareas)
* [`GetTextAreas`](../../groupdocs.parser/parser/gettextareas)
* [`GetTextAreas`](../../groupdocs.parser/parser/gettextareas)

Also an instance of [`PageTextArea`](../pagetextarea) class is used as value of [`PageArea`](../fielddata/pagearea) property.

See the usage examples there.

The text area can be single or composite. In the first case it contains a text which is bounded by a rectangular area. In the second case it contains other text areas; text and table properties are calculated by child text areas.
