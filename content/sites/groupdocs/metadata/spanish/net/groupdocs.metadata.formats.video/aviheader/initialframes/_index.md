## AviHeader.InitialFrames property

Obtiene el cuadro inicial para archivos intercalados.  Los archivos no intercalados deben especificar cero. Si está creando archivos intercalados, especifique el número de fotogramas en el archivo antes del fotograma inicial de la secuencia AVI en este miembro.

```csharp
public int InitialFrames { get; }
```

### El valor de la propiedad
