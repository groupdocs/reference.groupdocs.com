## AviHeader.SuggestedBufferSize property

Obtiene el tamaño de búfer sugerido para leer el archivo.  Por lo general, este tamaño debe ser lo suficientemente grande como para contener la parte más grande del archivo. Si se establece en cero, o si es demasiado pequeño, el software de reproducción tendrá que reasignar memoria durante la reproducción, lo que reducirá el rendimiento. Para un archivo intercalado, el tamaño del búfer debe ser lo suficientemente grande para leer un registro completo, y no solo un fragmento.

```csharp
public int SuggestedBufferSize { get; }
```

### El valor de la propiedad

El tamaño de búfer sugerido para leer el archivo.
