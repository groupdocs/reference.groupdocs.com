## MatroskaTrack.Language property

Obtiene el idioma de la pista en el formulario de idiomas Matroska. Este elemento DEBE ignorarse si el[`LanguageIetf`](../languageietf) El elemento se utiliza en el mismo TrackEntry.

```csharp
public string Language { get; }
```

### El valor de la propiedad

El idioma de la pista en la forma de idiomas Matroska.

### Observaciones

Los códigos de idioma pueden ser la forma bibliográfica ISO-639-2 de 3 letras (como "fre" para francés), o dicho código de idioma seguido de un guión y un código de país para especialidades en idiomas (como "fre-ca" para francés canadiense). Los códigos de país son los mismos que se utilizan para los dominios de Internet.
