## DublinCorePackage.Rights property

Obtiene los derechos del elemento Dublin Core.

```csharp
public string Rights { get; }
```

### El valor de la propiedad

Información sobre los derechos que se tienen en y sobre el recurso. Normalmente, un elemento de derechos contendrá una declaración de gestión de derechos para el recurso, o hará referencia a un servicio que proporciona dicha información. La información de derechos a menudo abarca los derechos de propiedad intelectual (DPI), los derechos de autor y varios derechos de propiedad. Si el elemento de derechos está ausente, no se pueden hacer suposiciones sobre el estado de estos y otros derechos con respecto al recurso.
