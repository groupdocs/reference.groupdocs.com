## MP3RootPackage.ID3V2 property

Obtiene o establece la etiqueta de metadatos ID3v2.

```csharp
public ID3V2Tag ID3V2 { get; set; }
```

### El valor de la propiedad

La etiqueta de metadatos ID3v2 adjunta al archivo de audio.

### Ejemplos

El ejemplo de código muestra cómo actualizar la etiqueta ID3v2 en un archivo MP3.

```csharp
using (Metadata metadata = new Metadata(Constants.MP3WithID3V2))
{
    var root = metadata.GetRootPackage<MP3RootPackage>();

    if (root.ID3V2 == null)
    {
        root.ID3V2 = new ID3V2Tag();
    }

    root.ID3V2.Album = "test album";
    root.ID3V2.Artist = "test artist";
    root.ID3V2.Band = "test band";
    root.ID3V2.TrackNumber = "1";
    root.ID3V2.MusicalKey = "C#";
    root.ID3V2.Title = "code sample";
    root.ID3V2.Date = "2019";

    // ...

    metadata.Save(Constants.OutputMp3);
}
```
