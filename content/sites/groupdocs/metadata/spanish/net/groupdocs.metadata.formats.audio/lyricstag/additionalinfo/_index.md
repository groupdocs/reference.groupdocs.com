## LyricsTag.AdditionalInfo property

Obtiene o establece la información adicional. Este valor está representado por el campo INF.

```csharp
public string AdditionalInfo { get; set; }
```

### El valor de la propiedad

La información adicional.

### Observaciones

Esto siempre tiene tres (3) caracteres de largo en v2.00, pero podría ser más largo en un estándar futuro. El primer byte indica si el campo de letras está presente o no. "1" para presente y "0" para lo contrario. El segundo carácter indica si hay una marca de tiempo en la letra. Nuevamente, "1" para sí y "0" para no. El tercer carácter inhibe las pistas para la selección aleatoria: "1" si está inhibido y "0" si no.
