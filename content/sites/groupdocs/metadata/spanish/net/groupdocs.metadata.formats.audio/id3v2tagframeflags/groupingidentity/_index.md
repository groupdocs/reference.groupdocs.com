## ID3V2TagFrameFlags.GroupingIdentity property

Obtiene un valor que indica si el marco pertenece a un grupo de marcos. Si se establece, se agrega un byte de identificador de grupo al encabezado del marco. Cada cuadro con el mismo identificador de grupo pertenece al mismo grupo.

```csharp
public bool GroupingIdentity { get; }
```

### El valor de la propiedad

`verdadero`, si el marco pertenece a un grupo de marcos; en caso contrario, falso.
