## ID3V2TagFrameFlags.DataLengthIndicator property

Obtiene un valor que indica si se ha agregado un indicador de longitud de datos a el marco. El indicador de longitud de datos es el valor que uno escribiría como 'Longitud de trama' si todas las banderas de formato de trama fueran puestas a cero, representadas como un entero sincronizado seguro de 32 bits.

```csharp
public bool DataLengthIndicator { get; }
```

### El valor de la propiedad

`verdadero`si se ha agregado un indicador de longitud de datos al marco; en caso contrario, falso.
