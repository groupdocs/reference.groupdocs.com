## XmpIptcCorePackage.Location property

Obtiene o establece la ubicación en la que se enfoca el contenido.

```csharp
public string Location { get; set; }
```

### El valor de la propiedad

La ubicación.

### Observaciones

Este nombre de ubicación podría ser el nombre de una sububicación a una ciudad o el nombre de una ubicación bien conocida o un monumento (natural) fuera de una ciudad. En el sentido de una sububicación a una ciudad, este elemento se encuentra en el cuarto nivel de una Jerarquía geográfica de arriba hacia abajo.
