## PropertyTypeTagCategory class

Proporciona etiquetas que contienen información adicional sobre el tipo de una propiedad en lugar de su propósito. Con estas etiquetas, puede detectar propiedades de metadatos que contienen enlaces URL a recursos externos, propiedades que describen fuentes, colores, ubicación geográfica, etc.

public class PropertyTypeTagCategory : TagCategory
