## ContentTagCategory class

Proporciona etiquetas que se adjuntan a las propiedades de metadatos que describen el contenido de un archivo. Las etiquetas son útiles para saber el idioma del contenido, tipo (género), tema, calificación, etc.

```csharp
public class ContentTagCategory : TagCategory
```
