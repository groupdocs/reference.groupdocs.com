## Tags class

Contiene varios conjuntos de etiquetas con las que se marcan las propiedades de metadatos más importantes. Las etiquetas le permiten buscar y actualizar las propiedades de metadatos en diferentes paquetes, independientemente del estándar de metadatos y el formato de archivo.
