## Tags.Person property

Obtiene un conjunto de etiquetas que marcan las propiedades de metadatos que contienen información sobre las personas que contribuyeron a la creación de archivos o contenido intelectual. Estas etiquetas pueden ayudarlo a encontrar al creador del documento, al editor o incluso al cliente para quien se realizó el trabajo. A pesar del nombre de la categoría, algunas propiedades de metadatos marcadas con las etiquetas pueden contener el nombre de una empresa en lugar del nombre de una persona.

```csharp
public static PersonTagCategory Person { get; }
```

### El valor de la propiedad

Un conjunto de etiquetas que marcan las propiedades de los metadatos que contienen información sobre las personas que contribuyeron a la creación de archivos o contenido intelectual.
