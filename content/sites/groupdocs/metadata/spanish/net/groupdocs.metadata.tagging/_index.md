El espacio de nombres contiene varios conjuntos de etiquetas con las que se marcan las propiedades de metadatos más importantes. Las etiquetas permiten a los usuarios encontrar y actualizar las propiedades de metadatos en diferentes paquetes, independientemente del estándar de metadatos y el formato de archivo.

## Clases

| Clase | Descripción |
| --- | --- |
| [ContentTagCategory](./contenttagcategory) | Proporciona etiquetas que se adjuntan a las propiedades de metadatos que describen el contenido de un archivo. Las etiquetas son útiles para saber el idioma del contenido, tipo (género), tema, calificación, etc. |
| [CorporateTagCategory](./corporatetagcategory) | Proporciona etiquetas destinadas a marcar propiedades de metadatos relacionadas con una empresa que participó en la creación del archivo. |
| [DocumentTagCategory](./documenttagcategory) | Proporciona etiquetas que se aplican solo a propiedades específicas del documento. Las etiquetas pueden ser útiles para determinar de qué parte de un documento de oficina se extrajo una propiedad. |
| [LegalTagCategory](./legaltagcategory) | Proporciona etiquetas que se adjuntan a las propiedades de metadatos que contienen información sobre los propietarios del contenido del archivo y las reglas bajo las cuales se puede usar el contenido. |
| [OriginTagCategory](./origintagcategory) | Proporciona etiquetas que ayudan al usuario a determinar el origen de un archivo (por ejemplo, una plantilla u otra fuente). |
| [PersonTagCategory](./persontagcategory) | Proporciona etiquetas que marcan las propiedades de metadatos que contienen información sobre las personas que contribuyeron a la creación de archivos o contenido intelectual. Estas etiquetas pueden ayudarlo a encontrar al creador del documento, al editor o incluso al cliente para quien se realizó el trabajo. A pesar del nombre de la categoría, algunas propiedades de metadatos marcadas con las etiquetas pueden contener el nombre de una empresa en lugar del nombre de una persona. |
| [PropertyTag](./propertytag) | Representa una etiqueta utilizada para marcar propiedades de metadatos. |
| [PropertyTypeTagCategory](./propertytypetagcategory) | Proporciona etiquetas que contienen información adicional sobre el tipo de una propiedad en lugar de su propósito. Con estas etiquetas, puede detectar propiedades de metadatos que contienen enlaces URL a recursos externos, propiedades que describen fuentes, colores, ubicación geográfica, etc. |
| [TagCategory](./tagcategory) | Representa un conjunto de etiquetas que comparten algunas características. |
| [Tags](./tags) | Contiene varios conjuntos de etiquetas con las que se marcan las propiedades de metadatos más importantes. Las etiquetas le permiten buscar y actualizar las propiedades de metadatos en diferentes paquetes, independientemente del estándar de metadatos y el formato de archivo. |
| [TimeTagCategory](./timetagcategory) | Proporciona etiquetas que marcan las propiedades de los metadatos que se utilizan para describir el ciclo de vida de un archivo. Las etiquetas se ocupan de los momentos en los que se creó, editó, imprimió, etc. un archivo o contenido intelectual. |
| [ToolTagCategory](./tooltagcategory) | Proporciona etiquetas destinadas a marcar propiedades de metadatos relacionadas con las herramientas (software y hardware) que se usaron para crear un archivo. |
