## XmpLangAlt class

representa la alternativa de lenguaje XMP.

Una matriz alternativa de elementos de texto simples. Las alternativas de idioma facilitan la selección de un elemento de texto simple basado en el idioma deseado. Cada elemento de la matriz tendrá un calificador xml:lang. Cada valor de xml:lang será único entre los elementos.

public sealed class XmpLangAlt : XmpArray
