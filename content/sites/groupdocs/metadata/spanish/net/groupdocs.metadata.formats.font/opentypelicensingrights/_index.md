Indica los derechos de licencia de incrustación de fuentes para la fuente.

public enum OpenTypeLicensingRights : ushort

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | `0` | Los derechos de licencia indefinidos. |
| UsagePermissionsMask | `F` | Máscara de permisos de uso. |
| InstallableEmbedding | `1` | Incrustación instalable. La fuente se puede incrustar y se puede instalar de forma permanente para usarla en sistemas remotos o para que la usen otros usuarios. |
| RestrictedLicenseEmbedding | `2` | Incrustación de licencia restringida. La fuente no debe modificarse, incrustarse ni intercambiarse de ninguna manera sin obtener primero el permiso explícito del propietario legal. |
| PreviewAndPrintEmbedding | `4` | Incrustación de vista previa e impresión. La fuente se puede incrustar y se puede cargar temporalmente en otros sistemas para ver o imprimir el documento. Los documentos que contienen fuentes de vista previa e impresión deben abrirse como "solo lectura"; no se pueden aplicar ediciones al documento. |
| EditableEmbedding | `8` | Incrustación editable. La fuente puede estar incrustada y puede cargarse temporalmente en otros sistemas. Al igual que con la incrustación de vista previa e impresión, los documentos que contienen fuentes editables se pueden abrir para leer. Además, se permite la edición, incluida la capacidad de formatear texto nuevo usando la fuente incrustada, y se pueden guardar los cambios. |
| NoSubsetting | `100` | Sin subconjunto. Cuando se establece este bit, la fuente no se puede subconjunto antes de la incrustación. También se aplican otras restricciones de incrustación especificadas en los bits 0 a 3 y el bit 9. |
| BitmapEmbeddingOnly | `200` | Solo incrustación de mapa de bits. Cuando se establece este bit, solo se pueden incrustar los mapas de bits contenidos en la fuente. No se pueden incrustar datos de contorno. Si no hay mapas de bits disponibles en la fuente, se considera que la fuente no se puede incrustar y los servicios de incrustación fallarán. También se aplican otras restricciones de incrustación especificadas en los bits 0-3 y 8. |
