## FileType.ASF field

Advanced Systems Format (ASF) — это цифровой мультимедийный контейнер, предназначенный в первую очередь для хранения и передачи медиапотоков. Microsoft Windows Media Video (WMV) — формат сжатого видео, а Microsoft Windows Media Audio (WMA) — формат с дополнительными метаданными в контейнере ASF, разработанном Microsoft. Подробнее об этом файле format [здесь](https://wiki.fileformat.com/video/wmv/) .
