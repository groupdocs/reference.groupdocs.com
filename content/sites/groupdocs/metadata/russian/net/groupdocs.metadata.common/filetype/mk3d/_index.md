## FileType.MK3D field

MK3D на самом деле представляет собой стереоскопическое 3D-видео, созданное с использованием формата Matroska 3D. Файловый контейнер MKV использует значение поля StereoMode для определения типа стереоскопического 3D-видеоматериала.
