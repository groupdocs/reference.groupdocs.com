## OpenTypeLicensingRights enumeration

Указывает лицензионные права на встраивание шрифта для шрифта.

```csharp
[Flags]
public enum OpenTypeLicensingRights : ushort
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| None | `0` | Неопределенные лицензионные права. |
| UsagePermissionsMask | `F` | Маска разрешений на использование. |
| InstallableEmbedding | `1` | Устанавливаемое встраивание. Шрифт может быть встроенным или постоянно установленным для использования в удаленных системах или для использования другими пользователями. |
| RestrictedLicenseEmbedding | `2` | Встраивание с ограниченной лицензией. Шрифт нельзя изменять, встраивать или заменять каким-либо образом без предварительного получения явного разрешения законного владельца. |
| PreviewAndPrintEmbedding | `4` | Встраивание для предварительного просмотра и печати. Шрифт может быть встроен и может быть временно загружен в другие системы для просмотра или печати документа. Документы, содержащие шрифты для предварительного просмотра и печати, должны быть открыты «только для чтения»; никакие изменения не могут быть применены к документу. |
| EditableEmbedding | `8` | Редактируемое встраивание. Шрифт может быть встроен и временно загружен в другие системы. Как и при встраивании предварительного просмотра и печати, документы, содержащие редактируемые шрифты, могут быть открыты для чтения. Кроме того, разрешено редактирование, включая возможность форматирования нового текста с использованием встроенного шрифта, и изменения могут быть сохранены. |
| NoSubsetting | `100` | Нет подмножества. Когда этот бит установлен, шрифт не может быть подмножеством перед внедрением. Также применяются другие ограничения внедрения, указанные в битах с 0 по 3 и в бите 9. |
| BitmapEmbeddingOnly | `200` | Только встраивание растровых изображений. Если этот бит установлен, могут быть внедрены только растровые изображения, содержащиеся в шрифте. Данные структуры не могут быть внедрены. Если в шрифте нет доступных растровых изображений, шрифт считается не встраиваемым, и службы встраивания не работают. Также применяются другие ограничения внедрения, указанные в битах 0-3 и 8. |
