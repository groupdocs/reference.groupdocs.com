## XmpLangAlt class

представляет собой альтернативу языку XMP.

Альтернативный массив простых текстовых элементов. Языковые альтернативы облегчают выбор простого текстового элемента на основе желаемого языка. Каждый элемент массива должен иметь квалификатор xml:lang. Каждое значение xml:lang должно быть уникальным среди элементов.

public sealed class XmpLangAlt : XmpArray
