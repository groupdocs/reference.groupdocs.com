## PersonTagCategory class

Предоставляет теги, которые отмечают свойства метаданных, содержащие информацию о людях, внесших свой вклад в создание файла или интеллектуального контента. Эти теги могут помочь вам найти создателя документа, редактора или даже клиента, для которого выполнялась работа. Несмотря на название категории, некоторые свойства метаданных, отмеченные тегами, могут содержать название компании, а не имя человека.

```csharp
public class PersonTagCategory : TagCategory
```
