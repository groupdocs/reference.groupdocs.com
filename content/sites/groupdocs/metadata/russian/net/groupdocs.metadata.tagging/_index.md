Пространство имен содержит различные наборы тегов, которыми отмечены наиболее важные свойства метаданных. Теги позволяют пользователям находить и обновлять свойства метаданных в различных пакетах независимо от стандарта метаданных и формата файла.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [ContentTagCategory](./contenttagcategory) | Предоставляет теги, которые присоединяются к свойствам метаданных, описывающим содержимое файла. Теги полезны для определения языка контента, типа (жанра), темы, рейтинга и т. д. |
| [CorporateTagCategory](./corporatetagcategory) | Предоставляет теги, предназначенные для обозначения свойств метаданных, связанных с компанией, участвовавшей в создании файла. |
| [DocumentTagCategory](./documenttagcategory) | Предоставляет теги, которые применяются только к свойствам документа. Теги могут быть полезны для определения того, из какой части офисного документа было извлечено свойство. |
| [LegalTagCategory](./legaltagcategory) | Предоставляет теги, прикрепленные к свойствам метаданных, содержащие информацию о владельцах содержимого файла и правилах, по которым может использоваться содержимое. |
| [OriginTagCategory](./origintagcategory) | Предоставляет теги, помогающие пользователю определить происхождение файла (например, шаблон или другой источник). |
| [PersonTagCategory](./persontagcategory) | Предоставляет теги, которые отмечают свойства метаданных, содержащие информацию о людях, внесших свой вклад в создание файла или интеллектуального контента. Эти теги могут помочь вам найти создателя документа, редактора или даже клиента, для которого выполнялась работа. Несмотря на название категории, некоторые свойства метаданных, отмеченные тегами, могут содержать название компании, а не имя человека. |
| [PropertyTag](./propertytag) | Представляет тег, используемый для обозначения свойств метаданных. |
| [PropertyTypeTagCategory](./propertytypetagcategory) | Предоставляет теги, которые несут дополнительную информацию о типе свойства, а не о его назначении. С помощью этих тегов вы можете обнаруживать свойства метаданных, которые содержат URL-ссылки на внешние ресурсы, свойства, описывающие шрифты, цвета, геолокацию и т. д.. |
| [TagCategory](./tagcategory) | Представляет набор тегов, имеющих некоторые общие характеристики. |
| [Tags](./tags) | Содержит различные наборы тегов, с помощью которых отмечены наиболее важные свойства метаданных. Теги позволяют находить и обновлять свойства метаданных в различных пакетах независимо от стандарта метаданных и формата файла. |
| [TimeTagCategory](./timetagcategory) | Предоставляет теги, которые отмечают свойства метаданных, используемые для описания жизненного цикла файла. Теги относятся к моментам времени, когда файл или интеллектуальный контент был создан, отредактирован, распечатан и т. д. |
| [ToolTagCategory](./tooltagcategory) | Предоставляет теги, предназначенные для обозначения свойств метаданных, связанных с инструментами (программным и аппаратным обеспечением), которые использовались для создания файла. |
