Содержимое может включать неструктурированные или полуструктурированные таблицы чисел, текста или как числа, так и текст, формулы, подключения к внешним данным, диаграммы и изображения. Стандарт OOXML ECMA-376, новый формат представляет собой ZIP-пакет, содержащий несколько XML-файлов. Подробнее об этом формате файлов[здесь](https://wiki.fileformat.com/spreadsheet/xlsx/) . |

С точки зрения приложения макрос представляет собой набор инструкций, которые используются для автоматизации процессов. Макрос используется для записи шаги, которые выполняются повторно, и облегчает выполнение действий путем повторного запуска макроса. Он используется для создания стандартного файла шаблона, который можно использовать для создания файлов XLSX, демонстрирующих те же настройки, что и в файле XLTX. Файлы XLTM аналогичны XLTX по структуре, за исключением того, что последний не поддерживает создание файлов шаблонов с macros. Такие файлы шаблонов используются для создания и установки макета, форматирования и других параметров вместе с макросами, чтобы облегчить создание аналогичных файлов XLSX. Данные хранятся в файле ODF в строках и столбцах. Это формат на основе XML, который является одним из нескольких подтипов в семействе форматов открытых документов (ODF). Microsoft Office 97-2003 поддерживает создание новых файлов XLT, а также открытие this. Последняя версия Excel по-прежнему способна открывать файлы шаблонов этого старого формата.
