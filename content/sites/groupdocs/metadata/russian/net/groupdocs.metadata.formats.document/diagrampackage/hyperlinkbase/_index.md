## DiagramPackage.HyperlinkBase property

Получает или задает путь, который будет использоваться для относительных гиперссылок (гиперссылок, для которых расположение связанного файла описано относительно диаграммы Microsoft Visio). По умолчанию путь гиперссылки относится к текущему документу, если не указан другой путь в этом элементе. Максимальная длина 256 символов.

public string HyperlinkBase { get; set; }
