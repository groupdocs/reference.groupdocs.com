## AviHeader.SuggestedBufferSize property

Получает рекомендуемый размер буфера для чтения файла.  Как правило, этот размер должен быть достаточно большим, чтобы вместить самый большой блок в файле. Если установлено значение 0 или слишком маленькое значение, программе воспроизведения придется перераспределять память во время воспроизведения, что снизит производительность. Для файла с чередованием размер буфера должен быть достаточно большим для чтения всей записи, а не только фрагмента.

```csharp
public int SuggestedBufferSize { get; }
```
