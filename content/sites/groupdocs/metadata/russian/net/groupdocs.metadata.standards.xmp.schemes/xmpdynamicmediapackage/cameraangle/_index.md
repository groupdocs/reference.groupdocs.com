## XmpDynamicMediaPackage.CameraAngle property

Получает или задает ориентацию камеры по отношению к объекту в статическом снимке из фиксированного набора стандартной отраслевой терминологии.

```csharp
public string CameraAngle { get; set; }
```

### Стоимость имущества

Ориентация камеры на объект в статическом снимке из фиксированного набора стандартных отраслевых терминов. Предопределенные значения включают: Низкий угол, Уровень глаз, Высокий угол, Съемка сверху, Съемка с высоты птичьего полета, Голландский угол, POV, Над Плечо, Реактивный выстрел.
