## ExifGpsPackage.Satellites property

Получает или устанавливает спутники GPS, используемые для измерений. Этот тег может использоваться для описания количества спутников, их идентификационного номера, угла места, азимута, SNR и другой информации в нотации ASCII. Формат не указан. Если приемник GPS не может выполнять измерения, значение тега должно быть установлено в NULL.

```csharp
public string Satellites { get; set; }
```
