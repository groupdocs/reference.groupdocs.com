Пространство имен предоставляет функциональные возможности, предназначенные для работы с цифровыми подписями на основе синтаксиса PKCS #7, указанного в RFC 5652.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [Cms](./cms) | Представляет цифровой знак, созданный с помощью криптографического синтаксиса сообщений (CMS) — стандарта IETF для криптографически защищенных сообщений. CMS основан на синтаксисе PKCS #7, указанном в RFC 5652. См.[https://tools.ietf.org/html/rfc5652](https://tools.ietf.org/html/rfc5652) для получения дополнительной информации. |
| [CmsAttribute](./cmsattribute) | Представляет атрибут информации о подписывающей стороне CMS. |
| [CmsCertificate](./cmscertificate) | Представляет сертификат CMS. |
| [CmsEncapsulatedContent](./cmsencapsulatedcontent) | Представляет подписанный контейнер содержимого, состоящий из идентификатора типа содержимого и самого содержимого. |
| [CmsPackage](./cmspackage) | Представляет пакет метаданных цифровой подписи CMS. |
| [CmsSigner](./cmssigner) | Представляет информацию CMS для каждого подписавшего. |
| [Oid](./oid) | Представляет идентификатор объекта (OID). Механизм идентификатора, стандартизированный Международным союзом электросвязи (ITU) и ISO/IEC для именования любого объекта, понятия или «предмета» глобальным недвусмысленным постоянным именем. |
