## ExifGpsPackage.Satellites property

測定に使用される GPS 衛星を取得または設定します。 このタグは、衛星の数、 ID 番号、仰角、方位角、SNR およびその他の情報を ASCII 表記で記述するために使用できます。フォーマットは指定されていません。 GPS 受信機が測定できない場合、タグの値は NULL. に設定されます。

```csharp
public string Satellites { get; set; }
```

### プロパティ値

測定に使用される GPS 衛星。
