名前空間には、最も重要なメタデータ プロパティがマークされているさまざまなタグのセットが含まれています. タグにより、ユーザーは、メタデータの標準とファイル形式に関係なく、さまざまなパッケージのメタデータ プロパティを検索および更新できます.

## クラス

| クラス | 説明 |
| --- | --- |
| [ContentTagCategory](./contenttagcategory) | ファイルの内容を説明するメタデータ プロパティに添付されるタグを提供します。 タグは、コンテンツの言語、タイプ (ジャンル)、件名、評価などを調べるのに役立ちます。 |
| [CorporateTagCategory](./corporatetagcategory) | ファイルの作成に参加した会社に関連するメタデータ プロパティをマークするためのタグを提供します。 |
| [DocumentTagCategory](./documenttagcategory) | ドキュメント固有のプロパティにのみ適用されるタグを提供します。 タグは、オフィス ドキュメントのどの部分からプロパティが抽出されたかを判断するのに役立ちます。 |
| [LegalTagCategory](./legaltagcategory) | ファイル コンテンツ の所有者とコンテンツを使用できるルールに関する情報を保持するメタデータ プロパティに添付されるタグを提供します。 |
| [OriginTagCategory](./origintagcategory) | ユーザーがファイルの出所を特定するのに役立つタグを提供します (例: テンプレートまたは別のソース). |
| [PersonTagCategory](./persontagcategory) | ファイルまたは知的コンテンツの作成に貢献した人々に関する情報を保持するメタデータ プロパティをマークするタグを提供します。 これらのタグは、ドキュメントの作成者、編集者、または作業が実行されたクライアントを見つけるのに役立ちます. カテゴリの名前にもかかわらず、タグでマークされた一部のメタデータ プロパティには、個人名ではなく会社名が含まれる場合があります. |
| [PropertyTag](./propertytag) | メタデータ プロパティをマークするために使用されるタグを表します。 |
| [PropertyTypeTagCategory](./propertytypetagcategory) | プロパティの目的ではなくタイプに関する追加情報を含むタグを提供します。 これらのタグを使用すると、外部リソースへの URL リンクを含むメタデータ プロパティ、フォント、色、地理位置情報などを説明する プロパティを検出できます。 |
| [TagCategory](./tagcategory) | いくつかの共通の特性を持つタグのセットを表します. |
| [Tags](./tags) | 最も重要なメタデータ プロパティがマークされているさまざまなタグのセットが含まれています. タグを使用すると、メタデータの標準とファイル形式に関係なく、さまざまなパッケージのメタデータ プロパティを検索および更新できます. |
| [TimeTagCategory](./timetagcategory) | ファイルのライフサイクルを記述するために使用されるメタデータ プロパティをマークするタグを提供します. タグは、ファイルまたは知的コンテンツが作成、編集、印刷などされた時点を扱います. |
| [ToolTagCategory](./tooltagcategory) | ファイルの作成に使用されたツール (ソフトウェアおよびハードウェア) に関連するメタデータ プロパティをマークするためのタグを提供します。 |
