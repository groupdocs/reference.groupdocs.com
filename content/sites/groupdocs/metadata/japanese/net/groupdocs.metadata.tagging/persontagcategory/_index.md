## PersonTagCategory class

ファイルまたは知的コンテンツの作成に貢献した人々に関する情報を保持するメタデータ プロパティをマークするタグを提供します。 これらのタグは、ドキュメントの作成者、編集者、または作業が実行されたクライアントを見つけるのに役立ちます. カテゴリの名前にもかかわらず、タグでマークされた一部のメタデータ プロパティには、個人名ではなく会社名が含まれる場合があります.

```csharp
public class PersonTagCategory : TagCategory
```
