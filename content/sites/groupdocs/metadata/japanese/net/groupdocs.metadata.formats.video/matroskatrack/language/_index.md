## MatroskaTrack.Language property

トラックの言語を Matroska 言語形式で取得します。 この要素は、[`LanguageIetf`](../languageietf)要素は同じ TrackEntry. で使用されています

```csharp
public string Language { get; }
```

### プロパティ値

Matroska 言語形式でのトラックの言語。

### 備考

言語コードは、3 文字の書誌 ISO-639-2 形式 (フランス語の「fre」など)、 、またはそのような言語コードの後にダッシュと言語の専門分野の国コード (「fre-ca」など) のいずれかです。カナダフランス語用）。 国コードは、インターネット ドメインに使用されるものと同じです。
