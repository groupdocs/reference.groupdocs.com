## Cms class

は、Cryptographic Message Syntax (CMS) (暗号で保護されたメッセージに対する IETF の標準) で作成されたデジタル署名を表します。 CMS は、RFC 5652 で指定されている PKCS #7 の構文に基づいています。 を参照してください。[https://tools.ietf.org/html/rfc5652](https://tools.ietf.org/html/rfc5652)詳しくは.

public class Cms : DigitalSignature
