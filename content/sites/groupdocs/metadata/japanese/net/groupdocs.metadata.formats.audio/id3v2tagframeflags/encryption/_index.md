## ID3V2TagFrameFlags.Encryption property

フレームが暗号化されているかどうかを示す値を取得します。 設定すると、暗号化された方法を示す 1 バイトがフレーム ヘッダーに追加されます。

```csharp
public bool Encryption { get; }
```

### プロパティ値

`真実`、フレームが暗号化されている場合。それ以外の場合は false。
