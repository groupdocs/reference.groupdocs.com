## DublinCorePackage.Rights property

ダブリン コア要素の権利を取得します。

```csharp
public string Rights { get; }
```

### プロパティ値

リソース内およびリソースに対して保持されている権利に関する情報。 通常、Rights 要素には、リソースに対する権利管理ステートメント が含まれるか、そのような情報を提供するサービスを参照します。権利情報 には、多くの場合、知的財産権 (IPR)、著作権、およびさまざまな財産権が含まれます。 権利要素が存在しない場合、リソースに関するこれらおよび 他の権利のステータスについて仮定を行うことはできません.
