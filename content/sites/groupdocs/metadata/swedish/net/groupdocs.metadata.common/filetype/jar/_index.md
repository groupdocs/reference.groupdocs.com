## FileType.JAR field

JAR är en Java Archive-fil som innehåller många olika programrelaterade filer som en enda fil. Detta filformat skapades för att minska hastigheten för att ladda en nedladdad Java-applet i webbläsaren via HTTP-transaktion, Läs mer om detta filformat [här](https://docs.fileformat.com/programming/jar/) .
