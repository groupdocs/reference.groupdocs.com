## ID3V2TagFrameFlags.DataLengthIndicator property

Får ett värde som indikerar om en datalängdsindikator har lagts till ramen. Datalängdsindikatorn är värdet man skulle skriva som 'Frame length' om alla ramformatsflaggor var nollställda, representerade som ett 32-bitars synchsafe heltal.

```csharp
public bool DataLengthIndicator { get; }
```

### Fastighetsvärde

`Sann`om en datalängdsindikator har lagts till i ramen; annars falskt.
