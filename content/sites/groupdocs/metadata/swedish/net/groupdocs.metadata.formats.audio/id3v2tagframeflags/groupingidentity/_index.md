## ID3V2TagFrameFlags.GroupingIdentity property

Får ett värde som indikerar om ramen tillhör en grupp av ramar. Om inställt läggs en gruppidentifierarbyte till i ramhuvudet. Varje ram med samma gruppidentifierare tillhör samma grupp.

```csharp
public bool GroupingIdentity { get; }
```

### Fastighetsvärde

`Sann`, om ramen tillhör en grupp av ramar; annars falskt.
