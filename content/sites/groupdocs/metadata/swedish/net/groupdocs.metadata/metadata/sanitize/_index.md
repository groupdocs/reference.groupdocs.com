## Metadata.Sanitize method

Tar bort skrivbara metadataegenskaper från alla upptäckta paket eller hela paket om möjligt. Operationen är rekursiv så den påverkar också alla kapslade paket.

```csharp
public int Sanitize()
```

### Returvärde

Antalet berörda fastigheter.

### Anmärkningar

**Läs mer**

* [Rensa metadata](https://docs.groupdocs.com/display/metadatanet/Clean+metadata)

### Exempel

Det här exemplet visar hur man tar bort alla upptäckta metadatapaket/egenskaper från en fil.

```csharp
using (Metadata metadata = new Metadata(Constants.InputPdf))
{
    // Ta bort upptäckta metadatapaket
    var affected = metadata.Sanitize();
    Console.WriteLine("Properties removed: {0}", affected);

    metadata.Save(Constants.OutputPdf);
}
```
