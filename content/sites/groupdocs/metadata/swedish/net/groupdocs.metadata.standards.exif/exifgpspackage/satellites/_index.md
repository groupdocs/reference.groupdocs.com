## ExifGpsPackage.Satellites property

Hämtar eller ställer in GPS-satelliterna som används för mätningar. Denna tagg kan användas för att beskriva antalet satelliter, deras ID-nummer, höjdvinkel, azimut, SNR och annan information i ASCII-notation. Formatet är inte specificerat. Om GPS-mottagaren inte kan göra mätningar ska värdet på taggen sättas till NULL.

public string Satellites { get; set; }
