## OpenTypeLicensingRights enumeration

Indikerar licensrättigheter för typsnittsinbäddning för teckensnittet.

[Flags]
public enum OpenTypeLicensingRights : ushort

### Värderingar

| namn | Värde | Beskrivning |
| --- | --- | --- |
| None | `0` | De odefinierade licensrättigheterna. |
| UsagePermissionsMask | `F` | Mask för användningsbehörigheter. |
| InstallableEmbedding | `1` | Installerbar inbäddning. Teckensnittet kan vara inbäddat och kan vara permanent installerat för användning på ett fjärrsystem eller för användning av andra användare. |
| RestrictedLicenseEmbedding | `2` | Begränsad licensinbäddning. Typsnittet får inte modifieras, bäddas in eller bytas ut på något sätt utan att först ha fått uttryckligt tillstånd från den juridiska ägaren. |
| PreviewAndPrintEmbedding | `4` | Förhandsgranska och skriv inbäddning. Teckensnittet kan vara inbäddat och kan laddas tillfälligt på andra system för att visa eller skriva ut dokumentet. Dokument som innehåller typsnitt för förhandsgranskning och utskrift måste öppnas "skrivskyddat"; inga redigeringar kan tillämpas på dokumentet. |
| EditableEmbedding | `8` | Redigerbar inbäddning. Teckensnittet kan vara inbäddat och kan laddas tillfälligt på andra system. Liksom med förhandsgranska och Skriv inbäddning kan dokument som innehåller redigerbara teckensnitt öppnas för läsning. Dessutom är redigering tillåten, inklusive möjligheten att formatera ny text med det inbäddade teckensnittet, och ändringar kan sparas. |
| NoSubsetting | `100` | Ingen underinställning. När den här biten är inställd kan det hända att teckensnittet inte är underställt före inbäddning. Andra inbäddningsbegränsningar specificerade i bitarna 0 till 3 och bit 9 gäller också. |
| BitmapEmbeddingOnly | `200` | Endast bitmappsinbäddning. När denna bit är inställd kan endast bitmappar som finns i teckensnittet bäddas in. Inga konturdata får bäddas in. Om det inte finns några tillgängliga bitmappar i teckensnittet anses teckensnittet vara obäddbart och inbäddningstjänsterna kommer att misslyckas. Andra inbäddningsbegränsningar specificerade i bitarna 0-3 och 8 gäller också. |
