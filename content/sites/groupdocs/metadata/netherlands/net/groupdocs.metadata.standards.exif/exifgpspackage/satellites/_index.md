## ExifGpsPackage.Satellites property

Ontvangt of stelt de GPS-satellieten in die voor metingen worden gebruikt. Deze tag kan worden gebruikt om het aantal satellieten, hun ID-nummer, elevatiehoek, azimut, SNR en andere informatie in ASCII-notatie te beschrijven. Het formaat is not gespecificeerd. Als de GPS-ontvanger geen metingen kan uitvoeren, wordt de waarde van de tag ingesteld op NULL.

```csharp
public string Satellites { get; set; }
```

### Eigendoms-waarde

De GPS-satellieten gebruikt voor metingen.
