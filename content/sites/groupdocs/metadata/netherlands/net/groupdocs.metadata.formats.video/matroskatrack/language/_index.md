## MatroskaTrack.Language property

Krijgt de taal van de track in de Matroska-taalvorm. Dit element MOET worden genegeerd als de[`LanguageIetf`](../languageietf) Element wordt gebruikt in dezelfde TrackEntry.

```csharp
public string Language { get; }
```

### Eigendoms-waarde

De taal van de track in het Matroska-talenformulier.

### Opmerkingen

Taalcodes kunnen de 3-letterige bibliografische ISO-639-2-vorm zijn (zoals "fre" voor Frans), of een dergelijke taalcode gevolgd door een streepje en een landcode voor specialiteiten in talen (zoals "fre-ca" voor Canadees Frans). Landcodes zijn dezelfde als voor internetdomeinen.
