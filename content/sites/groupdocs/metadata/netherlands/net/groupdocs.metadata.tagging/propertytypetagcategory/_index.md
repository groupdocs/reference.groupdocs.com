## PropertyTypeTagCategory class

Biedt tags die aanvullende informatie bevatten over het type eigenschap in plaats van over het doel ervan. Met behulp van deze tags kunt u metadata-eigenschappen detecteren die URL-links naar externe bronnen bevatten, eigenschappen die lettertypen, kleuren, geolocatie enzovoort beschrijven.
