## PersonTagCategory class

Biedt tags die metadata-eigenschappen markeren die informatie bevatten over de mensen die hebben bijgedragen aan het maken van bestanden of intellectuele inhoud. Deze tags kunnen u helpen de maker van het document, de editor of zelfs de klant te vinden voor wie het werk is uitgevoerd. Ondanks de naam van de categorie kunnen sommige metadata-eigenschappen gemarkeerd met de tags een bedrijfsnaam bevatten in plaats van de naam van een persoon.

```csharp
public class PersonTagCategory : TagCategory
```
