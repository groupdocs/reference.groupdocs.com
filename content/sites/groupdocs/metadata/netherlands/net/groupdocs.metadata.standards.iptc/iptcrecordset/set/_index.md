## IptcRecordSet.Set method

Voegt de gespecificeerde dataSet toe aan of werkt deze bij in het juiste record.

```csharp
public void Set(IptcDataSet dataSet)
```

| Parameter | Type | Beschrijving |

| dataSet | IptcDataSet | De IPTC dataSet om toe te voegen/bij te werken. |

### Voorbeelden

Dit voorbeeld laat zien hoe aangepaste IPTC-datasets in een bestand kunnen worden toegevoegd of bijgewerkt.

```csharp
using (Metadata metadata = new Metadata(Constants.PsdWithIptc))
{
    IIptc root = metadata.GetRootPackage() as IIptc;
    if (root != null)
    {
        // Stel het IPTC-pakket in als het ontbreekt
        if (root.IptcPackage == null)
        {
            root.IptcPackage = new IptcRecordSet();
        }

        // Voeg een know-eigenschap toe met behulp van de DataSet API
        root.IptcPackage.Set(new IptcDataSet((byte) IptcRecordType.ApplicationRecord, (byte) IptcApplicationRecordDataSet.BylineTitle, "test code sample"));

        // Voeg een aangepaste IPTC-gegevensset toe
        root.IptcPackage.Set(new IptcDataSet(255, 255, new byte[] { 1, 2, 3 }));

        metadata.Save(Constants.OutputPsd);
    }
}
```
