## ID3V2TagFrameFlags.DataLengthIndicator property

获取一个值，该值指示数据长度指示符是否已添加到 帧。如果所有帧格式标志都 归零，则数据长度指示符是一个将 写为“帧长度”的值，表示为32位同步安全整数。

```csharp
public bool DataLengthIndicator { get; }
```

### 适当的价值

`真的`如果数据长度指示符已添加到帧中；否则为假。
