## MetadataPackage.UpdateProperties method

更新满足指定谓词的已知元数据属性。 该操作是递归的，因此它也会影响所有嵌套包。

```csharp
public int UpdateProperties(Func<MetadataProperty, bool> predicate, PropertyValue value)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| predicate | Func`2 | 用于测试每个元数据属性的条件的函数。 |
| value | PropertyValue | 过滤属性的新值。 |

### 返回值

受影响属性的数量。

### 评论

请注意 GroupDocs.Metadata 隐式检查每个过滤属性的类型。 无法使用具有不适当类型的值更新属性。
