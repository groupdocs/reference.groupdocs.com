## XmpLangAlt class

表示 XMP 语言替代。

简单文本项的替代数组。语言选项有助于根据所需语言选择简单文本 item 。每个数组项都应有一个 xml:lang 限定符。每个 xml:lang 值在项目中应 唯一。

public sealed class XmpLangAlt : XmpArray
