## PersonTagCategory class

提供标记元数据属性的标签，其中包含有关为文件或知识内容创建做出贡献的人员的信息。 这些标签可以帮助您找到文档创建者、编辑者，甚至是为其执行工作的客户。 尽管有类别名称，但一些标有标签的元数据属性可以包含公司名称而不是个人姓名。

```csharp
public class PersonTagCategory : TagCategory
```
