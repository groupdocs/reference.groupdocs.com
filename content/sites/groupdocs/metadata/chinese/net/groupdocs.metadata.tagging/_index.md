命名空间包含各种标签集，其中最重要的元数据属性被标记。 标签允许用户在不同的包中查找和更新元数据属性，而不管元数据标准和文件格式如何。

## 课程

| 班级 | 描述 |
| --- | --- |
| [ContentTagCategory](./contenttagcategory) | 提供附加到描述文件内容的元数据属性的标签。 标签可用于查找内容语言、类型（流派）、主题、评级等。 |
| [CorporateTagCategory](./corporatetagcategory) | 提供用于标记与参与文件创建的公司相关的元数据属性的标签。 |
| [DocumentTagCategory](./documenttagcategory) | 提供仅应用于文档特定属性的标签。 标记可用于确定属性是从 office 文档的哪个部分提取的。 |
| [LegalTagCategory](./legaltagcategory) | 提供附加到元数据属性的标签，其中包含有关文件内容所有者的信息 以及可以使用该内容的规则。 |
| [OriginTagCategory](./origintagcategory) | 提供帮助用户确定文件来源（例如模板或其他来源）的标签。 |
| [PersonTagCategory](./persontagcategory) | 提供标记元数据属性的标签，这些元数据属性包含有关为文件或知识内容创建做出贡献的人员的信息。 这些标签可以帮助您找到文档创建者、编辑者，甚至是为其执行工作的客户。 尽管有类别名称，但一些标有标签的元数据属性可以包含公司名称而不是人名。 |
| [PropertyTag](./propertytag) | 表示用于标记元数据属性的标签。 |
| [PropertyTypeTagCategory](./propertytypetagcategory) | 提供包含有关属性类型而非其用途的附加信息的标签。 使用这些标签，您可以检测包含指向外部资源的 URL 链接的元数据属性， 描述字体、颜色、地理位置等的属性。 |
| [TagCategory](./tagcategory) | 表示一组具有某些共享特征的标签。 |
| [Tags](./tags) | 包含标记最重要的元数据属性的各种标签集。 标签允许您在不同的包中查找和更新元数据属性，而不管元数据标准和文件格式如何。 |
| [TimeTagCategory](./timetagcategory) | 提供标记用于描述文件生命周期的元数据属性的标签。 标签处理文件或知识内容创建、编辑、打印等的时间点。 |
| [ToolTagCategory](./tooltagcategory) | 提供用于标记与用于创建文件的工具（软件和硬件）相关的元数据属性的标签。 |
