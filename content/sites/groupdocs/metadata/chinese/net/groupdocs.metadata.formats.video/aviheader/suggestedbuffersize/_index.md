## AviHeader.SuggestedBufferSize property

获取建议的用于读取文件的缓冲区大小。  通常，这个大小应该足够大以包含文件中最大的块。 如果设置为零，或者太小，播放软件将不得不在播放过程中重新分配内存，这会降低性能。对于交错文件， 缓冲区大小应该足以读取整个记录，而不仅仅是一个块。

```csharp
public int SuggestedBufferSize { get; }
```
