## OpenTypeLicensingRights enumeration

表示字体的字体嵌入许可权。

```csharp
[Flags]
public enum OpenTypeLicensingRights : ushort
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| None | `0` | 未定义的许可权。 |
| UsagePermissionsMask | `F` | 使用权限掩码。 |
| InstallableEmbedding | `1` | 可安装嵌入。 字体可以嵌入，并且可以永久安装以供在远程系统上使用，或供其他用户使用。 |
| RestrictedLicenseEmbedding | `2` | 受限许可嵌入。 未经合法所有者明确许可，不得以任何方式修改、嵌入或交换字体。 |
| PreviewAndPrintEmbedding | `4` | 预览和打印嵌入。 字体可以嵌入，并且可以临时加载到其他系统上以查看或打印文档。 包含预览和打印字体的文档必须以“只读”方式打开；不能对文档应用任何编辑。 |
| EditableEmbedding | `8` | 可编辑嵌入。 字体可能是嵌入的，并且可能会临时加载到其他系统上。 与预览和打印嵌入一样，可以打开包含可编辑字体的文档以供阅读。 此外，允许编辑，包括使用嵌入字体格式化新文本的能力，并且可以保存更改。 |
| NoSubsetting | `100` | 无子集。 设置此位时，字体在嵌入之前可能不会被子集。位 0 到 3 和位 9 中指定的其他嵌入限制也适用。 |
| BitmapEmbeddingOnly | `200` | 仅嵌入位图。 设置此位时，只能嵌入字体中包含的位图。不得嵌入大纲数据。 如果字体中没有可用的位图，则认为字体不可嵌入，嵌入服务将失败。 位 0-3 和 8 中指定的其他嵌入限制也适用。 |
