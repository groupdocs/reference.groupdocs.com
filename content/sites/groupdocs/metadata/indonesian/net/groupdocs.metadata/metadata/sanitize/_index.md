## Metadata.Sanitize method

Menghapus properti metadata yang dapat ditulisi dari semua paket yang terdeteksi atau seluruh paket jika memungkinkan. Operasi bersifat rekursif sehingga memengaruhi semua paket bersarang juga.

```csharp
public int Sanitize()
```

### Nilai Pengembalian

Jumlah properti yang terpengaruh.

### Perkataan

**Belajarlah lagi**

* [Bersihkan metadata](https://docs.groupdocs.com/display/metadatanet/Clean+metadata)

### Contoh

Contoh ini menunjukkan cara menghapus semua paket/properti metadata yang terdeteksi dari sebuah file.

```csharp
using (Metadata metadata = new Metadata(Constants.InputPdf))
{
    // Hapus paket metadata yang terdeteksi
    var affected = metadata.Sanitize();
    Console.WriteLine("Properties removed: {0}", affected);

    metadata.Save(Constants.OutputPdf);
}
```
