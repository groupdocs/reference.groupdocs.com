## FileType.HEIF field

File HEIF adalah format file Gambar Kontainer Efisiensi Tinggi yang mampu menyimpan gambar tunggal atau urutan gambar dalam satu file. Format file HEIF mengompresi gambar menggunakan standar Pengodean Video Efisiensi Tinggi (HEVC). Pelajari lebih lanjut tentang format file ini [Di Sini](https://docs.fileformat.com/image/heif/) .
