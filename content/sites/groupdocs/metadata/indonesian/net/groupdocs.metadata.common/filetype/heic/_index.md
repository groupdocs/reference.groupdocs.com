## FileType.HEIC field

File HEIC adalah format file Gambar Kontainer Efisiensi Tinggi yang dapat menyimpan banyak gambar sebagai kumpulan dalam satu file. HEIC, seperti HEIF, dikompresi menggunakan standar Pengkodean Video Efisiensi Tinggi (HEVC) dan berukuran lebih kecil tanpa mengorbankan kualitas. Pelajari lebih lanjut tentang format file ini [Di Sini](https://docs.fileformat.com/image/heic/) .
