## PersonTagCategory class

Memberikan tag yang menandai properti metadata yang menyimpan informasi tentang orang yang berkontribusi pada pembuatan file atau konten intelektual. Tag ini dapat membantu Anda menemukan pembuat dokumen, editor, atau bahkan klien untuk siapa pekerjaan itu dilakukan. Terlepas dari nama kategori, beberapa properti metadata yang ditandai dengan tag dapat berisi nama perusahaan, bukan nama orang.

```csharp
public class PersonTagCategory : TagCategory
```
