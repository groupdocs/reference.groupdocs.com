Namespace berisi berbagai kumpulan tag yang menandai properti metadata paling penting. Tag memungkinkan pengguna menemukan dan memperbarui properti metadata dalam paket yang berbeda terlepas dari standar metadata dan format file.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [ContentTagCategory](./contenttagcategory) | Memberikan tag yang dilampirkan ke properti metadata yang mendeskripsikan konten file. Tag berguna untuk mengetahui konten bahasa, jenis (genre), subject, rating, dll |
| [CorporateTagCategory](./corporatetagcategory) | Menyediakan tag yang dimaksudkan untuk menandai properti metadata yang terkait dengan perusahaan yang berpartisipasi dalam pembuatan file. |
| [DocumentTagCategory](./documenttagcategory) | Memberikan tag yang hanya diterapkan pada properti khusus dokumen. Tag dapat berguna untuk menentukan dari bagian dokumen kantor mana sebuah properti diekstrak. |
| [LegalTagCategory](./legaltagcategory) | Menyediakan tag yang dilampirkan ke properti metadata yang menyimpan informasi tentang pemilik konten file dan aturan penggunaan konten tersebut. |
| [OriginTagCategory](./origintagcategory) | Memberikan tag yang membantu pengguna untuk menentukan asal file (mis. template atau sumber lain). |
| [PersonTagCategory](./persontagcategory) | Memberikan tag yang menandai properti metadata yang menyimpan informasi tentang orang yang berkontribusi pada pembuatan file atau konten intelektual. Tag ini dapat membantu Anda menemukan pembuat dokumen, editor, atau bahkan klien untuk siapa pekerjaan itu dilakukan. Terlepas dari nama kategori, beberapa properti metadata yang ditandai dengan tag dapat berisi nama perusahaan, bukan nama orang. |
| [PropertyTag](./propertytag) | Merupakan tag yang digunakan untuk menandai properti metadata. |
| [PropertyTypeTagCategory](./propertytypetagcategory) | Memberikan tag yang berisi informasi tambahan tentang jenis properti, bukan tentang tujuannya. Dengan menggunakan tag ini, Anda dapat mendeteksi properti metadata yang berisi tautan URL ke sumber daya eksternal, properti yang mendeskripsikan font, warna, geolokasi, dan sebagainya. |
| [TagCategory](./tagcategory) | Mewakili sekumpulan tag yang memiliki karakteristik yang sama. |
| [Tags](./tags) | Berisi berbagai rangkaian tag yang menandai properti metadata paling penting. Tag memungkinkan Anda menemukan dan memperbarui properti metadata dalam paket yang berbeda terlepas dari standar metadata dan format file. |
| [TimeTagCategory](./timetagcategory) | Menyediakan tag yang menandai properti metadata yang digunakan untuk menjelaskan daur hidup file. Tag berhubungan dengan titik waktu saat file atau konten intelektual dibuat, diedit, dicetak, dll. |
| [ToolTagCategory](./tooltagcategory) | Memberikan tag yang dimaksudkan untuk menandai properti metadata yang terkait dengan alat (perangkat lunak dan perangkat keras) yang digunakan untuk membuat file. |
