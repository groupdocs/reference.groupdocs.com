## ID3V2TagFrameFlags.DataLengthIndicator property

Mendapat nilai yang menunjukkan apakah indikator panjang data telah ditambahkan ke bingkai. Indikator panjang data adalah nilai yang akan ditulis sebagai 'Panjang bingkai' jika semua bendera format bingkai adalah dinolkan, direpresentasikan sebagai integer synchsafe 32 bit.

```csharp
public bool DataLengthIndicator { get; }
```

### Nilai properti

`BENAR`jika indikator panjang data telah ditambahkan ke bingkai; jika tidak, salah.
