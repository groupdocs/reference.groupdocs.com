## MP3RootPackage.ID3V1 property

Mendapatkan atau menyetel tag metadata ID3v1. Temukan informasi lebih lanjut di[http://id3.org/ID3v1](http://id3.org/ID3v1) .

```csharp
public ID3V1Tag ID3V1 { get; set; }
```

### Nilai properti

Tag metadata ID3v1 yang dilampirkan pada file audio.

### Perkataan

Tag ID3(v1) adalah bagian kecil dari data tambahan di akhir file MP3.

### Contoh

Contoh kode ini menunjukkan cara memperbarui tag ID3v1 dalam file MP3.

```csharp
using (Metadata metadata = new Metadata(Constants.MP3WithID3V1))
{
    var root = metadata.GetRootPackage<MP3RootPackage>();

    if (root.ID3V1 == null)
    {
        root.ID3V1 = new ID3V1Tag();
    }

    root.ID3V1.Album = "test album";
    root.ID3V1.Artist = "test artist";
    root.ID3V1.Title = "test title";
    root.ID3V1.Comment = "test comment";
    root.ID3V1.Year = "2019";

    // ...

    metadata.Save(Constants.OutputMp3);
}
```
