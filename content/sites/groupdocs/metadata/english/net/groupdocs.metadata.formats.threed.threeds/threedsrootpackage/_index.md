### Examples

The following code snippet shows how to get metadata from a 3D file.

```csharp
Encoding encoding = Encoding.GetEncoding(866);

using (Metadata metadata = new Metadata(Constants.InputThreeDS))
{
    var root = metadata.GetRootPackage<ThreeDSPackage>();

    Console.WriteLine(root.Name);

    foreach (var node in root.ThreeDSPackage.Nodes)
    {
        Console.WriteLine(node.Name);
    }
}
```
