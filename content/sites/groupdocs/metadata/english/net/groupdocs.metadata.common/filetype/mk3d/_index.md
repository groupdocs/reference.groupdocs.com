## FileType.MK3D field

MK3D is actually stereoscopic 3D video created using Matroska 3D format. The MKV file container uses a StereoMode field’s value to define the type of stereoscopic 3D video stuff. Learn more about this file format [here](https://docs.fileformat.com/video/mk3d/).
