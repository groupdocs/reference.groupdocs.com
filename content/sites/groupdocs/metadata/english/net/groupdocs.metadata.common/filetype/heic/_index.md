## FileType.HEIC field

An HEIC file is a High-Efficiency Container Image file format that can store multiple images as a collection in a single file. HEIC, like HEIF, are compressed using the High Efficiency Video Coding (HEVC) standard and are smaller in size without compromising the quality. Learn more about this file format [here](https://docs.fileformat.com/image/heic/).
