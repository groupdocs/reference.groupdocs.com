## ImageResourceBlock class

Represents a Photoshop Image Resource block.  Image resource blocks are the basic building unit of several file formats, including Photoshop's native file format, JPEG, and TIFF. Image resources are used to store non-pixel data associated with images, such as pen tool paths.
