Ο χώρος ονομάτων περιέχει διάφορα σύνολα ετικετών με τα οποία επισημαίνονται οι πιο σημαντικές ιδιότητες μεταδεδομένων. Οι ετικέτες επιτρέπουν στους χρήστες να βρίσκουν και να ενημερώνουν τις ιδιότητες μεταδεδομένων σε διαφορετικά πακέτα ανεξάρτητα από το πρότυπο μεταδεδομένων και τη μορφή αρχείου.

## Τάξεις

| Τάξη | Περιγραφή |
| --- | --- |
| [ContentTagCategory](./contenttagcategory) | Παρέχει ετικέτες που είναι προσαρτημένες σε ιδιότητες μεταδεδομένων που περιγράφουν το περιεχόμενο ενός αρχείου. Οι ετικέτες είναι χρήσιμες για να μάθετε τη γλώσσα περιεχομένου, τον τύπο (είδος), το θέμα, τη βαθμολογία κ.λπ. |
| [CorporateTagCategory](./corporatetagcategory) | Παρέχει ετικέτες που προορίζονται για την επισήμανση ιδιοτήτων μεταδεδομένων που σχετίζονται με μια εταιρεία που συμμετείχε στη δημιουργία αρχείων. |
| [DocumentTagCategory](./documenttagcategory) | Παρέχει ετικέτες που εφαρμόζονται μόνο σε ιδιότητες συγκεκριμένων εγγράφων. Οι ετικέτες μπορούν να είναι χρήσιμες για τον προσδιορισμό από ποιο τμήμα ενός εγγράφου γραφείου εξήχθη μια ιδιότητα. |
| [LegalTagCategory](./legaltagcategory) | Παρέχει ετικέτες που είναι προσαρτημένες σε ιδιότητες μεταδεδομένων που περιέχουν πληροφορίες σχετικά με τους κατόχους του περιεχομένου του αρχείου και τους κανόνες βάσει των οποίων μπορεί να χρησιμοποιηθεί το περιεχόμενο. |
| [OriginTagCategory](./origintagcategory) | Παρέχει ετικέτες που βοηθούν έναν χρήστη να προσδιορίσει την προέλευση ενός αρχείου (π.χ. πρότυπο ή άλλη πηγή). |
| [PersonTagCategory](./persontagcategory) | Παρέχει ετικέτες που επισημαίνουν ιδιότητες μεταδεδομένων που περιέχουν πληροφορίες σχετικά με τα άτομα που συνέβαλαν στη δημιουργία αρχείων ή πνευματικού περιεχομένου. Αυτές οι ετικέτες μπορούν να σας βοηθήσουν να βρείτε τον δημιουργό εγγράφων, τον επεξεργαστή ή ακόμα και τον πελάτη για τον οποίο εκτελέστηκε η εργασία. Παρά το όνομα της κατηγορίας, ορισμένες ιδιότητες μεταδεδομένων που επισημαίνονται με τις ετικέτες μπορεί να περιέχουν ένα όνομα εταιρείας και όχι το όνομα ενός ατόμου. |
| [PropertyTag](./propertytag) | Αντιπροσωπεύει μια ετικέτα που χρησιμοποιείται για την επισήμανση ιδιοτήτων μεταδεδομένων. |
| [PropertyTypeTagCategory](./propertytypetagcategory) | Παρέχει ετικέτες που φέρουν πρόσθετες πληροφορίες για τον τύπο μιας ιδιότητας και όχι για το σκοπό της. Χρησιμοποιώντας αυτές τις ετικέτες μπορείτε να εντοπίσετε ιδιότητες μεταδεδομένων που περιέχουν συνδέσμους URL προς εξωτερικούς πόρους, ιδιότητες που περιγράφουν γραμματοσειρές, χρώματα, γεωγραφική θέση κ.λπ. |
| [TagCategory](./tagcategory) | Αντιπροσωπεύει ένα σύνολο ετικετών με ορισμένα κοινά χαρακτηριστικά. |
| [Tags](./tags) | Περιέχει διάφορα σύνολα ετικετών με τα οποία επισημαίνονται οι πιο σημαντικές ιδιότητες μεταδεδομένων. Οι ετικέτες σάς επιτρέπουν να βρείτε και να ενημερώσετε τις ιδιότητες μεταδεδομένων σε διαφορετικά πακέτα, ανεξάρτητα από το πρότυπο μεταδεδομένων και τη μορφή αρχείου. |
| [TimeTagCategory](./timetagcategory) | Παρέχει ετικέτες που επισημαίνουν ιδιότητες μεταδεδομένων που χρησιμοποιούνται για την περιγραφή του κύκλου ζωής ενός αρχείου. Οι ετικέτες ασχολούνται με χρονικά σημεία κατά την οποία δημιουργήθηκε, επεξεργάστηκε, εκτυπώθηκε κ.λπ. ένα αρχείο ή πνευματικό περιεχόμενο. |
| [ToolTagCategory](./tooltagcategory) | Παρέχει ετικέτες που προορίζονται για την επισήμανση ιδιοτήτων μεταδεδομένων που σχετίζονται με τα εργαλεία (λογισμικό και υλικό) που χρησιμοποιήθηκαν για τη δημιουργία ενός αρχείου. |
