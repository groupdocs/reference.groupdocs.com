## PersonTagCategory class

Παρέχει ετικέτες που επισημαίνουν ιδιότητες μεταδεδομένων που περιέχουν πληροφορίες σχετικά με τα άτομα που συνέβαλαν στη δημιουργία αρχείων ή πνευματικού περιεχομένου. Αυτές οι ετικέτες μπορούν να σας βοηθήσουν να βρείτε τον δημιουργό εγγράφων, τον επεξεργαστή ή ακόμα και τον πελάτη για τον οποίο εκτελέστηκε η εργασία. Παρά το όνομα της κατηγορίας, ορισμένες ιδιότητες μεταδεδομένων που επισημαίνονται με τις ετικέτες μπορεί να περιέχουν ένα όνομα εταιρείας και όχι το όνομα ενός ατόμου.

```csharp
public class PersonTagCategory : TagCategory
```
