## PropertyTypeTagCategory class

Παρέχει ετικέτες που φέρουν πρόσθετες πληροφορίες για τον τύπο μιας ιδιότητας και όχι για το σκοπό της. Χρησιμοποιώντας αυτές τις ετικέτες μπορείτε να εντοπίσετε ιδιότητες μεταδεδομένων που περιέχουν συνδέσμους URL προς εξωτερικούς πόρους, ιδιότητες που περιγράφουν γραμματοσειρές, χρώματα, γεωγραφική θέση κ.λπ.
