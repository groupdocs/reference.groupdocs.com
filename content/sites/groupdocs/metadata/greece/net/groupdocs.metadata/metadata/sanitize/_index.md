## Metadata.Sanitize method

Καταργεί τις εγγράψιμες ιδιότητες μεταδεδομένων από όλα τα πακέτα που ανιχνεύονται ή ολόκληρα πακέτα, εάν είναι δυνατόν. Η λειτουργία είναι αναδρομική, επομένως επηρεάζει επίσης όλα τα ένθετα πακέτα.

```csharp
public int Sanitize()
```

### Παραδείγματα

Αυτό το παράδειγμα δείχνει πώς να αφαιρέσετε όλα τα πακέτα/ιδιότητες μεταδεδομένων που εντοπίστηκαν από ένα αρχείο.

```csharp
using (Metadata metadata = new Metadata(Constants.InputPdf))
{
    // Κατάργηση πακέτων μεταδεδομένων που εντοπίστηκαν
    var affected = metadata.Sanitize();
    Console.WriteLine("Properties removed: {0}", affected);

    metadata.Save(Constants.OutputPdf);
}
```
