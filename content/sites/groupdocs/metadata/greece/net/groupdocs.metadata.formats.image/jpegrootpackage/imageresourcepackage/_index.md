## JpegRootPackage.ImageResourcePackage property

Λαμβάνει το πακέτο μεταδεδομένων του Photoshop Image Resource. Τα μπλοκ πόρων εικόνας είναι η βασική δομική μονάδα της εγγενούς μορφής αρχείου Photoshop.

```csharp
public ImageResourcePackage ImageResourcePackage { get; }
```

### Αξία περιουσίας

Το πακέτο μεταδεδομένων πόρων εικόνας.

* [Εργασία με μεταδεδομένα σε εικόνες JPEG](https://docs.groupdocs.com/display/metadatanet/Working+with+metadata+in+JPEG+images)

### Παραδείγματα

Το παρακάτω δείγμα κώδικα δείχνει πώς να εξαγάγετε μπλοκ πόρων εικόνας (δομικά στοιχεία της μορφής αρχείου Photoshop) από μια εικόνα JPEG.

```csharp
using (Metadata metadata = new Metadata(Constants.JpegWithIrb))
{
    var root = metadata.GetRootPackage<JpegRootPackage>();

    if (root.ImageResourcePackage != null)
    {
        foreach (var block in root.ImageResourcePackage.ToList())
        {
            Console.WriteLine(block.Signature);
            Console.WriteLine(block.ID);
            Console.WriteLine(block.Name);
            Console.WriteLine(block.Data);
        }
    }
}
```
