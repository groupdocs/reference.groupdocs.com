### Παραδείγματα

Το ακόλουθο δείγμα κώδικα δείχνει πώς να ενημερώσετε τις ενσωματωμένες ιδιότητες μεταδεδομένων σε ένα έγγραφο διαγράμματος.

```csharp
using (Metadata metadata = new Metadata(Constants.InputVdx))
{
    var root = metadata.GetRootPackage<DiagramRootPackage>();

    root.DocumentProperties.Creator = "test author";
    root.DocumentProperties.TimeCreated = DateTime.Now;
    root.DocumentProperties.Company = "GroupDocs";
    root.DocumentProperties.Category = "test category";
    root.DocumentProperties.Keywords = "metadata, built-in, update";

    //... 

    metadata.Save(Constants.OutputVdx);
}
```
