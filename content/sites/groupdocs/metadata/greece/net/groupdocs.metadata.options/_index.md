Ο χώρος ονομάτων περιέχει όλες τις κλάσεις που αντιπροσωπεύουν ρυθμίσεις που επιτρέπουν στους χρήστες να ελέγχουν την επεξεργασία αρχείων και μεταδεδομένων.

## Τάξεις

| Τάξη | Περιγραφή |
| --- | --- |
| [CreatePageStream](./createpagestream) | Αντιπροσωπεύει μια μέθοδο που επιστρέφει μια ροή για την εγγραφή δεδομένων προεπισκόπησης σελίδας. |
| [LoadOptions](./loadoptions) | Επιτρέπει σε έναν προγραμματιστή να καθορίσει πρόσθετες επιλογές (όπως κωδικό πρόσβασης) κατά τη φόρτωση ενός αρχείου. |
| [PreviewOptions](./previewoptions) | Παρέχει επιλογές για ορισμό απαιτήσεων και ροή εκπροσώπων για δημιουργία προεπισκόπησης. |
| [ReleasePageStream](./releasepagestream) | Αντιπροσωπεύει μια μέθοδο που απελευθερώνει τη ροή που δημιουργήθηκε από το[`CreatePageStream`](../groupdocs.metadata.options/createpagestream) αντιπρόσωπος. |
