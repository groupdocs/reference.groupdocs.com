## ExifGpsPackage.Satellites property

Λαμβάνει ή ορίζει τους δορυφόρους GPS που χρησιμοποιούνται για μετρήσεις. Αυτή η ετικέτα μπορεί να χρησιμοποιηθεί για να περιγράψει τον αριθμό των δορυφόρων, τον αριθμό ID τους, τη γωνία ανύψωσης, το αζιμούθιο, το SNR και άλλες πληροφορίες σε συμβολισμό ASCII. Η μορφή δεν έχει καθοριστεί . Εάν ο δέκτης GPS δεν μπορεί να πραγματοποιήσει μετρήσεις, η τιμή της ετικέτας θα οριστεί σε NULL.

```csharp
public string Satellites { get; set; }
```
