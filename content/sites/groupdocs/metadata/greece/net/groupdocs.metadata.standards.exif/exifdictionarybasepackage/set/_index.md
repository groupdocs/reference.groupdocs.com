### Παραδείγματα

Αυτό το δείγμα κώδικα δείχνει πώς να προσθέσετε μια προσαρμοσμένη ετικέτα σε ένα πακέτο EXIF.

```csharp
using (Metadata metadata = new Metadata(Constants.TiffWithExif))
{
    IExif root = metadata.GetRootPackage() as IExif;
    if (root != null)
    {
        // Ορίστε το πακέτο EXIF εάν λείπει
        if (root.ExifPackage == null)
        {
            root.ExifPackage = new ExifPackage();
        }

        // Προσθήκη γνωστής ιδιότητας
        root.ExifPackage.Set(new TiffAsciiTag(TiffTagID.Artist, "test artist"));

        // Προσθέστε μια πλήρως προσαρμοσμένη ιδιότητα (η οποία δεν περιγράφεται στην προδιαγραφή EXIF).
        // Λάβετε υπόψη ότι το επιλεγμένο αναγνωριστικό μπορεί να τέμνεται με τα αναγνωριστικά που χρησιμοποιούνται από ορισμένα εργαλεία τρίτων.
        root.ExifPackage.Set(new TiffAsciiTag((TiffTagID)65523, "custom"));

        metadata.Save(Constants.OutputTiff);
    }
}
```
