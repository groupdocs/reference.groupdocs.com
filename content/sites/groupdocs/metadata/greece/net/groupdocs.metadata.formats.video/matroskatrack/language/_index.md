## MatroskaTrack.Language property

Λαμβάνει τη γλώσσα του κομματιού στη φόρμα γλωσσών Matroska. Αυτό το στοιχείο ΠΡΕΠΕΙ να αγνοηθεί εάν[`LanguageIetf`](../languageietf) Το στοιχείο χρησιμοποιείται στο ίδιο TrackEntry.

```csharp
public string Language { get; }
```

### Αξία περιουσίας

Η γλώσσα του κομματιού στη φόρμα των γλωσσών Matroska.

### Παρατηρήσεις

Οι κωδικοί γλώσσας μπορεί να είναι είτε η βιβλιογραφική μορφή ISO-639-2 3 γραμμάτων (όπως "fre" για τα γαλλικά), είτε ένας τέτοιος κωδικός γλώσσας ακολουθούμενος από μια παύλα και έναν κωδικό χώρας για ειδικότητες σε γλώσσες (όπως "fre-ca" για καναδικά γαλλικά). Οι κωδικοί χώρας είναι οι ίδιοι με αυτούς που χρησιμοποιούνται για τους τομείς Διαδικτύου.
