# GroupDocs.Metadata — API Reference Source Files

This repository holds the Markdown source files used to generate the **GroupDocs.Metadata**
API reference, published at <https://reference.groupdocs.com/metadata/>.

## About GroupDocs.Metadata

[GroupDocs.Metadata](https://products.groupdocs.com/metadata/) is a developer API for document
processing: read, edit, and remove metadata in documents, images, audio, and video. It is available for the following platforms, each with its own
API reference:

- **.NET** — <https://reference.groupdocs.com/metadata/net/>
- **Java** — <https://reference.groupdocs.com/metadata/java/>
- **Node.js via Java** — <https://reference.groupdocs.com/metadata/nodejs-java/>
- **Python via .NET** — <https://reference.groupdocs.com/metadata/python-net/>

## Repository purpose

The files in this repository are used to:

- Build and maintain the official **GroupDocs.Metadata** API reference.
- Keep the reference in sync with the latest product release.
- Provide a consistent structure for automated Hugo builds and publication.

Content is English-only and compiled into a static site with [Hugo](https://gohugo.io/).
