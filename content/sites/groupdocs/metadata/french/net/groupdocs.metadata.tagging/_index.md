L'espace de noms contient divers ensembles de balises avec lesquelles les propriétés de métadonnées les plus importantes sont marquées. Les balises permettent aux utilisateurs de rechercher et de mettre à jour les propriétés des métadonnées dans différents packages, quels que soient la norme de métadonnées et le format de fichier.

## Des classes

| Classer | La description |
| --- | --- |
| [ContentTagCategory](./contenttagcategory) | Fournit des balises qui sont attachées aux propriétés de métadonnées décrivant le contenu d'un fichier. Les balises sont utiles pour connaître la langue du contenu, le type (genre), le sujet, la note, etc. |
| [CorporateTagCategory](./corporatetagcategory) | Fournit des balises destinées à marquer les propriétés de métadonnées liées à une entreprise qui a participé à la création du fichier. |
| [DocumentTagCategory](./documenttagcategory) | Fournit des balises qui sont appliquées uniquement aux propriétés spécifiques au document. Les balises peuvent être utiles pour déterminer de quelle partie d'un document bureautique une propriété a été extraite. |
| [LegalTagCategory](./legaltagcategory) | Fournit des balises attachées aux propriétés des métadonnées contenant des informations sur les propriétaires du contenu du fichier et les règles selon lesquelles le contenu peut être utilisé. |
| [OriginTagCategory](./origintagcategory) | Fournit des balises qui aident un utilisateur à déterminer l'origine d'un fichier (par exemple, un modèle ou une autre source). |
| [PersonTagCategory](./persontagcategory) | Fournit des balises qui marquent les propriétés des métadonnées contenant des informations sur les personnes ayant contribué à la création de fichiers ou de contenu intellectuel. Ces balises peuvent vous aider à trouver le créateur du document, l'éditeur ou même le client pour lequel le travail a été effectué. Malgré le nom de la catégorie, certaines propriétés de métadonnées marquées avec les balises peuvent contenir un nom de société plutôt que le nom d'une personne. |
| [PropertyTag](./propertytag) | Représente une balise utilisée pour marquer les propriétés des métadonnées. |
| [PropertyTypeTagCategory](./propertytypetagcategory) | Fournit des balises qui contiennent des informations supplémentaires sur le type d'une propriété plutôt que sur son objectif. En utilisant ces balises, vous pouvez détecter les propriétés de métadonnées qui contiennent des liens URL vers des ressources externes, des propriétés décrivant les polices, les couleurs, la géolocalisation, etc. |
| [TagCategory](./tagcategory) | Représente un ensemble de balises ayant des caractéristiques communes. |
| [Tags](./tags) | Contient divers ensembles de balises avec lesquelles les propriétés de métadonnées les plus importantes sont marquées. Les balises vous permettent de rechercher et de mettre à jour les propriétés des métadonnées dans différents packages, quels que soient la norme de métadonnées et le format de fichier. |
| [TimeTagCategory](./timetagcategory) | Fournit des balises qui marquent les propriétés des métadonnées utilisées pour décrire le cycle de vie d'un fichier. Les balises traitent des moments où un fichier ou un contenu intellectuel a été créé, modifié, imprimé, etc. |
| [ToolTagCategory](./tooltagcategory) | Fournit des balises destinées à marquer les propriétés des métadonnées liées aux outils (logiciels et matériels) qui ont été utilisés pour créer un fichier. |
