## PropertyTypeTagCategory class

Fournit des balises qui contiennent des informations supplémentaires sur le type d'une propriété plutôt que sur son objectif. En utilisant ces balises, vous pouvez détecter les propriétés de métadonnées qui contiennent des liens URL vers des ressources externes, des propriétés décrivant les polices, les couleurs, la géolocalisation, etc.

```csharp
public class PropertyTypeTagCategory : TagCategory
```
