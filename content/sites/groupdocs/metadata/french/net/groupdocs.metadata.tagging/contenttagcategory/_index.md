## ContentTagCategory class

Fournit des balises qui sont attachées aux propriétés de métadonnées décrivant le contenu d'un fichier. Les balises sont utiles pour connaître la langue du contenu, le type (genre), le sujet, la note, etc.

public class ContentTagCategory : TagCategory
