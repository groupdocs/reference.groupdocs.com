## Tags.Person property

Obtient un ensemble de balises qui marquent les propriétés des métadonnées contenant des informations sur les personnes ayant contribué à la création de fichiers ou de contenu intellectuel. Ces balises peuvent vous aider à trouver le créateur du document, l'éditeur ou même le client pour qui le travail a été effectué. Malgré le nom de la catégorie, certaines propriétés de métadonnées marquées avec les balises peuvent contenir un nom de société plutôt que le nom d'une personne.

```csharp
public static PersonTagCategory Person { get; }
```

### Valeur de la propriété

Ensemble de balises qui marquent les propriétés des métadonnées contenant des informations sur les personnes ayant contribué à la création de fichiers ou de contenu intellectuel.
