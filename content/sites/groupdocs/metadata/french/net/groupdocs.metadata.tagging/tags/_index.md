## Tags class

Contient divers ensembles de balises avec lesquelles les propriétés de métadonnées les plus importantes sont marquées. Les balises vous permettent de rechercher et de mettre à jour les propriétés des métadonnées dans différents packages, quels que soient la norme de métadonnées et le format de fichier.
