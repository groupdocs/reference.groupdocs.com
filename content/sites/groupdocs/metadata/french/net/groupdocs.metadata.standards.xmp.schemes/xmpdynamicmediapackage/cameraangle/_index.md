## XmpDynamicMediaPackage.CameraAngle property

Obtient ou définit l'orientation de la caméra par rapport au sujet dans une prise de vue statique, à partir d'un ensemble fixe de terminologie standard de l'industrie.

```csharp
public string CameraAngle { get; set; }
```

### Valeur de la propriété

L'orientation de l'appareil photo par rapport au sujet dans une prise de vue statique, à partir d'un ensemble fixe de terminologie standard de l'industrie. Les valeurs prédéfinies incluent : angle bas, niveau des yeux, angle élevé, prise de vue aérienne, prise de vue à vol d'oiseau, angle néerlandais, POV, sur l'épaule, tir de réaction.
