## XmpDynamicMediaPackage.CameraMove property

Obtient ou définit le mouvement de la caméra pendant la prise de vue, à partir d'un ensemble fixe de terminologie standard de l'industrie.

```csharp
public string CameraMove { get; set; }
```

### Valeur de la propriété

Le mouvement de la caméra pendant la prise de vue, à partir d'un ensemble fixe de terminologie standard de l'industrie. Les valeurs prédéfinies incluent : Aérien, Boom Up, Boom Down, Crane Up, Crane Down, Dolly In, Dolly Out, Pan Left, Pan Right, Piédestal vers le haut, Piédestal vers le bas, Inclinaison vers le haut, Inclinaison vers le bas, Suivi, Camion à gauche, Camion à droite, Zoom avant, Zoom arrière.
