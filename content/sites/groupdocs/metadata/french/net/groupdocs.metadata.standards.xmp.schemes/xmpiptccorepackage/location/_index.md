## XmpIptcCorePackage.Location property

Obtient ou définit l'emplacement sur lequel le contenu se concentre.

```csharp
public string Location { get; set; }
```

### Valeur de la propriété

L'emplacement.

### Remarques

Ce nom de lieu peut être soit le nom d'un sous-lieu vers une ville, soit le nom d'un lieu bien connu ou d'un monument (naturel) à l'extérieur d'une ville. Dans le sens d'un sous-lieu vers une ville, cet élément se trouve au quatrième niveau d'un hiérarchie géographique descendante.
