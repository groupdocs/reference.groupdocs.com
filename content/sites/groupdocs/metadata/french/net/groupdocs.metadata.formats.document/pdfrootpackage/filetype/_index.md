## PdfRootPackage.FileType property

Obtient le package de métadonnées de type de fichier.

```csharp
public PdfTypePackage FileType { get; }
```

### Valeur de la propriété

Le package de métadonnées de type de fichier.

### Exemples

Cet extrait de code montre comment détecter la version PDF d'un document chargé et extraire des informations supplémentaires sur le format de fichier.

```csharp
using (Metadata metadata = new Metadata(Constants.InputPdf))
{
    var root = metadata.GetRootPackage<PdfRootPackage>();

    Console.WriteLine(root.FileType.FileFormat);
    Console.WriteLine(root.FileType.Version);
    Console.WriteLine(root.FileType.MimeType);
    Console.WriteLine(root.FileType.Extension);
}
```
