## WordProcessingPackage.Version property

Obtient le numéro de version de l'application qui a créé le document.

```csharp
public int? Version { get; }
```

### Valeur de la propriété

Le numéro de version de l'application qui a créé le document.

### Remarques

Lorsqu'un document a été créé par Microsoft Word, les 16 bits supérieurs représentent la version principale et les 16 bits inférieurs représentent le numéro de build.
