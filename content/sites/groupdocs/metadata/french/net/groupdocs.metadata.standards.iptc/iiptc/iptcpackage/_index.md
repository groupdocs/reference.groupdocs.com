## IIptc.IptcPackage property

Obtient ou définit le package de métadonnées IPTC associé au fichier.

```csharp
public IptcRecordSet IptcPackage { get; set; }
```

### Valeur de la propriété

Le package de métadonnées IPTC associé au fichier.

### Remarques

**Apprendre encore plus**

* [Utilisation des métadonnées IPTC IIM](https://docs.groupdocs.com/display/metadatanet/Working+with+IPTC+IIM+metadata)

### Exemples

Cet exemple de code montre comment supprimer les métadonnées IPTC d'un fichier.

```csharp
using (Metadata metadata = new Metadata(Constants.JpegWithIptc))
{
    IIptc root = metadata.GetRootPackage() as IIptc;
    if (root != null)
    {
        root.IptcPackage = null;

        metadata.Save(Constants.OutputJpeg);
    }
}
```
