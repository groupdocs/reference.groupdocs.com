## FileType.DOCX field

DOCX est un format bien connu pour les documents Microsoft Word. Introduit à partir de 2007 avec la release de Microsoft Office 2007, la structure de ce nouveau format de document est passée de plain binary à une combinaison de fichiers XML et binaires. En savoir plus sur ce format de fichier [ici](https://wiki.fileformat.com/word-processing/docx/) .
