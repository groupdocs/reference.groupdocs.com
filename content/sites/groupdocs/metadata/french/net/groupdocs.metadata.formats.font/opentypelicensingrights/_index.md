## OpenTypeLicensingRights enumeration

Indique les droits de licence d'incorporation de polices pour la police.

```csharp
[Flags]
public enum OpenTypeLicensingRights : ushort
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| None | `0` | Les droits de licence indéfinis. |
| UsagePermissionsMask | `F` | Masque d'autorisations d'utilisation. |
| InstallableEmbedding | `1` | Incorporation installable. La police peut être intégrée et peut être installée de manière permanente pour être utilisée sur des systèmes distants ou pour être utilisée par d'autres utilisateurs. |
| RestrictedLicenseEmbedding | `2` | Incorporation de licence restreinte. La police ne doit pas être modifiée, incorporée ou échangée de quelque manière que ce soit sans avoir obtenu au préalable l'autorisation explicite du propriétaire légal. |
| PreviewAndPrintEmbedding | `4` | Incorporation de la prévisualisation et de l'impression. La police peut être intégrée et peut être temporairement chargée sur d'autres systèmes à des fins de visualisation ou d'impression du document. Les documents contenant des polices de prévisualisation et d'impression doivent être ouverts en « lecture seule » ; aucune modification ne peut être appliquée au document. |
| EditableEmbedding | `8` | Incorporation modifiable. La police peut être intégrée et temporairement chargée sur d'autres systèmes. Comme pour l'intégration de l'aperçu et de l'impression, les documents contenant des polices modifiables peuvent être ouverts pour lecture. De plus, l'édition est autorisée, y compris la possibilité de formater un nouveau texte à l'aide de la police intégrée, et les modifications peuvent être enregistrées. |
| NoSubsetting | `100` | Pas de sous-ensemble. Lorsque ce bit est défini, la police ne peut pas être sous-ensemble avant l'incorporation. D'autres restrictions d'intégration spécifiées dans les bits 0 à 3 et le bit 9 s'appliquent également. |
| BitmapEmbeddingOnly | `200` | Incorporation de bitmap uniquement. Lorsque ce bit est défini, seuls les bitmaps contenus dans la police peuvent être incorporés. Aucune donnée de contour ne peut être intégrée. S'il n'y a pas de bitmaps disponibles dans la police, la police est considérée comme non incorporable et les services d'incorporation échoueront. D'autres restrictions d'intégration spécifiées dans les bits 0-3 et 8 s'appliquent également. |
