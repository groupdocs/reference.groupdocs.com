## XmpTimecode.TimeValue property

Obtient ou définit la valeur de l'heure dans le format spécifié. Les valeurs temporelles utilisent un délimiteur deux-points dans tous les formats sauf 2997drop et 5994drop, qui utilisent un point-virgule. Les quatre champs indiquent les heures, les minutes, les secondes et les images : hh:mm:ss:ff

```csharp
public string TimeValue { get; set; }
```

### Valeur de la propriété

La valeur d'heure dans le format spécifié. Les valeurs d'heure utilisent un séparateur deux-points dans tous les formats sauf 2997drop et 5994drop, qui utilisent un point-virgule. Les quatre champs indiquent les heures, les minutes, les secondes et les images : hh:mm:ss:ff
