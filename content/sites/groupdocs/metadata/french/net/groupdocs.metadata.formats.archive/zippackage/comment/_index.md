## ZipPackage.Comment property

Obtient ou définit le commentaire d'archive ZIP créé par un utilisateur.

```csharp
public string Comment { get; set; }
```

### Valeur de la propriété

Le commentaire de l'utilisateur.

### Exemples

L'extrait de code suivant montre comment supprimer le commentaire de l'utilisateur d'une archive ZIP.

```csharp
using (Metadata metadata = new Metadata(Constants.InputZip))
{
    var root = metadata.GetRootPackage<ZipRootPackage>();

    root.ZipPackage.Comment = null;

    metadata.Save(Constants.OutputZip);
}
```
