## ExifGpsPackage.Satellites property

Obtient ou définit les satellites GPS utilisés pour les mesures. Cette balise peut être utilisée pour décrire le nombre de satellites, leur numéro d'identification, l'angle d'élévation, l'azimut, le SNR et d'autres informations en notation ASCII. Le format n'est pas spécifié. Si le récepteur GPS est incapable de prendre des mesures, la valeur de la balise doit être définie sur NULL.

```csharp
public string Satellites { get; set; }
```

### Valeur de la propriété

Les satellites GPS utilisés pour les mesures.
