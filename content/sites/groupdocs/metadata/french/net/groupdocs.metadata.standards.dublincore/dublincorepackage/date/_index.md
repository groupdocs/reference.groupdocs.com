## DublinCorePackage.Date property

Obtient l'élément de date Dublin Core.

```csharp
public string Date { get; }
```

### Valeur de la propriété

Une date associée à un événement dans le cycle de vie de la ressource. Généralement, la date sera associée à la création ou à la disponibilité de la ressource. La meilleure pratique recommandée pour l'encodage de la valeur de date est définie dans un profil ISO 8601 et suit le format AAAA-MM-JJ.
