## DublinCorePackage.Source property

Obtient l'élément source Dublin Core.

```csharp
public string Source { get; }
```

### Valeur de la propriété

Une référence à une ressource à partir de laquelle la ressource actuelle est dérivée. La ressource actuelle peut être dérivée de la ressource source en tout ou en partie. La meilleure pratique recommandée consiste à référencer la ressource au moyen d'une chaîne ou d'un numéro conforme à un système d'identification formel.
