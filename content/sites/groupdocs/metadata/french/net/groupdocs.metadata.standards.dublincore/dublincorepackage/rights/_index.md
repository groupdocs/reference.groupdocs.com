## DublinCorePackage.Rights property

Obtient les droits de l'élément Dublin Core.

```csharp
public string Rights { get; }
```

### Valeur de la propriété

Informations sur les droits détenus dans et sur la ressource. Généralement, un élément Droits contiendra une déclaration de gestion des droits pour la ressource, ou fera référence à un service fournissant de telles informations. Les informations sur les droits englobent souvent les droits de propriété intellectuelle (DPI), le droit d'auteur et divers droits de propriété. Si l'élément droits est absent, aucune hypothèse ne peut être faite sur le statut de ces droits et d'autres droits par rapport à la ressource.
