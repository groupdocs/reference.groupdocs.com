## MP3RootPackage.ID3V1 property

Obtient ou définit la balise de métadonnées ID3v1. Veuillez trouver plus d'informations sur[http://id3.org/ID3v1](http://id3.org/ID3v1) .

```csharp
public ID3V1Tag ID3V1 { get; set; }
```

### Valeur de la propriété

La balise de métadonnées ID3v1 jointe au fichier audio.

### Remarques

La balise ID3(v1) est un petit morceau de données supplémentaires à la fin d'un fichier MP3.

### Exemples

Cet exemple de code montre comment mettre à jour la balise ID3v1 dans un fichier MP3.

```csharp
using (Metadata metadata = new Metadata(Constants.MP3WithID3V1))
{
    var root = metadata.GetRootPackage<MP3RootPackage>();

    if (root.ID3V1 == null)
    {
        root.ID3V1 = new ID3V1Tag();
    }

    root.ID3V1.Album = "test album";
    root.ID3V1.Artist = "test artist";
    root.ID3V1.Title = "test title";
    root.ID3V1.Comment = "test comment";
    root.ID3V1.Year = "2019";

    // ...

    metadata.Save(Constants.OutputMp3);
}
```
