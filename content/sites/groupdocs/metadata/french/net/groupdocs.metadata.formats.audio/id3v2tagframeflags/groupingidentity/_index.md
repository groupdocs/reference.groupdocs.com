## ID3V2TagFrameFlags.GroupingIdentity property

Obtient une valeur indiquant si le cadre appartient à un groupe de cadres. S'il est défini, un octet d'identifiant de groupe est ajouté à l'en-tête de trame. Chaque trame avec le même identifiant de groupe appartient au même groupe.

```csharp
public bool GroupingIdentity { get; }
```

### Valeur de la propriété

`vrai`, si la trame appartient à un groupe de trames ; sinon, faux.
