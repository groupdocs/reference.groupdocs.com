## ID3V2TagFrameFlags.DataLengthIndicator property

Obtient une valeur indiquant si un indicateur de longueur de données a été ajouté à la trame. L'indicateur de longueur de données est la valeur que l'on écrirait comme "Longueur de trame" si tous les drapeaux de format de trame étaient mis à zéro, représentés comme un entier synchsafe 32 bits.

```csharp
public bool DataLengthIndicator { get; }
```

### Valeur de la propriété

`vrai`si un indicateur de longueur de données a été ajouté à la trame ; sinon, faux.
