## AviHeader.SuggestedBufferSize property

Obtient la taille de tampon suggérée pour lire le fichier.  Généralement, cette taille doit être suffisamment grande pour contenir le plus gros morceau du fichier. S'il est défini sur zéro, ou s'il est trop petit, le logiciel de lecture devra réallouer de la mémoire pendant la lecture, ce qui réduira les performances. Pour un fichier entrelacé, la taille de la mémoire tampon doit être suffisamment grande pour lire un enregistrement entier, et pas seulement un morceau.

```csharp
public int SuggestedBufferSize { get; }
```

### Valeur de la propriété

La taille de tampon suggérée pour lire le fichier.
