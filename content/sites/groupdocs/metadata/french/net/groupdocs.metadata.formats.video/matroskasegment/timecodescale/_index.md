## MatroskaSegment.TimecodeScale property

Obtient la valeur d'échelle du code temporel. Chaque code temporel mis à l'échelle dans un fichier MATROSKA est multiplié par TIMECODESCALE pour obtenir le code temporel en nanosecondes. Notez que tous les codes temporels ne sont pas mis à l'échelle !

```csharp
public ulong TimecodeScale { get; }
```

### Valeur de la propriété

La valeur d'échelle du code temporel.
