## Metadata.Sanitize method

यदि संभव हो तो सभी ज्ञात पैकेजों या पूरे पैकेजों से लिखने योग्य मेटाडेटा गुणों को हटा देता है। ऑपरेशन पुनरावर्ती है इसलिए यह सभी नेस्टेड पैकेजों को भी प्रभावित करता है।

```csharp
public int Sanitize()
```

### उदाहरण

यह उदाहरण दर्शाता है कि किसी फ़ाइल से सभी पहचाने गए मेटाडेटा पैकेज/गुणों को कैसे हटाया जाए।

```csharp
using (Metadata metadata = new Metadata(Constants.InputPdf))
{
    // पता लगाए गए मेटाडेटा पैकेज हटाएं
    var affected = metadata.Sanitize();
    Console.WriteLine("Properties removed: {0}", affected);

    metadata.Save(Constants.OutputPdf);
}
```
