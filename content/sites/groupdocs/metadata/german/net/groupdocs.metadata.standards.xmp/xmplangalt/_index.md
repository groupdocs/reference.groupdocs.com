## XmpLangAlt class

Stellt die XMP-Sprachalternative dar.

Ein alternatives Array einfacher Textelemente. Sprachalternativen erleichtern die Auswahl eines einfachen Textelements basierend auf einer gewünschten Sprache. Jedes Array-Element muss einen xml:lang-Qualifizierer haben. Jeder xml:lang-Wert muss unter den Elementen eindeutig sein.

```csharp
public sealed class XmpLangAlt : XmpArray
```
