| Unknown | `0` | Das Format wird nicht erkannt. |
| Xls | `1` | Stellt das Excel-Format .XLS dar. Dateien mit der Erweiterung XLS stellen das Excel-Binärdateiformat dar. Solche Dateien können von Microsoft Excel sowie anderen ähnlichen Tabellenkalkulationsprogrammen wie OpenOffice Calc oder Apple Numbers erstellt werden. Von Excel gespeicherte Dateien sind bekannt als Arbeitsmappe, wobei jede Arbeitsmappe ein oder mehrere Arbeitsblätter enthalten kann. Erfahren Sie mehr über dieses Dateiformat[Hier](https://wiki.fileformat.com/spreadsheet/xls/) . |
| Xlsb | `2` | Stellt das .XLSB-Excel-Format dar. Das XLSB-Dateiformat gibt das Excel-Binärdateiformat an, bei dem es sich um eine Sammlung von Datensätzen und Strukturen handelt, die den Inhalt von Excel-Arbeitsmappen angeben. Der Inhalt kann unstrukturierte oder halbstrukturierte Tabellen mit Zahlen, Text oder sowohl Zahlen als auch Text, Formeln, externe Datenverbindungen, Diagramme und Bilder. Erfahren Sie mehr über dieses Dateiformat[Hier](https://wiki.fileformat.com/spreadsheet/xlsb/) . |
| Xlsx | `3` | Stellt das Excel-Format .XLSX dar. XLSX ist ein bekanntes Format für Microsoft Excel-Dokumente, das von Microsoft mit der Veröffentlichung von Microsoft Office 2007 eingeführt wurde OOXML-Standard ECMA-376, das neue Format ist ein ZIP-Paket, das eine Reihe von XML-Dateien enthält. Erfahren Sie mehr über dieses Dateiformat[Hier](https://wiki.fileformat.com/spreadsheet/xlsx/) . |
| Xlsm | `4` | Stellt das Excel-Format .XLSM dar. Dateien mit der Erweiterung XLSM sind Tabellenkalkulationsdateien, die Makros unterstützen. Aus Anwendungssicht ist ein Makro eine Reihe von Anweisungen, die zum Automatisieren von Prozessen verwendet werden. Ein Makro wird zum Aufzeichnen verwendet die Schritte, die wiederholt ausgeführt werden, und erleichtert das Ausführen der Aktionen durch erneutes Ausführen des Makros. Erfahren Sie mehr über dieses Dateiformat[Hier](https://wiki.fileformat.com/spreadsheet/xlsm/) . |
| Xltx | `5` | Stellt das Excel-Format .XLTX dar. Dateien mit der Erweiterung XLTX stellen Microsoft Excel-Vorlagendateien dar, die auf den Office OpenXML-Dateiformatspezifikationen basieren. Sie wird verwendet, um eine Standardvorlagendatei zu erstellen, die zum Generieren von XLSX-Dateien verwendet werden kann, die das aufweisen dieselben Einstellungen wie in der XLTX-Datei angegeben. Erfahren Sie mehr über dieses Dateiformat[Hier](https://wiki.fileformat.com/spreadsheet/xltx/) . |
| Xltm | `6` | Repräsentiert das .XLTM-Excel-Format. Die XLTM-Dateierweiterung stellt Dateien dar, die von Microsoft Excel als makrofähige Vorlagendateien generiert werden. XLTM-Dateien ähneln XLTX in der Struktur, abgesehen davon, dass letztere das Erstellen von Vorlagendateien nicht unterstützen Makros. Solche Vorlagendateien werden zum Generieren und Festlegen des Layouts, der Formatierung und anderer Einstellungen zusammen mit den Makros verwendet, um dann das Erstellen ähnlicher XLSX-Dateien zu erleichtern. Erfahren Sie mehr über dieses Dateiformat[Hier](https://wiki.fileformat.com/spreadsheet/xltm/) . |
| Ods | `7` | Stellt das Opendocument Spreadsheet-Format dar. Dateien mit ODS-Erweiterung stehen für OpenDocument Spreadsheet Document-Format, das vom Benutzer bearbeitet werden kann. Daten werden in der ODF-Datei in Zeilen und Spalten gespeichert. Es ist ein XML-basiertes Format und einer von mehreren Untertypen in der Familie der Open Document Formats (ODF). Erfahren Sie mehr über dieses Dateiformat[Hier](https://wiki.fileformat.com/spreadsheet/ods/) . |
| Xlt | `8` | Stellt das Excel-Format .XLT dar. Dateien mit der Erweiterung .XLT sind Vorlagendateien, die mit Microsoft Excel erstellt wurden, einer Tabellenkalkulationsanwendung, die Teil der Microsoft Office-Suite ist. Microsoft Office 97-2003 unterstützte das Erstellen neuer XLT-Dateien sowie das Öffnen these. Die neueste Version von Excel kann immer noch Vorlagendateien dieses alten Formats öffnen. Erfahren Sie mehr über dieses Dateiformat[Hier](https://wiki.fileformat.com/spreadsheet/xlt/) . |
