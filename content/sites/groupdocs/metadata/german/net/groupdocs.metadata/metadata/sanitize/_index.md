## Metadata.Sanitize method

Entfernt beschreibbare Metadateneigenschaften von allen erkannten Paketen oder ganzen Paketen, falls möglich. Die Operation ist rekursiv, sodass sie sich auch auf alle verschachtelten Pakete auswirkt.

```csharp
public int Sanitize()
```

### Rückgabewert

Die Anzahl der betroffenen Eigenschaften.

### Bemerkungen

**Erfahren Sie mehr**

* [Metadaten bereinigen](https://docs.groupdocs.com/display/metadatanet/Clean+metadata)

### Beispiele

Dieses Beispiel zeigt, wie alle erkannten Metadatenpakete/Eigenschaften aus einer Datei entfernt werden.

```csharp
using (Metadata metadata = new Metadata(Constants.InputPdf))
{
    // Erkannte Metadatenpakete entfernen
    var affected = metadata.Sanitize();
    Console.WriteLine("Properties removed: {0}", affected);

    metadata.Save(Constants.OutputPdf);
}
```
