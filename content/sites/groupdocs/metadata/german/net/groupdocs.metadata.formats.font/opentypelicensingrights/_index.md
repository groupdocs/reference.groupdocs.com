## OpenTypeLicensingRights enumeration

Gibt die Lizenzrechte zum Einbetten von Schriftarten für die Schriftart an.

```csharp
[Flags]
public enum OpenTypeLicensingRights : ushort
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Die undefinierten Lizenzrechte. |
| UsagePermissionsMask | `F` | Nutzungsberechtigungsmaske. |
| InstallableEmbedding | `1` | Installierbare Einbettung. Die Schriftart kann eingebettet sein und dauerhaft zur Verwendung auf einem entfernten System oder zur Verwendung durch andere Benutzer installiert werden. |
| RestrictedLicenseEmbedding | `2` | Einbettung mit eingeschränkter Lizenz. Die Schriftart darf ohne vorherige ausdrückliche Genehmigung des rechtmäßigen Eigentümers in keiner Weise modifiziert, eingebettet oder ausgetauscht werden. |
| PreviewAndPrintEmbedding | `4` | Vorschau- und Druckeinbettung. Die Schriftart kann eingebettet und vorübergehend auf andere Systeme geladen werden, um das Dokument anzuzeigen oder zu drucken. Dokumente, die Vorschau- und Druckschriftarten enthalten, müssen „schreibgeschützt“ geöffnet werden; Das Dokument kann nicht bearbeitet werden. |
| EditableEmbedding | `8` | Bearbeitbare Einbettung. Die Schriftart kann eingebettet und vorübergehend auf anderen Systemen geladen werden. Wie bei der Vorschau- und Druckeinbettung können Dokumente, die bearbeitbare Schriftarten enthalten, zum Lesen geöffnet werden. Darüber hinaus ist die Bearbeitung zulässig, einschließlich der Möglichkeit, neuen Text mit der eingebetteten Schriftart zu formatieren, und Änderungen können gespeichert werden. |
| NoSubsetting | `100` | No subsetting. Wenn dieses Bit gesetzt ist, darf der Font vor dem Einbetten nicht subseted werden. Es gelten auch andere Einbettungsbeschränkungen, die in den Bits 0 bis 3 und Bit 9 angegeben sind. |
| BitmapEmbeddingOnly | `200` | Nur Bitmap-Einbettung. Wenn dieses Bit gesetzt ist, dürfen nur im Font enthaltene Bitmaps eingebettet werden. Es dürfen keine Gliederungsdaten eingebettet werden. Wenn in der Schriftart keine Bitmaps verfügbar sind, gilt die Schriftart als nicht einbettbar und die Einbettungsdienste schlagen fehl. Es gelten auch andere Einbettungsbeschränkungen, die in den Bits 0-3 und 8 angegeben sind. |
