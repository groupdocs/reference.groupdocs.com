## MatroskaTrack.Language property

Ruft die Sprache des Tracks im Matroska-Sprachenformat ab. Dieses Element MUSS ignoriert werden, wenn die[`LanguageIetf`](../languageietf) Element wird im selben TrackEntry verwendet.

```csharp
public string Language { get; }
```

### Eigentumswert

Die Sprache der Spur im Matroska-Sprachenformular.

### Bemerkungen

Sprachcodes können entweder die aus 3 Buchstaben bestehende bibliografische ISO-639-2-Form sein (wie "fre" für Französisch), oder ein solcher Sprachcode, gefolgt von einem Bindestrich und einem Ländercode für Fachgebiete in Sprachen (wie "fre-ca" für kanadisches Französisch). Ländercodes sind dieselben wie für Internetdomänen.
