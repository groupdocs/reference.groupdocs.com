public enum AviHeaderFlags

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| HasIndex | `10` | Zeigt an, dass die AVI-Datei einen Index hat. |
| MustUseIndex | `20` | Gibt an, dass die Anwendung den Index und nicht die physische Reihenfolge der Chunks in der Datei verwenden soll, um die Reihenfolge der Darstellung der Daten zu bestimmen. Dieses Flag könnte beispielsweise verwendet werden, um eine Liste von Frames zur Bearbeitung zu erstellen. |
| IsInterleaved | `100` | Zeigt an, dass die AVI-Datei interleaved ist. |
| TrustCkType | `800` | Verwenden Sie CKType, um Keyframes zu finden. |
| WasCaptureFile | `10000` | Zeigt an, dass die AVI-Datei eine speziell zugewiesene Datei ist, die zum Erfassen von Echtzeitvideos verwendet wird. Anwendungen sollten den Benutzer warnen, bevor sie eine Datei mit diesem Flag überschreiben, da der Benutzer diese Datei wahrscheinlich defragmentiert hat. |
| Copyrighted | `20000` | Gibt an, dass die AVI-Datei urheberrechtlich geschützte Daten und Software enthält. Wenn dieses Flag verwendet wird, sollte die Software nicht zulassen, dass die Daten dupliziert werden. |
