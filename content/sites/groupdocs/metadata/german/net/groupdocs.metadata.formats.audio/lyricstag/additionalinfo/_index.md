## LyricsTag.AdditionalInfo property

Ruft die zusätzlichen Informationen ab oder setzt sie. Dieser Wert wird durch das INF-Feld repräsentiert.

```csharp
public string AdditionalInfo { get; set; }
```

### Eigentumswert

Die zusätzlichen Informationen.

### Bemerkungen

Dies ist in v2.00 immer drei (3) Zeichen lang, könnte aber in einem zukünftigen Standard länger sein. Das erste Byte zeigt an, ob oder ob kein Textfeld vorhanden ist. „1“ für vorhanden und „0“ für sonst. Das zweite Zeichen gibt an, ob der Text einen Zeitstempel enthält. Wieder "1" für ja und "0" für nein. Das dritte Zeichen sperrt Spuren für die Zufallsauswahl - "1" wenn gesperrt und "0" wenn nicht.
