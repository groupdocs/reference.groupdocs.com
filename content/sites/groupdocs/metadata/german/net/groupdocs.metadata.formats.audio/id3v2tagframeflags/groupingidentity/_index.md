## ID3V2TagFrameFlags.GroupingIdentity property

Ruft einen Wert ab, der angibt, ob der Frame zu einer Gruppe von Frames gehört. Wenn gesetzt, wird dem Frame-Header ein Gruppenkennungsbyte hinzugefügt. Jeder Frame mit derselben Gruppenkennung gehört zu derselben Gruppe.

```csharp
public bool GroupingIdentity { get; }
```

### Eigentumswert

`WAHR`, wenn der Frame zu einer Gruppe von Frames gehört; andernfalls falsch.
