## ID3V2TagFrameFlags.DataLengthIndicator property

Ruft einen Wert ab, der angibt, ob ein Datenlängenindikator zu dem Frame hinzugefügt wurde. Der Datenlängenindikator ist der Wert, den man als 'Frame-Länge' schreiben würde wenn alle Frame-Format-Flags auf Null gesetzt wären, dargestellt als synchronsichere 32-Bit-Ganzzahl.

```csharp
public bool DataLengthIndicator { get; }
```

### Eigentumswert

`WAHR`wenn dem Rahmen ein Datenlängenindikator hinzugefügt wurde; andernfalls falsch.
