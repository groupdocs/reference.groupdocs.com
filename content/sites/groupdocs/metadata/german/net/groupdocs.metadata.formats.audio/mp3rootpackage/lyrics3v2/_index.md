## MP3RootPackage.Lyrics3V2 property

### Eigentumswert

Das Lyrics3v2-Metadaten-Tag.

### Beispiele

Dieses Beispiel zeigt, wie das Liedtext-Tag in einer MP3-Datei aktualisiert wird

```csharp
using (Metadata metadata = new Metadata(Constants.MP3WithLyrics))
{
    var root = metadata.GetRootPackage<MP3RootPackage>();

    if (root.Lyrics3V2 == null)
    {
        root.Lyrics3V2 = new LyricsTag();
    }

    root.Lyrics3V2.Lyrics = "[00:01]Test lyrics";
    root.Lyrics3V2.Artist = "test artist";
    root.Lyrics3V2.Album = "test album";
    root.Lyrics3V2.Track = "test track";

    // Sie können dem Tag ein vollständig benutzerdefiniertes Feld hinzufügen
    root.Lyrics3V2.Set(new LyricsField("ABC", "custom value"));

    // ...

    metadata.Save(Constants.OutputMp3);
}
```
