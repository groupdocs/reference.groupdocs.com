## MP3RootPackage.ID3V1 property

Ruft das ID3v1-Metadaten-Tag ab oder setzt es. Weitere Informationen finden Sie unter[http://id3.org/ID3v1](http://id3.org/ID3v1) .

```csharp
public ID3V1Tag ID3V1 { get; set; }
```

### Eigentumswert

Das an die Audiodatei angehängte ID3v1-Metadaten-Tag.

### Bemerkungen

Das ID3(v1)-Tag ist ein kleines Stück zusätzlicher Daten am Ende einer MP3-Datei.

### Beispiele

Dieses Codebeispiel zeigt, wie das ID3v1-Tag in einer MP3-Datei aktualisiert wird.

```csharp
using (Metadata metadata = new Metadata(Constants.MP3WithID3V1))
{
    var root = metadata.GetRootPackage<MP3RootPackage>();

    if (root.ID3V1 == null)
    {
        root.ID3V1 = new ID3V1Tag();
    }

    root.ID3V1.Album = "test album";
    root.ID3V1.Artist = "test artist";
    root.ID3V1.Title = "test title";
    root.ID3V1.Comment = "test comment";
    root.ID3V1.Year = "2019";

    // ...

    metadata.Save(Constants.OutputMp3);
}
```
