## DocumentTagCategory class

Stellt Tags bereit, die nur auf dokumentspezifische Eigenschaften angewendet werden. Die Tags können nützlich sein, um festzustellen, aus welchem Teil eines Office-Dokuments eine Eigenschaft extrahiert wurde.
