## ContentTagCategory class

Stellt Tags bereit, die an Metadateneigenschaften angehängt sind, die den Inhalt einer Datei beschreiben. Die Tags sind nützlich, um die Inhaltssprache, den Typ (Genre), das Thema, die Bewertung usw. herauszufinden.
