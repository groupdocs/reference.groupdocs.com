## PropertyTypeTagCategory class

Stellt Tags bereit, die zusätzliche Informationen über den Typ einer Eigenschaft und nicht über ihren Zweck enthalten. Mithilfe dieser Tags können Sie Metadateneigenschaften erkennen, die URL-Links zu externen Ressourcen enthalten, Eigenschaften, die Schriftarten, Farben, Geolokalisierung usw. beschreiben.

public class PropertyTypeTagCategory : TagCategory
