Der Namensraum enthält verschiedene Sätze von Tags, mit denen die wichtigsten Metadateneigenschaften gekennzeichnet sind. Die Tags ermöglichen es Benutzern, Metadateneigenschaften in verschiedenen Paketen unabhängig vom Metadatenstandard und Dateiformat zu finden und zu aktualisieren.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [ContentTagCategory](./contenttagcategory) | Stellt Tags bereit, die an Metadateneigenschaften angehängt sind, die den Inhalt einer Datei beschreiben. Die Tags sind nützlich, um die Inhaltssprache, den Typ (Genre), das Thema, die Bewertung usw. herauszufinden. |
| [CorporateTagCategory](./corporatetagcategory) | Stellt Tags bereit, die Metadateneigenschaften markieren sollen, die sich auf ein Unternehmen beziehen, das an der Dateierstellung beteiligt war. |
| [DocumentTagCategory](./documenttagcategory) | Stellt Tags bereit, die nur auf dokumentspezifische Eigenschaften angewendet werden. Die Tags können nützlich sein, um festzustellen, aus welchem Teil eines Office-Dokuments eine Eigenschaft extrahiert wurde. |
| [LegalTagCategory](./legaltagcategory) | Stellt Tags bereit, die an Metadateneigenschaften angehängt sind und Informationen über die Eigentümer des Dateiinhalts und die Regeln enthalten, unter denen der Inhalt verwendet werden kann. |
| [OriginTagCategory](./origintagcategory) | Stellt Tags bereit, die einem Benutzer helfen, die Herkunft einer Datei (z. B. Vorlage oder andere Quelle) zu bestimmen. |
| [PersonTagCategory](./persontagcategory) | Stellt Tags bereit, die Metadateneigenschaften markieren, die Informationen über die Personen enthalten, die an der Erstellung von Dateien oder geistigen Inhalten mitgewirkt haben. Diese Tags können Ihnen helfen, den Ersteller, Bearbeiter oder sogar den Kunden des Dokuments zu finden, für den die Arbeit ausgeführt wurde. Trotz des Namens der Kategorie können einige mit den Tags gekennzeichnete Metadateneigenschaften einen Firmennamen anstelle eines Personennamens enthalten. |
| [PropertyTag](./propertytag) | Stellt ein Tag dar, das zum Markieren von Metadateneigenschaften verwendet wird. |
| [PropertyTypeTagCategory](./propertytypetagcategory) | Stellt Tags bereit, die zusätzliche Informationen über den Typ einer Eigenschaft und nicht über ihren Zweck enthalten. Mithilfe dieser Tags können Sie Metadateneigenschaften erkennen, die URL-Links zu externen Ressourcen enthalten, Eigenschaften, die Schriftarten, Farben, Geolokalisierung usw. beschreiben. |
| [TagCategory](./tagcategory) | Stellt eine Reihe von Tags dar, die einige gemeinsame Merkmale aufweisen. |
| [Tags](./tags) | Enthält verschiedene Sätze von Tags, mit denen die wichtigsten Metadateneigenschaften gekennzeichnet sind. Die Tags ermöglichen es Ihnen, Metadateneigenschaften in verschiedenen Paketen unabhängig vom Metadatenstandard und Dateiformat zu finden und zu aktualisieren. |
| [TimeTagCategory](./timetagcategory) | Stellt Tags bereit, die Metadateneigenschaften markieren, die zur Beschreibung des Lebenszyklus einer Datei verwendet werden. Die Tags beziehen sich auf Zeitpunkte, zu denen eine Datei oder geistige Inhalte erstellt, bearbeitet, gedruckt usw. wurden. |
| [ToolTagCategory](./tooltagcategory) | Stellt Tags bereit, die dazu bestimmt sind, Metadateneigenschaften in Bezug auf die Tools (Software und Hardware) zu markieren, die zum Erstellen einer Datei verwendet wurden. |
