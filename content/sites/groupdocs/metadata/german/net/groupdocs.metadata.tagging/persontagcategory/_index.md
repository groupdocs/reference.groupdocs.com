## PersonTagCategory class

Stellt Tags bereit, die Metadateneigenschaften markieren, die Informationen über die Personen enthalten, die an der Erstellung von Dateien oder geistigen Inhalten mitgewirkt haben. Diese Tags können Ihnen helfen, den Ersteller, Bearbeiter oder sogar den Kunden des Dokuments zu finden, für den die Arbeit ausgeführt wurde. Trotz des Namens der Kategorie können einige mit den Tags gekennzeichnete Metadateneigenschaften einen Firmennamen anstelle eines Personennamens enthalten.

```csharp
public class PersonTagCategory : TagCategory
```
