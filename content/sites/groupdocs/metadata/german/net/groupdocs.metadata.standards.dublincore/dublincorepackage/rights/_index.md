## DublinCorePackage.Rights property

Ruft die Rechte des Dublin Core-Elements ab.

```csharp
public string Rights { get; }
```

### Eigentumswert

Informationen über Rechte, die an und über die Ressource gehalten werden. Typischerweise enthält ein Rights-Element eine Rechteverwaltungsanweisung für die Ressource oder verweist auf einen Dienst, der solche Informationen bereitstellt. Rechteinformationen umfassen oft geistige Eigentumsrechte (IPR), Urheberrechte und verschiedene Eigentumsrechte. Wenn das Rechteelement fehlt, können keine Annahmen über den Status dieser und anderer Rechte in Bezug auf die Ressource getroffen werden.
