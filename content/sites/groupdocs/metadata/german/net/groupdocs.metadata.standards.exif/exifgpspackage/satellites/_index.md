## ExifGpsPackage.Satellites property

Ruft die für Messungen verwendeten GPS-Satelliten ab oder setzt sie. Dieses Tag kann verwendet werden, um die Anzahl der Satelliten, ihre ID-Nummer, Höhenwinkel, Azimut, SNR und andere Informationen in ASCII-Notation zu beschreiben. Das Format ist not angegeben. Wenn der GPS-Empfänger keine Messungen vornehmen kann, muss der Wert des Tags auf NULL gesetzt werden.

```csharp
public string Satellites { get; set; }
```
