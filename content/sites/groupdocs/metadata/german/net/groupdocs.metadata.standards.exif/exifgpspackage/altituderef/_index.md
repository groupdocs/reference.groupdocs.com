## ExifGpsPackage.AltitudeRef property

Ermittelt oder setzt die Höhe, die als Referenzhöhe verwendet wird. Wenn die Referenz der Meeresspiegel ist und die Höhe über dem Meeresspiegel liegt, wird 0 angegeben. Wenn die Höhe unter dem Meeresspiegel liegt, wird der Wert 1 angegeben und die Höhe wird als absoluter Wert in angegeben[`Altitude`](../altitude) tag.

public ExifGpsAltitudeRef? AltitudeRef { get; set; }
