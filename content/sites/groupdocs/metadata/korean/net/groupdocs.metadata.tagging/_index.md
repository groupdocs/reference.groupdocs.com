네임스페이스에는 가장 중요한 메타데이터 속성이 표시되는 다양한 태그 집합이 포함되어 있습니다. 태그를 사용하면 메타데이터 표준 및 파일 형식에 관계없이 다른 패키지에서 메타데이터 속성을 찾고 업데이트할 수 있습니다.

## 클래스

| 수업 | 설명 |
| --- | --- |
| [ContentTagCategory](./contenttagcategory) | 파일 내용을 설명하는 메타데이터 속성에 첨부된 태그를 제공합니다. 태그는 콘텐츠 언어, 유형(장르), 주제, 등급 등을 찾는 데 유용합니다. |
| [CorporateTagCategory](./corporatetagcategory) | 파일 생성에 참여한 회사와 관련된 메타데이터 속성을 표시하기 위한 태그를 제공합니다. |
| [DocumentTagCategory](./documenttagcategory) | 문서별 속성에만 적용되는 태그를 제공합니다. 태그는 Office 문서의 어느 부분에서 속성이 추출되었는지 확인하는 데 유용할 수 있습니다. |
| [LegalTagCategory](./legaltagcategory) | 파일 콘텐츠 의 소유자에 대한 정보와 콘텐츠를 사용할 수 있는 규칙을 포함하는 메타데이터 속성에 첨부된 태그를 제공합니다. |
| [OriginTagCategory](./origintagcategory) | 사용자가 파일의 원본(예: 템플릿 또는 다른 원본)을 확인하는 데 도움이 되는 태그를 제공합니다. |
| [PersonTagCategory](./persontagcategory) | 파일 또는 지적 콘텐츠 생성에 기여한 사람들에 대한 정보를 포함하는 메타데이터 속성을 표시하는 태그를 제공합니다. 이러한 태그는 문서 작성자, 편집자 또는 작업을 수행한 클라이언트를 찾는 데 도움이 될 수 있습니다. 범주 이름에도 불구하고 태그로 표시된 일부 메타데이터 속성에는 사람 이름이 아닌 회사 이름이 포함될 수 있습니다. |
| [PropertyTag](./propertytag) | 메타데이터 속성을 표시하는 데 사용되는 태그를 나타냅니다. |
| [PropertyTypeTagCategory](./propertytypetagcategory) | 용도보다는 속성 유형에 대한 추가 정보가 포함된 태그를 제공합니다. 이러한 태그를 사용하면 외부 리소스에 대한 URL 링크가 포함된 메타데이터 속성, 글꼴, 색상, 지리적 위치 등을 설명하는 속성을 감지할 수 있습니다. |
| [TagCategory](./tagcategory) | 일부 공유 특성이 있는 태그 집합을 나타냅니다. |
| [Tags](./tags) | 가장 중요한 메타데이터 속성이 표시되는 다양한 태그 세트를 포함합니다. 태그를 사용하면 메타데이터 표준 및 파일 형식에 관계없이 다른 패키지에서 메타데이터 속성을 찾고 업데이트할 수 있습니다. |
| [TimeTagCategory](./timetagcategory) | 파일의 수명 주기를 설명하는 데 사용되는 메타데이터 속성을 표시하는 태그를 제공합니다. 태그는 파일 또는 지적 콘텐츠가 생성, 편집, 인쇄된 시점을 처리합니다. |
| [ToolTagCategory](./tooltagcategory) | 파일을 만드는 데 사용된 도구(소프트웨어 및 하드웨어)와 관련된 메타데이터 속성을 표시하기 위한 태그를 제공합니다. |
