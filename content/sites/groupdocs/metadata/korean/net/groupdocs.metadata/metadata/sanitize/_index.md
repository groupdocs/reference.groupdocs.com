## Metadata.Sanitize method

감지된 모든 패키지 또는 가능한 경우 전체 패키지에서 쓰기 가능한 메타데이터 속성을 제거합니다. 이 작업은 재귀적이므로 중첩된 모든 패키지에도 영향을 미칩니다.

```csharp
public int Sanitize()
```

### 반환 값

영향을 받는 속성의 수입니다.

### 비고

**더 알아보기**

* [깨끗한 메타데이터](https://docs.groupdocs.com/display/metadatanet/Clean+metadata)

### 예

이 예는 파일에서 감지된 모든 메타데이터 패키지/속성을 제거하는 방법을 보여줍니다.

```csharp
using (Metadata metadata = new Metadata(Constants.InputPdf))
{
    // 감지된 메타데이터 패키지 제거
    var affected = metadata.Sanitize();
    Console.WriteLine("Properties removed: {0}", affected);

    metadata.Save(Constants.OutputPdf);
}
```
