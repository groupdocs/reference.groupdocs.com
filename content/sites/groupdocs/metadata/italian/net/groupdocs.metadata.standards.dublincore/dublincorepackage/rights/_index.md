## DublinCorePackage.Rights property

Ottiene i diritti dell'elemento Dublin Core.

```csharp
public string Rights { get; }
```

### Valore della proprietà

Informazioni sui diritti detenuti nella e sulla risorsa. In genere un elemento Rights conterrà una dichiarazione di gestione dei diritti per la risorsa, o farà riferimento a un servizio che fornisce tali informazioni. Le informazioni sui diritti comprendono spesso i diritti di proprietà intellettuale (DPI), il copyright e vari diritti di proprietà. Se l'elemento diritti è assente, non è possibile fare ipotesi sullo stato di questi e altri diritti rispetto alla risorsa.
