Questo esempio di codice mostra come aggiornare le statistiche del documento per un documento di elaborazione di testi.

```csharp
using (Metadata metadata = new Metadata(Constants.InputDoc))
{

   root.UpdateDocumentStatistics();

   metadata.Save(Constants.OutputDoc);
}
```
