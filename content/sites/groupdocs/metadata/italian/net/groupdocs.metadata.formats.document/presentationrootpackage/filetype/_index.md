## PresentationRootPackage.FileType property

Ottiene il pacchetto di metadati del tipo di file.

```csharp
public PresentationTypePackage FileType { get; }
```

### Valore della proprietà

Il pacchetto di metadati del tipo di file.

### Esempi

Questo esempio mostra come rilevare il tipo esatto di una presentazione ed estrarre alcune informazioni aggiuntive sul formato del file.

```csharp
using (Metadata metadata = new Metadata(Constants.InputPptx))
{
    var root = metadata.GetRootPackage<PresentationRootPackage>();

    Console.WriteLine(root.FileType.FileFormat);
    Console.WriteLine(root.FileType.PresentationFormat);
    Console.WriteLine(root.FileType.MimeType);
    Console.WriteLine(root.FileType.Extension);
}
```
