## DiagramRootPackage.FileType property

Ottiene il pacchetto di metadati del tipo di file.

```csharp
public DiagramTypePackage FileType { get; }
```

### Valore della proprietà

Il pacchetto di metadati del tipo di file.

### Esempi

Questo esempio di codice mostra come rilevare il tipo esatto di un diagramma caricato ed estrarre alcune informazioni aggiuntive sul formato del file.

```csharp
using (Metadata metadata = new Metadata(Constants.InputVdx))
{
    var root = metadata.GetRootPackage<DiagramRootPackage>();

    Console.WriteLine(root.FileType.FileFormat);
    Console.WriteLine(root.FileType.DiagramFormat);
    Console.WriteLine(root.FileType.MimeType);
    Console.WriteLine(root.FileType.Extension);
}
```
