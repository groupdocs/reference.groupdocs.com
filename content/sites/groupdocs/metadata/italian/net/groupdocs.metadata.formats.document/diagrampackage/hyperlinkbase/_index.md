## DiagramPackage.HyperlinkBase property

Ottiene o imposta il percorso da utilizzare per i collegamenti ipertestuali relativi (collegamenti ipertestuali per i quali è descritta la posizione del file collegato in relazione al diagramma di Microsoft Visio). Per impostazione predefinita, un percorso di collegamento ipertestuale è relativo al documento corrente a meno che non sia specificato un percorso diverso in questo elemento. La lunghezza massima è di 256 caratteri.

```csharp
public string HyperlinkBase { get; set; }
```

### Valore della proprietà

Il percorso da utilizzare per i collegamenti ipertestuali relativi.
