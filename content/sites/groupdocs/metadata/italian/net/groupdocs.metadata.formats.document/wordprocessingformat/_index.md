### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Unknown | `0` | Il formato non è riconosciuto. |
| Doc | `1` | Rappresenta il formato .DOC Word. I file con estensione .doc rappresentano documenti generati da Microsoft Word o altre applicazioni di elaborazione testi in formato file binario. L'estensione è stata inizialmente utilizzata per la documentazione di testo semplice su diversi sistemi operativi. Può contenere diversi tipi di dati come immagini, formattato così come testo semplice, grafici, diagrammi, oggetti incorporati, collegamenti, pagine, formattazione della pagina, impostazioni di stampa e molto altro. Ulteriori informazioni su questo formato di file[qui](https://wiki.fileformat.com/word-processing/doc/) . |
| Docm | `2` | Rappresenta il formato .DOCM Word. I file DOCM sono documenti generati da Microsoft Word 2007 o superiore con la possibilità di eseguire macro. È simile al formato di file DOCX ma la possibilità di eseguire macro lo rende diverso da DOCX. Come DOCX, i file DOCM possono contenere testo, immagini, tabelle, forme, grafici e altri contenuti. Ulteriori informazioni su questo formato di file[qui](https://wiki.fileformat.com/word-processing/docm/) . |
| Docx | `3` | Rappresenta il formato .DOCX Word. DOCX è un formato ben noto per i documenti Microsoft Word. Introdotto a partire dal 2007 con il rilascio di Microsoft Office 2007, la struttura di questo nuovo formato Documento è stata modificata da binario semplice a una combinazione di XML e file binari. I file Docx possono essere aperti con Word 2007 e versioni laterali ma non con le versioni precedenti di MS Word che supportano le estensioni di file DOC. Ulteriori informazioni su questo formato di file[qui](https://wiki.fileformat.com/word-processing/docx/) . |
| Dot | `4` | Rappresenta il formato .DOT Word. I file con estensione .DOT sono file modello creati da Microsoft Word per avere impostazioni preformattate per la generazione di ulteriori file DOC o DOCX. Viene creato un file modello per avere impostazioni utente specifiche che dovrebbe essere applicato ai file successivi creati da questi. Ulteriori informazioni su questo formato di file[qui](https://wiki.fileformat.com/word-processing/dot/) . |
| Dotx | `5` | Rappresenta il formato .DOTX Word. I file con estensione DOTX sono file modello creati da Microsoft Word per avere impostazioni preformattate per la generazione di ulteriori file DOCX. Viene creato un file modello per avere impostazioni utente specifiche che devono essere applicate alle mosche successive create da questi. Ulteriori informazioni su questo formato di file[qui](https://wiki.fileformat.com/word-processing/dotx/) . |
| Dotm | `6` | Rappresenta il formato .DOTM Word. Un file con estensione DOTM rappresenta il file modello creato con Microsoft Word 2007 o versioni successive. È simile al popolare formato file DOCX ma conserva le impostazioni definite dall'utente per il riutilizzo in caso di creazione di nuovi documents. Ulteriori informazioni su questo formato di file[qui](https://wiki.fileformat.com/word-processing/dotm/) . |
| Odt | `7` | Rappresenta il formato .ODT. I file ODT sono tipi di documenti creati con applicazioni di elaborazione testi basate sul formato OpenDocument Text File. Vengono creati con applicazioni di elaborazione testi come OpenOffice Writer gratuito e possono contenere contenuti come testo, immagini , oggetti e stili. Ulteriori informazioni su questo formato di file[qui](https://wiki.fileformat.com/word-processing/odt/) . |
