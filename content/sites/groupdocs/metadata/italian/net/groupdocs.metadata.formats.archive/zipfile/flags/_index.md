## ZipFile.Flags property

Ottiene i flag di immissione ZIP.

```csharp

```

### Valore della proprietà

I contrassegni di immissione ZIP.

### Osservazioni

Bit 00: file crittografato. Bit 01: opzione di compressione. Bit 02: opzione di compressione. Bit 03: descrittore dati. Bit 04: deflazione potenziata. Bit 05: dati patch compressi. Bit 06: crittografia forte. Bit 07-10: non utilizzato. Bit 11: codifica lingua. Bit 12: riservato. Bit 13: maschera i valori di intestazione. Bit 14-15: riservato.
