## ZipPackage.Comment property

Ottiene o imposta il commento dell'archivio ZIP creato da un utente.

```csharp
public string Comment { get; set; }
```

### Valore della proprietà

Il commento dell'utente.

### Esempi

Il seguente frammento di codice mostra come rimuovere il commento dell'utente da un archivio ZIP.

```csharp
using (Metadata metadata = new Metadata(Constants.InputZip))
{
    var root = metadata.GetRootPackage<ZipRootPackage>();

    root.ZipPackage.Comment = null;

    metadata.Save(Constants.OutputZip);
}
```
