## ID3V2TagFrameFlags.GroupingIdentity property

Ottiene un valore che indica se il fotogramma appartiene a un gruppo di fotogrammi. Se impostato, un byte identificativo di gruppo viene aggiunto all'intestazione del frame. Ogni fotogramma con lo stesso identificatore di gruppo appartiene allo stesso gruppo.

```csharp
public bool GroupingIdentity { get; }
```

### Valore della proprietà

`VERO`, se il frame appartiene a un gruppo di frame; altrimenti, falso.
