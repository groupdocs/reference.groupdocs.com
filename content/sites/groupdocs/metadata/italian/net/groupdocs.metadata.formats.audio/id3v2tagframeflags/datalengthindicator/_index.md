## ID3V2TagFrameFlags.DataLengthIndicator property

Ottiene un valore che indica se un indicatore di lunghezza dei dati è stato aggiunto a il fotogramma. L'indicatore di lunghezza dei dati è il valore che si scriverebbe come 'Lunghezza del frame' se tutti i flag del formato del frame fossero azzerati, rappresentati come un numero intero synchsafe a 32 bit.

```csharp
public bool DataLengthIndicator { get; }
```

### Valore della proprietà

`VERO`se al frame è stato aggiunto un indicatore di lunghezza dei dati; altrimenti, falso.
