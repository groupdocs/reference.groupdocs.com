## XmpLangAlt class

Rappresenta l'alternativa del linguaggio XMP.

Una matrice alternativa di semplici elementi di testo. Le alternative linguistiche facilitano la selezione di un semplice elemento di testo basato sulla lingua desiderata. Ogni elemento dell'array deve avere un qualificatore xml:lang. Ogni valore xml:lang deve essere unico tra gli elementi.

public sealed class XmpLangAlt : XmpArray
