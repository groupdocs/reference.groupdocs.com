## MatroskaTrack.Language property

Ottiene la lingua della traccia nella forma delle lingue Matroska. Questo elemento DEVE essere ignorato se il[`LanguageIetf`](../languageietf) L'elemento è utilizzato nella stessa TrackEntry.

```csharp
public string Language { get; }
```

### Valore della proprietà

La lingua della traccia nella forma delle lingue Matroska.

### Osservazioni

I codici lingua possono essere sia il formato bibliografico ISO-639-2 di 3 lettere (come "fre" per il francese), o un codice lingua seguito da un trattino e un codice paese per specialità nelle lingue (come "fre-ca" per il francese canadese). I codici paese sono gli stessi utilizzati per i domini Internet.
