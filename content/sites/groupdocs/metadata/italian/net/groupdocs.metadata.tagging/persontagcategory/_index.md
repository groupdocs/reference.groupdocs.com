## PersonTagCategory class

Fornisce tag che contrassegnano le proprietà dei metadati contenenti informazioni sulle persone che hanno contribuito alla creazione di file o contenuti intellettuali. Questi tag possono aiutarti a trovare il creatore del documento, l'editore o anche il cliente per il quale è stato eseguito il lavoro. Nonostante il nome della categoria, alcune proprietà di metadati contrassegnate con i tag possono contenere un nome di società anziché il nome di una persona.

```csharp
public class PersonTagCategory : TagCategory
```
