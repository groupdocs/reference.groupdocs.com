Lo spazio dei nomi contiene vari set di tag con i quali vengono contrassegnate le proprietà dei metadati più importanti. I tag consentono agli utenti di trovare e aggiornare le proprietà dei metadati in diversi pacchetti indipendentemente dallo standard dei metadati e dal formato del file.

## Classi

| Classe | Descrizione |
| --- | --- |
| [ContentTagCategory](./contenttagcategory) | Fornisce tag collegati alle proprietà dei metadati che descrivono il contenuto di un file. I tag sono utili per scoprire la lingua del contenuto, il tipo (genere), l'oggetto, la valutazione, ecc. |
| [CorporateTagCategory](./corporatetagcategory) | Fornisce tag destinati a contrassegnare le proprietà dei metadati relative a un'azienda che ha partecipato alla creazione del file. |
| [DocumentTagCategory](./documenttagcategory) | Fornisce i tag che vengono applicati solo alle proprietà specifiche del documento. I tag possono essere utili per determinare da quale parte di un documento d'ufficio è stata estratta una proprietà. |
| [LegalTagCategory](./legaltagcategory) | Fornisce tag collegati alle proprietà dei metadati contenenti informazioni sui proprietari del contenuto del file e le regole in base alle quali il contenuto può essere utilizzato. |
| [OriginTagCategory](./origintagcategory) | Fornisce tag che aiutano un utente a determinare l'origine di un file (ad es. modello o altra fonte). |
| [PersonTagCategory](./persontagcategory) | Fornisce tag che contrassegnano le proprietà dei metadati contenenti informazioni sulle persone che hanno contribuito alla creazione di file o contenuti intellettuali. Questi tag possono aiutarti a trovare il creatore del documento, l'editore o anche il cliente per il quale è stato eseguito il lavoro. Nonostante il nome della categoria, alcune proprietà di metadati contrassegnate con i tag possono contenere un nome di società anziché il nome di una persona. |
| [PropertyTag](./propertytag) | Rappresenta un tag utilizzato per contrassegnare le proprietà dei metadati. |
| [PropertyTypeTagCategory](./propertytypetagcategory) | Fornisce tag che contengono informazioni aggiuntive sul tipo di una proprietà piuttosto che sul suo scopo. Utilizzando questi tag è possibile rilevare proprietà di metadati che contengono collegamenti URL a risorse esterne, proprietà che descrivono caratteri, colori, geolocalizzazione e così via. |
| [TagCategory](./tagcategory) | Rappresenta un insieme di tag con alcune caratteristiche condivise. |
| [Tags](./tags) | Contiene vari set di tag con i quali vengono contrassegnate le proprietà dei metadati più importanti. I tag consentono di trovare e aggiornare le proprietà dei metadati in diversi pacchetti indipendentemente dallo standard dei metadati e dal formato del file. |
| [TimeTagCategory](./timetagcategory) | Fornisce tag che contrassegnano le proprietà dei metadati utilizzate per descrivere il ciclo di vita di un file. I tag riguardano i momenti in cui un file o contenuto intellettuale è stato creato, modificato, stampato, ecc. |
| [ToolTagCategory](./tooltagcategory) | Fornisce tag destinati a contrassegnare le proprietà dei metadati relative agli strumenti (software e hardware) utilizzati per creare un file. |
