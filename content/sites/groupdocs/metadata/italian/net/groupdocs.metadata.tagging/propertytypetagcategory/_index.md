## PropertyTypeTagCategory class

Fornisce tag che contengono informazioni aggiuntive sul tipo di una proprietà piuttosto che sul suo scopo. Utilizzando questi tag è possibile rilevare proprietà di metadati che contengono collegamenti URL a risorse esterne, proprietà che descrivono caratteri, colori, geolocalizzazione e così via.

```csharp
public class PropertyTypeTagCategory : TagCategory
```
