## Tags class

Contiene vari set di tag con i quali vengono contrassegnate le proprietà dei metadati più importanti. I tag consentono di trovare e aggiornare le proprietà dei metadati in diversi pacchetti indipendentemente dallo standard dei metadati e dal formato del file.
