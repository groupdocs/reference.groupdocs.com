## ExifDictionaryBasePackage.Set method

Belirtilen etiketi ekler veya değiştirir.

```csharp
public void Set(TiffTag tag)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| tag | TiffTag | Ayarlanacak etiket. |

### Örnekler

Bu kod örneği, bir EXIF paketine nasıl özel etiket ekleneceğini gösterir.

```csharp
using (Metadata metadata = new Metadata(Constants.TiffWithExif))
{
    IExif root = metadata.GetRootPackage() as IExif;
    if (root != null)
    {
        // EXIF paketi yoksa ayarlayın
        if (root.ExifPackage == null)
        {
            root.ExifPackage = new ExifPackage();
        }

        // Bilinen bir özellik ekleyin
        root.ExifPackage.Set(new TiffAsciiTag(TiffTagID.Artist, "test artist"));

        // Tamamen özel bir özellik ekleyin (EXIF belirtiminde açıklanmayan).
        // Lütfen seçilen kimliğin bazı üçüncü taraf araçlar tarafından kullanılan kimliklerle kesişebileceğini unutmayın.
        root.ExifPackage.Set(new TiffAsciiTag((TiffTagID)65523, "custom"));

        metadata.Save(Constants.OutputTiff);
    }
}
```
