## MP3RootPackage.ID3V1 property

ID3v1 meta veri etiketini alır veya ayarlar. Lütfen daha fazla bilgiyi şu adreste bulabilirsiniz:[http://id3.org/ID3v1](http://id3.org/ID3v1) .

```csharp
public ID3V1Tag ID3V1 { get; set; }
```

### Mülk değeri

Ses dosyasına eklenmiş ID3v1 meta veri etiketi.

### Notlar

ID3(v1) etiketi, bir MP3 dosyasının sonundaki küçük bir ekstra veri yığınıdır.

### Örnekler

Bu kod örneği, bir MP3 dosyasındaki ID3v1 etiketinin nasıl güncelleneceğini gösterir.

```csharp
using (Metadata metadata = new Metadata(Constants.MP3WithID3V1))
{
    var root = metadata.GetRootPackage<MP3RootPackage>();

    if (root.ID3V1 == null)
    {
        root.ID3V1 = new ID3V1Tag();
    }

    root.ID3V1.Album = "test album";
    root.ID3V1.Artist = "test artist";
    root.ID3V1.Title = "test title";
    root.ID3V1.Comment = "test comment";
    root.ID3V1.Year = "2019";

    // ...

    metadata.Save(Constants.OutputMp3);
}
```
