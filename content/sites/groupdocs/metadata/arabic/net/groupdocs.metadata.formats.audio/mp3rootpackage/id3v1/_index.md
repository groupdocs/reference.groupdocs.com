## MP3RootPackage.ID3V1 property

الحصول على أو تعيين علامة البيانات الوصفية ID3v1 . يرجى العثور على مزيد من المعلومات على[http://id3.org/ID3v1](http://id3.org/ID3v1) .

```csharp
public ID3V1Tag ID3V1 { get; set; }
```

### Property_Value

علامة واصفات البيانات ID3v1 المرفقة بملف الصوت.

### ملاحظات

علامة ID3 (v1) هي جزء صغير من البيانات الإضافية في نهاية ملف MP3.

### أمثلة

يوضح نموذج التعليمات البرمجية هذا كيفية تحديث علامة ID3v1 في ملف MP3.

```csharp
using (Metadata metadata = new Metadata(Constants.MP3WithID3V1))
{
    var root = metadata.GetRootPackage<MP3RootPackage>();

    if (root.ID3V1 == null)
    {
        root.ID3V1 = new ID3V1Tag();
    }

    root.ID3V1.Album = "test album";
    root.ID3V1.Artist = "test artist";
    root.ID3V1.Title = "test title";
    root.ID3V1.Comment = "test comment";
    root.ID3V1.Year = "2019";

    // ...

    metadata.Save(Constants.OutputMp3);
}
```
