# GroupDocs.Redaction — API Reference Source Files

This repository holds the Markdown source files used to generate the **GroupDocs.Redaction**
API reference, published at <https://reference.groupdocs.com/redaction/>.

## About GroupDocs.Redaction

[GroupDocs.Redaction](https://products.groupdocs.com/redaction/) is a developer API for document
processing: remove sensitive text, metadata, and annotations from documents. It is available for the following platforms, each with its own
API reference:

- **.NET** — <https://reference.groupdocs.com/redaction/net/>
- **Java** — <https://reference.groupdocs.com/redaction/java/>
- **Python via .NET** — <https://reference.groupdocs.com/redaction/python-net/>

## Repository purpose

The files in this repository are used to:

- Build and maintain the official **GroupDocs.Redaction** API reference.
- Keep the reference in sync with the latest product release.
- Provide a consistent structure for automated Hugo builds and publication.

Content is English-only and compiled into a static site with [Hugo](https://gohugo.io/).
