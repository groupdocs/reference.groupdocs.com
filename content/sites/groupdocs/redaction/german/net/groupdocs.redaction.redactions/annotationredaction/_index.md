### Beispiele

Das folgende Beispiel zeigt, wie der Name „John“ in allen Anmerkungen durch „[redigiert]“ ersetzt wird.

```csharp
using (Redactor redactor = new Redactor(@"C:\test.pdf"))
{
   redactor.Apply(new AnnotationRedaction("(?im:john)", "[redacted]"));
   redactor.Save()
}
```
