## EraseMetadataRedaction class

### Exemples

L'exemple suivant montre comment effacer (définir égal à des valeurs vides) toutes les métadonnées ou des métadonnées spécifiques.

```csharp
using (Redactor redactor = new Redactor(@"C:\sample.docx"))
{
   // Effacer l'auteur, le responsable et l'entreprise
   redactor.Apply(new EraseMetadataRedaction(MetadataFilters.Author | MetadataFilters.Manager | MetadataFilters.Company));
   // Efface toutes les métadonnées
   redactor.Apply(new EraseMetadataRedaction(MetadataFilters.All));
   redactor.Save();
}
```
