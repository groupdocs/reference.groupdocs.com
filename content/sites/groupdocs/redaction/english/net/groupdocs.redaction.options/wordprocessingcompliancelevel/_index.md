## WordProcessingComplianceLevel enumeration

Represents the supported OOXML compliance levels for word processing documents. The Strict compliance level can only be downgraded to Transitional, not to Ecma.

```csharp
public enum WordProcessingComplianceLevel
```

### Values

| Name | Value | Description |
| --- | --- | --- |
| Ecma | `0` | Specifies the ECMA-376 1st Edition, 2006 compliance level. |
| Transitional | `1` | Specifies the ISO/IEC 29500:2008 Transitional compliance level. |
| Strict | `2` | Specifies the ISO/IEC 29500:2008 Strict compliance level. |
