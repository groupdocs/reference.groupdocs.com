The Redaction namespace provides classes for redacting sensitive information from documents in PDF, raster image and office document formats. The main classes in this namespace are:

## Classes

| Class | Description |
| --- | --- |
| [DocumentInfo](./documentinfo) | Represents an information about document. Implements IDocumentInfo interface. See [`IDocumentInfo`](../groupdocs.redaction/idocumentinfo) for examples. |
| [FileType](./filetype) | Represents a file type. Provides methods to obtain a list of all file types supported by GroupDocs.Redaction, detect file type by extension, etc. |
| [License](./license) | Provides methods for applying license. |
| [Metered](./metered) | Provides methods which allow to activate product with Metered license and retrieve amount of MBs processed. Learn more about Metered licenses [here](https://purchase.groupdocs.com/faqs/licensing/metered). |
| [PageInfo](./pageinfo) | Represents a brief page information. |
| [Redaction](./redaction) | Represents a base abstract class for all redaction types. |
| [RedactionPolicy](./redactionpolicy) | Represents a sanitization policy, containing a set of specific redactions to apply. |
| [RedactionResult](./redactionresult) | Represents a result of the redaction operation. |
| [Redactor](./redactor) | Represents a main class that controls document redaction process, allowing to open, redact and save documents. |
| [RedactorChangeLog](./redactorchangelog) | Represents results for a list of redactions, passed to Apply() method of [`Redactor`](../groupdocs.redaction/redactor) class. |
| [RedactorLogEntry](./redactorlogentry) | Represents results of applying redaction. |
