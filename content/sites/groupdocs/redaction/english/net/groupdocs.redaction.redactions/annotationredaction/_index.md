### Examples

The following example demonstrates how to replace the name "John" with "[redacted]" in all annotations.

```csharp
using (Redactor redactor = new Redactor(@"C:\test.pdf"))
{
   redactor.Apply(new AnnotationRedaction("(?im:john)", "[redacted]"));
   redactor.Save()
}
```
