### Voorbeelden

Het volgende voorbeeld laat zien hoe u de naam "John" kunt vervangen door "[geredigeerd]" in alle annotaties.

```csharp
using (Redactor redactor = new Redactor(@"C:\test.pdf"))
{
   redactor.Apply(new AnnotationRedaction("(?im:john)", "[redacted]"));
   redactor.Save()
}
```
