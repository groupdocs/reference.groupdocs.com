Het volgende voorbeeld laat zien hoe u de laatste pagina van het document kunt verwijderen.

```csharp
using (Redactor redactor = new Redactor(@"C:\test.pdf"))
{
   redactor.Apply(new RemovePageRedaction(PageSeekOrigin.End, 0, 1));
   redactor.Save()
}
```
