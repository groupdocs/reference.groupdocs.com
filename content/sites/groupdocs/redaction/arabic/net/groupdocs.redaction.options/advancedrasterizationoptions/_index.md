## AdvancedRasterizationOptions enumeration

تعداد الإشارات لإدارة خيارات التنقيط المتقدمة المطلوب تطبيقها.

```csharp
public enum AdvancedRasterizationOptions
```

### قيم

| اسم | قيمة | وصف |
| --- | --- | --- |
| None | `0` | لا توجد خيارات متقدمة للتطبيق . |
| Tilt | `1` | إمالة لإمالة الصورة المنقطة إلى زاوية عشوائية. |
| Noise | `2` | أضف بقعًا عشوائية إلى صور الصفحة المنقطة. |
| Border | `4` | أضف خط حد لتقليد تأثير مسح الصفحة. |
| Grayscale | `8` | جعل صور الصفحة ذات تدرج رمادي لتقليد المسح الضوئي بالتدرج الرمادي. |

### أمثلة

يوضح المثال التالي كيفية تطبيق خيارات التنقيط المتقدمة بالإعدادات الافتراضية.

```csharp
    using (Redactor redactor = new Redactor(@"C:\sample.docx"))
    {
      // احفظ المستند بالخيارات الافتراضية (تحويل الصفحات إلى صور ، حفظ كملف PDF)
      var so = new SaveOptions();
      so.Rasterization.Enabled = true;
      so.RedactedFileSuffix = "_scan";
      so.Rasterization.AddAdvancedOption(AdvancedRasterizationOptions.Border);
      so.Rasterization.AddAdvancedOption(AdvancedRasterizationOptions.Noise);
      so.Rasterization.AddAdvancedOption(AdvancedRasterizationOptions.Grayscale);
      so.Rasterization.AddAdvancedOption(AdvancedRasterizationOptions.Tilt);
      redactor.Save(so);
    }
```
