### Exempel

Följande exempel visar hur man ersätter namnet "John" med "[redacted]" i alla anteckningar.

```csharp
using (Redactor redactor = new Redactor(@"C:\test.pdf"))
{
   redactor.Apply(new AnnotationRedaction("(?im:john)", "[redacted]"));
   redactor.Save()
}
```
