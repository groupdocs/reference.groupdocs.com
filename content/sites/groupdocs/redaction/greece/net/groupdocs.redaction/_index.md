ΤοRedaction Το namespace παρέχει κλάσεις για τη διόρθωση ευαίσθητων πληροφοριών από έγγραφα σε μορφή PDF, εικόνας ράστερ και εγγράφων γραφείου. Οι κύριες κλάσεις σε αυτόν τον χώρο ονομάτων είναι:

## Τάξεις

| Τάξη | Περιγραφή |
| --- | --- |
| [DocumentInfo](./documentinfo) | Αντιπροσωπεύει μια πληροφορία σχετικά με το έγγραφο. Υλοποιεί τη διεπαφή IDocumentInfo. Βλέπω[`IDocumentInfo`](../groupdocs.redaction/idocumentinfo) για παραδείγματα. |
| [FileType](./filetype) | Αντιπροσωπεύει έναν τύπο αρχείου. Παρέχει μεθόδους για τη λήψη μιας λίστας όλων των τύπων αρχείων που υποστηρίζονται από το GroupDocs.Redaction, εντοπισμός τύπου αρχείου κατά επέκταση κ.λπ. |
| [License](./license) | Παρέχει μεθόδους για την εφαρμογή άδειας. |
| [Metered](./metered) | Παρέχει μεθόδους που επιτρέπουν την ενεργοποίηση προϊόντος με άδεια Metered και την ανάκτηση του όγκου των MB που έχουν υποστεί επεξεργασία. Μάθετε περισσότερα για τις άδειες Metered[εδώ](https://purchase.groupdocs.com/faqs/licensing/metered) . |
| [PageInfo](./pageinfo) | Αντιπροσωπεύει μια σύντομη πληροφορία σελίδας. |
| [Redaction](./redaction) | Αντιπροσωπεύει μια βασική αφηρημένη κλάση για όλους τους τύπους διόρθωσης. |
| [RedactionPolicy](./redactionpolicy) | Αντιπροσωπεύει μια πολιτική απολύμανσης, η οποία περιέχει ένα σύνολο συγκεκριμένων διορθώσεων προς εφαρμογή. |
| [RedactionResult](./redactionresult) | Αντιπροσωπεύει ένα αποτέλεσμα της λειτουργίας διόρθωσης. |
| [Redactor](./redactor) | Αντιπροσωπεύει μια κύρια κλάση που ελέγχει τη διαδικασία επεξεργασίας εγγράφων, επιτρέποντας το άνοιγμα, την επεξεργασία και την αποθήκευση εγγράφων. |
| [RedactorChangeLog](./redactorchangelog) | Αντιπροσωπεύει τα αποτελέσματα για μια λίστα διορθώσεων, που μεταβιβάστηκαν στη μέθοδο Apply() του[`Redactor`](../groupdocs.redaction/redactor) τάξη. |
| [RedactorLogEntry](./redactorlogentry) | Αντιπροσωπεύει τα αποτελέσματα της εφαρμογής της διόρθωσης. |
