## DeleteAnnotationRedaction class

### Παραδείγματα

Το ακόλουθο παράδειγμα δείχνει πώς να αφαιρέσετε όλους τους σχολιασμούς που περιέχουν λέξεις "χρήση", "εμφάνιση" ή "περιγραφή" από το έγγραφο (και αφήστε άλλα).

```csharp
using (Redactor redactor = new Redactor(@"D:\test.docx"))
{
   redactor.Apply(new DeleteAnnotationRedaction("(?im:(use|show|describe))"));
   redactor.Save()
}
```
