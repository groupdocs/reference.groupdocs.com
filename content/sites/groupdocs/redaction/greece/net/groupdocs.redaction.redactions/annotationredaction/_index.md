### Παραδείγματα

Το ακόλουθο παράδειγμα δείχνει πώς να αντικαταστήσετε το όνομα "John" με το "[redacted]" σε όλους τους σχολιασμούς.

```csharp
using (Redactor redactor = new Redactor(@"C:\test.pdf"))
{
   redactor.Apply(new AnnotationRedaction("(?im:john)", "[redacted]"));
   redactor.Save()
}
```
