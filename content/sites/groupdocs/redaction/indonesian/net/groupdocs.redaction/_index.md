ItuRedaction namespace menyediakan kelas untuk menyunting informasi sensitif dari dokumen dalam format PDF, gambar raster dan dokumen kantor. Kelas utama dalam ruang nama ini adalah:

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [DocumentInfo](./documentinfo) | Merupakan informasi tentang dokumen. Menerapkan antarmuka IDocumentInfo. Melihat[`IDocumentInfo`](../groupdocs.redaction/idocumentinfo) untuk contoh. |
| [FileType](./filetype) | Merupakan jenis file. Menyediakan metode untuk mendapatkan daftar semua jenis file yang didukung oleh GroupDocs.Redaction, mendeteksi jenis file dengan ekstensi, dll. |
| [License](./license) | Menyediakan metode untuk menerapkan lisensi. |
| [Metered](./metered) | Menyediakan metode yang memungkinkan untuk mengaktifkan produk dengan lisensi Terukur dan mengambil jumlah MB yang diproses. Pelajari lebih lanjut tentang Lisensi Terukur[Di Sini](https://purchase.groupdocs.com/faqs/licensing/metered) . |
| [PageInfo](./pageinfo) | Mewakili informasi halaman singkat. |
| [Redaction](./redaction) | Merupakan kelas abstrak dasar untuk semua jenis redaksi. |
| [RedactionPolicy](./redactionpolicy) | Mewakili kebijakan sanitasi, berisi kumpulan redaksi khusus untuk diterapkan. |
| [RedactionResult](./redactionresult) | Mewakili hasil operasi redaksi. |
| [Redactor](./redactor) | Mewakili kelas utama yang mengontrol proses penyuntingan dokumen, memungkinkan untuk membuka, menyunting, dan menyimpan dokumen. |
| [RedactorChangeLog](./redactorchangelog) | Mewakili hasil untuk daftar redaksi, diteruskan ke metode Apply()[`Redactor`](../groupdocs.redaction/redactor) kelas. |
| [RedactorLogEntry](./redactorlogentry) | Mewakili hasil penerapan redaksi. |
