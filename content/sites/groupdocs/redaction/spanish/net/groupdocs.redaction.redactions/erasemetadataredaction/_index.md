### Ejemplos

El siguiente ejemplo muestra cómo borrar (establecer valores iguales a vacíos) todos o metadatos específicos.

```csharp
using (Redactor redactor = new Redactor(@"C:\sample.docx"))
{
   // Borrar Autor, Gerente y Empresa
   redactor.Apply(new EraseMetadataRedaction(MetadataFilters.Author | MetadataFilters.Manager | MetadataFilters.Company));
   // Borrar todos los metadatos
   redactor.Apply(new EraseMetadataRedaction(MetadataFilters.All));
   redactor.Save();
}
```
