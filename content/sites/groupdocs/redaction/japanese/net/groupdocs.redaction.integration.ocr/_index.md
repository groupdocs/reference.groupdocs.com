Ocr名前空間は、光学式文字認識 (OCR) のサポートに必要なインターフェイスとクラスを提供します。

## クラス

| クラス | 説明 |
| --- | --- |
| [RecognizedImage](./recognizedimage) | 認識プロセスの結果として画像から抽出されたテキストを表します。 |
| [TextFragment](./textfragment) | OCR エンジンによって抽出された、認識されたテキスト (単語、記号など) の一部を表します。 |
| [TextLine](./textline) | OCR エンジンによって画像から抽出されたテキスト行を表します。 |

| インターフェース | 説明 |
| --- | --- |
