Redaction名前空間は、PDF、ラスター イメージ、およびオフィス ドキュメント形式のドキュメントの機密情報を編集するためのクラスを提供します。 この名前空間の主なクラスは次のとおりです:

## クラス

| クラス | 説明 |
| --- | --- |
| [DocumentInfo](./documentinfo) | ドキュメントに関する情報を表します。 IDocumentInfo インターフェイスを実装します。見る[`IDocumentInfo`](../groupdocs.redaction/idocumentinfo)例. |
| [FileType](./filetype) | ファイルの種類を表します。 GroupDocs.Redaction でサポートされているすべてのファイル タイプのリストを取得するメソッド、拡張子でファイル タイプを検出するメソッドなどを提供します。 |
| [License](./license) | ライセンスを適用する方法を提供します。 |
| [Metered](./metered) | 従量制ライセンスで製品をアクティベートし、処理された MB の量を取得できるメソッドを提供します。 従量制ライセンスの詳細[ここ](https://purchase.groupdocs.com/faqs/licensing/metered) . |
| [PageInfo](./pageinfo) | 簡単なページ情報を表します. |
| [Redaction](./redaction) | すべての編集タイプの基本抽象クラスを表します. |
| [RedactionPolicy](./redactionpolicy) | 適用する特定のリダクションのセットを含むサニタイズ ポリシーを表します. |
| [RedactionResult](./redactionresult) | リダクション操作の結果を表します。 |
| [Redactor](./redactor) | ドキュメントの編集プロセスを制御するメイン クラスを表し、ドキュメントを開く、編集する、保存することができます。 |
| [RedactorChangeLog](./redactorchangelog) | の Apply() メソッドに渡されたリダクションのリストの結果を表します[`Redactor`](../groupdocs.redaction/redactor)class. |
| [RedactorLogEntry](./redactorlogentry) | リダクションを適用した結果を表します。 |
