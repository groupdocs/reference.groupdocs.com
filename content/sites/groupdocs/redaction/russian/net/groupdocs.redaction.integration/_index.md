Integrationnamespace предоставляет интерфейсы и классы, используемые для интеграции документов разных форматов с GroupDocs.Redaction.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [DocumentFormatInstance](./documentformatinstance) | Представляет определенный формат документа. Реализуйте этот класс, чтобы добавить свои собственные типы документов. |
| [MetadataCollection](./metadatacollection) | Представляет словарь[`MetadataItem`](../groupdocs.redaction.integration/metadataitem) с его заголовком в качестве ключа. |
| [MetadataItem](./metadataitem) | Представляет элемент метаданных, общий для всех поддерживаемых форматов и используемый при редакциях метаданных. |
## Интерфейсы

| Интерфейс | Описание |
| --- | --- |
| [IAnnotatedDocument](./iannotateddocument) | Определяет методы, необходимые для доступа к аннотациям, например комментариям. Должен быть реализован[`DocumentFormatInstance`](../groupdocs.redaction.integration/documentformatinstance) производный класс для редактирования аннотаций. |
| [ICellularFormatInstance](./icellularformatinstance) | Определяет методы, необходимые для доступа к форматам электронных таблиц, имеющим один или несколько рабочих листов. |
| [IImageFormatInstance](./iimageformatinstance) | Определяет методы, необходимые для редактирования формата растрового изображения. |
| [IMetadataAccess](./imetadataaccess) | Определяет методы, необходимые для доступа к метаданным документа, если формат поддерживает это. |
| [IPaginatedDocument](./ipaginateddocument) | Определяет методы, необходимые для управления страницами документа. Должен быть реализован[`DocumentFormatInstance`](../groupdocs.redaction.integration/documentformatinstance) производный класс для редактирования страниц. |
| [IPreviewable](./ipreviewable) | Определяет методы для создания предварительного просмотра документа. |
| [IRasterizableDocument](./irasterizabledocument) | Определяет методы, необходимые для сохранения документа в любой двоичной форме. Встроенные типы сохраняют документ в формате PDF с изображениями его страниц. |
| [ITextualFormatInstance](./itextualformatinstance) | Определяет методы, необходимые для редактирования текстовых данных в любом документе, содержащем текст. |
