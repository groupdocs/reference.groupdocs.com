Ocr namespace предоставляет интерфейсы и классы, необходимые для поддержки оптического распознавания символов (OCR).

## Классы

| Учебный класс | Описание |
| --- | --- |
| [RecognizedImage](./recognizedimage) | Представляет собой текст, извлеченный из изображения в результате процесса его распознавания. |
| [TextFragment](./textfragment) | Представляет часть распознанного текста (слово, символ и т. д.), извлеченную механизмом OCR. |
| [TextLine](./textline) | Представляет строку текста, извлеченную механизмом OCR из изображения. |
