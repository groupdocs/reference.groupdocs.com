## AdvancedRasterizationOptions enumeration

Перечисление флагов для управления применяемыми дополнительными параметрами растеризации.

```csharp
public enum AdvancedRasterizationOptions
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| None | `0` | Нет дополнительных параметров для применения. |
| Tilt | `1` | Наклон, чтобы наклонить растровое изображение под произвольным углом. |
| Noise | `2` | Добавить случайные точки к растровым изображениям страниц. |
| Border | `4` | Добавить границу для имитации эффекта сканирования страницы. |
| Grayscale | `8` | Сделать изображения страниц в градациях серого, чтобы имитировать сканирование в градациях серого. |

### Примеры

В следующем примере показано, как применить расширенные параметры растеризации с настройками по умолчанию.

```csharp
    using (Redactor redactor = new Redactor(@"C:\sample.docx"))
    {
      // Сохраняем документ с параметрами по умолчанию (преобразование страниц в изображения, сохранение в формате PDF)
      var so = new SaveOptions();
      so.Rasterization.Enabled = true;
      so.RedactedFileSuffix = "_scan";
      so.Rasterization.AddAdvancedOption(AdvancedRasterizationOptions.Border);
      so.Rasterization.AddAdvancedOption(AdvancedRasterizationOptions.Noise);
      so.Rasterization.AddAdvancedOption(AdvancedRasterizationOptions.Grayscale);
      so.Rasterization.AddAdvancedOption(AdvancedRasterizationOptions.Tilt);
      redactor.Save(so);
    }
```
