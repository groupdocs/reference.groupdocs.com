This page serves as the entry point to the **GroupDocs.Annotation API references**.

**GroupDocs.Annotation** is a set of on-premise APIs that add, render, and manage annotations — text markups, graphic shapes, comments and replies, watermarks, and image stamps — across popular document formats such as **PDF, Word, Excel, PowerPoint, images, and CAD**, on multiple platforms. The same annotation engine powers .NET, Java, and Python (via .NET), so you get consistent behavior across your whole stack.

## Additional Resources

### Product Overview

**GroupDocs.Annotation** is a comprehensive document-annotation SDK that gives developers full programmatic control over adding, rendering, and managing annotations — without requiring Microsoft Office, Adobe Acrobat, or any other third-party software. The same annotation engine is available for .NET, Java, and Python (via .NET), so you get consistent behavior across your whole stack.

#### Key Features

- **Text Markup**: Highlight, underline, strike out, squiggly-underline, redact, or replace text.
- **Graphic Annotations**: Add area, ellipse, point, arrow, distance, and polyline shapes.
- **Rich Annotations**: Add watermarks, text fields, hyperlinks, and image stamps.
- **Comments & Replies**: Attach an author and threaded replies to any annotation.
- **Manage Annotations**: Add, update, get, and remove annotations by id, type, or instance.
- **Import / Export**: Load existing annotations and export them to another document.
