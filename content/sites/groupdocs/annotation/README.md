# GroupDocs.Annotation — API Reference Source Files

This repository holds the Markdown source files used to generate the **GroupDocs.Annotation**
API reference, published at <https://reference.groupdocs.com/annotation/>.

## About GroupDocs.Annotation

[GroupDocs.Annotation](https://products.groupdocs.com/annotation/) is a developer API for document
processing: add, edit, and remove annotations — comments, highlights, shapes, and stamps. It is available for the following platforms, each with its own
API reference:

- **.NET** — <https://reference.groupdocs.com/annotation/net/>
- **Java** — <https://reference.groupdocs.com/annotation/java/>
- **Python via .NET** — <https://reference.groupdocs.com/annotation/python-net/>

## Repository purpose

The files in this repository are used to:

- Build and maintain the official **GroupDocs.Annotation** API reference.
- Keep the reference in sync with the latest product release.
- Provide a consistent structure for automated Hugo builds and publication.

Content is English-only and compiled into a static site with [Hugo](https://gohugo.io/).
