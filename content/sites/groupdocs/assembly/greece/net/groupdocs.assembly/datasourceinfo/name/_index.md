## DataSourceInfo.Name property

Λαμβάνει ή ορίζει το όνομα του αντικειμένου πηγής δεδομένων που θα χρησιμοποιηθεί για την πρόσβαση στο αντικείμενο προέλευσης δεδομένων σε ένα έγγραφο προτύπου.

```csharp
public string Name { get; set; }
```

### Παρατηρήσεις

Όταν καθοριστεί το όνομα του αντικειμένου πηγής δεδομένων, μπορείτε να αποκτήσετε πρόσβαση στο αντικείμενο προέλευσης δεδομένων και στα μέλη του σε ένα πρότυπο έγγραφο χρησιμοποιώντας το όνομα.

Όταν το όνομα του αντικειμένου της πηγής δεδομένων είναι μηδενικό ή κενό, μπορείτε ακόμα να αποκτήσετε πρόσβαση σε μέλη του αντικειμένου πηγής δεδομένων σε ένα έγγραφο προτύπου χρησιμοποιώντας την πρόσβαση μέλους αντικειμένου περιβάλλοντος (δείτε Αναφορά σύνταξης προτύπου για περισσότερες πληροφορίες), αλλά δεν μπορείτε να αποκτήσετε πρόσβαση στο Το ίδιο το αντικείμενο πηγής δεδομένων.

Όταν περνάτε πολλαπλά[`DataSourceInfo`](../../datasourceinfo) περιπτώσεις σε[`DocumentAssembler`](../../documentassembler) , μόνο το όνομα του του πρώτου αντικειμένου πηγής δεδομένων μπορεί να είναι μηδενικό ή κενό. Τα ονόματα των υπολοίπων πρέπει να είναι καθορισμένα και μοναδικά.
