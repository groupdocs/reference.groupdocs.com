## LoadSaveOptions.ResourceSaveFolder property

Λαμβάνει ή ορίζει μια διαδρομή προς έναν φάκελο για την αποθήκευση εξωτερικών αρχείων πόρων, ενώ ένα συναρμολογημένο έγγραφο που έχει φορτωθεί από μορφή μη HTML αποθηκεύεται σε HTML. Η προεπιλεγμένη τιμή είναι μια κενή συμβολοσειρά.

```csharp
public string ResourceSaveFolder { get; set; }
```

### Παρατηρήσεις

Από προεπιλογή, κατά την αποθήκευση ενός συναρμολογημένου εγγράφου σε ένα αρχείο HTML, τα αρχεία εξωτερικών πόρων αποθηκεύονται σε έναν φάκελο που έχει το ίδιο όνομα με το αρχείο HTML χωρίς επέκταση συν το επίθημα "_files". Αυτός ο φάκελος βρίσκεται στον στον ίδιο φάκελο με το αρχείο HTML. Ωστόσο, αυτό δεν μπορεί να γίνει κατά την αποθήκευση ενός συναρμολογημένου εγγράφου σε μια ροή HTML. Ορίστε αυτήν την ιδιότητα για να καθορίσετε μια διαδρομή προς έναν φάκελο για την αποθήκευση αρχείων εξωτερικών πόρων κατά την αποθήκευση ενός συναρμολογημένου document σε μια ροή HTML ή να παρακάμψετε τον προεπιλεγμένο φάκελο κατά την αποθήκευση ενός συναρμολογημένο έγγραφο σε αρχείο HTML.

Μια τιμή αυτής της ιδιότητας αγνοείται εάν ένα συναρμολογημένο έγγραφο που αποθηκεύεται σε HTML φορτώθηκε επίσης από HTML (τα αρχεία εξωτερικών πόρων δεν αποθηκεύονται και οι σύνδεσμοι προς αυτά δεν αλλάζουν τότε).
