## LoadSaveOptions.ResourceLoadBaseUri property

Λαμβάνει ή ορίζει ένα βασικό URI για την επίλυση των σχετικών URI των αρχείων εξωτερικών πόρων σε απόλυτα κατά τη φόρτωση ενός εγγράφου προτύπου HTML που πρόκειται να συναρμολογηθεί και να αποθηκευτεί σε μορφή μη HTML. Η προεπιλεγμένη τιμή είναι μια κενή συμβολοσειρά.

```csharp
public string ResourceLoadBaseUri { get; set; }
```

### Παρατηρήσεις

Κατά τη φόρτωση ενός εγγράφου HTML από ένα αρχείο, ο φάκελος που περιέχει χρησιμοποιείται ως βασικό URI από προεπιλογή, κάτι που δεν μπορεί να συμβεί κατά τη φόρτωση ενός εγγράφου HTML από μια ροή. Ρυθμίστε αυτήν την ιδιότητα για να καθορίσετε ένα βασικό URI κατά τη φόρτωση ενός εγγράφου HTML από μια ροή ή για να παρακάμψετε το προεπιλεγμένο βασικό URI κατά τη φόρτωση ενός εγγράφου HTML από ένα αρχείο.

Μια τιμή αυτής της ιδιότητας αγνοείται στις ακόλουθες περιπτώσεις:

* Ένα έγγραφο HTML που φορτώνεται περιέχει ένα στοιχείο BASE HTML που παρέχει ένα βασικό URI.
* Ένα έγγραφο HTML που φορτώνεται πρέπει να συναρμολογηθεί και να αποθηκευτεί σε HTML (τα αρχεία εξωτερικών πόρων δεν φορτώνονται και τα σχετικά URI δεν αλλάζουν τότε).
