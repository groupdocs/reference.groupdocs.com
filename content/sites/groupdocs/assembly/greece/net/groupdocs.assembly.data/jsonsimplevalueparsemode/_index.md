## JsonSimpleValueParseMode enumeration

Καθορίζει μια λειτουργία ανάλυσης απλών τιμών JSON (null, boolean, αριθμός, ακέραιος και συμβολοσειρά) κατά τη φόρτωση του JSON. Μια τέτοια λειτουργία δεν επηρεάζει την ανάλυση των τιμών ημερομηνίας-ώρας.

```csharp
public enum JsonSimpleValueParseMode
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| Loose | `0` | Καθορίζει τη λειτουργία όπου οι τύποι απλών τιμών JSON καθορίζονται κατά την ανάλυση των αναπαραστάσεων συμβολοσειράς τους. Για παράδειγμα, ο τύπος 'prop' από το απόσπασμα JSON '{ prop: "123" }' καθορίζεται ως ακέραιος σε αυτήν τη λειτουργία. |
| Strict | `1` | Καθορίζει τη λειτουργία όπου οι τύποι απλών τιμών JSON καθορίζονται από την ίδια τη σημείωση JSON. Για παράδειγμα, ο τύπος του 'prop' από το απόσπασμα JSON '{ prop: "123" }' καθορίζεται ως συμβολοσειρά σε αυτήν τη λειτουργία. |
