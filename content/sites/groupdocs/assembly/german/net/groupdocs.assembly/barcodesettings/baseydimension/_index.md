## BarcodeSettings.BaseYDimension property

Ruft eine Basis-y-Abmessung ab oder legt sie fest, d. h. die kleinste Höhe der Einheit von 2D-Barcodemodulen. Gemessen in[`GraphicsUnit`](../graphicsunit) .

```csharp
public float BaseYDimension { get; set; }
```

### Bemerkungen

Barcodes einiger Typen (z. B. Data Matrix) können eine y-Dimension ignorieren und eine x-Dimension sowohl für Breiten- als auch für Höheneinheiten verwenden.

Wenn die Barcode-Skalierung über eine Vorlage angewendet wird, wird eine tatsächliche y-Abmessung anhand der Basis-y-Abmessung und eines Skalierungsfaktors berechnet.
