## BarcodeSettings.UseAutoCorrection property

Erhält oder setzt einen Wert, der angibt, ob ein ungültiger Barcode-Wert automatisch (falls möglich) korrigiert werden soll, damit er der Spezifikation des Barcodes entspricht, oder ob eine Ausnahme ausgelöst werden soll, um den Fehler anzuzeigen. Der Standardwert ist wahr.

```csharp
public bool UseAutoCorrection { get; set; }
```

### Bemerkungen

Die Autokorrektur ist für Databar-Barcodes nicht möglich.
