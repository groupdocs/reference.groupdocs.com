## DocumentTableSet class

Bietet Zugriff auf Daten mehrerer Tabellen (oder Arbeitsblätter) in einem externen Dokument, die beim Zusammenstellen eines Dokuments verwendet werden. Ermöglicht außerdem die Definition von Eltern-Kind-Beziehungen für die Dokumenttabellen und vereinfacht so den Zugriff auf zugehörige Daten in Vorlagendokumenten.

### Bemerkungen

Für Dokumente im Spreadsheet-Dateiformat a[`DocumentTableSet`](../documenttableset) Instanz repräsentiert einen Satz Blätter. Für Dokumente anderer Dateiformate, a[`DocumentTableSet`](../documenttableset) Instanz stellt eine Reihe von Tabellen dar.

Um beim Zusammenstellen eines Dokuments auf Daten der entsprechenden Tabellen zuzugreifen, übergeben Sie eine Instanz dieser Klasse als eine Datenquelle an eine von[`DocumentAssembler`](../../groupdocs.assembly/documentassembler) .AssembleDocument Überladungen.

In Vorlagendokumenten a[`DocumentTableSet`](../documenttableset) Instanz sollte genauso behandelt werden, als wäre es aDataSet Beispiel. Weitere Informationen finden Sie in der Referenz zur Vorlagensyntax.
