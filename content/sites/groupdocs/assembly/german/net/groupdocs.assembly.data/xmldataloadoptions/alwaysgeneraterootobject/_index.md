## XmlDataLoadOptions.AlwaysGenerateRootObject property

Ruft ein Flag ab oder setzt es, das angibt, ob eine generierte Datenquelle immer ein Objekt für ein XML-Root- -Element enthält. Wenn ein XML-Stammelement keine Attribute hat und alle untergeordneten Elemente dieselben Namen haben, wird ein solches Objekt standardmäßig nicht erstellt.

```csharp
public bool AlwaysGenerateRootObject { get; set; }
```

### Bemerkungen

Der Standardwert ist**FALSCH** .
