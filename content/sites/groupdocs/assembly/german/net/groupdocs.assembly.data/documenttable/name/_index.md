## DocumentTable.Name property

Ruft den Namen dieser Tabelle ab oder legt ihn fest, die verwendet wird, um auf die Daten der Tabelle in einem Vorlagendokument zuzugreifen, das an übergeben wird.[`DocumentAssembler`](../../../groupdocs.assembly/documentassembler) .

```csharp
public string Name { get; set; }
```

### Bemerkungen

Wenn der Name der Tabelle aus einem Dokument gelesen wird, wird der Name automatisch korrigiert, damit er gültig ist. Wenn der Name der Tabelle jedoch manuell über diese Eigenschaft festgelegt wird und der Name ungültig ist, wird eine Ausnahme ausgelöst.

Der Name der Tabelle gilt als gültig, wenn folgende Bedingungen erfüllt sind:

* Der Name ist nicht leer.
* Das erste Zeichen des Namens ist ein Buchstabe oder ein Unterstrich.
* Die restlichen Zeichen des Namens sind Buchstaben, Unterstriche, Ziffern oder die folgenden Zeichen: '@', '#', '$'.
* Das entsprechende[`DocumentTableSet`](../../documenttableset) Objekt enthält kein a[`DocumentTable`](../../documenttable)Instanz mit demselben Namen.
