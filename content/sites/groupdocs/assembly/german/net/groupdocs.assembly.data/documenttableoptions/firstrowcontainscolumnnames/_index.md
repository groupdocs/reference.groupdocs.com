## DocumentTableOptions.FirstRowContainsColumnNames property

Ruft einen Wert ab oder legt einen Wert fest, der angibt, ob Spaltennamen aus der ersten extrahierten Zeile einer Dokumenttabelle abgerufen werden sollen. Der Standardwert ist false.

```csharp
public bool FirstRowContainsColumnNames { get; set; }
```

### Bemerkungen

Wenn Spaltennamen nicht so eingestellt sind, dass sie aus der ersten extrahierten Zeile einer Dokumenttabelle abgerufen werden, werden stattdessen Standardspaltennamen verwendet. Für Dokumente im Spreadsheet-Dateiformat sind standardmäßige -Spaltennamen als A, B, C, ... Z, AA, AB usw. definiert. Für Dokumente in anderen -Dateiformaten sind die Standardspaltennamen als Spalte1, Spalte2, Spalte3 usw. definiert.
