## DocumentTableColumn.Name property

Ruft den Namen dieser Spalte ab oder legt diesen fest, der verwendet wird, um auf die Daten der Spalte in einem Vorlagendokument zuzugreifen, das an übergeben wird.[`DocumentAssembler`](../../../groupdocs.assembly/documentassembler) .

```csharp
public string Name { get; set; }
```

### Bemerkungen

Wenn der Name der Spalte aus einem Dokument gelesen wird (vgl[`FirstRowContainsColumnNames`](../../documenttableoptions/firstrowcontainscolumnnames) ), wird der Name automatisch korrigiert, damit er gültig ist. Wenn der Name der Spalte jedoch manuell über diese Eigenschaft festgelegt wird und der Name ungültig ist, wird eine Ausnahme ausgelöst.

Der Name der Spalte gilt als gültig, wenn die folgenden Bedingungen erfüllt sind:

* Der Name ist nicht leer.
* Das erste Zeichen des Namens ist ein Buchstabe oder ein Unterstrich.
* Die restlichen Zeichen des Namens sind Buchstaben, Unterstriche, Ziffern oder die folgenden Zeichen: '@', '#', '$'.
* Das entsprechende[`DocumentTable`](../../documenttable) Objekt enthält kein a[`DocumentTableColumn`](../../documenttablecolumn) Instanz mit demselben Namen.
