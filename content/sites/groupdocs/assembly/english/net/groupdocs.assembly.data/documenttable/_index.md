### Remarks

For documents of Spreadsheet file formats, a [`DocumentTable`](../documenttable) instance represents a single sheet. For documents of other file formats, a [`DocumentTable`](../documenttable) instance represents a single table.

To access data of the corresponding table while assembling a document, pass an instance of this class as a data source to one of [`DocumentAssembler`](../../groupdocs.assembly/documentassembler).AssembleDocument overloads.

In template documents, a [`DocumentTable`](../documenttable) instance should be treated in the same way as if it was a DataTable instance. See template syntax reference for more information.
