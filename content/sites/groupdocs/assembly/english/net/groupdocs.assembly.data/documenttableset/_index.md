## DocumentTableSet class

Provides access to data of multiple tables (or spreadsheets) located in an external document to be used while assembling a document. Also, enables to define parent-child relations for the document tables thus simplifying access to related data within template documents.

### Remarks

For documents of Spreadsheet file formats, a [`DocumentTableSet`](../documenttableset) instance represents a set of sheets. For documents of other file formats, a [`DocumentTableSet`](../documenttableset) instance represents a set of tables.

To access data of the corresponding tables while assembling a document, pass an instance of this class as a data source to one of [`DocumentAssembler`](../../groupdocs.assembly/documentassembler).AssembleDocument overloads.

In template documents, a [`DocumentTableSet`](../documenttableset) instance should be treated in the same way as if it was a DataSet instance. See template syntax reference for more information.
