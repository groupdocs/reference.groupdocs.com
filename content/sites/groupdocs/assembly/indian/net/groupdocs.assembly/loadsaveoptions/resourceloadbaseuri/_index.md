## LoadSaveOptions.ResourceLoadBaseUri property

एक HTML टेम्पलेट दस्तावेज़ को असेंबल करने और गैर-HTML प्रारूप में सहेजने के लिए लोड करते समय बाहरी संसाधन फ़ाइलों के सापेक्ष URI को हल करने के लिए एक आधार URI प्राप्त या सेट करता है। डिफ़ॉल्ट मान एक रिक्त स्ट्रिंग है.

```csharp
public string ResourceLoadBaseUri { get; set; }
```

### टिप्पणियों

किसी फ़ाइल से HTML दस्तावेज़ लोड करते समय, इसके युक्त फ़ोल्डर को डिफ़ॉल्ट रूप से आधार URI के रूप में उपयोग किया जाता है, जो कि स्ट्रीम से HTML दस्तावेज़ लोड करते समय नहीं हो सकता है। स्ट्रीम से HTML दस्तावेज़ लोड करते समय या किसी फ़ाइल से HTML दस्तावेज़ लोड करते समय डिफ़ॉल्ट आधार URI को ओवरराइड करने के लिए इस गुण को आधार URI निर्दिष्ट करने के लिए सेट करें.

निम्नलिखित मामलों में इस गुण के मान को अनदेखा किया जाता है:

* लोड किए जा रहे एक HTML दस्तावेज़ में एक आधार HTML तत्व होता है जो एक आधार URI प्रदान करता है।
* एक HTML दस्तावेज़ लोड किया जा रहा है जिसे इकट्ठा किया जाना है और HTML में सहेजा जाना है (बाहरी संसाधन फ़ाइलें लोड नहीं होती हैं और सापेक्ष URI तब नहीं बदले जाते हैं)।
