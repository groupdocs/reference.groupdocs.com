## LoadSaveOptions.ResourceSaveFolder property

बाहरी संसाधन फ़ाइलों को संग्रहीत करने के लिए एक फ़ोल्डर के लिए एक पथ प्राप्त करता है या सेट करता है, जबकि एक गैर-HTML प्रारूप से लोड किए गए इकट्ठे दस्तावेज़ को HTML में सहेजा जा रहा है। डिफ़ॉल्ट मान एक रिक्त स्ट्रिंग है.

```csharp
public string ResourceSaveFolder { get; set; }
```

### टिप्पणियों

डिफ़ॉल्ट रूप से, एक इकट्ठे दस्तावेज़ को एक HTML फ़ाइल में सहेजते समय, बाहरी संसाधन फ़ाइलों को एक फ़ोल्डर में संग्रहीत किया जाता है, जिसका नाम बिना एक्सटेंशन के HTML फ़ाइल के समान नाम और "_files" प्रत्यय होता है। यह फ़ोल्डर उसी फ़ोल्डर में स्थित है जिसमें HTML फ़ाइल है। हालाँकि, इकट्ठे दस्तावेज़ को HTML स्ट्रीम में सहेजते समय ऐसा नहीं किया जा सकता है। एक HTML फ़ाइल के लिए इकट्ठे दस्तावेज़.

इस संपत्ति के एक मूल्य को नजरअंदाज कर दिया जाता है यदि HTML में सहेजे जा रहे इकट्ठे दस्तावेज़ को HTML से भी लोड किया गया था (बाहरी संसाधन फ़ाइलें संग्रहीत नहीं की जाती हैं और उनके लिंक तब नहीं बदले जाते हैं)।
