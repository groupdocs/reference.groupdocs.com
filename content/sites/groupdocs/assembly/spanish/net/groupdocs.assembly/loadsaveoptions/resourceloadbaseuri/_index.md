## LoadSaveOptions.ResourceLoadBaseUri property

Obtiene o establece un URI base para convertir los URI relativos de los archivos de recursos externos en absolutos al cargar un documento de plantilla HTML para ensamblarlo y guardarlo en un formato que no sea HTML. El valor predeterminado es una cadena vacía.

```csharp
public string ResourceLoadBaseUri { get; set; }
```

### Observaciones

Cuando se carga un documento HTML desde un archivo, la carpeta que lo contiene se usa como URI base de manera predeterminada, lo que no puede suceder cuando se carga un documento HTML desde una transmisión. Establezca esta propiedad para especificar un URI base al cargar un documento HTML desde una transmisión o para anular el URI base predeterminado al cargar un documento HTML desde un archivo.

Un valor de esta propiedad se ignora en los siguientes casos:

* Un documento HTML que se está cargando contiene un elemento HTML BASE que proporciona un URI base.
* Un documento HTML que se está cargando debe ensamblarse y guardarse en HTML (los archivos de recursos externos no se cargan y los URI relativos no se cambian en ese momento).
