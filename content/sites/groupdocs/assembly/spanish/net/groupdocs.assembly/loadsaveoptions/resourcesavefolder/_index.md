## LoadSaveOptions.ResourceSaveFolder property

Obtiene o establece una ruta a una carpeta para almacenar archivos de recursos externos mientras un documento ensamblado cargado desde un formato que no es HTML se guarda en HTML. El valor predeterminado es una cadena vacía.

```csharp
public string ResourceSaveFolder { get; set; }
```

### Observaciones

De manera predeterminada, al guardar un documento ensamblado en un archivo HTML, los archivos de recursos externos se almacenan en una carpeta que tiene el mismo nombre que el archivo HTML sin extensión más el sufijo "_files". Esta carpeta se encuentra en la misma carpeta que el archivo HTML. Sin embargo, esto no se puede hacer cuando se guarda un documento ensamblado en una secuencia HTML. Establezca esta propiedad para especificar una ruta a una carpeta para almacenar archivos de recursos externos al guardar un documento ensamblado en una secuencia HTML o para anular la carpeta predeterminada al guardar un documento ensamblado a un archivo HTML.

Se ignora un valor de esta propiedad si un documento ensamblado que se está guardando en HTML también se cargó desde HTML (los archivos de recursos externos no se almacenan y los enlaces a ellos no se modifican en ese momento).
