## DataSourceInfo.Name property

Obtiene o establece el nombre del objeto de origen de datos que se utilizará para acceder al objeto de origen de datos en un documento de plantilla.

```csharp
public string Name { get; set; }
```

### Observaciones

Cuando se especifica el nombre del objeto de origen de datos, puede acceder al objeto de origen de datos y sus miembros en un documento de plantilla utilizando el nombre.

Cuando el nombre del objeto de fuente de datos es nulo o está vacío, aún puede acceder a los miembros del objeto de fuente de datos en un documento de plantilla mediante el acceso a miembros de objetos de contexto (consulte Referencia de sintaxis de plantilla para obtener más información), pero no puede acceder a la objeto de origen de datos en sí mismo.

Al pasar múltiples[`DataSourceInfo`](../../datasourceinfo) instancias a[`DocumentAssembler`](../../documentassembler) , solo el nombre de el primer objeto de fuente de datos puede ser nulo o estar vacío. Los nombres del resto deben ser especificados y únicos.
