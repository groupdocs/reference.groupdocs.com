## BarcodeSettings.UseAutoCorrection property

Obtiene o establece un valor que indica si un valor de código de barras no válido debe corregirse automáticamente (si es posible) para ajustarse a la especificación del código de barras o si debe lanzarse una excepción para indicar el error. El valor predeterminado es verdadero.

```csharp
public bool UseAutoCorrection { get; set; }
```

### Observaciones

La corrección automática no es posible para los códigos de barras Databar.
