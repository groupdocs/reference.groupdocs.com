## DocumentTableColumn.Type property

Obtiene o establece el tipo de valores de celda en esta columna.

```csharp
public Type Type { get; set; }
```

### Observaciones

Para documentos de formatos de archivo que no son hojas de cálculo, el tipo inicial siempre se determina automáticamente como cadena. Para documentos de formatos de archivo de hoja de cálculo, el tipo inicial se determina automáticamente según los valores de celda correspondientes.

Si las celdas de una columna de hoja de cálculo en particular contienen valores de diferentes tipos, entonces el tipo inicial de la columna también se determina automáticamente como cadena.
