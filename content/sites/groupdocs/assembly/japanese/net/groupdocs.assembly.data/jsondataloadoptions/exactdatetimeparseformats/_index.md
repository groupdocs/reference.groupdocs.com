## JsonDataLoadOptions.ExactDateTimeParseFormats property

JSON の読み込み中に JSON の日時値を解析するための正確な形式を取得または設定します。デフォルトは**ヌル**.

```csharp
public IEnumerable<string> ExactDateTimeParseFormats { get; set; }
```

### 備考

Microsoft® JSON 日時形式 (たとえば、"/Date(1224043200000)/") を使用してエンコードされた文字列は、このプロパティの値に関係なく、常に 日時値として認識されます。このプロパティは、次の方法で文字列から日時値を解析する際に使用される追加の 形式を定義します:

* いつ`ExactDateTimeParseFormats`は**ヌル**、ISO-8601 形式、および現在の英語の USA カルチャと英語のニュージーランドカルチャでサポートされているすべての日時形式 が、前述の順序で 追加で使用されます.
* いつ`ExactDateTimeParseFormats`文字列が含まれている場合、現在のカルチャを利用する追加の date-time 形式として使用されます。
* いつ`ExactDateTimeParseFormats`空です。追加の日時形式は使用されません.
