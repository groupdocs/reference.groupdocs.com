## JsonDataLoadOptions.AlwaysGenerateRootObject property

生成されたデータ ソースに常に JSON root 要素のオブジェクトが含まれるかどうかを示すフラグを取得または設定します。 JSON ルート要素に単一の複雑なプロパティが含まれている場合、そのようなオブジェクトはデフォルトでは作成されません.

```csharp
public bool AlwaysGenerateRootObject { get; set; }
```

### 備考
