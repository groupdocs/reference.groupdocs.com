## DocumentTableOptions.FirstRowContainsColumnNames property

ドキュメント テーブルの最初の 抽出された行から列名を取得するかどうかを示す値を取得または設定します。デフォルト値は false. です。

```csharp
public bool FirstRowContainsColumnNames { get; set; }
```

### 備考

列名がドキュメント テーブルの最初に抽出された行から取得されるように設定されていない場合、 デフォルトの列名が代わりに使用されます。スプレッドシート ファイル形式のドキュメントの場合、デフォルトの 列名は、A、B、C、... Z、AA、AB などとして定義されます。他の ファイル形式のドキュメントの場合、デフォルトの列名は、Column1、Column2、Column3 などとして定義されます。
