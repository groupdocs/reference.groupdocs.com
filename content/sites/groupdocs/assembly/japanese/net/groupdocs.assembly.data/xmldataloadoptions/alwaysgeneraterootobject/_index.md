## XmlDataLoadOptions.AlwaysGenerateRootObject property

生成されたデータ ソースに常に XML ルート 要素のオブジェクトが含まれるかどうかを示すフラグを取得または設定します。 XML ルート要素に属性がなく、そのすべての子要素が同じ名前である場合、そのようなオブジェクト はデフォルトでは作成されません.

```csharp
public bool AlwaysGenerateRootObject { get; set; }
```
