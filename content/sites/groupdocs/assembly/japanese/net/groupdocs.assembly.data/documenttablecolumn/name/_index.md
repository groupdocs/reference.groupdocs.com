## DocumentTableColumn.Name property

に渡されたテンプレート ドキュメント内の列のデータにアクセスするために使用されるこの列の名前を取得または設定します。[`DocumentAssembler`](../../../groupdocs.assembly/documentassembler).

```csharp
public string Name { get; set; }
```

### 備考

列の名前がドキュメントから読み取られた場合 ([`FirstRowContainsColumnNames`](../../documenttableoptions/firstrowcontainscolumnnames) ), 名前が有効になるように自動的に修正されます。ただし、列の名前が このプロパティを介して手動で設定され、名前が無効である場合、例外がスローされます.

次の条件が満たされている場合、列の名前は有効であると見なされます:

* 名前が空ではありません.
* 名前の最初の文字は文字またはアンダースコアです。
* 名前の残りの文字は、文字、アンダースコア、数字、または次の文字です: '@'、'#'、'$'.
* 対応する[`DocumentTable`](../../documenttable)オブジェクトに含まれていない[`DocumentTableColumn`](../../documenttablecolumn)同じ名前のインスタンス .
