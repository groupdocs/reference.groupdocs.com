## BarcodeSettings.BaseYDimension property

ベースの y 寸法、つまり 2D バーコード モジュールのユニットの最小の高さを取得または設定します。 で測定[`GraphicsUnit`](../graphicsunit).

```csharp
public float BaseYDimension { get; set; }
```

### 備考

一部のタイプ (データ マトリックスなど) のバーコードは、y 次元を無視し、 幅と高さの両方の単位に x 次元を使用する場合があります。

テンプレートを介してバーコードのスケーリングが適用されると、実際の y 寸法は、ベース y 寸法とスケーリング係数に基づいて計算されます.
