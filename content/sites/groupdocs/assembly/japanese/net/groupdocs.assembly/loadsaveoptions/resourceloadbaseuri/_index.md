## LoadSaveOptions.ResourceLoadBaseUri property

ベース URI を取得または設定して、HTML テンプレート ドキュメントを読み込み、HTML 以外の形式にアセンブルして保存する際に、外部リソース ファイルの相対 URI を絶対 URI に解決します。デフォルト値は空の文字列です。

```csharp
public string ResourceLoadBaseUri { get; set; }
```

### 備考

ファイルから HTML ドキュメントをロードするとき、それを含むフォルダーがデフォルトでベース URI として使用されますが、ストリームから HTML ドキュメントをロードするときに が発生することはありません。ストリームから HTML ドキュメントをロードするときにベース URI を指定するか、ファイルから HTML ドキュメントをロードするときにデフォルトのベース URI をオーバーライドするには、このプロパティを設定します。

次の場合、このプロパティの値は無視されます:

* 読み込まれている HTML ドキュメントには、ベース URI を提供する BASE HTML 要素が含まれています。
* 読み込まれている HTML ドキュメントは、HTML にアセンブルされ、保存されます (外部リソース ファイルは読み込まれず、 相対 URI は変更されません)。
