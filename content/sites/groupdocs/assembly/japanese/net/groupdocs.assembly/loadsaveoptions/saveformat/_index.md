## LoadSaveOptions.SaveFormat property

アセンブルされたドキュメントを保存するファイル形式を取得または設定します。Unspecifiedがデフォルトです。

```csharp
public FileFormat SaveFormat { get; set; }
```

### 備考

このプロパティの値が指定されていない場合、[`DocumentAssembler`](../../documentassembler)次のように動作します:

- 組み立てられたドキュメントを保存するためにファイル パスを指定すると、保存ファイル形式はパスからのファイル 拡張子に基づいて決定されます。

- ストリームを指定してアセンブルされたドキュメントを保存する場合、保存ファイル形式は、読み込まれたテンプレート ドキュメントのファイル 形式と同じままです。

GroupDocs.Assembly を使用して、アセンブルされたドキュメントを任意のファイル形式で保存できるとは限らないことに注意してください。 たとえば、ワード プロセッシング ファイル形式 (DOCX など) からロードされたドキュメントを スプレッドシート ファイル形式 (XLSX など) に保存することはできません。 GroupDocs.Assembly でサポートされているロードおよび保存ファイル フォーマットの可能な組み合わせの詳細については、GroupDocs.Assembly のオンライン ドキュメント. を確認してください。
