## DocumentAssembler.UseReflectionOptimization property

リフレクション API を介して実行されるカスタム型メンバーの呼び出しが動的クラス生成を使用して 最適化されているかどうかを示す値を取得または設定します。デフォルト値は true. です。

```csharp
public static bool UseReflectionOptimization { get; set; }
```

### 備考

この最適化を無効にすることが望ましいシナリオがいくつかあります。たとえば、 データ項目の小さなコレクションを常に処理している場合、動的クラス生成のオーバーヘッドは、直接的なリフレクション API 呼び出しのオーバーヘッドよりも 顕著になる可能性があります.
