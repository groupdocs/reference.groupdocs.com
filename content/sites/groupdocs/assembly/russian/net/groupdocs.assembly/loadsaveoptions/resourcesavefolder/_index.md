## LoadSaveOptions.ResourceSaveFolder property

Получает или задает путь к папке для хранения файлов внешних ресурсов, пока собранный документ, загруженный из формата , отличного от HTML, сохраняется в HTML. Значение по умолчанию — пустая строка.

```csharp
public string ResourceSaveFolder { get; set; }
```

### Примечания

По умолчанию при сохранении собранного документа в файл HTML внешние файлы ресурсов сохраняются в папку , имеющую то же имя, что и файл HTML, без расширения плюс суффикс "_files". Эта папка находится в той же папке, что и файл HTML. Однако это невозможно сделать при сохранении собранного документа в поток HTML. Установите это свойство, чтобы указать путь к папке для хранения файлов внешних ресурсов при сохранении собранного документа в поток HTML или переопределить папку по умолчанию при сохранении собранный документ в файл HTML.

Значение этого свойства игнорируется, если собранный документ, сохраняемый в HTML, был также загружен из HTML (в этом случае файлы внешних ресурсов не сохраняются и ссылки на них не изменяются).
