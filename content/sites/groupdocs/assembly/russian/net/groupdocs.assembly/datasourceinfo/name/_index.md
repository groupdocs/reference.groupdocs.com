## DataSourceInfo.Name property

Получает или задает имя объекта источника данных, который будет использоваться для доступа к объекту источника данных в шаблоне документа.

```csharp
public string Name { get; set; }
```

### Примечания

Когда имя объекта источника данных указано, вы можете получить доступ к объекту источника данных и его членам в шаблоне документа, используя имя.

Когда имя объекта источника данных равно null или пусто, вы все равно можете получить доступ к членам объекта источника данных в документе шаблона, используя доступ к членам объекта контекста (дополнительную информацию см. в Справочнике по синтаксису шаблона), , но вы не можете получить доступ к сам объект источника данных.

При передаче нескольких[`DataSourceInfo`](../../datasourceinfo) случаи[`DocumentAssembler`](../../documentassembler) , только имя первого объекта источника данных может быть нулевым или пустым. Имена остальных должны быть указаны и уникальны.
