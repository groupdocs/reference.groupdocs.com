## DocumentAssemblyOptions enumeration

Задает параметры, управляющие поведением[`DocumentAssembler`](../documentassembler) при сборке документа.

```csharp
[Flags]
public enum DocumentAssemblyOptions
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| None | `0` | Задает параметры по умолчанию. |
| AllowMissingMembers | `1` | Указывает, что отсутствующие члены объекта должны рассматриваться ассемблером как пустые литералы. Этот параметр влияет только на доступ к экземплярам (то есть нестатическим) членам объекта и методам расширения. Если этот параметр не установлен, ассемблер выдает исключение при обнаружении отсутствующего члена объекта. |
| UpdateFieldsAndFormulas | `2` | Указывает, что поля результирующих документов Word Processing и формулы результирующих электронных таблиц должны обновляться ассемблером. |
| RemoveEmptyParagraphs | `4` | Указывает, что ассемблер должен удалять абзацы, ставшие пустыми после того, как теги синтаксиса шаблона удалены или заменены пустыми значениями. |
| InlineErrorMessages | `8` | Указывает, что ассемблер должен встраивать сообщения об ошибках синтаксиса шаблона в выходные документы. Если этот параметр не установлен, ассемблер выдает исключение при обнаружении синтаксической ошибки. |
| UseSpreadsheetDataTypes | `10` | Относится только к электронным таблицам. Указывает, что результаты оцениваемых выражений должны быть сопоставлены с соответствующими типами данных электронной таблицы, что также влияет на их форматирование по умолчанию в ячейках. Если эта опция не установлена, результаты выражения всегда записываются ассемблером в виде строк. Этот параметр не действует, когда результаты выражения форматируются с использованием синтаксиса шаблона — тогда результаты выражения также всегда записываются в виде строк. |
