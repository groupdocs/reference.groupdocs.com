## BarcodeSettings.BaseYDimension property

Получает или задает базовый размер по оси Y, то есть наименьшую высоту блока модулей 2D-штрихкода. Измеряется в[`GraphicsUnit`](../graphicsunit) .

```csharp
public float BaseYDimension { get; set; }
```

### Примечания

Штрих-коды некоторых типов (например, матрица данных) могут игнорировать размер y и использовать размер x как для единиц ширины, так и для единиц высоты.

Когда масштабирование штрих-кода применяется с помощью шаблона, фактический размер y рассчитывается на основе базового размера y и коэффициента масштабирования.
