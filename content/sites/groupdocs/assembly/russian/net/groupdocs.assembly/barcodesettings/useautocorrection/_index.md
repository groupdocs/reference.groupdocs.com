## BarcodeSettings.UseAutoCorrection property

Получает или задает значение, указывающее, должно ли недопустимое значение штрих-кода автоматически исправляться (если возможно) в соответствии со спецификацией штрих-кода или должно быть выдано исключение, указывающее на ошибку. Значение по умолчанию — true.

```csharp
public bool UseAutoCorrection { get; set; }
```

### Примечания

Автокоррекция штрих-кодов панели данных невозможна.
