## DocumentTable.Name property

Получает или задает имя этой таблицы, используемое для доступа к данным таблицы в документе шаблона, переданном to [`DocumentAssembler`](../../../groupdocs.assembly/documentassembler) .

```csharp
public string Name { get; set; }
```

### Примечания

Если имя таблицы считывается из документа, имя автоматически корректируется, чтобы оно было допустимым. Однако, если имя таблицы задано вручную с помощью этого свойства и это имя недействительно, создается исключение.

Имя таблицы считается допустимым, если выполняются следующие условия:

* Имя не пустое.
* Первый символ имени — буква или подчеркивание.
* Остальные символы имени — это буквы, символы подчеркивания, цифры или следующие символы: '@', '#', '$'.
* Соответствующий[`DocumentTableSet`](../../documenttableset) объект не содержит[`DocumentTable`](../../documenttable) экземпляр с тем же именем.
