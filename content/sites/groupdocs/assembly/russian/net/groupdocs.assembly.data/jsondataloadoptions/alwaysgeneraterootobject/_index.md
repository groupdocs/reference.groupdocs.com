## JsonDataLoadOptions.AlwaysGenerateRootObject property

Получает или задает флаг, указывающий, всегда ли сгенерированный источник данных будет содержать объект для элемента JSON root . Если корневой элемент JSON содержит одно сложное свойство, такой объект по умолчанию не создается.

public bool AlwaysGenerateRootObject { get; set; }
