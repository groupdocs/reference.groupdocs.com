## DocumentAssembler.UseReflectionOptimization property

Obtient ou définit une valeur indiquant si les appels de membres de type personnalisé effectués via l'API de réflexion sont optimisés à l'aide de la génération de classe dynamique ou non. La valeur par défaut est true.

```csharp
public static bool UseReflectionOptimization { get; set; }
```

### Remarques

Il existe certains scénarios où il est préférable de désactiver cette optimisation. Par exemple, si vous traitez avec de petites collections d'éléments de données tout le temps, une surcharge de génération de classe dynamique peut être plus perceptible qu'une surcharge d'appels d'API de réflexion directe.
