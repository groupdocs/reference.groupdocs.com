## LoadSaveOptions.ResourceLoadBaseUri property

Obtient ou définit un URI de base pour résoudre les URI relatifs des fichiers de ressources externes en URI absolus lors du chargement d'un document de modèle HTML à assembler et à enregistrer dans un format non HTML. La valeur par défaut est une chaîne vide.

```csharp
public string ResourceLoadBaseUri { get; set; }
```

### Remarques

Lors du chargement d'un document HTML à partir d'un fichier, son dossier contenant est utilisé comme URI de base par défaut, ce qui ne peut pas se produire lors du chargement d'un document HTML à partir d'un flux. Définissez cette propriété pour spécifier un URI de base lors du chargement d'un document HTML à partir d'un flux ou pour remplacer l'URI de base par défaut lors du chargement d'un document HTML à partir d'un fichier.

Une valeur de cette propriété est ignorée dans les cas suivants :

* Un document HTML en cours de chargement contient un élément HTML BASE fournissant un URI de base.
* Un document HTML en cours de chargement doit être assemblé et enregistré au format HTML (les fichiers de ressources externes ne sont pas chargés et les URI relatifs ne sont alors pas modifiés).
