## LoadSaveOptions.SaveFormat property

Obtient ou définit un format de fichier dans lequel enregistrer un document assemblé.Unspecified est la valeur par défaut.

```csharp
public FileFormat SaveFormat { get; set; }
```

### Remarques

Lorsque la valeur de cette propriété n'est pas spécifiée,[`DocumentAssembler`](../../documentassembler) se comporte comme suit :

- Lorsque vous spécifiez un chemin de fichier pour enregistrer un document assemblé, le format du fichier d'enregistrement est déterminé par l'extension de fichier du chemin.

- Lorsque vous spécifiez un flux pour enregistrer un document assemblé, le format du fichier d'enregistrement reste le même que le format de fichier d'un document modèle chargé.

Attention, il n'est pas toujours possible d'enregistrer un document assemblé dans n'importe quel format de fichier à l'aide de GroupDocs.Assembly. Par exemple, il est impossible d'enregistrer un document chargé à partir d'un format de fichier de traitement de texte (tel que DOCX) vers un format de fichier de feuille de calcul (tel que XLSX). Pour plus d'informations sur les combinaisons possibles de formats de fichiers de chargement et d'enregistrement pris en charge par GroupDocs.Assembly, veuillez consulter la documentation en ligne de GroupDocs.Assembly.
