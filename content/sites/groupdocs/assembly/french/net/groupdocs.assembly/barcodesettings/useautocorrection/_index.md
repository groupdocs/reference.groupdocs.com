## BarcodeSettings.UseAutoCorrection property

Obtient ou définit une valeur indiquant si une valeur de code-barres non valide doit être corrigée automatiquement (si possible) pour correspondre à la spécification du code-barres ou si une exception doit être levée pour indiquer l'erreur. La valeur par défaut est true.

```csharp
public bool UseAutoCorrection { get; set; }
```

### Remarques

La correction automatique n'est pas possible pour les codes-barres Databar.
