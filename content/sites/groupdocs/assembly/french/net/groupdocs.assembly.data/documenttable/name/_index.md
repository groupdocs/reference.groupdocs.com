## DocumentTable.Name property

Obtient ou définit le nom de cette table utilisée pour accéder aux données de la table dans un modèle de document transmis à [`DocumentAssembler`](../../../groupdocs.assembly/documentassembler) .

```csharp
public string Name { get; set; }
```

### Remarques

Si le nom de la table est lu à partir d'un document, le nom est automatiquement corrigé afin qu'il soit valide. Cependant, si le nom de la table est défini manuellement via cette propriété et que le nom n'est pas valide, une exception est levée.

Le nom de la table est considéré comme valide si les conditions suivantes sont remplies :

* Le nom n'est pas vide.
* Le premier caractère du nom est une lettre ou un trait de soulignement.
* Les autres caractères du nom sont des lettres, des traits de soulignement, des chiffres ou les caractères suivants : '@', '#', '$'.
* Le correspondant[`DocumentTableSet`](../../documenttableset) l'objet ne contient pas de[`DocumentTable`](../../documenttable) instance avec le même nom.
