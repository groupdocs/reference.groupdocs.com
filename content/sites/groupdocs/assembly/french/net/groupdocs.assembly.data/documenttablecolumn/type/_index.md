## DocumentTableColumn.Type property

Obtient ou définit le type de valeurs de cellule dans cette colonne.

```csharp
public Type Type { get; set; }
```

### Remarques

Pour les documents de formats de fichier autres que les feuilles de calcul, le type initial est toujours automatiquement déterminé en tant que chaîne. Pour les documents au format de fichier Feuille de calcul, le type initial est automatiquement déterminé en fonction des valeurs de cellule correspondantes.

Si les cellules d'une colonne particulière de la feuille de calcul contiennent des valeurs de types différents, le type initial de la colonne est également automatiquement déterminé comme chaîne.
