## LoadSaveOptions.ResourceLoadBaseUri property

Ottiene o imposta un URI di base per risolvere gli URI relativi dei file di risorse esterni in valori assoluti durante il caricamento di un documento modello HTML da assemblare e salvare in un formato non HTML. Il valore predefinito è una stringa vuota.

```csharp
public string ResourceLoadBaseUri { get; set; }
```

### Osservazioni

Quando si carica un documento HTML da un file, la cartella che lo contiene viene utilizzata come URI di base per impostazione predefinita, cosa che non può accadere quando si carica un documento HTML da uno stream. Imposta questa proprietà per specificare un URI di base durante il caricamento di un documento HTML da uno stream o per sovrascrivere l'URI di base predefinito durante il caricamento di un documento HTML da un file.

Un valore di questa proprietà viene ignorato nei seguenti casi:

* Un documento HTML caricato contiene un elemento BASE HTML che fornisce un URI di base.
* Un documento HTML in fase di caricamento deve essere assemblato e salvato in HTML (i file di risorse esterni non vengono caricati e gli URI relativi non vengono quindi modificati).
