## LoadSaveOptions.ResourceSaveFolder property

Ottiene o imposta un percorso di una cartella per archiviare file di risorse esterne mentre un documento assemblato caricato da un formato non HTML viene salvato in HTML. Il valore predefinito è una stringa vuota.

```csharp
public string ResourceSaveFolder { get; set; }
```

### Osservazioni

Per impostazione predefinita, quando si salva un documento assemblato in un file HTML, i file di risorse esterni vengono archiviati in una cartella con lo stesso nome del file HTML senza estensione più il suffisso "_files". Questa cartella si trova in la stessa cartella del file HTML. Tuttavia, non è possibile eseguire questa operazione quando si salva un documento assemblato in un flusso HTML. Impostare questa proprietà per specificare un percorso a una cartella per archiviare i file di risorse esterni durante il salvataggio di un documento assemblato in un flusso HTML o per sovrascrivere la cartella predefinita durante il salvataggio di un documento assemblato in un file HTML.

Un valore di questa proprietà viene ignorato se un documento assemblato salvato in HTML è stato caricato anche da HTML (i file di risorse esterni non vengono archiviati e i collegamenti ad essi non vengono modificati).
