## BarcodeSettings.UseAutoCorrection property

Ottiene o imposta un valore che indica se un valore di codice a barre non valido deve essere corretto automaticamente (se possibile) per adattarsi alla specifica del codice a barre o deve essere generata un'eccezione per indicare l'errore. Il valore predefinito è true.

```csharp
public bool UseAutoCorrection { get; set; }
```

### Osservazioni

La correzione automatica non è possibile per i codici a barre Databar.
