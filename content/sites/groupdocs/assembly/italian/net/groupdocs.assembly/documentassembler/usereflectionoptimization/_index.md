## DocumentAssembler.UseReflectionOptimization property

Ottiene o imposta un valore che indica se le chiamate di membri di tipo personalizzato eseguite tramite l'API di riflessione sono ottimizzate utilizzando la generazione di classi dinamiche o meno. Il valore predefinito è true.

```csharp
public static bool UseReflectionOptimization { get; set; }
```

### Osservazioni

Ci sono alcuni scenari in cui è preferibile disabilitare questa ottimizzazione. Ad esempio, se hai sempre a che fare con piccole raccolte di elementi di dati, un sovraccarico di generazione di classi dinamiche può essere più evidente di un sovraccarico di chiamate API di riflessione diretta.
