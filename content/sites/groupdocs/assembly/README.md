# GroupDocs.Assembly — API Reference Source Files

This repository holds the Markdown source files used to generate the **GroupDocs.Assembly**
API reference, published at <https://reference.groupdocs.com/assembly/>.

## About GroupDocs.Assembly

[GroupDocs.Assembly](https://products.groupdocs.com/assembly/) is a developer API for document
processing: generate documents and reports from templates and data using LINQ syntax. It is available for the following platforms, each with its own
API reference:

- **.NET** — <https://reference.groupdocs.com/assembly/net/>
- **Java** — <https://reference.groupdocs.com/assembly/java/>
- **Python via .NET** — <https://reference.groupdocs.com/assembly/python-net/>

## Repository purpose

The files in this repository are used to:

- Build and maintain the official **GroupDocs.Assembly** API reference.
- Keep the reference in sync with the latest product release.
- Provide a consistent structure for automated Hugo builds and publication.

Content is English-only and compiled into a static site with [Hugo](https://gohugo.io/).
