## LoadSaveOptions.ResourceLoadBaseUri property

الحصول على أو تعيين URI أساسي لحل URIs النسبية لملفات الموارد الخارجية إلى تلك المطلقة أثناء تحميل مستند قالب HTML ليتم تجميعه وحفظه بتنسيق غير HTML. القيمة الافتراضية هي سلسلة فارغة.

```csharp
public string ResourceLoadBaseUri { get; set; }
```

### ملاحظات

عند تحميل مستند HTML من ملف ، يتم استخدام المجلد المحتوي الخاص به كمعرف URI أساسي بشكل افتراضي ، والذي لا يمكن أن يحدث عند تحميل مستند HTML من دفق. قم بتعيين هذه الخاصية لتحديد عنوان URI أساسي عند تحميل مستند HTML من دفق أو لتجاوز عنوان URI الأساسي الافتراضي عند تحميل مستند HTML من ملف.

يتم تجاهل قيمة هذه الخاصية في الحالات التالية:

* يحتوي مستند HTML الجاري تحميله على عنصر BASE HTML يوفر URI أساسي.
* يجب تجميع مستند HTML الذي يتم تحميله وحفظه في HTML (لا يتم تحميل ملفات الموارد الخارجية و لم يتم تغيير URIs النسبية بعد ذلك) .
