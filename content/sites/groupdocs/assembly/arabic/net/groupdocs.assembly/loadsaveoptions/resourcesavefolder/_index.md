## LoadSaveOptions.ResourceSaveFolder property

الحصول على مسار أو تعيينه لمجلد لتخزين ملفات الموارد الخارجية أثناء حفظ المستند المجمع الذي تم تحميله من تنسيق غير HTML إلى HTML. القيمة الافتراضية هي سلسلة فارغة.

```csharp
public string ResourceSaveFolder { get; set; }
```

### ملاحظات

افتراضيًا ، عند حفظ مستند مجمع في ملف HTML ، يتم تخزين ملفات الموارد الخارجية في folder التي لها نفس اسم ملف HTML بدون امتداد بالإضافة إلى اللاحقة "_files". يوجد هذا المجلد في نفس المجلد مثل ملف HTML. ومع ذلك ، لا يمكن القيام بذلك عند حفظ مستند مجمع إلى تدفق HTML . قم بتعيين هذه الخاصية لتحديد مسار لمجلد لتخزين ملفات الموارد الخارجية عند حفظ مستند تم تجميعه إلى تدفق HTML أو لتجاوز المجلد الافتراضي عند حفظ تم تجميع المستند إلى ملف HTML.

يتم تجاهل قيمة هذه الخاصية إذا تم تحميل مستند مجمع يتم حفظه في HTML من HTML وكذلك (لا يتم تخزين ملفات الموارد الخارجية ولا يتم تغيير الروابط المؤدية إليها بعد ذلك) .
