템플릿 문서에서 다양한 유형의 데이터 소스로 작업하는 방법에 대한 자세한 내용은 템플릿 구문 참조 (https://docs.groupdocs.com/display/assemblynet/Template+Syntax+-+Part+1+of+2# TemplateSyntax-Part1of2-UsingDataSources).

데이터 원본 개체의 이름을 지정하면 name. 를 사용하여 템플릿 문서에서 데이터 원본 개체와 해당 멤버 에 액세스할 수 있습니다.

데이터 소스 개체의 이름이 null이거나 비어 있는 경우에도 컨텍스트 개체 멤버 액세스(자세한 내용은 템플릿 구문 참조 참조), 를 사용하여 템플릿 문서에서 데이터 소스 개체 의 멤버에 액세스할 수 있지만 데이터 원본 개체 자체.

여러 개를 전달할 때[`DataSourceInfo`](../../datasourceinfo) 인스턴스[`DocumentAssembler`](../../documentassembler) , 첫 번째 데이터 원본 개체의 이름만 null이거나 비어 있을 수 있습니다. 나머지 이름은 고유해야 합니다.
