## DataSourceInfo.Name property

템플릿 문서에서 데이터 원본 개체에 액세스하는 데 사용할 데이터 원본 개체의 이름을 가져오거나 설정합니다.

```csharp
public string Name { get; set; }
```

### 비고

데이터 원본 개체의 이름을 지정하면 name. 를 사용하여 템플릿 문서에서 데이터 원본 개체와 해당 멤버 에 액세스할 수 있습니다.

데이터 소스 개체의 이름이 null이거나 비어 있는 경우에도 컨텍스트 개체 멤버 액세스(자세한 내용은 템플릿 구문 참조 참조), 를 사용하여 템플릿 문서에서 데이터 소스 개체 의 멤버에 액세스할 수 있지만 데이터 원본 개체 자체.
