## LoadSaveOptions.ResourceSaveFolder property

HTML이 아닌 형식에서 로드된 어셈블된 문서가 HTML로 저장되는 동안 외부 리소스 파일을 저장할 폴더의 경로를 가져오거나 설정합니다. 기본값은 빈 문자열입니다.

```csharp
public string ResourceSaveFolder { get; set; }
```

### 비고

기본적으로 어셈블된 문서를 HTML 파일로 저장할 때 외부 리소스 파일은 확장자에 "_files" 접미사를 추가하지 않고 HTML 파일과 동일한 이름을 가진 폴더 에 저장됩니다. 이 폴더는 HTML 파일과 같은 폴더인 에 있습니다. 그러나 어셈블된 문서를 HTML 스트림으로 저장할 때는 이 작업을 수행할 수 없습니다. 어셈블된 문서 를 HTML 스트림으로 저장할 때 외부 리소스 파일을 저장할 폴더의 경로를 지정하거나 파일을 저장할 때 기본 폴더를 재정의하려면 이 속성을 설정하십시오. 어셈블된 문서를 HTML 파일로.

HTML로 저장되는 어셈블된 문서가 HTML에서도 로드된 경우 이 속성 값은 무시됩니다. (외부 리소스 파일이 저장되지 않고 해당 파일에 대한 링크가 변경되지 않음)
