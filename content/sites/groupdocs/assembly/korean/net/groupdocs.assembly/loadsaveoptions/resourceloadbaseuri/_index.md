## LoadSaveOptions.ResourceLoadBaseUri property

HTML 템플릿 문서를 로드하여 비HTML 형식으로 저장하는 동안 외부 리소스 파일의 상대 URI를 절대 URI로 확인하기 위한 기본 URI를 가져오거나 설정합니다. 기본값은 빈 문자열입니다.

```csharp
public string ResourceLoadBaseUri { get; set; }
```

### 비고

파일에서 HTML 문서를 로드할 때 포함하는 폴더는 기본적으로 기본 URI로 사용되며 스트림에서 HTML 문서를 로드할 때 발생할 수 없습니다. 스트림에서 HTML 문서를 로드할 때 기본 URI를 지정하거나 파일에서 HTML 문서를 로드할 때 기본 기본 URI를 대체하려면 이 특성을 설정하십시오.

이 속성의 값은 다음과 같은 경우에 무시됩니다.

* 로드되는 HTML 문서에는 기본 URI를 제공하는 BASE HTML 요소가 포함되어 있습니다.
* 로드되는 HTML 문서는 어셈블되어 HTML로 저장됩니다(외부 리소스 파일이 로드되지 않고 상대 URI가 변경되지 않음).
