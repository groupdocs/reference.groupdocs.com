## DocumentTableOptions.FirstRowContainsColumnNames property

문서 테이블의 first 추출된 행에서 열 이름을 가져올지 여부를 나타내는 값을 가져오거나 설정합니다. 기본값은 false입니다.

```csharp
public bool FirstRowContainsColumnNames { get; set; }
```

### 비고

문서 테이블의 첫 번째 추출 행에서 열 이름을 가져오도록 설정하지 않은 경우 기본 열 이름이 대신 사용됩니다. 스프레드시트 파일 형식 문서의 경우 기본 열 이름은 A, B, C, ... Z, AA, AB 등으로 정의됩니다. 다른 파일 형식의 문서의 경우 기본 열 이름은 Column1, Column2, Column3 등으로 정의됩니다.
