## LoadSaveOptions.ResourceLoadBaseUri property

Haalt een basis-URI op of stelt deze in om de relatieve URI's van externe bronbestanden om te zetten in absolute URI's tijdens het laden van een HTML -sjabloondocument dat moet worden samengesteld en opgeslagen in een niet-HTML-indeling. De standaardwaarde is een lege tekenreeks.

```csharp
public string ResourceLoadBaseUri { get; set; }
```

### Opmerkingen

Bij het laden van een HTML-document uit een bestand, wordt de bijbehorende map standaard gebruikt als basis-URI, wat niet kan gebeuren bij het laden van een HTML-document uit een stream. Stel deze eigenschap in om een basis-URI op te geven bij het laden van een HTML -document uit een stream of om de standaard basis-URI te overschrijven bij het laden van een HTML-document uit een bestand.

Een waarde van deze eigenschap wordt genegeerd in de volgende gevallen:

* Een HTML-document dat wordt geladen bevat een BASE HTML-element dat een basis-URI levert.
* Een HTML-document dat wordt geladen, moet worden samengesteld en opgeslagen in HTML (externe bronbestanden worden niet geladen en relatieve URI's worden dan niet gewijzigd).
