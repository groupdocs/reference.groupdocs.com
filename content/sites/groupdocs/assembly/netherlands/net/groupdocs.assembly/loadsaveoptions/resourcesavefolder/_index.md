## LoadSaveOptions.ResourceSaveFolder property

Hiermee wordt een pad naar een map opgehaald of ingesteld om externe bronbestanden op te slaan terwijl een samengesteld document dat is geladen vanuit een niet-HTML -indeling wordt opgeslagen in HTML. De standaardwaarde is een lege tekenreeks.

```csharp
public string ResourceSaveFolder { get; set; }
```

### Opmerkingen

Wanneer een samengesteld document wordt opgeslagen in een HTML-bestand, worden externe bronbestanden standaard opgeslagen in een map met dezelfde naam als het HTML-bestand zonder extensie plus het achtervoegsel "_files". Deze map bevindt zich in dezelfde map als het HTML-bestand. Dit is echter niet mogelijk bij het opslaan van een samengesteld document in een HTML-stream. Stel deze eigenschap in om een pad naar een map op te geven om externe bronbestanden op te slaan bij het opslaan van een samengesteld document in een HTML-stream of om de standaardmap te overschrijven bij het opslaan van een samengesteld document naar een HTML-bestand.

Een waarde van deze eigenschap wordt genegeerd als een samengesteld document dat in HTML wordt opgeslagen, ook vanuit HTML is geladen (externe bronbestanden worden niet opgeslagen en links daarnaar worden niet gewijzigd).
