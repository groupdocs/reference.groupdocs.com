## JsonDataLoadOptions.ExactDateTimeParseFormats property

Mendapat atau menyetel format yang tepat untuk mem-parsing nilai tanggal-waktu JSON saat memuat JSON. Standarnya adalah**batal** .

```csharp
public IEnumerable<string> ExactDateTimeParseFormats { get; set; }
```

### Perkataan

String yang dikodekan menggunakan format tanggal-waktu Microsoft® JSON (misalnya, "/Tanggal(1224043200000)/") selalu dikenali sebagai nilai tanggal-waktu terlepas dari nilai properti ini. Properti menentukan format tambahan untuk digunakan saat mem-parsing nilai tanggal-waktu dari string dengan cara berikut:

* Kapan`ExactDateTimeParseFormats` adalah**batal** format ISO-8601 dan semua format tanggal-waktu didukung untuk budaya saat ini, Inggris AS, dan Inggris Selandia Baru digunakan tambahan dalam urutan yang disebutkan.
* Kapan`ExactDateTimeParseFormats` berisi string, mereka digunakan sebagai format tanggal-waktu tambahan yang memanfaatkan budaya saat ini.
* Kapan`ExactDateTimeParseFormats` kosong, tidak ada format tanggal-waktu tambahan yang digunakan.
