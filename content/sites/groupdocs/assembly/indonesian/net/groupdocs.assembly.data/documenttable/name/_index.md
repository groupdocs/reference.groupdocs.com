## DocumentTable.Name property

Mendapat atau menyetel nama tabel ini yang digunakan untuk mengakses data tabel dalam dokumen template yang diteruskan ke [`DocumentAssembler`](../../../groupdocs.assembly/documentassembler) .

```csharp
public string Name { get; set; }
```

### Perkataan

Jika nama tabel terbaca dari dokumen, otomatis nama tersebut dikoreksi sehingga menjadi valid. Namun, jika nama tabel diatur secara manual melalui properti ini dan namanya tidak valid, pengecualian akan terjadi.

Nama tabel dianggap valid, jika kondisi berikut terpenuhi:

* Nama tidak kosong.
* Karakter pertama nama adalah huruf atau garis bawah.
* Karakter nama lainnya adalah huruf, garis bawah, angka, atau karakter berikut: '@', '#', '$'.
* Yang sesuai[`DocumentTableSet`](../../documenttableset) objek tidak mengandung a[`DocumentTable`](../../documenttable) contoh dengan nama yang sama.
