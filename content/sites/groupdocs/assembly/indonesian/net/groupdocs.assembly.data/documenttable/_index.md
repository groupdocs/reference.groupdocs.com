### Perkataan

Untuk dokumen dengan format file Spreadsheet, a[`DocumentTable`](../documenttable) instance mewakili satu lembar. Untuk dokumen dengan format file lain, a[`DocumentTable`](../documenttable) instance mewakili satu tabel.

Untuk mengakses data dari tabel yang sesuai saat menyusun dokumen, berikan instance kelas ini sebagai sumber data ke salah satu[`DocumentAssembler`](../../groupdocs.assembly/documentassembler) .AssembleDocument kelebihan beban.

Dalam dokumen template, a[`DocumentTable`](../documenttable) instance harus diperlakukan dengan cara yang sama seolah-olah was aDataTable contoh. Lihat referensi sintaks template untuk informasi lebih lanjut.
