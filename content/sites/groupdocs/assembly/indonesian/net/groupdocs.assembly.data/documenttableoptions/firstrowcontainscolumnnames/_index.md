## DocumentTableOptions.FirstRowContainsColumnNames property

Mendapat atau menetapkan nilai yang menunjukkan apakah nama kolom diperoleh dari baris pertama yang diekstraksi dari tabel dokumen. Nilai defaultnya adalah false.

```csharp
public bool FirstRowContainsColumnNames { get; set; }
```

### Perkataan

Jika nama kolom tidak disetel untuk diperoleh dari baris pertama yang diekstrak dari tabel dokumen, nama kolom default akan digunakan sebagai gantinya. Untuk dokumen dengan format file Spreadsheet, nama kolom default ditentukan sebagai A, B, C, ... Z, AA, AB, dan seterusnya. Untuk dokumen dengan format file lainnya, nama kolom default ditetapkan sebagai Kolom1, Kolom2, Kolom3, dan seterusnya.
