## DocumentTableColumn.Name property

Mendapat atau menyetel nama kolom ini yang digunakan untuk mengakses data kolom dalam dokumen template yang diteruskan ke [`DocumentAssembler`](../../../groupdocs.assembly/documentassembler) .

```csharp
public string Name { get; set; }
```

### Perkataan

Jika nama kolom dibaca dari dokumen (lihat[`FirstRowContainsColumnNames`](../../documenttableoptions/firstrowcontainscolumnnames) ), nama otomatis terkoreksi sehingga menjadi valid. Namun, jika nama kolom diatur secara manual melalui properti ini dan namanya tidak valid, pengecualian akan dilemparkan.

Nama kolom dianggap valid, jika kondisi berikut terpenuhi:

* Nama tidak kosong.
* Karakter pertama nama adalah huruf atau garis bawah.
* Karakter nama lainnya adalah huruf, garis bawah, angka, atau karakter berikut: '@', '#', '$'.
* Yang sesuai[`DocumentTable`](../../documenttable) objek tidak mengandung a[`DocumentTableColumn`](../../documenttablecolumn) contoh dengan nama yang sama.
