Mendapat atau menetapkan jenis nilai sel di kolom ini.

### Perkataan

Untuk dokumen dengan format file non-Spreadsheet, tipe awal selalu ditentukan secara otomatis sebagai string. Untuk dokumen format file Spreadsheet, tipe awal ditentukan secara otomatis tergantung pada nilai sel yang sesuai.

Jika sel dari kolom Spreadsheet tertentu berisi nilai dari tipe yang berbeda, maka tipe awal kolom juga secara otomatis ditentukan sebagai string.
