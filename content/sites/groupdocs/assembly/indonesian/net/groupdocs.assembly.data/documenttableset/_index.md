## DocumentTableSet class

Menyediakan akses ke data beberapa tabel (atau spreadsheet) yang terletak di dokumen eksternal untuk digunakan saat menyusun dokumen. Juga, memungkinkan untuk menentukan hubungan induk-anak untuk tabel dokumen sehingga menyederhanakan akses ke data terkait dalam dokumen template.

### Perkataan

Untuk dokumen dengan format file Spreadsheet, a[`DocumentTableSet`](../documenttableset) instance mewakili sekumpulan sheets. Untuk dokumen dengan format file lain, a[`DocumentTableSet`](../documenttableset) instance mewakili sekumpulan tabel.

Untuk mengakses data dari tabel yang sesuai saat menyusun dokumen, berikan instance kelas ini sebagai sumber data ke salah satu[`DocumentAssembler`](../../groupdocs.assembly/documentassembler) .AssembleDocument kelebihan beban.

Dalam dokumen template, a[`DocumentTableSet`](../documenttableset) instance harus diperlakukan dengan cara yang sama seolah-olah was aDataSet contoh. Lihat referensi sintaks template untuk informasi lebih lanjut.
