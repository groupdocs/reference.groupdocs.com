## JsonSimpleValueParseMode enumeration

Menentukan mode untuk mengurai nilai sederhana JSON (null, boolean, angka, bilangan bulat, dan string) saat memuat JSON. Mode seperti itu tidak memengaruhi penguraian nilai tanggal-waktu.

```csharp
public enum JsonSimpleValueParseMode
```

### Nilai

| Nama | Nilai | Keterangan |
| --- | --- | --- |
| Loose | `0` | Menentukan mode di mana jenis nilai sederhana JSON ditentukan setelah penguraian representasi stringnya. Misalnya, jenis 'prop' dari cuplikan JSON '{ prop: "123" }' ditentukan sebagai bilangan bulat dalam mode ini. |
| Strict | `1` | Menentukan mode di mana jenis nilai sederhana JSON ditentukan dari notasi JSON itu sendiri. Misalnya, jenis 'penyangga' dari cuplikan JSON '{ prop: "123" }' ditentukan sebagai string dalam mode ini. |
