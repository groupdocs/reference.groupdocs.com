## DataSourceInfo.Name property

Mendapat atau menetapkan nama objek sumber data yang akan digunakan untuk mengakses objek sumber data dalam dokumen template.

```csharp
public string Name { get; set; }
```

### Perkataan

Ketika nama objek sumber data ditentukan, Anda dapat mengakses objek sumber data dan anggotanya dalam dokumen template menggunakan nama.

Ketika nama objek sumber data null atau kosong, Anda masih dapat mengakses anggota objek sumber data dalam dokumen template menggunakan akses anggota objek konteks (lihat Referensi Sintaks Template untuk informasi lebih lanjut), tetapi Anda tidak dapat mengakses objek sumber data itu sendiri.

Saat melewati beberapa[`DataSourceInfo`](../../datasourceinfo) contoh ke[`DocumentAssembler`](../../documentassembler) , hanya nama objek sumber data pertama yang boleh nol atau kosong. Nama yang lainnya harus ditentukan dan unik.
