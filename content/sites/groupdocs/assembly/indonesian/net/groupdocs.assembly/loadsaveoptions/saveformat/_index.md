## LoadSaveOptions.SaveFormat property

Mendapat atau mengatur format file untuk menyimpan dokumen rakitan.Unspecified adalah default.

```csharp
public FileFormat SaveFormat { get; set; }
```

### Perkataan

Ketika nilai properti ini tidak ditentukan,[`DocumentAssembler`](../../documentassembler) berperilaku sebagai berikut:

- Saat Anda menentukan jalur file untuk menyimpan dokumen rakitan, format file penyimpanan ditentukan berdasarkan ekstensi file dari jalur.

- Saat Anda menentukan aliran untuk menyimpan dokumen rakitan, format file penyimpanan tetap sama dengan format file dari dokumen template yang dimuat.

Berhati-hatilah karena tidak selalu mungkin untuk menyimpan dokumen rakitan ke format file apa pun menggunakan GroupDocs.Assembly. Misalnya, tidak mungkin menyimpan dokumen yang dimuat dari format file Pemrosesan Kata (seperti DOCX) ke format file Spreadsheet (seperti XLSX). Untuk informasi selengkapnya tentang kemungkinan kombinasi memuat dan menyimpan file format yang didukung oleh GroupDocs.Assembly, silakan periksa dokumentasi online GroupDocs.Assembly.
