## LoadSaveOptions.ResourceSaveFolder property

Mendapat atau menyetel jalur ke folder untuk menyimpan file sumber daya eksternal sementara dokumen rakitan dimuat dari format non-HTML disimpan ke HTML. Nilai default adalah string kosong.

```csharp
public string ResourceSaveFolder { get; set; }
```

### Perkataan

Secara default, saat menyimpan dokumen rakitan ke file HTML, file sumber daya eksternal disimpan ke folder dengan nama yang sama dengan file HTML tanpa ekstensi plus akhiran "_files". Folder ini terletak di folder yang sama dengan file HTML. Namun, ini tidak dapat dilakukan saat menyimpan dokumen rakitan ke aliran HTML. Setel properti ini untuk menentukan jalur ke folder untuk menyimpan file sumber eksternal saat menyimpan dokumen rakitan ke aliran HTML atau untuk mengganti folder default saat menyimpan dokumen rakitan ke file HTML.

Nilai properti ini diabaikan jika dokumen rakitan yang disimpan ke HTML juga dimuat dari HTML (file sumber daya eksternal tidak disimpan dan tautan ke sana tidak diubah).
