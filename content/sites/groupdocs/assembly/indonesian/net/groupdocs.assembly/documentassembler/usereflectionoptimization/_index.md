## DocumentAssembler.UseReflectionOptimization property

Mendapat atau menetapkan nilai yang menunjukkan apakah pemanggilan anggota tipe khusus yang dilakukan melalui API refleksi dioptimalkan menggunakan pembuatan kelas dinamis atau tidak. Nilai defaultnya adalah true.

```csharp
public static bool UseReflectionOptimization { get; set; }
```

### Perkataan

Ada beberapa skenario yang lebih disukai untuk menonaktifkan pengoptimalan ini. Misalnya, jika Anda berurusan dengan kumpulan kecil item data sepanjang waktu, maka overhead pembuatan kelas dinamis dapat lebih terlihat daripada overhead panggilan API refleksi langsung.
