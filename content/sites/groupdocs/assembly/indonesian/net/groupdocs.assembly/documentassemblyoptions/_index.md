## DocumentAssemblyOptions enumeration

Menentukan opsi yang mengontrol perilaku[`DocumentAssembler`](../documentassembler) saat menyusun dokumen.

```csharp
[Flags]
public enum DocumentAssemblyOptions
```

### Nilai

| Nama | Nilai | Keterangan |
| --- | --- | --- |
| None | `0` | Menentukan opsi default. |
| AllowMissingMembers | `1` | Menentukan bahwa anggota objek yang hilang harus diperlakukan sebagai null literal oleh assembler. Opsi ini hanya memengaruhi akses ke anggota objek dan metode ekstensi instance (yaitu, non-statis). Jika opsi ini tidak disetel, assembler melempar pengecualian saat menemukan anggota objek yang hilang. |
| UpdateFieldsAndFormulas | `2` | Menentukan bahwa bidang hasil dokumen Pemrosesan Kata dan rumus hasil dokumen Spreadsheet harus diperbarui oleh assembler. |
| RemoveEmptyParagraphs | `4` | Menentukan bahwa assembler harus menghapus paragraf yang menjadi kosong setelah tag sintaks template dihapus atau diganti dengan nilai kosong. |
| InlineErrorMessages | `8` | Menentukan bahwa assembler harus memasukkan pesan kesalahan sintaks template ke dalam dokumen keluaran. Jika opsi ini tidak disetel, assembler akan melontarkan pengecualian saat menemukan kesalahan sintaksis. |
| UseSpreadsheetDataTypes | `10` | Berkaitan dengan dokumen Spreadsheet saja. Menentukan bahwa hasil ekspresi yang dievaluasi harus dipetakan ke tipe data Spreadsheet yang sesuai, yang juga memengaruhi pemformatan defaultnya di dalam sel. Jika opsi ini tidak disetel, hasil ekspresi selalu ditulis sebagai string oleh assembler. Opsi ini tidak berpengaruh ketika hasil ekspresi diformat menggunakan sintaks template - hasil ekspresi juga selalu ditulis sebagai string. |
