## Watermarker.SearchableObjects property

الحصول على أو تعيين كائنات المحتوى التي سيتم تضمينها في بحث العلامة المائية.

```csharp
public SearchableObjects SearchableObjects { get; set; }
```

### Property_Value

الكائنات التي سيتم تضمينها في بحث العلامة المائية.

### ملاحظات

تحدد هذه الخاصية أيضًا كائنات المحتوى المستخدمة في البحث عن الصور . لمزيد من المعلومات ، راجع[`Search`](../search) و[`GetImages`](../getimages) طُرق.

تعرف على المزيد حول البحث عن العلامات المائية: [البحث عن العلامات المائية](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### أمثلة

قم بإزالة كل XObjects و Artifacts من مستند pdf.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.pdf"))
{
    // حدد أنه يجب البحث عن XObjects و Artifacts فقط.
    watermarker.SearchableObjects.PdfSearchableObjects
        = PdfSearchableObjects.XObjects | PdfSearchableObjects.Artifacts; 

    PossibleWatermarkCollection watermarks = watermarker.Search();

    // إزالة جميع الكائنات التي تم العثور عليها.
    watermarks.Clear();

    // حفظ التغييرات.
    watermarker.Save();
}
```
