## GetImages() {#getimages}

البحث عن كافة الصور الموجودة في المستند.

```csharp
public WatermarkableImageCollection GetImages()
```

### قيمة الإرجاع

مجموعة الصور التي تم العثور عليها.

### ملاحظات

يتم البحث في الأشياء المحددة في[`SearchableObjects`](../searchableobjects).

تعرف على المزيد حول البحث عن العلامات المائية [البحث عن العلامات المائية](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### أمثلة

أضف علامة مائية إلى كافة الصور في مستند من أي نوع مدعوم.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    // تهيئة نص أو صورة العلامة المائية.
    TextWatermark watermark = new TextWatermark("DRAFT", new Font("Arial", 19));

    // البحث عن جميع الصور في المستند.
    WatermarkableImageCollection images = watermarker.GetImages();

    // أضف علامة مائية.
    foreach (WatermarkableImage watermarkableImage in images)
    {
        watermarkableImage.AddWatermark(watermark);
    }

    // حفظ التغييرات.
    watermarker.Save(@"D:\output.doc");
}
```
