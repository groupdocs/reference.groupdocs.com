## WordProcessingWatermarkPagesOptions.PageNumbers property

Ottiene o imposta i numeri di pagina per aggiungere la filigrana.

```csharp
public int[] PageNumbers { get; set; }
```

### Valore della proprietà

I numeri di pagina per aggiungere la filigrana.

### Osservazioni

Tutti i numeri devono essere maggiori o uguali a 1. Questa proprietà viene utilizzata solo quando si aggiunge la filigrana a un documento. Se questo valore è`nullo` o vuoto, la filigrana viene aggiunta a tutte le pagine.
