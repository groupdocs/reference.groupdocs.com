## OfficeImageEffects.Brightness property

Ottiene o imposta la luminosità dell'immagine. Il valore per questa proprietà deve essere un numero compreso tra 0,0 (il più debole) e 1,0 (il più luminoso).

public double Brightness { get; set; }

### Valore della proprietà

Il valore predefinito è 0,5.
