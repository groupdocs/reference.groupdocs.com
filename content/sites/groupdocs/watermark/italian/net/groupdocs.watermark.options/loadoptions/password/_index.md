## LoadOptions.Password property

Ottiene o imposta la password per l'apertura di un documento crittografato.

```csharp
public string Password { get; set; }
```

### Valore della proprietà

La password per aprire un documento crittografato.

### Osservazioni

La password deve essere nulla o una stringa vuota. Il valore predefinito è null. Se il contenuto non è crittografato, impostalo su null o stringa vuota.

### Esempi

Carica un contenuto protetto da password.

```csharp
LoadOptions loadOptions = new LoadOptions
{
    Password = "pwd123"
};
using (Watermarker watermarker = new Watermarker(@"D:\doc.vsdx", loadOptions))
{
    //...
}
```
