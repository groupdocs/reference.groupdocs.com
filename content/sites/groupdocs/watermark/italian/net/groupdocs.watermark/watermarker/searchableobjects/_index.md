## Watermarker.SearchableObjects property

Ottiene o imposta gli oggetti contenuto da includere in una ricerca filigrana.

```csharp
public SearchableObjects SearchableObjects { get; set; }
```

### Valore della proprietà

Gli oggetti che devono essere inclusi in una ricerca filigrana.

### Osservazioni

Questa proprietà specifica anche gli oggetti di contenuto utilizzati nella ricerca di immagini. Per ulteriori informazioni, vedere[`Search`](../search) e[`GetImages`](../getimages) metodi.

Ulteriori informazioni sulla ricerca di filigrane: [Ricerca di filigrane](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Esempi

Rimuovi tutti gli XObject e gli artefatti da un documento pdf.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.pdf"))
{
    // Specifica che devono essere cercati solo XObjects e Artifacts.
    watermarker.SearchableObjects.PdfSearchableObjects
        = PdfSearchableObjects.XObjects | PdfSearchableObjects.Artifacts; 

    PossibleWatermarkCollection watermarks = watermarker.Search();

    // Rimuove tutti gli oggetti trovati.
    watermarks.Clear();

    // Salvare le modifiche.
    watermarker.Save();
}
```
