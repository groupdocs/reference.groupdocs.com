## Watermarker.GetDocumentInfo method

Recupera le informazioni sul formato del documento caricato.

```csharp
public IDocumentInfo GetDocumentInfo()
```

### Valore di ritorno

IL[`IDocumentInfo`](../../../groupdocs.watermark.common/idocumentinfo) istanza che contiene le informazioni rilevate.

### Osservazioni

Ulteriori informazioni su come ottenere le informazioni sul documento [Ottieni informazioni sul documento](https://docs.groupdocs.com/display/watermarknet/Get+document+info) .

### Esempi

Ottieni informazioni su un documento di qualsiasi tipo supportato.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.ppt"))
{
    IDocumentInfo docInfo = watermarker.GetDocumentInfo();
    Console.WriteLine("Document size: {0}", docInfo.Size);
    Console.WriteLine("Document format: {0}", docInfo.FileType.FileFormat);
    Console.WriteLine("Document contains {0} pages", docInfo.PageCount);
}
```
