## Watermark.ConsiderParentMargins property

Ottiene o imposta un valore che indica se le dimensioni e le coordinate della filigrana vengono calcolate considerando i margini padre.

```csharp
public bool ConsiderParentMargins { get; set; }
```

### Valore della proprietà

Se il valore è vero, le dimensioni e le coordinate della filigrana vengono calcolate considerando i margini principali. Per impostazione predefinita, il valore è falso, i margini principali vengono ignorati.
