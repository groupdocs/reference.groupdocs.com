## WordProcessingHeaderFooter.IsLinkedToPrevious property

Ottiene o imposta un valore che indica se questa intestazione/piè di pagina è collegata a l'intestazione/piè di pagina corrispondente nella sezione precedente.

```csharp
public bool IsLinkedToPrevious { get; set; }
```

### Valore della proprietà

True se questa intestazione/piè di pagina è collegata all'intestazione/piè di pagina corrispondente nella sezione precedente; false in caso contrario.

### Osservazioni

L'impostazione predefinita è true. Nota, quando il tuo collegamento è un'intestazione o un piè di pagina, il suo contenuto viene cancellato.
