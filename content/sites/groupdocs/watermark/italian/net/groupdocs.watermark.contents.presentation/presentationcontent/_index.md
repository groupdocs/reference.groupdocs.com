### Esempi

Carica e salva un documento PowerPoint di qualsiasi tipo supportato.

```csharp
PresentationLoadOptions loadOptions = new PresentationLoadOptions();
using (Watermarker watermarker = new Watermarker(@"D:\input.ppt", loadOptions))
{
    // Usa il metodo Aggiungi per aggiungere filigrana a una particolare diapositiva o a tutte le diapositive.

    // Salvare le modifiche.
    watermarker.Save(@"D:\output.ppt");
}
```
