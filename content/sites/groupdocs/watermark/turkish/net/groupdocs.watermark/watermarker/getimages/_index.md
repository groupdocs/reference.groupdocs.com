## GetImages(ImageSearchCriteria) {#getimages_1}

Belirtilen arama kriterlerine göre resimleri bulur.

```csharp
public WatermarkableImageCollection GetImages(ImageSearchCriteria searchCriteria)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| searchCriteria | ImageSearchCriteria | Kullanılacak arama kriterleri. |

### Geri dönüş değeri

Bulunan görüntülerin toplanması.

### Notlar

Arama belirtilen nesnelerde gerçekleştirilir.[`SearchableObjects`](../searchableobjects).

Watermarks arama hakkında daha fazla bilgi edinin[Filigran arama](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Örnekler

Desteklenen herhangi bir belge türünden referansa benzer tüm resimleri kaldırın.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    ImageThumbnailSearchCriteria criteria = new ImageThumbnailSearchCriteria("reference.png");
    WatermarkableImageCollection images = watermarker.GetImages(criteria);
    images.Clear();
    watermarker.Save(@"D:\output.doc");
}
```

## GetImages() {#getimages}

Belgedeki tüm görüntüleri bulur.

```csharp
public WatermarkableImageCollection GetImages()
```

### Geri dönüş değeri

Bulunan görüntülerin toplanması.

### Notlar

Arama belirtilen nesnelerde gerçekleştirilir.[`SearchableObjects`](../searchableobjects).

Watermarks arama hakkında daha fazla bilgi edinin[Filigran arama](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Örnekler

Desteklenen herhangi bir türdeki bir belgedeki tüm görüntülere filigran ekleyin.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    // Metin veya resim filigranını başlat.
    TextWatermark watermark = new TextWatermark("DRAFT", new Font("Arial", 19));

    // Belgedeki tüm görüntüleri bulun.
    WatermarkableImageCollection images = watermarker.GetImages();

    // Filigran ekle.
    foreach (WatermarkableImage watermarkableImage in images)
    {
        watermarkableImage.AddWatermark(watermark);
    }

    // Değişiklikleri Kaydet.
    watermarker.Save(@"D:\output.doc");
}
```
