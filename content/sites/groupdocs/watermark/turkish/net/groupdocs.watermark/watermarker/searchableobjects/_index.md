## Watermarker.SearchableObjects property

Bir filigran aramasına dahil edilecek içerik nesnelerini alır veya ayarlar.

```csharp
public SearchableObjects SearchableObjects { get; set; }
```

### Mülk değeri

Bir filigran aramasına dahil edilecek nesneler.

### Notlar

Bu özellik aynı zamanda görsel aramada kullanılan içerik nesnelerini de belirtir. Daha fazla bilgi için bkz.[`Search`](../search) ve[`GetImages`](../getimages) yöntemler.

Filigran arama hakkında daha fazla bilgi edinin: [Filigran arama](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Örnekler

Bir pdf belgesinden tüm XObject'leri ve Yapıları kaldırın.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.pdf"))
{
    // Yalnızca XObjects ve Artifacts'ın aranması gerektiğini belirtin.
    watermarker.SearchableObjects.PdfSearchableObjects
        = PdfSearchableObjects.XObjects | PdfSearchableObjects.Artifacts; 

    PossibleWatermarkCollection watermarks = watermarker.Search();

    // Bulunan tüm nesneleri kaldırın.
    watermarks.Clear();

    // Değişiklikleri Kaydet.
    watermarker.Save();
}
```
