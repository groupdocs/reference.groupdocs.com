## Watermark.ConsiderParentMargins property

Hämtar eller ställer in ett värde som anger om vattenstämpelns storlek och koordinater beräknas med hänsyn till överordnade marginaler.

```csharp
public bool ConsiderParentMargins { get; set; }
```

### Fastighetsvärde

Om värdet är sant, beräknas vattenstämpelns storlek och koordinater med hänsyn till överordnade marginaler. Som standard är värdet falskt, överordnade marginaler ignoreras.
