## Watermarker.SearchableObjects property

Hämtar eller ställer in innehållsobjekten som ska inkluderas i en vattenstämpelsökning.

```csharp
public SearchableObjects SearchableObjects { get; set; }
```

### Fastighetsvärde

De objekt som ska ingå i en vattenstämpelsökning.

### Anmärkningar

Den här egenskapen anger också innehållsobjekt som används i bildsökning. För mer information se[`Search`](../search) och[`GetImages`](../getimages) metoder.

Läs mer om att söka efter vattenstämplar: [Söker efter vattenstämplar](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Exempel

Ta bort alla XObjects and Artefakter från ett pdf-dokument.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.pdf"))
{
    // Ange att endast XObjects och Artifacts ska sökas efter.
    watermarker.SearchableObjects.PdfSearchableObjects
        = PdfSearchableObjects.XObjects | PdfSearchableObjects.Artifacts; 

    PossibleWatermarkCollection watermarks = watermarker.Search();

    // Ta bort alla hittade objekt.
    watermarks.Clear();

    // Spara ändringar.
    watermarker.Save();
}
```
