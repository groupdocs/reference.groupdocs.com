### Returvärde

### Anmärkningar

Läs mer om att söka vattenmärken [Söker efter vattenstämplar](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Exempel

Ta bort alla bilder som liknar referensen från ett dokument av vilken typ som helst.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    ImageThumbnailSearchCriteria criteria = new ImageThumbnailSearchCriteria("reference.png");
    WatermarkableImageCollection images = watermarker.GetImages(criteria);
    images.Clear();
    watermarker.Save(@"D:\output.doc");
}
```

## GetImages() {#getimages}

Hittar alla bilder i dokumentet.

```csharp
public WatermarkableImageCollection GetImages()
```

### Returvärde

Samlingen av hittade bilder.

### Anmärkningar

Sökningen utförs i objekt som anges i[`SearchableObjects`](../searchableobjects).

Läs mer om att söka vattenmärken [Söker efter vattenstämplar](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Exempel

Lägg till vattenstämpel till alla bilder i ett dokument av alla typer som stöds.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    // Initiera text- eller bildvattenstämpel.
    TextWatermark watermark = new TextWatermark("DRAFT", new Font("Arial", 19));

    // Hitta alla bilder i dokumentet.
    WatermarkableImageCollection images = watermarker.GetImages();

    // Lägg till vattenstämpel.
    foreach (WatermarkableImage watermarkableImage in images)
    {
        watermarkableImage.AddWatermark(watermark);
    }

    // Spara ändringar.
    watermarker.Save(@"D:\output.doc");
}
```
