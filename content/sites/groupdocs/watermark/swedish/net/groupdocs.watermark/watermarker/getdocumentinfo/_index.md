## Watermarker.GetDocumentInfo method

Får information om formatet på det laddade dokumentet.

```csharp
public IDocumentInfo GetDocumentInfo()
```

### Returvärde

De[`IDocumentInfo`](../../../groupdocs.watermark.common/idocumentinfo) instans som innehåller upptäckt information.

### Anmärkningar

Läs mer om att få dokumentinformationen [Få dokumentinformation](https://docs.groupdocs.com/display/watermarknet/Get+document+info) .

### Exempel

Få information om ett dokument av vilken typ som helst som stöds.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.ppt"))
{
    IDocumentInfo docInfo = watermarker.GetDocumentInfo();
    Console.WriteLine("Document size: {0}", docInfo.Size);
    Console.WriteLine("Document format: {0}", docInfo.FileType.FileFormat);
    Console.WriteLine("Document contains {0} pages", docInfo.PageCount);
}
```
