```csharp
public PossibleWatermarkCollection Search(SearchCriteria searchCriteria)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| searchCriteria | SearchCriteria | Sökkriterierna att använda. |

### Returvärde

Samlingen av möjliga vattenstämplar.

### Anmärkningar

Sökningen utförs i objekt som anges i[`SearchableObjects`](../searchableobjects).

Läs mer om att söka vattenmärken [Söker efter vattenstämplar](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Exempel

Hitta och ta bort alla möjliga vattenstämplar som innehåller viss text eller bild från ett document av vilken typ som helst som stöds.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    ImageSearchCriteria imageSearchCriteria = new ImageDctHashSearchCriteria(@"D:\logo.png");
    Regex regex = new Regex(@"^Company\sName$", RegexOptions.IgnoreCase);
    TextSearchCriteria textSearchCriteria = new TextSearchCriteria(regex);
    PossibleWatermarkCollection watermarks = watermarker.Search(textSearchCriteria.Or(imageSearchCriteria));
    watermarks.Clear();
    watermarker.Save(@"D:\output.doc");
}
```
