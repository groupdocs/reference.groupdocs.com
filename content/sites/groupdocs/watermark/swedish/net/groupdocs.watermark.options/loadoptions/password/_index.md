## LoadOptions.Password property

Hämtar eller ställer in lösenordet för att öppna ett krypterat dokument.

### Fastighetsvärde

Lösenordet för att öppna ett krypterat dokument.

### Anmärkningar

Lösenordet är null eller tom sträng. Standardvärdet är null. Om innehållet inte är krypterat, ställ in detta på null eller tom sträng.

### Exempel

Ladda ett innehåll skyddat med ett lösenord.

{
    Password = "pwd123"
};

{
    // ...
}
