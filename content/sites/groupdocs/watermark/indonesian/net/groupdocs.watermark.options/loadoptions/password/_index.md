## LoadOptions.Password property

Mendapatkan atau menyetel kata sandi untuk membuka dokumen terenkripsi.

public string Password { get; set; }

### Nilai properti

Kata sandi untuk membuka dokumen terenkripsi.

### Perkataan

Kata sandi berupa string nol atau kosong. Nilai defaultnya adalah null. Jika konten tidak dienkripsi, setel ini ke null atau string kosong.

### Contoh

Memuat konten yang dilindungi dengan kata sandi.

```csharp
LoadOptions loadOptions = new LoadOptions
{
    Password = "pwd123"
};
using (Watermarker watermarker = new Watermarker(@"D:\doc.vsdx", loadOptions))
{
    // ...
}
```
