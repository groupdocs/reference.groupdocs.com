## Watermarker.GetDocumentInfo method

Mendapat informasi tentang format dokumen yang dimuat.

```csharp
public IDocumentInfo GetDocumentInfo()
```

### Nilai Pengembalian

Itu[`IDocumentInfo`](../../../groupdocs.watermark.common/idocumentinfo) contoh yang berisi informasi yang terdeteksi.

### Perkataan

Pelajari lebih lanjut tentang mendapatkan informasi dokumen [Dapatkan info dokumen](https://docs.groupdocs.com/display/watermarknet/Get+document+info) .

### Contoh

Dapatkan informasi tentang dokumen jenis apa pun yang didukung.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.ppt"))
{
    IDocumentInfo docInfo = watermarker.GetDocumentInfo();
    Console.WriteLine("Document size: {0}", docInfo.Size);
    Console.WriteLine("Document format: {0}", docInfo.FileType.FileFormat);
    Console.WriteLine("Document contains {0} pages", docInfo.PageCount);
}
```
