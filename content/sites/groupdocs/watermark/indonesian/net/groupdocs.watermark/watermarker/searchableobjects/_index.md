## Watermarker.SearchableObjects property

Mendapat atau menyetel objek konten yang akan disertakan dalam pencarian watermark.

```csharp
public SearchableObjects SearchableObjects { get; set; }
```

### Nilai properti

Objek yang akan dimasukkan dalam pencarian watermark.

### Perkataan

Properti ini juga menentukan objek konten yang digunakan dalam pencarian gambar. Untuk informasi selengkapnya, lihat[`Search`](../search) dan[`GetImages`](../getimages) metode.

Pelajari lebih lanjut tentang mencari tanda air: [Mencari tanda air](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Contoh

Hapus semua XObjects dan Artefak dari dokumen pdf.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.pdf"))
{
    // Tentukan bahwa hanya XObjects dan Artifacts yang harus dicari.
    watermarker.SearchableObjects.PdfSearchableObjects
        = PdfSearchableObjects.XObjects | PdfSearchableObjects.Artifacts; 

    PossibleWatermarkCollection watermarks = watermarker.Search();

    // Hapus semua objek yang ditemukan.
    watermarks.Clear();

    // Simpan perubahan.
    watermarker.Save();
}
```
