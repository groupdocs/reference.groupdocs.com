public PossibleWatermarkCollection Search(SearchCriteria searchCriteria)

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| searchCriteria | SearchCriteria | Kriteria pencarian yang akan digunakan. |

### Nilai Pengembalian

Pengumpulan kemungkinan tanda air.

### Perkataan

Pencarian dilakukan di objek yang ditentukan di[`SearchableObjects`](../searchableobjects).

Pelajari lebih lanjut tentang mencari tanda air [Mencari tanda air](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Contoh

Temukan dan hapus semua tanda air yang mungkin berisi teks atau gambar tertentu dari dokumen jenis apa pun yang didukung.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    ImageSearchCriteria imageSearchCriteria = new ImageDctHashSearchCriteria(@"D:\logo.png");
    Regex regex = new Regex(@"^Company\sName$", RegexOptions.IgnoreCase);
    TextSearchCriteria textSearchCriteria = new TextSearchCriteria(regex);
    PossibleWatermarkCollection watermarks = watermarker.Search(textSearchCriteria.Or(imageSearchCriteria));
    watermarks.Clear();
    watermarker.Save(@"D:\output.doc");
}
```
