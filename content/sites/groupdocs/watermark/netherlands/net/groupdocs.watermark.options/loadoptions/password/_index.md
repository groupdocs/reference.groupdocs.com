## LoadOptions.Password property

Haalt het wachtwoord op of stelt het in voor het openen van een gecodeerd document.

```csharp
public string Password { get; set; }
```

### Eigendoms-waarde

Het wachtwoord voor het openen van een versleuteld document.

### Opmerkingen

Het wachtwoord is null of een lege string. De standaardwaarde is null. Als de inhoud niet is gecodeerd, stelt u dit in op null of lege tekenreeks.

### Voorbeelden

Laad inhoud die is beveiligd met een wachtwoord.

```csharp
LoadOptions loadOptions = new LoadOptions
{
    Password = "pwd123"
};
using (Watermarker watermarker = new Watermarker(@"D:\doc.vsdx", loadOptions))
{
    // ...
}
```
