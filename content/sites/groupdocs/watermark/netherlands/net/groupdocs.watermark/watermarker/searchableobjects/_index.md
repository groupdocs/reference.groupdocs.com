## Watermarker.SearchableObjects property

Haalt of stelt de inhoudsobjecten op die moeten worden opgenomen in een watermerkzoekopdracht.

```csharp
public SearchableObjects SearchableObjects { get; set; }
```

### Eigendoms-waarde

De objecten die moeten worden opgenomen in een watermerkzoekopdracht.

### Opmerkingen

Deze eigenschap specificeert ook inhoudsobjecten die worden gebruikt bij het zoeken naar afbeeldingen. Zie voor meer informatie[`Search`](../search) en[`GetImages`](../getimages) methoden.

Meer informatie over het zoeken naar watermerken: [Watermerken zoeken](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Voorbeelden

Verwijder alle XObjects en artefacten uit een pdf-document.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.pdf"))
{
    // Specificeer dat er alleen naar XObjects en Artifacts moet worden gezocht.
    watermarker.SearchableObjects.PdfSearchableObjects
        = PdfSearchableObjects.XObjects | PdfSearchableObjects.Artifacts; 

    PossibleWatermarkCollection watermarks = watermarker.Search();

    // Verwijder alle gevonden voorwerpen.
    watermarks.Clear();

    // Wijzigingen opslaan.
    watermarker.Save();
}
```
