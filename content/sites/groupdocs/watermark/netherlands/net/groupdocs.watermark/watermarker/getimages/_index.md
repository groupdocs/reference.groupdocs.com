Meer informatie over zoeken naar watermerken [Watermerken zoeken](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Voorbeelden

Verwijder alle afbeeldingen die lijken op de referentie uit een document van elk ondersteund type.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    ImageThumbnailSearchCriteria criteria = new ImageThumbnailSearchCriteria("reference.png");
    WatermarkableImageCollection images = watermarker.GetImages(criteria);
    images.Clear();
    watermarker.Save(@"D:\output.doc");
}
```

## GetImages() {#getimages}

Vindt alle afbeeldingen in het document.

```csharp
public WatermarkableImageCollection GetImages()
```

### Winstwaarde

De verzameling gevonden afbeeldingen.

### Opmerkingen

De zoekopdracht wordt uitgevoerd in objecten gespecificeerd in[`SearchableObjects`](../searchableobjects).

Meer informatie over zoeken naar watermerken [Watermerken zoeken](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Voorbeelden

Voeg een watermerk toe aan alle afbeeldingen in een document van elk ondersteund type.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    // Initialiseer tekst- of afbeeldingswatermerk.
    TextWatermark watermark = new TextWatermark("DRAFT", new Font("Arial", 19));

    // Vind alle afbeeldingen in het document.
    WatermarkableImageCollection images = watermarker.GetImages();

    // Voeg een watermerk toe.
    foreach (WatermarkableImage watermarkableImage in images)
    {
        watermarkableImage.AddWatermark(watermark);
    }

    // Wijzigingen opslaan.
    watermarker.Save(@"D:\output.doc");
}
```
