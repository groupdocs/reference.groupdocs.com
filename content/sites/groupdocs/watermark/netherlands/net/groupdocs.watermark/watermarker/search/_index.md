### Winstwaarde

### Opmerkingen

De zoekopdracht wordt uitgevoerd in objecten gespecificeerd in[`SearchableObjects`](../searchableobjects).

Meer informatie over zoeken naar watermerken [Watermerken zoeken](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Voorbeelden

Zoek en verwijder alle mogelijke watermerken met bepaalde tekst of afbeelding uit een document van elk ondersteund type.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    ImageSearchCriteria imageSearchCriteria = new ImageDctHashSearchCriteria(@"D:\logo.png");
    Regex regex = new Regex(@"^Company\sName$", RegexOptions.IgnoreCase);
    TextSearchCriteria textSearchCriteria = new TextSearchCriteria(regex);
    PossibleWatermarkCollection watermarks = watermarker.Search(textSearchCriteria.Or(imageSearchCriteria));
    watermarks.Clear();
    watermarker.Save(@"D:\output.doc");
}
```
