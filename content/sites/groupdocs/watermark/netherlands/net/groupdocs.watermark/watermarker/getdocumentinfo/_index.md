## Watermarker.GetDocumentInfo method

Krijgt de informatie over de indeling van het geladen document.

```csharp
public IDocumentInfo GetDocumentInfo()
```

### Winstwaarde

De[`IDocumentInfo`](../../../groupdocs.watermark.common/idocumentinfo) instantie die gedetecteerde informatie bevat.

### Opmerkingen

Meer informatie over het ophalen van de documentinformatie [Documentinformatie ophalen](https://docs.groupdocs.com/display/watermarknet/Get+document+info) .

### Voorbeelden

Krijg informatie over een document van elk ondersteund type.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.ppt"))
{
    IDocumentInfo docInfo = watermarker.GetDocumentInfo();
    Console.WriteLine("Document size: {0}", docInfo.Size);
    Console.WriteLine("Document format: {0}", docInfo.FileType.FileFormat);
    Console.WriteLine("Document contains {0} pages", docInfo.PageCount);
}
```
