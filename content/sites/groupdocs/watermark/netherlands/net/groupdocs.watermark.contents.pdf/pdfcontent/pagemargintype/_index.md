## PdfContent.PageMarginType property

Haalt PDF-paginamarges op of stelt deze in voor gebruik tijdens het toevoegen van watermerken.

```csharp
public PdfPageMarginType PageMarginType { get; set; }
```

### Eigendoms-waarde

Pdf-paginamarges die moeten worden gebruikt tijdens het toevoegen van watermerken.

### Opmerkingen

Deze eigenschap werkt alleen wanneer[`ConsiderParentMargins`](../../../groupdocs.watermark/watermark/considerparentmargins) is ingesteld op waar. Als[`ConsiderParentMargins`](../../../groupdocs.watermark/watermark/considerparentmargins) is onwaar, wanneer pdf CropBox wordt gebruikt als watermerkgebied.

De standaardwaarde isTrimBox.
