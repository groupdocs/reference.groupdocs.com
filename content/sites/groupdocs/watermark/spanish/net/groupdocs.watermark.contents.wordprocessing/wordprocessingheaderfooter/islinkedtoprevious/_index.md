## WordProcessingHeaderFooter.IsLinkedToPrevious property

Obtiene o establece un valor que indica si este encabezado/pie de página está vinculado a el encabezado/pie de página correspondiente en la sección anterior.

```csharp
public bool IsLinkedToPrevious { get; set; }
```

### El valor de la propiedad

True si este encabezado/pie de página está vinculado al encabezado/pie de página correspondiente en la sección anterior; false de lo contrario.

### Observaciones

El valor predeterminado es verdadero. Tenga en cuenta que cuando vincula un encabezado o pie de página, su contenido se borra.
