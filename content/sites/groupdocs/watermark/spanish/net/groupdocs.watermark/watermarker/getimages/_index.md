## GetImages(ImageSearchCriteria) {#getimages_1}

Encuentra imágenes de acuerdo con los criterios de búsqueda especificados.

```csharp
public WatermarkableImageCollection GetImages(ImageSearchCriteria searchCriteria)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| searchCriteria | ImageSearchCriteria | Los criterios de búsqueda a utilizar. |

### Valor_devuelto

La colección de imágenes encontradas.

### Observaciones

La búsqueda se lleva a cabo en los objetos especificados en[`SearchableObjects`](../searchableobjects).

Más información sobre la búsqueda de marcas de agua [Búsqueda de marcas de agua](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Ejemplos

Eliminar todas las imágenes similares a la referencia de un documento de cualquier tipo compatible.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    ImageThumbnailSearchCriteria criteria = new ImageThumbnailSearchCriteria("reference.png");
    WatermarkableImageCollection images = watermarker.GetImages(criteria);
    images.Clear();
    watermarker.Save(@"D:\output.doc");
}
```

## GetImages() {#getimages}

Encuentra todas las imágenes en el documento.

```csharp
public WatermarkableImageCollection GetImages()
```

### Valor_devuelto

La colección de imágenes encontradas.

### Observaciones

La búsqueda se lleva a cabo en los objetos especificados en[`SearchableObjects`](../searchableobjects).

Más información sobre la búsqueda de marcas de agua [Búsqueda de marcas de agua](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Ejemplos

Agregar marca de agua a todas las imágenes en un documento de cualquier tipo compatible.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    // Inicializa la marca de agua de texto o imagen.
    TextWatermark watermark = new TextWatermark("DRAFT", new Font("Arial", 19));

    // Encuentra todas las imágenes en el documento.
    WatermarkableImageCollection images = watermarker.GetImages();

    // Añadir marca de agua.
    foreach (WatermarkableImage watermarkableImage in images)
    {
        watermarkableImage.AddWatermark(watermark);
    }

    // Guardar cambios.
    watermarker.Save(@"D:\output.doc");
}
```
