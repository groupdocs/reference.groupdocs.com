## Watermarker.SearchableObjects property

Obtiene o establece los objetos de contenido que se incluirán en una búsqueda de marca de agua.

```csharp
public SearchableObjects SearchableObjects { get; set; }
```

### El valor de la propiedad

Los objetos que se van a incluir en una búsqueda de marca de agua.

### Observaciones

Esta propiedad también especifica los objetos de contenido que se utilizan en la búsqueda de imágenes. Para obtener más información, consulte[`Search`](../search) y[`GetImages`](../getimages) métodos.

Más información sobre la búsqueda de marcas de agua: [Búsqueda de marcas de agua](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Ejemplos

Eliminar todos los XObjects y Artifacts de un documento pdf.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.pdf"))
{
    // Especificar que solo se deben buscar XObjects y Artifacts.
    watermarker.SearchableObjects.PdfSearchableObjects
        = PdfSearchableObjects.XObjects | PdfSearchableObjects.Artifacts; 

    PossibleWatermarkCollection watermarks = watermarker.Search();

    // Elimina todos los objetos encontrados.
    watermarks.Clear();

    // Guardar cambios.
    watermarker.Save();
}
```
