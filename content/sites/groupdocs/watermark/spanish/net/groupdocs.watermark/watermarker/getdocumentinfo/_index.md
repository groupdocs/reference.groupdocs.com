## Watermarker.GetDocumentInfo method

Obtiene la información sobre el formato del documento cargado.

```csharp
public IDocumentInfo GetDocumentInfo()
```

### Valor_devuelto

los[`IDocumentInfo`](../../../groupdocs.watermark.common/idocumentinfo) instancia que contiene información detectada.

### Observaciones

Obtenga más información sobre cómo obtener la información del documento [Obtener información del documento](https://docs.groupdocs.com/display/watermarknet/Get+document+info) .

### Ejemplos

Obtener información sobre un documento de cualquier tipo compatible.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.ppt"))
{
    IDocumentInfo docInfo = watermarker.GetDocumentInfo();
    Console.WriteLine("Document size: {0}", docInfo.Size);
    Console.WriteLine("Document format: {0}", docInfo.FileType.FileFormat);
    Console.WriteLine("Document contains {0} pages", docInfo.PageCount);
}
```
