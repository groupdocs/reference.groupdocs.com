## WordProcessingShapeSettings.PageNumbers property

Obtiene o establece los números de página para agregar la marca de agua.

```csharp
public int[] PageNumbers { get; set; }
```

### El valor de la propiedad

Los números de página para agregar la marca de agua.

### Observaciones

Todos los números deben ser mayores o iguales a 1. Esta propiedad solo se usa cuando se agrega la marca de agua a un documento. Si este valor es`nulo` , la marca de agua se agrega a todas las páginas.
