## LoadOptions.Password property

Obtiene o establece la contraseña para abrir un documento cifrado.

```csharp
public string Password { get; set; }
```

### El valor de la propiedad

La contraseña para abrir un documento cifrado.

### Observaciones

La contraseña puede ser nula o una cadena vacía. El valor predeterminado es nulo. Si el contenido no está encriptado, configúrelo como nulo o cadena vacía.

### Ejemplos

Carga un contenido protegido con contraseña.

```csharp
LoadOptions loadOptions = new LoadOptions
{
    Password = "pwd123"
};
using (Watermarker watermarker = new Watermarker(@"D:\doc.vsdx", loadOptions))
{
    // ...
}
```
