### Ejemplos

Agregue una marca de agua de anotación de imagen a un documento PDF.

```csharp
PdfLoadOptions loadOptions = new PdfLoadOptions();
using (Watermarker watermarker = new Watermarker(@"D:\test.pdf", loadOptions))
{
    using (ImageWatermark watermark = new ImageWatermark(@"D:\icon.png"))
    {
        PdfAnnotationWatermarkOptions options = new PdfAnnotationWatermarkOptions();
        options.PageIndex = -1; // por defecto - todas las paginas

        watermarker.Add(watermark, options);
    }

    watermarker.Save();
}
```
