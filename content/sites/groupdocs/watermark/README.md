# GroupDocs.Watermark — API Reference Source Files

This repository holds the Markdown source files used to generate the **GroupDocs.Watermark**
API reference, published at <https://reference.groupdocs.com/watermark/>.

## About GroupDocs.Watermark

[GroupDocs.Watermark](https://products.groupdocs.com/watermark/) is a developer API for document
processing: add, find, edit, and remove watermarks in documents, images, and PDFs. It is available for the following platforms, each with its own
API reference:

- **.NET** — <https://reference.groupdocs.com/watermark/net/>
- **Java** — <https://reference.groupdocs.com/watermark/java/>
- **Python via .NET** — <https://reference.groupdocs.com/watermark/python-net/>

## Repository purpose

The files in this repository are used to:

- Build and maintain the official **GroupDocs.Watermark** API reference.
- Keep the reference in sync with the latest product release.
- Provide a consistent structure for automated Hugo builds and publication.

Content is English-only and compiled into a static site with [Hugo](https://gohugo.io/).
