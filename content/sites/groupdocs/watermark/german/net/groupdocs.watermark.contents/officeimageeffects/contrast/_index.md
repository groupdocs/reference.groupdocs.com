## OfficeImageEffects.Contrast property

Holt oder setzt den Kontrast für das angegebene Bild. Der Wert für diese Eigenschaft muss eine Zahl zwischen 0,0 (geringster Kontrast) und 1,0 (größter Kontrast) sein.

```csharp
public double Contrast { get; set; }
```

### Eigentumswert

Der Standardwert ist 0,5.
