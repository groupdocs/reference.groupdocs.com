## PdfContent.PageMarginType property

Ruft die beim Hinzufügen von Wasserzeichen zu verwendenden PDF-Seitenränder ab oder legt sie fest.

```csharp
public PdfPageMarginType PageMarginType { get; set; }
```

### Eigentumswert

Pdf-Seitenränder, die beim Hinzufügen von Wasserzeichen verwendet werden sollen.

### Bemerkungen

Diese Eigenschaft funktioniert nur, wenn[`ConsiderParentMargins`](../../../groupdocs.watermark/watermark/considerparentmargins) auf true gesetzt ist. Wenn[`ConsiderParentMargins`](../../../groupdocs.watermark/watermark/considerparentmargins) ist falsch, wenn pdf CropBox als Wasserzeichenbereich verwendet wird.

Der Standardwert istTrimBox.
