## Watermarker.SearchableObjects property

Holt oder setzt die Inhaltsobjekte, die in eine Wasserzeichensuche eingeschlossen werden sollen.

```csharp
public SearchableObjects SearchableObjects { get; set; }
```

### Eigentumswert

Die Objekte, die in eine Wasserzeichensuche eingeschlossen werden sollen.

### Bemerkungen

Diese Eigenschaft gibt auch Inhaltsobjekte an, die bei der Bildersuche verwendet werden. Weitere Informationen finden Sie unter[`Search`](../search) und[`GetImages`](../getimages) Methoden.

Weitere Informationen zum Suchen von Wasserzeichen: [Suche nach Wasserzeichen](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Beispiele

Alle XObjects und Artefakte aus einem PDF-Dokument entfernen.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.pdf"))
{
    // Geben Sie an, dass nur nach XObjects und Artifacts gesucht werden soll.
    watermarker.SearchableObjects.PdfSearchableObjects
        = PdfSearchableObjects.XObjects | PdfSearchableObjects.Artifacts; 

    PossibleWatermarkCollection watermarks = watermarker.Search();

    // Alle gefundenen Objekte entfernen.
    watermarks.Clear();

    // Änderungen speichern.
    watermarker.Save();
}
```
