## GetImages(ImageSearchCriteria) {#getimages_1}

Findet Bilder nach festgelegten Suchkriterien.

```csharp
public WatermarkableImageCollection GetImages(ImageSearchCriteria searchCriteria)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| searchCriteria | ImageSearchCriteria | Die zu verwendenden Suchkriterien. |

### Rückgabewert

Die Sammlung gefundener Bilder.

### Bemerkungen

Die Suche wird in Objekten durchgeführt, die in angegeben sind[`SearchableObjects`](../searchableobjects).

Weitere Informationen zum Suchen von watermarks [Suche nach Wasserzeichen](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Beispiele

Entfernen Sie alle Bilder, die der Referenz ähnlich sind, aus einem Dokument eines beliebigen unterstützten Typs.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    ImageThumbnailSearchCriteria criteria = new ImageThumbnailSearchCriteria("reference.png");
    WatermarkableImageCollection images = watermarker.GetImages(criteria);
    images.Clear();
    watermarker.Save(@"D:\output.doc");
}
```

## GetImages() {#getimages}

Findet alle Bilder im Dokument.

```csharp
public WatermarkableImageCollection GetImages()
```

### Rückgabewert

Die Sammlung gefundener Bilder.

### Bemerkungen

Die Suche wird in Objekten durchgeführt, die in angegeben sind[`SearchableObjects`](../searchableobjects).

Weitere Informationen zum Suchen von watermarks [Suche nach Wasserzeichen](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Beispiele

Wasserzeichen zu allen Bildern in einem Dokument eines beliebigen unterstützten Typs hinzufügen.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    // Text- oder Bildwasserzeichen initialisieren.
    TextWatermark watermark = new TextWatermark("DRAFT", new Font("Arial", 19));

    // Finde alle Bilder im Dokument.
    WatermarkableImageCollection images = watermarker.GetImages();

    // Wasserzeichen hinzufügen.
    foreach (WatermarkableImage watermarkableImage in images)
    {
        watermarkableImage.AddWatermark(watermark);
    }

    // Änderungen speichern.
    watermarker.Save(@"D:\output.doc");
}
```
