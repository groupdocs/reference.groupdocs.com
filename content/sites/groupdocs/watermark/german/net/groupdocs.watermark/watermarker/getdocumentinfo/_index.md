## Watermarker.GetDocumentInfo method

Ruft die Informationen über das Format des geladenen Dokuments ab.

```csharp
public IDocumentInfo GetDocumentInfo()
```

### Rückgabewert

Der[`IDocumentInfo`](../../../groupdocs.watermark.common/idocumentinfo) Instanz, die erkannte Informationen enthält.

### Bemerkungen

Weitere Informationen zum Abrufen der Dokumentinformationen [Dokumentinformationen erhalten](https://docs.groupdocs.com/display/watermarknet/Get+document+info) .

### Beispiele

Abrufen von Informationen zu einem Dokument eines beliebigen unterstützten Typs.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.ppt"))
{
    IDocumentInfo docInfo = watermarker.GetDocumentInfo();
    Console.WriteLine("Document size: {0}", docInfo.Size);
    Console.WriteLine("Document format: {0}", docInfo.FileType.FileFormat);
    Console.WriteLine("Document contains {0} pages", docInfo.PageCount);
}
```
