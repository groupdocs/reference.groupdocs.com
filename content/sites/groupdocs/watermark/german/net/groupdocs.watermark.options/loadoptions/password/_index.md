## LoadOptions.Password property

Ruft das Passwort zum Öffnen eines verschlüsselten Dokuments ab oder legt es fest.

```csharp
public string Password { get; set; }
```

### Eigentumswert

Das Passwort zum Öffnen eines verschlüsselten Dokuments.

### Bemerkungen

Das Passwort ist null oder eine leere Zeichenfolge. Der Standardwert ist null. Wenn der Inhalt nicht verschlüsselt ist, setzen Sie dies auf null oder eine leere Zeichenfolge.

### Beispiele

Laden Sie einen passwortgeschützten Inhalt.

```csharp
LoadOptions loadOptions = new LoadOptions
{
    Password = "pwd123"
};
using (Watermarker watermarker = new Watermarker(@"D:\doc.vsdx", loadOptions))
{
    // ...
}
```
