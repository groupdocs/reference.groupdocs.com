## Watermarker.SearchableObjects property

Λαμβάνει ή ορίζει τα αντικείμενα περιεχομένου που πρόκειται να συμπεριληφθούν σε μια αναζήτηση υδατογραφήματος.

```csharp
public SearchableObjects SearchableObjects { get; set; }
```

### Αξία περιουσίας

Τα αντικείμενα που πρόκειται να συμπεριληφθούν σε μια αναζήτηση υδατογραφήματος.

### Παρατηρήσεις

Αυτή η ιδιότητα καθορίζει επίσης αντικείμενα περιεχομένου που χρησιμοποιούνται στην αναζήτηση εικόνων. Για περισσότερες πληροφορίες βλ.[`Search`](../search) και[`GetImages`](../getimages) μεθόδους.

Μάθετε περισσότερα σχετικά με την αναζήτηση υδατογραφημάτων: [Αναζήτηση υδατογραφημάτων](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Παραδείγματα

Καταργήστε όλα τα XObjects και Artifacts από ένα έγγραφο pdf.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.pdf"))
{
    // Καθορίστε ότι θα πρέπει να γίνεται αναζήτηση μόνο για XObjects και Artifacts.
    watermarker.SearchableObjects.PdfSearchableObjects
        = PdfSearchableObjects.XObjects | PdfSearchableObjects.Artifacts; 

    PossibleWatermarkCollection watermarks = watermarker.Search();

    // Αφαιρέστε όλα τα αντικείμενα που βρέθηκαν.
    watermarks.Clear();

    // Αποθήκευσε τις αλλαγές.
    watermarker.Save();
}
```
