## PresentationShapeSettings.IsLocked property

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν απαγορεύεται η επεξεργασία του σχήματος στο PowerPoint.

```csharp
public bool IsLocked { get; set; }
```

### Αξία περιουσίας

Εάν η τιμή είναι αληθής, η επεξεργασία σχήματος θα απαγορεύεται. Από προεπιλογή, η τιμή είναι false, το σχήμα μπορεί να επεξεργαστεί στο PowerPoint.
