## PdfContent.PageMarginType property

Λαμβάνει ή ορίζει περιθώρια σελίδας pdf που θα χρησιμοποιηθούν κατά την προσθήκη υδατογραφήματος.

```csharp
public PdfPageMarginType PageMarginType { get; set; }
```

### Αξία περιουσίας

Περιθώρια σελίδας Pdf που θα χρησιμοποιηθούν κατά την προσθήκη υδατογραφήματος.

### Παρατηρήσεις

Αυτή η ιδιοκτησία λειτουργεί μόνο όταν[`ConsiderParentMargins`](../../../groupdocs.watermark/watermark/considerparentmargins) ορίζεται σε true. Αν[`ConsiderParentMargins`](../../../groupdocs.watermark/watermark/considerparentmargins) είναι ψευδής, όταν το pdf CropBox χρησιμοποιείται ως περιοχή υδατογράφησης .

Η προεπιλεγμένη τιμή είναιTrimBox.
