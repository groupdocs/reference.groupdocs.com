## WordProcessingWatermarkPagesOptions.PageNumbers property

Получает или задает номера страниц для добавления водяного знака.

```csharp
public int[] PageNumbers { get; set; }
```

### Стоимость имущества

Номера страниц для добавления водяного знака.

### Примечания

Все числа должны быть больше или равны 1. Это свойство используется только при добавлении водяного знака в документ. Если это значение`нулевой` или пустой, водяной знак добавляется на все страницы.
