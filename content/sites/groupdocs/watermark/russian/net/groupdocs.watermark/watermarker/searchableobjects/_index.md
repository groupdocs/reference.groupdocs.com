## Watermarker.SearchableObjects property

Получает или задает объекты содержимого, которые должны быть включены в поиск водяных знаков.

```csharp
public SearchableObjects SearchableObjects { get; set; }
```

### Стоимость имущества

Объекты, которые должны быть включены в поиск водяных знаков.

### Примечания

Это свойство также определяет объекты содержимого, используемые при поиске изображений. Дополнительные сведения см.[`Search`](../search) и[`GetImages`](../getimages) методы.

Подробнее о поиске водяных знаков: [Поиск водяных знаков](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Примеры

Удалить все объекты XObject и артефакты из документа PDF.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.pdf"))
{
    // Указываем, что следует искать только XObjects и Artifacts.
    watermarker.SearchableObjects.PdfSearchableObjects
        = PdfSearchableObjects.XObjects | PdfSearchableObjects.Artifacts; 

    PossibleWatermarkCollection watermarks = watermarker.Search();

    // Удалить все найденные объекты.
    watermarks.Clear();

    // Сохранить изменения.
    watermarker.Save();
}
```
