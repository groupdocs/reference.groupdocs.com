## Watermark.IsBackground property

Получает или задает значение, указывающее, следует ли размещать водяной знак на фоне.

```csharp
public bool IsBackground { get; set; }
```

### Стоимость имущества

Если значение равно true, водяной знак будет размещен внизу. По умолчанию значение равно false, водяной знак будет размещен вверху.
