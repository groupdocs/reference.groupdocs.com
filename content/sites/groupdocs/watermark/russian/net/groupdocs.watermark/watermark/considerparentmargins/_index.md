## Watermark.ConsiderParentMargins property

Получает или задает значение, указывающее, вычисляются ли размер и координаты водяного знака с учетом родительских полей.

```csharp
public bool ConsiderParentMargins { get; set; }
```

### Стоимость имущества

Если значение равно true, размер и координаты водяного знака рассчитываются с учетом родительских полей. По умолчанию значение равно false, родительские поля игнорируются.
