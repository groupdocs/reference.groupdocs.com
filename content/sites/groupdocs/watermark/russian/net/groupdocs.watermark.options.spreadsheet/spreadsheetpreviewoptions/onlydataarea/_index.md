## SpreadsheetPreviewOptions.OnlyDataArea property

Получает или устанавливает флаг для отображения только области данных без верхних, нижних колонтитулов и полей.

```csharp
public bool OnlyDataArea { get; set; }
```

### Стоимость имущества

Флаг для отрисовки области данных только без заголовков, футеров, полей.

### Примечания

Этот флаг приводит к отображению всего рабочего листа на одной странице. Значение по умолчанию:`ЛОЖЬ` .
