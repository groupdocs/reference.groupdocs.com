## LoadOptions.Password property

Получает или задает пароль для открытия зашифрованного документа.

```csharp
public string Password { get; set; }
```

### Стоимость имущества

Пароль для открытия зашифрованного документа.

### Примечания

Пароль может быть нулевым или пустой строкой. Значение по умолчанию — null. Если содержимое не зашифровано, установите для него значение null или пустую строку.

### Примеры

Загрузить содержимое, защищенное паролем.

```csharp
LoadOptions loadOptions = new LoadOptions
{
    Password = "pwd123"
};
using (Watermarker watermarker = new Watermarker(@"D:\doc.vsdx", loadOptions))
{
    // ...
}
```
