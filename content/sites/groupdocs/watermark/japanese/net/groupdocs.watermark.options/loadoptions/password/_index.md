## LoadOptions.Password property

暗号化されたドキュメントを開くためのパスワードを取得または設定します。

```csharp
public string Password { get; set; }
```

### プロパティ値

暗号化されたドキュメントを開くためのパスワード。

### 備考

パスワードは null または空の文字列です。デフォルト値は null です。 コンテンツが暗号化されていない場合は、これを null または空の文字列に設定します。

### 例

パスワードで保護されたコンテンツを読み込みます。

```csharp
LoadOptions loadOptions = new LoadOptions
{
    Password = "pwd123"
};
using (Watermarker watermarker = new Watermarker(@"D:\doc.vsdx", loadOptions))
{
    // ...
}
```
