## Watermarker.SearchableObjects property

透かし検索に含まれるコンテンツ オブジェクトを取得または設定します。

```csharp
public SearchableObjects SearchableObjects { get; set; }
```

### プロパティ値

透かし検索に含まれるオブジェクト。

### 備考

このプロパティは、画像検索で使用されるコンテンツ オブジェクトも指定します。 詳細については、[`Search`](../search) と[`GetImages`](../getimages)メソッド。

透かしの検索の詳細: [透かしの検索](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### 例

PDF ドキュメントからすべての XObjects と Artifacts を削除します。

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.pdf"))
{
    // XObject と Artifacts のみを検索するように指定します。
    watermarker.SearchableObjects.PdfSearchableObjects
        = PdfSearchableObjects.XObjects | PdfSearchableObjects.Artifacts; 

    PossibleWatermarkCollection watermarks = watermarker.Search();

    // 見つかったすべてのオブジェクトを削除します。
    watermarks.Clear();

    // 変更内容を保存。
    watermarker.Save();
}
```
