## Watermark.ConsiderParentMargins property

透かしのサイズと座標が 親マージンを考慮して計算されるかどうかを示す値を取得または設定します.

```csharp
public bool ConsiderParentMargins { get; set; }
```

### プロパティ値

値が true の場合、透かしのサイズと座標は親マージンを考慮して計算されます。 デフォルトでは、値は false で、親マージンは無視されます。
