## Watermarker.SearchableObjects property

वॉटरमार्क खोज में शामिल किए जाने वाले सामग्री ऑब्जेक्ट को प्राप्त या सेट करता है।

```csharp
public SearchableObjects SearchableObjects { get; set; }
```

### संपत्ति मूल्य

वॉटरमार्क खोज में शामिल किए जाने वाले ऑब्जेक्ट।

### टिप्पणियों

यह गुण उन सामग्री वस्तुओं को भी निर्दिष्ट करता है जिनका उपयोग छवि खोज में किया जाता है। अधिक जानकारी के लिए देखें[`Search`](../search) और[`GetImages`](../getimages) तरीके।

वॉटरमार्क खोजने के बारे में अधिक जानें: [वॉटरमार्क खोज रहे हैं](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### उदाहरण

एक पीडीएफ दस्तावेज़ से सभी एक्सऑब्जेक्ट्स और कलाकृतियों को हटा दें।

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.pdf"))
{
    // निर्दिष्ट करें कि केवल XObjects और कलाकृतियों की खोज की जानी चाहिए।
    watermarker.SearchableObjects.PdfSearchableObjects
        = PdfSearchableObjects.XObjects | PdfSearchableObjects.Artifacts; 

    PossibleWatermarkCollection watermarks = watermarker.Search();

    // सभी मिली वस्तुओं को हटा दें।
    watermarks.Clear();

    // परिवर्तनों को सुरक्षित करें।
    watermarker.Save();
}
```
