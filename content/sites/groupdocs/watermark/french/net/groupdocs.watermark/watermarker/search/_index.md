| --- | --- | --- |
| searchCriteria | SearchCriteria | Les critères de recherche à utiliser. |

### Return_Value

La collection de filigranes possibles.

### Remarques

La recherche est effectuée dans les objets spécifiés dans[`SearchableObjects`](../searchableobjects).

En savoir plus sur la recherche de filigranes [Recherche de filigranes](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Exemples

Rechercher et supprimer tous les filigranes possibles contenant un texte ou une image particulière d'un document de tout type pris en charge.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    ImageSearchCriteria imageSearchCriteria = new ImageDctHashSearchCriteria(@"D:\logo.png");
    Regex regex = new Regex(@"^Company\sName$", RegexOptions.IgnoreCase);
    TextSearchCriteria textSearchCriteria = new TextSearchCriteria(regex);
    PossibleWatermarkCollection watermarks = watermarker.Search(textSearchCriteria.Or(imageSearchCriteria));
    watermarks.Clear();
    watermarker.Save(@"D:\output.doc");
}
```
