## GetImages(ImageSearchCriteria) {#getimages_1}

Trouve des images selon des critères de recherche spécifiés.

```csharp
public WatermarkableImageCollection GetImages(ImageSearchCriteria searchCriteria)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| searchCriteria | ImageSearchCriteria | Les critères de recherche à utiliser. |

### Return_Value

La collection d'images trouvées.

### Remarques

La recherche est effectuée dans les objets spécifiés dans[`SearchableObjects`](../searchableobjects).

En savoir plus sur la recherche de filigranes [Recherche de filigranes](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Exemples

Supprimez toutes les images similaires à la référence d'un document de tout type pris en charge.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    ImageThumbnailSearchCriteria criteria = new ImageThumbnailSearchCriteria("reference.png");
    WatermarkableImageCollection images = watermarker.GetImages(criteria);
    images.Clear();
    watermarker.Save(@"D:\output.doc");
}
```

## GetImages() {#getimages}

Recherche toutes les images du document.

```csharp
public WatermarkableImageCollection GetImages()
```

### Return_Value

La collection d'images trouvées.

### Remarques

La recherche est effectuée dans les objets spécifiés dans[`SearchableObjects`](../searchableobjects).

En savoir plus sur la recherche de filigranes [Recherche de filigranes](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### Exemples

Ajoutez un filigrane à toutes les images d'un document de tout type pris en charge.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\input.doc"))
{
    // Initialise le filigrane de texte ou d'image.
    TextWatermark watermark = new TextWatermark("DRAFT", new Font("Arial", 19));

    // Trouve toutes les images dans le document.
    WatermarkableImageCollection images = watermarker.GetImages();

    // Ajouter un filigrane.
    foreach (WatermarkableImage watermarkableImage in images)
    {
        watermarkableImage.AddWatermark(watermark);
    }

    // Sauvegarder les modifications.
    watermarker.Save(@"D:\output.doc");
}
```
