## LoadOptions.Password property

Obtient ou définit le mot de passe pour ouvrir un document chiffré.

```csharp
public string Password { get; set; }
```

### Valeur de la propriété

Le mot de passe pour ouvrir un document crypté.

### Remarques

Le mot de passe doit être nul ou une chaîne vide. La valeur par défaut est null. Si le contenu n'est pas chiffré, définissez-le sur null ou sur une chaîne vide.

### Exemples

Charger un contenu protégé par un mot de passe.

```csharp
LoadOptions loadOptions = new LoadOptions
{
    Password = "pwd123"
};
using (Watermarker watermarker = new Watermarker(@"D:\doc.vsdx", loadOptions))
{
    // ...
}
```
