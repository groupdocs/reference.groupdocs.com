## Watermarker.SearchableObjects property

워터마크 검색에 포함될 콘텐츠 개체를 가져오거나 설정합니다.

```csharp
public SearchableObjects SearchableObjects { get; set; }
```

### 자산 가치

워터마크 검색에 포함될 개체입니다.

### 비고

이 속성은 이미지 검색에 사용되는 콘텐츠 개체도 지정합니다. 자세한 내용은 다음을 참조하세요.[`Search`](../search) 및[`GetImages`](../getimages) 행동 양식.

워터마크 검색에 대해 자세히 알아보기: [워터마크 검색](https://docs.groupdocs.com/display/watermarknet/Searching+watermarks).

### 예

PDF 문서에서 모든 XObject 및 아티팩트를 제거합니다.

```csharp
using (Watermarker watermarker = new Watermarker(@"D:\test.pdf"))
{
    // XObject 및 아티팩트만 검색하도록 지정합니다.
    watermarker.SearchableObjects.PdfSearchableObjects
        = PdfSearchableObjects.XObjects | PdfSearchableObjects.Artifacts; 

    PossibleWatermarkCollection watermarks = watermarker.Search();

    // 발견된 모든 객체를 제거합니다.
    watermarks.Clear();

    // 변경 사항을 저장하다.
    watermarker.Save();
}
```
