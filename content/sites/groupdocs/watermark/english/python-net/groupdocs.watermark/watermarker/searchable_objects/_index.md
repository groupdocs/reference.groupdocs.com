## searchable_objects property

The objects that are to be included in a watermark search.

This property also specifies content objects which are used in image search. For more information see [`Watermarker.search`](/watermark/python-net/groupdocs.watermark/watermarker/search/) and [`Watermarker.get_images`](/watermark/python-net/groupdocs.watermark/watermarker/get_images/) methods.
