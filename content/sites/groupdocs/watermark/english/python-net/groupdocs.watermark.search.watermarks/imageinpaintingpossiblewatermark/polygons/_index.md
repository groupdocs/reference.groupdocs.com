## polygons property

The polygonal regions to be inpainted, expressed as a list of polygons in image pixels.

Each polygon is represented as a list of vertices.

Either [`ImageInpaintingPossibleWatermark.Rectangles`](/watermark/python-net/groupdocs.watermark.search.watermarks/imageinpaintingpossiblewatermark/rectangles/) or [`ImageInpaintingPossibleWatermark.Polygons`](/watermark/python-net/groupdocs.watermark.search.watermarks/imageinpaintingpossiblewatermark/polygons/) must be specified, but not both.
