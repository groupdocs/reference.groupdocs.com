## OfficeLineStyle enumeration

Represents the compound line style of a [`OfficeLineFormat`](/watermark/python-net/groupdocs.watermark.contents/officelineformat).



The OfficeLineStyle type exposes the following members:

### Fields
| Field | Description |
| :- | :- |
| SINGLE | Single line. |
| DEFAULT | Default value is [`OfficeLineStyle.SINGLE`](/watermark/python-net/groupdocs.watermark.contents/officelinestyle#SINGLE). |
| DOUBLE | Double lines of equal width. |
| THICK_THIN | Double lines, one thick, one thin. |
| THIN_THICK | Double lines, one thin, one thick. |
| TRIPLE | Three lines, thin, thick, thin. |
