## WordProcessingHeaderFooter.IsLinkedToPrevious property

获取或设置一个值，该值指示此页眉/页脚是否链接到 上一节中的相应页眉/页脚。

```csharp
public bool IsLinkedToPrevious { get; set; }
```

### 适当的价值

如果此页眉/页脚链接到上一节中的相应页眉/页脚，则为 true；否则为 false。

### 评论

默认为真。请注意，当您链接页眉或页脚时，其内容将被清除。
