## InvalidFontFormatException class

Undantaget som skapas när man försöker öppna, ladda, spara eller bearbeta något innehåll på något annat sätt, som antagligen är ett teckensnitt med stödet (känt) format, men som faktiskt är ett teckensnitt av oväntat och/eller ostödd format eller inte ett teckensnitt på alla.
