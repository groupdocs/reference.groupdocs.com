## EbookDocumentInfo.PageCount property

Returnerar antal sidor i fallet med MOBI eller AZW3 eller antal kapitel i fallet med ePub.

```csharp
public int PageCount { get; }
```

### Anmärkningar

e-boksdokument har vanligtvis inga fasta sidor och därmed sidantal. När det gäller ePub är det möjligt att beräkna ett antal kapitel. Men formaten MOBI och AZW3 har inga kapitel heller, så detta antal beräknas från standard sidstorlek inställd på A4 i stående orientering.
