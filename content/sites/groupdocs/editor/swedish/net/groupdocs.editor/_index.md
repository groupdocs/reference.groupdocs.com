| [Editor](./editor) | Main class, som kapslar in konverteringsmetoder. Editor class tillhandahåller metoder för att ladda, redigera och spara dokument i alla format som stöds. Det är disponibelt, så använd ett "using"-direktiv eller kassera dess resurser manuellt via 'Dispose()' metodanrop. Dokumentladdning utförs genom konstruktörer.
