## EditableDocument.FromFile method

Static factory, som skapar en instans av EditableDocument från en HTML-fil, som anges av en sökväg till själva *.html-filen och en mapp med länkade resurser

```csharp
public static EditableDocument FromFile(string htmlFilePath, string resourceFolderPath)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| htmlFilePath | String | Sträng, som innehåller en fullständig sökväg till HTML-filen. Kan inte vara null, bör vara en giltig sökväg och själva filen bör finnas. |
| resourceFolderPath | String | Valfri sökväg till mappen med HTML-resurser. Om NULL, ogiltig eller sådan mapp inte finns, kommer Editor att försöka hitta den här mappen själv, analysera HTML-uppmärkningen |

### Returvärde

Ny icke-null-instans av EditableDocument

### Undantag

| undantag | skick |
| --- | --- |
| ArgumentException | HTML-filsökväg och/eller resursmappsökväg är ogiltiga |
| FileNotFoundException | Angiven HTML-fil kan inte hittas |
