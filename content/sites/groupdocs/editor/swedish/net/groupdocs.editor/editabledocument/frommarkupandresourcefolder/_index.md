## EditableDocument.FromMarkupAndResourceFolder method

Statisk fabrik, som skapar en instans av EditableDocument från en specificerad HTML-uppmärkning och från resurser, som finns i mappen, specificerad av hela path

```csharp
public static EditableDocument FromMarkupAndResourceFolder(string newHtmlContent, 
    string resourceFolderPath)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| newHtmlContent | String | Sträng, som innehåller rå HTML-uppmärkning, som bör tolkas. Kan inte vara NULL, tomt eller ogiltigt. |
| resourceFolderPath | String | Obligatorisk sökväg till mappen med resurser. Alla stilmallar, som finns i den här mappen, kommer att användas. Kan inte vara NULL eller tom sträng, och den här mappen bör finnas. |

### Returvärde

Ny icke-null-instans av EditableDocument

### Anmärkningar

Denna statiska fabrik är användbar när innehållet i HTML-dokumentet presenteras som en sträng, men alla resurser finns i någon mapp, och ofta är länkar till dessa resurser i HTML-uppmärkning ogiltiga och saknas. När den här metoden anropas skannar den angivna mappar och tillämpar automatiskt alla hittade stilmallar på dokumentet. Den här metoden är mycket användbar när du skaffar innehåll från olika HTML-redigerare, som vanligtvis skär bort dokumentets metadata och så vidare.
