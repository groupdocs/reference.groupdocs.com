## XmlEditOptions.FixIncorrectStructure property

Tillåter att aktivera eller inaktivera mekanism för att fixa skadad XML-struktur. Som standard är inaktiverad (false).

```csharp
public bool FixIncorrectStructure { get; set; }
```

### Anmärkningar

Som standard accepteras endast korrekt giltiga välformade XML-dokument. När det här alternativet är aktiverat kommer GroupDocs.Editor att försöka fixa skadad XML-struktur om möjligt.
