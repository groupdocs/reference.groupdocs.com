## PresentationEditOptions.SlideNumber property

Tillåter att ange bildnummer, som ska öppnas för redigering

```csharp
public int SlideNumber { get; set; }
```

### Anmärkningar

Bildnummer är ett nollbaserat index för en bild, som gör det möjligt att specificera och välja en viss bild från en presentation att redigera. Om det är mindre än 0, kommer den första bilden att väljas (samma som SlideNumber = 0). Om det är större än antalet av alla bilder i presentationen kommer den sista bilden att väljas. Om inmatningspresentationen endast innehåller en bild, ignoreras det här alternativet, och denna enstaka bild kommer att redigeras. Om du försöker öppna för redigering av en dold bild, medan[`ShowHiddenSlides`](../showhiddenslides) alternativet är inställt på "false", kommer undantaget att kastas.
