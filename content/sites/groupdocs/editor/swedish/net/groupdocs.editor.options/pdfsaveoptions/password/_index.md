## PdfSaveOptions.Password property

Lösenord, som kommer att tillämpas på det genererade PDF-dokumentet som användarlösenord, krävs för att öppna. Om NULL eller tomt, kommer inget lösenord att tillämpas på dokumentet. Annars kommer dokumentet att krypteras med RC4 (nyckellängd på 128 bitar). Som standard är NULL — lösenordet tillämpas inte.
