## SpreadsheetEditOptions.WorksheetIndex property

Gör det möjligt att specificera det 0-baserade indexet för kalkylbladet (fliken) i inmatningsdokumentet för kalkylark, som ska konverteras till HTML (se anmärkningar).

```csharp
public int WorksheetIndex { get; set; }
```

### Anmärkningar

De flesta kalkylarksdokument stöder ett koncept med flikar, dvs de kan ha flera flikar. Å andra sidan stöder inte HTML-format en sådan struktur. På grund av detta kan GroupDocs.Editor endast konvertera en specifik flik i indatadokumentet till HTML, och det här alternativet gör det möjligt att specificera den. Tabindex är 0-baserat negativa värden är förbjudna. Om det angivna indexet överskrider antalet alla flikar, kommer undantaget att kastas. Om inmatat kalkylarksdokument endast innehåller en flik, kommer detta alternativ att ignoreras. Standardvärdet är 0 (första fliken).
