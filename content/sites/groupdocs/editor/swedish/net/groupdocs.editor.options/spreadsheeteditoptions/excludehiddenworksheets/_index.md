## SpreadsheetEditOptions.ExcludeHiddenWorksheets property

Tillåter att utesluta dolda kalkylblad i kalkylarksdokumentet, så att de ignoreras totalt. Standard är falskt - dolda kalkylblad är tillgängliga och bearbetas som vanligt.

```csharp
public bool ExcludeHiddenWorksheets { get; set; }
```

### Anmärkningar

Flera binära kalkylbladsformat (som XLSX) stöder konceptet för dolda kalkylblad (flikar). Dokument av sådant format, om det har mer än ett kalkylblad, kan innehålla ytterligare dolda kalkylblad. Som standard är sådana dolda kalkylblad tillgängliga för bearbetning, men med det här alternativet kan den ignorera dem, som dessa dolda kalkylblad saknas och existerar inte. När det här alternativet är aktiverat kan du inte välja dolt kalkylblad med '[`WorksheetIndex`](../worksheetindex) egenskap.
