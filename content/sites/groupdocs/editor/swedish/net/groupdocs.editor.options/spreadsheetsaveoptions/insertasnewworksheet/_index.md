## SpreadsheetSaveOptions.InsertAsNewWorksheet property

Boolesk flagga, som anger om det redigerade kalkylbladet ska ersätta det befintliga kalkylbladet i det ursprungliga kalkylbladet på positionen, specificerad av[`WorksheetNumber`](../worksheetnumber) egenskap, eller så ska det injiceras mellan befintligt kalkylblad och föregående, utan att ersätta dess innehåll. Som standard är false — befintligt kalkylblad kommer att ersättas. Denna egenskap ignoreras, om värdet på[`WorksheetNumber`](../worksheetnumber) egenskapen är inställd på '0'.

```csharp
public bool InsertAsNewWorksheet { get; set; }
```

### Anmärkningar

Som standard ersätts kalkylbladet. Detta betyder att om ett kalkylblad har 5 kalkylblad, och[`WorksheetNumber`](../worksheetnumber) =4, då kommer det fjärde kalkylbladet att ersättas med det nya redigerade kalkylbladet, medan den totala mängden kalkylblad i kalkylbladet (5) kommer att förbli orörd. Men om värdet för den här egenskapen är inställd påSannkommer det nya redigerade kalkylbladet att injiceras som 4:e kalkylblad, och alla efterföljande kalkylblad kommer att flyttas till slutet: "gamla" 4:e kalkylbladet blir 5:e, och 5:e blir 6:e, och det totala antalet kalkylblad i kalkylbladet kommer att ökas med en och lika med 6.
