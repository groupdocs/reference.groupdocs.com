## WordProcessingEditOptions.ExtractOnlyUsedFont property

Hämtar eller ställer in ett värde som anger om endast teckensnittsresurser som används i dokumentets textinnehåll extraheras.

```csharp
public bool ExtractOnlyUsedFont { get; set; }
```

### Fastighetsvärde

`Sann` om det krävs att endast extrahera de teckensnittsresurser som används i textinnehållet i dokumentet; annat,`falsk` . Standardvärdet är`falsk` .

### Anmärkningar

Alla typsnitt som används i WordProcessing-dokumentet används inte till 100 % direkt (tillämpas på viss text). Det kan finnas en situation när teckensnittet refereras till i dokumentet och till och med kan vara inbäddat, men det tillämpas inte på någon text. Till exempel kan ett teckensnitt vara kopplat till en stil, men den här stilen tillämpas inte på någon del av texten. Det här alternativet styr hur sådana ärenden ska behandlas.
