## PresentationSaveOptions.InsertAsNewSlide property

부울 플래그는 편집된 슬라이드가[`SlideNumber`](../slidenumber) 또는 내용을 교체하지 않고 기존 슬라이드와 이전 슬라이드 사이에 삽입해야 합니다. 기본적으로 false — 기존 슬라이드가 교체됩니다. 값이[`SlideNumber`](../slidenumber) 속성이 '0'으로 설정되었습니다.

```csharp
public bool InsertAsNewSlide { get; set; }
```

### 비고

기본적으로 슬라이드가 대체됩니다. 즉, 주어진 프레젠테이션에 5개의 슬라이드가 있고[`SlideNumber`](../slidenumber) =4이면 4번째 슬라이드가 새로 편집된 슬라이드로 대체되고 프레젠테이션의 총 슬라이드 수(5)는 그대로 유지됩니다. 단, 이 속성의 값을 로 설정하면진실, 새로 편집된 슬라이드가 4번째 슬라이드로 삽입되고 이후의 모든 슬라이드는 끝으로 이동합니다. "이전" 4번째 슬라이드는 5번째가 되고 5번째는 6번째가 되며 프레젠테이션의 총 슬라이드 수는 1씩 증가하고 6과 같습니다.
