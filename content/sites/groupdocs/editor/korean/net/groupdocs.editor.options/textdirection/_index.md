## TextDirection enumeration

일반 텍스트 문서 에서 텍스트 방향을 처리하는 방법에 대한 3가지 가능한 변형을 나타냅니다.

```csharp
public enum TextDirection
```

### 가치

| 이름 | 값 | 설명 |
| --- | --- | --- |
| LeftToRight | `0` | 왼쪽에서 오른쪽 방향, 일반 텍스트, 기본값. |
| RightToLeft | `1` | 오른쪽에서 왼쪽 방향 |
| Auto | `2` | 자동 감지 방향. 이 옵션을 선택하고 텍스트에 RTL 스크립트에 속하는 문자가 포함되어 있으면 문서 방향이 자동으로 RTL로 설정됩니다. |
