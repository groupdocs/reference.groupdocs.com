## Azw3SaveOptions.SplitHeadingLevel property

AZW3 전자책의 콘텐츠를 패키지로 분할할지 여부를 지정하고, 분할할 경우 AZW3 콘텐츠를 분할할 제목의 최대 수준을 지정합니다. 기본값은`2` . 로 설정`0`분할을 비활성화하므로 전자책의 모든 콘텐츠가 AZW3 내부의 단일 패키지로 통합됩니다.

```csharp
public int SplitHeadingLevel { get; set; }
```

### 비고

경우에 따라 전자책의 콘텐츠를 출력 AZW3 파일 내에 있는 여러 개의 작은 패키지로 분할하는 것이 좋습니다. 이 속성은 이러한 분리를 수행해야 하는지 여부와 수행해야 하는 경우 방법을 제어합니다.

이 속성을 1에서 9 사이의 값으로 설정하면 using 형식의 단락에서 문서가 분할됩니다.**제목 1** ,**제목 2** ,**제목 3** 등 스타일을 지정된 제목 수준까지 지정합니다.

기본적으로(값`2` ), 오직**제목 1** 그리고**제목 2** 단락으로 인해 문서가 분할됩니다. 이 속성을 0으로 설정하면 문서가 제목 단락에서 전혀 분할되지 않습니다.
