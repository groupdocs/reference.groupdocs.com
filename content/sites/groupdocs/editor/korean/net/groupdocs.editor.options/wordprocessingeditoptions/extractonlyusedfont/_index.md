## WordProcessingEditOptions.ExtractOnlyUsedFont property

문서의 텍스트 내용에 사용된 글꼴 리소스만 추출할지 여부를 나타내는 값을 가져오거나 설정합니다.

```csharp
public bool ExtractOnlyUsedFont { get; set; }
```

### 자산 가치

`진실` 문서의 텍스트 콘텐츠에 사용되는 글꼴 리소스만 추출해야 하는 경우 그렇지 않으면,`거짓` . 기본값은`거짓` .

### 비고

WordProcessing 문서에서 사용되는 모든 글꼴이 100% 직접 사용되는 것은 아닙니다(일부 텍스트에 적용됨). 문서에서 글꼴이 참조되고 내장될 수도 있지만 어떤 텍스트에도 적용되지 않는 상황이 있을 수 있습니다. 예를 들어, 일부 글꼴은 일부 스타일에 첨부될 수 있지만 이 스타일은 텍스트의 어떤 부분에도 적용되지 않습니다. 이 옵션은 이러한 사례를 처리하는 방법을 제어합니다.
