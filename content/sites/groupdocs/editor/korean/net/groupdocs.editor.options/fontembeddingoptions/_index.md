## FontEmbeddingOptions enumeration

글꼴 포함 옵션은 출력 WordProcessing 또는 PDF에 포함되어야 하는 글꼴 리소스를 제어합니다. document

```csharp
public enum FontEmbeddingOptions : byte
```

### 가치

| 이름 | 값 | 설명 |
| --- | --- | --- |
| NotEmbed | `0` | EditableDocument나 시스템의 글꼴 리소스를 포함하지 마십시오. 기본값. |
| EmbedAll | `1` | 입력 EditableDocument에서 문서 내용을 분석하고 사용된 모든 글꼴을 찾아 출력 WordProcessing 또는 PDF 문서에 포함합니다. 우선 GroupDocs.Editor는 EditableDocument 내의 글꼴 리소스에서 글꼴을 가져옵니다. 글꼴이 부족하거나 누락된 경우 GroupDocs.Editor는 OS. 에서 글꼴을 가져옵니다. |
| EmbedWithoutSystem | `2` | 정확함EmbedAll , 그러나 OS에서 시스템 fonts 로 취급되는 글꼴은 제외합니다. |

### 비고

글꼴 포함 옵션은 문서 저장 중에 적용됩니다(중간 EditableDocument에서 출력 WordProcessing 또는 PDF 형식으로). 이 열거형은 사용해야 하는 WordProcessingSaveOptions 및 PdfSaveOptions의 속성으로 포함됩니다.
