## WordProcessingSaveOptions.LocaleBi property

작성 중에 적용될 RTL(오른쪽에서 왼쪽) 텍스트에 대한 WordProcessing 문서의 재정의 로케일(언어)을 설정할 수 있습니다. 지정되지 않은 경우(기본값), MS Word(또는 다른 프로그램) 자체 설정 또는 기타 요인에 따라 문서 RTL locale 를 감지(또는 선택)합니다.

```csharp
public CultureInfo LocaleBi { get; set; }
```

### 비고

이 옵션은 지정된 로케일을 문서의 전체 RTL 텍스트에 강제로 적용합니다. 문서에 다른 언어로 작성된 텍스트의 다른 부분이 포함된 경우 사용하지 마십시오.
