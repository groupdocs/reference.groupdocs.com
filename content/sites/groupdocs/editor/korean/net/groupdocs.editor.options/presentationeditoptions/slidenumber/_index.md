## PresentationEditOptions.SlideNumber property

편집을 위해 열어야 하는 슬라이드 번호를 지정할 수 있습니다.

```csharp
public int SlideNumber { get; set; }
```

### 비고

슬라이드 번호는 프레젠테이션에서 편집할 특정 슬라이드 하나를 지정하고 선택할 수 있는 슬라이드의 0부터 시작하는 인덱스입니다. 0보다 작으면 첫 번째 슬라이드가 선택됩니다(SlideNumber = 0과 동일). 프레젠테이션에 있는 모든 슬라이드의 양이 크면 마지막 슬라이드가 선택됩니다. 입력 프레젠테이션에 단일 슬라이드만 포함된 경우 이 옵션은 무시되고 이 단일 슬라이드가 편집됩니다. 숨겨진 슬라이드를 편집하기 위해 열려고 하면[`ShowHiddenSlides`](../showhiddenslides) 옵션이 'false'로 설정되면 예외가 발생합니다.
