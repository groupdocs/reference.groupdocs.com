```csharp
public static PageRange FromStartPageWithCount(ushort startPageNumber, ushort pageCount)
```

| startPageNumber | UInt16 | 페이지 범위가 시작되는 페이지 번호입니다. 페이지 번호는 1부터 시작하므로 반드시 0보다 커야 합니다. |
