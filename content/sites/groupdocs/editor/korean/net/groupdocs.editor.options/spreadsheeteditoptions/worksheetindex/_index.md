## SpreadsheetEditOptions.WorksheetIndex property

HTML로 변환해야 하는 입력 스프레드시트 문서의 워크시트(탭)의 0 기반 인덱스를 지정할 수 있습니다(설명 참조).

```csharp
public int WorksheetIndex { get; set; }
```

### 비고

대부분의 스프레드시트 문서는 탭 개념을 지원합니다. 즉, 다중 탭이 가능합니다. 반면에 HTML 형식은 이러한 구조를 지원하지 않습니다. 이 때문에 GroupDocs.Editor는 입력 문서의 특정 탭 하나만 HTML로 변환할 수 있으며 이 옵션을 사용하면 이를 지정할 수 있습니다. 탭 인덱스는 0 기반입니다. , 음수 값은 금지됩니다. 지정된 인덱스가 모든 탭의 수를 초과하면 예외가 발생합니다. 입력 스프레드시트 문서에 탭이 하나만 포함된 경우 이 옵션은 무시됩니다. 기본값은 0(첫 번째 탭)입니다.
