```csharp
public interface IImage
```

### 비고

https://developer.mozilla.org/en-US/docs/Web/CSS/image CSS 참조는 처리할 수 있는 4가지 유형의 이미지를 선언합니다. 모든 래스터 이미지를 포함하는 종횡비. 2. ICON(*.ico) 파일을 포함하는 단일 파일 내 여러 버전에 존재하는 여러 고유 치수가 있는 이미지. 이것은 하나의 파일 내에서 여러 이미지를 지원하므로 TIFF일 수도 있지만 TIFF 형식은 web. 에서 지원되지 않습니다. 3. 고유 치수가 없지만 너비와 높이 사이에 고유 종횡비가 있는 이미지. 여기에는 모든 벡터 이미지, 우선 SVG가 포함됩니다. 4. 고유 치수나 고유 종횡비가 없는 이미지. 여기에는 모든 그라디언트가 포함됩니다.
