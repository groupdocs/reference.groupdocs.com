## EditableDocument.FromFile method

HTML 파일에서 EditableDocument의 인스턴스를 만드는 정적 팩터리 는 *.html 파일 자체의 경로와 연결된 resources 가 있는 폴더로 지정됩니다.

```csharp
public static EditableDocument FromFile(string htmlFilePath, string resourceFolderPath)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| htmlFilePath | String | HTML 파일의 전체 경로를 포함하는 문자열입니다. null일 수 없으며 유효한 파일 경로여야 하며 파일 자체가 존재해야 합니다. |
| resourceFolderPath | String | HTML 리소스가 있는 폴더의 선택적 경로입니다. NULL, 유효하지 않거나 해당 폴더가 존재하지 않는 경우 편집기는 HTML 마크업을 분석하여 이 폴더를 스스로 찾으려고 시도합니다. |

### 반환 값

EditableDocument의 null이 아닌 새 인스턴스

### 예외

| 예외 | 상태 |
| --- | --- |
| ArgumentException | HTML 파일 경로 및/또는 리소스 폴더 경로가 유효하지 않습니다. |
| FileNotFoundException | 지정된 HTML 파일을 찾을 수 없습니다. |
