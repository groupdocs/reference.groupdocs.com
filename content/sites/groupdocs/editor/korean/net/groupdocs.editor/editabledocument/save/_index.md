## Save(string, string) {#save_1}

```csharp
public void Save(string htmlFilePath, string resourcesFolderPath)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| htmlFilePath | String | HTML 마크업이 저장될 파일의 전체 경로입니다. NULL이거나 비어 있을 수 없습니다. 파일이 있으면 파일을 생성하거나 덮어씁니다. |
