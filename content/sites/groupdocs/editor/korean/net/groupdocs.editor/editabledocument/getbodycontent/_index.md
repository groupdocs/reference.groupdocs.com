## GetBodyContent() {#getbodycontent}

HTML 문서의 본문(이러한 태그 없이 여는 BODY 태그와 닫는 BODY 태그 사이의 내부 콘텐츠)을 문자열로 반환합니다.

```csharp
public string GetBodyContent()
```

### 반환 값

HTML 문서의 본문을 포함하는 문자열(BODY 태그를 열고 닫지 않음)

### 비고

대부분의 WYSIWYG 편집기는 일반적으로 문서의 BODY 내부 내용으로 작동하며 HEAD 블록의 메타 정보를 올바르게 처리할 수 없습니다. 이 방법은 이러한 경우를 위해 설계되었습니다. 이 오버로드는 외부 리소스 요청에 대한 URI를 조정할 수 없습니다.
