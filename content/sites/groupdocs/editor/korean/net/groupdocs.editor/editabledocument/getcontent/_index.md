## GetContent(string, string) {#getcontent_1}

HTML 문서의 전체 내용을 문자열로 반환합니다. 여기서 외부 리소스에 대한 링크는 지정된 접두사를 포함합니다.

```csharp
public string GetContent(string externalImagesPrefix, string externalCssPrefix)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| externalImagesPrefix | String | 사용되는 이 매개변수를 통해 결과 HTML 문자열에 표시될 IMG 요소의 모든 외부 이미지에 대한 links 에 추가될 접두어를 지정할 수 있습니다. NULL이거나 비어 있으면 접두사가 추가되지 않습니다. |
| externalCssPrefix | String | 사용되는 이 매개변수를 통해 결과 HTML 문자열에 표시될 LINK 요소의 모든 외부 스타일시트에 링크 에 추가될 접두어를 지정할 수 있습니다. NULL이거나 비어 있으면 접두사가 추가되지 않습니다. |

### 반환 값

외부 리소스에 맞게 조정된 링크가 있는 HTML 문서의 내용을 포함하는 문자열
