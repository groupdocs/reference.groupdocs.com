## MarkdownSaveOptions.ExportImagesAsBase64 property

Указывает, сохраняются ли изображения в формате Base64 в выходной файл. По умолчанию`ЛОЖЬ` .

```csharp
public bool ExportImagesAsBase64 { get; set; }
```

### Примечания

Когда это свойство установлено в`истинный` то данные изображений экспортируются непосредственно в элементы изображения the ![]() и отдельные файлы не создаются. Это свойство, если установлено`истинный` , имеет более высокий приоритет, чем[`ImagesFolder`](../imagesfolder) свойство.
