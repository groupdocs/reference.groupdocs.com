## XmlEditOptions.FixIncorrectStructure property

Позволяет включить или отключить механизм исправления поврежденной структуры XML. По умолчанию отключено (false).

```csharp
public bool FixIncorrectStructure { get; set; }
```

### Примечания

По умолчанию допустимы только правильно сформированные XML-документы. Когда эта опция включена, GroupDocs.Editor попытается исправить поврежденную структуру XML, если это возможно.
