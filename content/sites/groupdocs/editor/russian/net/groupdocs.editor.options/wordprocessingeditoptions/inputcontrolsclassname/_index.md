## WordProcessingEditOptions.InputControlsClassName property

Позволяет указать имя класса, которое будет помещено в атрибуты class в каждом элементе HTML, представляющем некоторое поле во входном документе WordProcessing. По умолчанию NULL - атрибуты класса не применяются.

```csharp
public string InputControlsClassName { get; set; }
```

### Примечания

Почти все форматы из семейства форматов WordProcessing содержат поля — определенные сущности документа, которые позволяют получать входные данные от пользователей. Существует большое разнообразие полей: текстовые поля, флажки, поля со списками, выпадающие списки, кнопки, средства выбора даты/времени и т. д. Все они переведены в наиболее подходящие HTML-структуры и элементы, с сохранением введенного пользователем данные, если они присутствуют во входном документе. В определенных случаях требуется только сбор введенных данных на стороне клиента, а не редактирование всего содержимого документа. В таком случае необходимо каким-то образом идентифицировать элементы управления вводом, чтобы получить их вместе с их данными на стороне клиента. Это свойство позволяет указать имя класса, которое будет применяться для каждого элемента управления вводом в HTML-разметке, чтобы клиентский код мог просматривать структуру HTML-документа и собирать данные.
