## PresentationEditOptions.SlideNumber property

Позволяет указать номера слайдов, которые необходимо открыть для редактирования

```csharp
public int SlideNumber { get; set; }
```

### Примечания

Номер слайда — это отсчитываемый от нуля индекс слайда, который позволяет указать и выбрать один конкретный слайд из презентации для редактирования. Если меньше 0, будет выбран первый слайд (аналогично SlideNumber = 0). Если больше, чем количество всех слайдов в презентации, будет выбран последний слайд. Если входная презентация содержит только один слайд, эта опция будет проигнорирована, и этот единственный слайд будет отредактирован.[`ShowHiddenSlides`](../showhiddenslides) для параметра установлено значение «false», будет выдано исключение.
