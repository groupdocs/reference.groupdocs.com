## Характеристики

| Имя | Описание |

| [InsertAsNewWorksheet](../../groupdocs.editor.options/spreadsheetsaveoptions/insertasnewworksheet) { get; set; } | Логический флаг, который указывает, должен ли редактируемый рабочий лист заменить существующий рабочий лист в исходной электронной таблице в позиции, указанной[`WorksheetNumber`](./worksheetnumber) свойство, , либо оно должно быть введено между существующим рабочим листом и предыдущим, без замены его содержимого. По умолчанию false — существующий рабочий лист будет заменен. Это свойство игнорируется, если значение[`WorksheetNumber`](./worksheetnumber) свойство установлено в '0'. |
| [OutputFormat](../../groupdocs.editor.options/spreadsheetsaveoptions/outputformat) { get; set; } | Позволяет указать формат электронной таблицы, который будет использоваться для сохранения документа |
| [Password](../../groupdocs.editor.options/spreadsheetsaveoptions/password) { get; set; } | Позволяет указать, изменить, получить или удалить пароль, который будет использоваться для кодирования сгенерированного документа электронной таблицы, , если формат этого документа поддерживает защиту паролем. Укажите NULL или пустую строку для удаления (очистки) пароля. |
| [WorksheetNumber](../../groupdocs.editor.options/spreadsheetsaveoptions/worksheetnumber) { get; set; } | Позволяет вставить отредактированный рабочий лист в копию существующей электронной таблицы вместо создания новой электронной таблицы с одним рабочим листом (поведение по умолчанию). WorksheetNumber — это номер рабочего листа в электронной таблице, начинающийся с 1, загруженный в класс Editor. Если он равен 0 (значение по умолчанию), новая электронная таблица будет создана с одним редактируемым листом. входной экземпляр EditableDocument, будет вставлен в эту электронную таблицу. |
| [WorksheetProtection](../../groupdocs.editor.options/spreadsheetsaveoptions/worksheetprotection) { get; set; } | Позволяет включить защиту рабочего листа для выходного документа электронной таблицы. По умолчанию NULL - защита не применяется. Не все форматы поддерживают защиту листа. |
