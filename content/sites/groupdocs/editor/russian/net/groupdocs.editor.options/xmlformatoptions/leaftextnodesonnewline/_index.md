## XmlFormatOptions.LeafTextNodesOnNewline property

Если включено, конечные текстовые узлы (текстовое содержимое внутри XML-элементов, не имеющее дочерних элементов) будут отображаться на новой строке с большим левым отступом. По умолчанию false (отключено) — конечные текстовые узлы размещаются на той же их родителей, без нового отступа.

public bool LeafTextNodesOnNewline { get; set; }
