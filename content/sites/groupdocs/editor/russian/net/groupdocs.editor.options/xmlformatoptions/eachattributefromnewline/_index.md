## XmlFormatOptions.EachAttributeFromNewline property

Если этот параметр включен, каждая пара атрибут-значение в каждом элементе XML будет помещена в новую строку. По умолчанию установлено значение false (отключено) — все пары атрибут-значение помещаются в одну строку.

public bool EachAttributeFromNewline { get; set; }
