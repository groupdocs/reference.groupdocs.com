## WordProcessingSaveOptions.OptimizeMemoryUsage property

Включает механизмы оптимизации памяти во время создания документа из HTML, что снижает производительность за счет уменьшения использования памяти. Установка этого параметра в значение true может значительно снизить потребление памяти при создании больших документов за счет более медленной экономии времени. Значение по умолчанию — false (оптимизация памяти отключена для повышения производительности).

```csharp
public bool OptimizeMemoryUsage { get; set; }
```
