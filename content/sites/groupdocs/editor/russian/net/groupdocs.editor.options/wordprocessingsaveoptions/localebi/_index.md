## WordProcessingSaveOptions.LocaleBi property

Позволяет установить переопределение локали (языка) для документа WordProcessing для текста RTL (справа налево), который будет применяться при его создании. Если не указано (значение по умолчанию), MS Word (или другая программа) обнаружит (или выберет) документ RTL locale в соответствии со своими настройками или другими факторами.

```csharp
public CultureInfo LocaleBi { get; set; }
```

### Примечания

Эта опция принудительно применяет указанную локаль ко всему тексту справа налево в документе. Не используйте ее, если документ содержит разные части текста, написанные на разных языках.
