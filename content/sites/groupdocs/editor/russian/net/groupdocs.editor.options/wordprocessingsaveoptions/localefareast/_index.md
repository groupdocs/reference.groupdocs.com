## WordProcessingSaveOptions.LocaleFarEast property

Позволяет переопределить локаль (язык) документа WordProcessing для восточноазиатского текста, который будет применен при его создании. Если не указано (значение по умолчанию), MS Word (или другая программа) обнаружит (или выберет ) документ Восточно-Азиатская локаль в соответствии со своими настройками или другими факторами.

```csharp
public CultureInfo LocaleFarEast { get; set; }
```

### Примечания

Эта опция принудительно применяет указанную локаль ко всему восточноазиатскому тексту в документе. Не используйте ее, если документ содержит разные части текста, написанные на разных языках.
