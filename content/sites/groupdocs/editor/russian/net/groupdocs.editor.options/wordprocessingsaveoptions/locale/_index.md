## WordProcessingSaveOptions.Locale property

Позволяет установить переопределение локали (языка) по умолчанию для документа WordProcessing, который будет применяться при его создании. собственным настройкам или другим факторам.

```csharp
public CultureInfo Locale { get; set; }
```

### Примечания

Эта опция принудительно применяет указанную локаль ко всему тексту в документе. Не используйте ее, если документ содержит разные части текста, написанные на разных языках.
