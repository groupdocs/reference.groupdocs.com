## WordProcessingSaveOptions.EnablePagination property

Позволяет включить или отключить нумерацию страниц, которая будет использоваться для сохранения документа WordProcessing. Если исходный документ был открыт и отредактирован в режиме нумерации страниц, эта опция также должна быть включена. По умолчанию отключено.

```csharp
public bool EnablePagination { get; set; }
```
