## PdfSaveOptions.Password property

Пароль, который будет применен к сгенерированному PDF-документу в качестве пароля пользователя, необходимого для открытия. Если значение NULL или пусто, пароль к документу применяться не будет. В противном случае документ будет зашифрован RC4 (длина ключа 128 бит). По умолчанию NULL — пароль не применяется.

public string Password { get; set; }
