| [FontEmbedding](../../groupdocs.editor.options/pdfsaveoptions/fontembedding) { get; set; } | Отвечает за встраивание ресурсов шрифта, которые используются в исходном документе, в результирующий документ PDF. По умолчанию не встраивает шрифты (NotEmbed). |
| [OptimizeMemoryUsage](../../groupdocs.editor.options/pdfsaveoptions/optimizememoryusage) { get; set; } | Включает механизмы оптимизации памяти во время создания документа из HTML, что снижает производительность за счет уменьшения использования памяти. Установка этого параметра в значение true может значительно снизить потребление памяти при создании больших документов за счет более медленной экономии времени. Значение по умолчанию — false (оптимизация памяти отключена для повышения производительности). |

Если значение NULL или пусто, пароль к документу применяться не будет. В противном случае документ будет зашифрован RC4 (длина ключа 128 бит). По умолчанию NULL — пароль не применяется. |
