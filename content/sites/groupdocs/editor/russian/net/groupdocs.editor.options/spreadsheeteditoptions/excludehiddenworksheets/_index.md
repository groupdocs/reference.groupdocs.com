## SpreadsheetEditOptions.ExcludeHiddenWorksheets property

Позволяет исключить скрытые рабочие листы во входном документе электронной таблицы, поэтому они будут полностью проигнорированы. Значение по умолчанию false — скрытые рабочие листы доступны и обрабатываются как обычно.

```csharp
public bool ExcludeHiddenWorksheets { get; set; }
```

### Примечания

Некоторые двоичные форматы электронных таблиц (например, XLSX) поддерживают концепцию скрытых рабочих листов (вкладок). Документ такого формата, если в нем больше одного рабочего листа, может содержать дополнительные скрытые рабочие листы. По умолчанию такие скрытые рабочие листы доступны для обработки, но с этой опцией можно их игнорировать, вроде этих скрытых рабочих листов нет и не существует. Когда эта опция включена, вы не можете выбрать скрытый рабочий лист с '[`WorksheetIndex`](../worksheetindex) свойство.
