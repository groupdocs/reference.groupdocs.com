## PresentationLoadOptions.Password property

Позволяет указать, изменить и получить пароль, который будет использоваться для открытия документа Презентация, если он зашифрован. Установите значение NULL или пустую строку, чтобы удалить пароль.

```csharp
public string Password { get; set; }
```

### Примечания

По умолчанию это свойство имеет значение NULL — пароль не задан. Если входной документ презентации защищен паролем, пароль является обязательным, и если пароль не указан или недействителен, будет выдано исключение. Если входной документ презентации НЕ защищен паролем, но пароль установлен, он будет проигнорирован.
