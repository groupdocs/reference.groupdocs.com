## FixedLayoutEditOptionsBase.EnablePagination property

Позволяет включить (true) или отключить (false) нумерацию страниц в результирующем HTML-документе. По умолчанию отключено (false).

```csharp
public bool EnablePagination { get; set; }
```

### Примечания

Документы формата фиксированной верстки (в частности, PDF и XPS) по своей сути являются строго постраничными, их содержимое имеет фиксированную верстку и разбито на страницы. Но результирующий редактируемый HTML может быть представлен либо в бесстраничном, либо в постраничном представлении.
