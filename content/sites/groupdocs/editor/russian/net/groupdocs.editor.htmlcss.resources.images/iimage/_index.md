## IImage interface

Тип данных CSS Image представляет двумерное изображение

```csharp
public interface IImage
```

### Примечания

https://developer.mozilla.org/en-US/docs/Web/CSS/image Справочник по CSS объявляет 4 типа изображений, которые можно обрабатывать: соотношение сторон, включая все растровые изображения. 2. Изображения с несколькими внутренними размерами, существующие в нескольких версиях в одном файле, включая файлы ICON (*.ico). Это также может быть TIFF, так как они поддерживают несколько изображений в одном файле, однако формат TIFF не поддерживается в web. 3. Изображения без внутренних размеров, но с внутренним соотношением сторон между шириной и высотой. Сюда входят все векторные изображения, в первую очередь SVG. 4. Изображения без собственных размеров и собственного соотношения сторон. Сюда входят все градиенты.
