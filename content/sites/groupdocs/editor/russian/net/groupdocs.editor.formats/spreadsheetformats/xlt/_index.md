## SpreadsheetFormats.Xlt field

Шаблон Excel 97-2003 (XLT) представляет собой файлы шаблонов, созданные с помощью Microsoft Excel, приложения для работы с электронными таблицами, входящего в состав пакета Microsoft Office. Microsoft Office 97-2003 поддерживает создание новых файлов XLT, а также их открытие. Узнайте больше об этом формате файла[здесь](https://wiki.fileformat.com/spreadsheet/xlt) .
