## FontSize.IsRelativeSize property

Указывает, определен ли этот размер шрифта с относительным размером в качестве ключевого слова. Шрифт будет больше или меньше относительно размера шрифта родительского элемента, примерно на соотношение, используемое для разделения ключевых слов абсолютного размера.
