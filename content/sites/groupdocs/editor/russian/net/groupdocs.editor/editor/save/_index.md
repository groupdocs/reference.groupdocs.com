## Save(EditableDocument, Stream, ISaveOptions) {#save}

Преобразует указанный отредактированный документ, представленный как экземпляр '[`EditableDocument`](../../editabledocument) , в результирующий документ указанного формата и сохраняет его содержимое в указанный stream

```csharp
public void Save(EditableDocument inputDocument, Stream outputDocument, ISaveOptions saveOptions)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| inputDocument | EditableDocument | Версия входного документа, которая была отредактирована в HTML-редакторе WYSIWYG и хранится как экземпляр '[`EditableDocument`](../../editabledocument)class, который должен быть преобразован в выходной документ определенного формата. Не должен быть нулевым или удаленным. |
| outputDocument | Stream | Выходной поток, в который будет записано содержимое результирующего документа. Не должен быть нулевым, удаленным, должен поддерживать запись. |
| saveOptions | ISaveOptions | Параметры сохранения документа, которые определяют формат результирующего документа, , а также общие и специфичные для формата параметры сохранения. Не должен быть нулевым. |
