## EditableDocument.FromFile method

Статическая фабрика, создающая экземпляр EditableDocument из файла HTML, указанный путем к самому файлу *.html и папке со связанными ресурсами

```csharp
public static EditableDocument FromFile(string htmlFilePath, string resourceFolderPath)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| htmlFilePath | String | Строка, содержащая полный путь к файлу HTML. Не может быть нулевым, должен быть допустимым путем к файлу, а сам файл должен существовать. |
| resourceFolderPath | String | Необязательный путь к папке с HTML-ресурсами. Если NULL, неверный или такой папки не существует, Редактор попытается найти эту папку самостоятельно, анализируя HTML-разметку |

### Возвращаемое значение

Новый ненулевой экземпляр EditableDocument

### Исключения

| исключение | условие |
| --- | --- |
| ArgumentException | Путь к файлу HTML и/или путь к папке ресурсов недействителен. |
| FileNotFoundException | Указанный файл HTML не может быть найден |
