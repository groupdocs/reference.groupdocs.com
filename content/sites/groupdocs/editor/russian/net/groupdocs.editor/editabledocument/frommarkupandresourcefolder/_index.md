## EditableDocument.FromMarkupAndResourceFolder method

Статическая фабрика, создающая экземпляр EditableDocument из указанной HTML-разметки и из ресурсов, расположенных в папке, указанной полным путем

```csharp
public static EditableDocument FromMarkupAndResourceFolder(string newHtmlContent, 
    string resourceFolderPath)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| newHtmlContent | String | Строка, содержащая необработанную HTML-разметку, которую необходимо проанализировать. Не может быть NULL, пустым или недействительным. |
| resourceFolderPath | String | Обязательный путь к папке с ресурсами. Будут использованы все таблицы стилей, находящиеся в этой папке. Не может быть NULL или пустой строкой, и эта папка должна существовать. |

### Возвращаемое значение

Новый ненулевой экземпляр EditableDocument

### Примечания

Эта статическая фабрика полезна, когда содержимое HTML-документа представлено в виде строки, но все ресурсы находятся в какой-то папке, и часто ссылки на эти ресурсы в HTML-разметке недействительны и отсутствуют. При вызове этого метода он сканирует указанную папку и автоматически применяет к документу все найденные таблицы стилей. Этот метод очень удобен при получении контента из разных HTML-редакторов, которые обычно обрезают метаданные документа и так далее.
