## GetContent(string, string) {#getcontent_1}

Возвращает общее содержимое HTML-документа в виде строки, в которой ссылки на внешние ресурсы содержат указанный префикс.

```csharp
public string GetContent(string externalImagesPrefix, string externalCssPrefix)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| externalImagesPrefix | String | Через этот параметр можно указать префикс, который будет добавляться к links ко всем внешним изображениям в элементах IMG, которые будут присутствовать в результирующей HTML-строке. Если значение NULL или пустое, префиксы добавляться не будут. |
| externalCssPrefix | String | С помощью этого параметра можно указать префикс, который будет добавлен к links ко всем внешним таблицам стилей в элементах LINK, которые будут присутствовать в результирующей HTML-строке. Если значение NULL или пустое, префиксы добавляться не будут. |

### Возвращаемое значение

Строка, содержащая содержимое HTML-документа со ссылками, настроенными на внешние ресурсы
