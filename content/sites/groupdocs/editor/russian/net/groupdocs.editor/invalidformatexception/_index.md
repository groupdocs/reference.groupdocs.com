## InvalidFormatException class

Исключение, которое возникает, когда пользователь пытается открыть какой-либо документ с параметрами, специфичными для формата, которые несовместимы с исходным форматом документа.

```csharp
public sealed class InvalidFormatException : Exception
```

### Примечания

Например, это исключение будет выдано при попытке открыть документ электронной таблицы с параметрами документа WordProcessing.
