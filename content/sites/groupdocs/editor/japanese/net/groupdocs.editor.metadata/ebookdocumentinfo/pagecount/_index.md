## EbookDocumentInfo.PageCount property

MOBI または AZW3 の場合はページ数、ePub の場合は章数を返します。

```csharp
public int PageCount { get; }
```

### 備考

e-Book 文書には通常固定ページがないため、ページ数が決まっています。 ePub の場合、チャプター数を計算することができます。ただし、MOBI および AZW3 形式にも章がないため、この数は縦方向の A4 に設定された標準ページ サイズから計算されます.
