## EditableDocument.GetEmbeddedHtml method

この HTML ドキュメントのすべてのコンテンツと関連するすべてのリソースを 1 つの文字列の形式で返します。すべてのリソースは base64 でエンコードされた形式で HTML マークアップ内に埋め込まれます。

```csharp
public string GetEmbeddedHtml()
```

### 戻り値

いずれの場合も NULL または空ではない文字列

### 例外

| 例外 | 調子 |
| --- | --- |
| ObjectDisposedException | この EditableDocument インスタンスは既に破棄されています |

### 備考

このメソッドは、この EditableDocument を HTML に変換し、すべてのリソースが HTML マークアップと共に文字列に埋め込まれた単一の文字列 にシリアル化します。

* HTML-&gt; BODY のすべての画像は base64 形式に変換され、IMG の 'src' 属性に配置されます
* すべてのスタイルシートは、HTML-&gt;HEAD セクション内の STYLE 要素に格納されます
* スタイルシートのすべての画像は base64 形式に変換され、適切な CSS 宣言に配置されます
* スタイルシートのすべてのフォントは base64 形式に変換され、適切な @font-face @-rules に配置されます
