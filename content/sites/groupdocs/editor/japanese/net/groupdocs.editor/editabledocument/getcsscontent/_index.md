## GetCssContent(string, string) {#getcsscontent_1}

すべての外部スタイルシートのコンテンツを文字列のリストとして返します。ここで、1 つの文字列は 1 つのスタイルシートを表します. 指定されたプレフィックスは、結果として得られるすべてのスタイルシートの外部リソースへのすべてのリンクに適用されます. このための CSS がない場合、空のリストを返しますドキュメント.

```csharp
public List<string> GetCssContent(string externalImagesPrefix, string externalFontsPrefix)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| externalImagesPrefix | String | このパラメーターを使用してプレフィックスを指定できます。プレフィックスはすべての外部画像の links に追加され、結果の CSS 文字列の CSS 宣言に含まれます。 NULL または空の場合、プレフィックスは追加されません。 |
| externalFontsPrefix | String | このパラメーターを使用してプレフィックスを指定できます。このプレフィックスは、結果の CSS 文字列の @font-face @-rules 内のすべての外部フォントへの links に追加されます。 NULL または空の場合、プレフィックスは追加されません。 |

### 戻り値

各文字列が 1 つの CSS ドキュメントのコンテンツを保持する文字列のリスト
