## GetContent(string, string) {#getcontent_1}

HTML ドキュメントの全体的なコンテンツを文字列として返します。外部リソースへのリンクには指定されたプレフィックスが含まれます。

```csharp
public string GetContent(string externalImagesPrefix, string externalCssPrefix)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| externalImagesPrefix | String | このパラメーターを使用してプレフィックスを指定できます。これは、結果の HTML 文字列に存在する IMG 要素のすべての外部画像の links に追加されます。 NULL または空の場合、プレフィックスは追加されません。 |
| externalCssPrefix | String | このパラメーターを使用して接頭辞を指定できます。この接頭辞は、LINK 要素のすべての外部スタイルシートの links に追加され、結果の HTML 文字列に表示されます。 NULL または空の場合、プレフィックスは追加されません。 |

### 戻り値

外部リソースに合わせて調整された、リンク付きの HTML ドキュメントのコンテンツを含む文字列
