## GetBodyContent() {#getbodycontent}

HTML ドキュメントの本文 (BODY の開始タグと終了タグの間の内側のコンテンツで、これらのタグを含まないもの) を文字列として返します。

```csharp
public string GetBodyContent()
```

### 戻り値

HTML ドキュメントの本文を含む文字列 (BODY タグの開始と終了を除く)

### 備考

ほとんどの WYSIWYG エディターは、通常、ドキュメントの BODY の内部コンテンツで動作し、HEAD ブロックからのメタ情報を正しく処理できません。 このメソッドは、そのような場合のために設計されています。このオーバーロードでは、外部リソース リクエストの URI を調整できません。

## GetBodyContent(string) {#getbodycontent_1}

HTML ドキュメントの本文 (BODY タグの開始タグと終了タグの間の内部コンテンツ) を文字列として返します。 外部リソースへのリンクには指定されたプレフィックスが含まれます。

```csharp
public string GetBodyContent(string externalImagesPrefix)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| externalImagesPrefix | String | このパラメーターを使用してプレフィックスを指定できます。これは、結果の HTML 文字列に存在する IMG 要素のすべての外部画像の links に追加されます。 NULL または空の場合、プレフィックスは追加されません。 |

### 戻り値

外部画像に合わせて調整された、リンク付きの HTML ドキュメントの本文 (BODY タグの開始タグと終了タグを除く) を含む文字列

### 備考

ほとんどの WYSIWYG エディターは、通常、ドキュメントの BODY の内部コンテンツで動作し、HEAD ブロックからのメタ情報を正しく処理できません。 このメソッドは、そのような場合のために設計されています。このオーバーロードにより、外部リソース リクエストの URI を調整できます。
