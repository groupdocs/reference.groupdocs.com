## EditableDocument.FromFile method

HTML ファイルから EditableDocument のインスタンスを作成する静的ファクトリ。

```csharp
public static EditableDocument FromFile(string htmlFilePath, string resourceFolderPath)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| htmlFilePath | String | HTML ファイルへのフル パスを含む文字列。 null にすることはできません。有効なファイル パスである必要があり、ファイル自体が存在する必要があります。 |
| resourceFolderPath | String | HTML リソースを含むフォルダーへのオプションのパス。 NULL、無効、またはそのようなフォルダーが存在しない場合、エディターは HTML マークアップを分析して、このフォルダーを自分で見つけようとします。 |

### 戻り値

EditableDocument の新しい非 null インスタンス

### 例外

| 例外 | 調子 |
| --- | --- |
| ArgumentException | HTML ファイル パス、および/またはリソース フォルダ パスが無効です |
| FileNotFoundException | 指定された HTML ファイルが見つかりません |
