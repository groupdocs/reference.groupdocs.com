## Editor.GetDocumentInfo method

この「エディタ」インスタンスにロードされたドキュメントに関するメタデータを返します

```csharp
public IDocumentInfo GetDocumentInfo(string password)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| password | String | ドキュメントがパスワードで暗号化されている場合、ユーザーはドキュメントのパスワードを指定できます。 NULL または空の文字列の場合があります。これは、パスワードがない場合と同等です。パスワード保護機能を持たないドキュメント形式の場合、この引数は無視されます。ドキュメントが暗号化されていて、パスワードがこのパラメーターで指定されていないが、これを作成するときにロード オプションで以前に指定された場合[`Editor`](../../editor)たとえば、使用されます。 |

### 戻り値

フォーマット固有の継承者[`IDocumentInfo`](../../../groupdocs.editor.metadata/idocumentinfo)フォーマット固有のメタデータを含む検出されたフォーマットを示すインターフェイス、またはドキュメントがサポート可能であると認識されなかった場合、または破損している場合は NULL, 。
