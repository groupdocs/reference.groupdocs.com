## IImage interface

画像 CSS データ型は 2 次元の画像を表します

```csharp
public interface IImage
```

### 備考

https://developer.mozilla.org/en-US/docs/Web/CSS/image CSS リファレンスは、処理可能な 4 種類の画像を宣言しています: 1. 固有の寸法 (自然なサイズ) を持ち、固定されている画像すべてのラスター イメージを含む縦横比. 2. ICON (*.ico) ファイルを含む単一ファイル内の複数のバージョンに存在する複数の固有寸法を持つイメージ。 1 つのファイル内で複数の画像をサポートするため、これも TIFF の場合がありますが、web. では TIFF 形式はサポートされていません。これにはすべてのベクター画像が含まれ、まず SVG が含まれます。 4. 固有の寸法も固有の縦横比も持たない画像。これにはすべてのグラデーションが含まれます.
