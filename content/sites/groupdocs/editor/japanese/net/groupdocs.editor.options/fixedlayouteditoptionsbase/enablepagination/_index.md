## FixedLayoutEditOptionsBase.EnablePagination property

結果の HTML ドキュメントでページネーションを有効 (true) または無効 (false) にすることができます。デフォルトでは無効になっています (false).

```csharp
public bool EnablePagination { get; set; }
```

### 備考

固定レイアウト形式のドキュメント (特に PDF と XPS) は本質的に厳密にページ化されており、コンテンツは固定レイアウトでページに分割されています。ただし、結果として得られる編集可能な HTML は、ページレス ビューまたはページ ビューで表示できます。
