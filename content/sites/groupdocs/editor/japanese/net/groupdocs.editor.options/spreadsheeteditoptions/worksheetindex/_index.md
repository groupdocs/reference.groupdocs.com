## SpreadsheetEditOptions.WorksheetIndex property

HTML に変換する入力スプレッドシート ドキュメントのワークシート (タブ) の 0 から始まるインデックスを指定できます (備考を参照)。

```csharp
public int WorksheetIndex { get; set; }
```

### 備考

ほとんどのスプレッドシート ドキュメントは、タブの概念をサポートしています。つまり、マルチタブにすることができます。一方、HTML 形式はこのような構造をサポートしていません。 このため、GroupDocs.Editor は入力ドキュメントの特定の 1 つのタブのみを HTML に変換できます。 、負の値は禁止されています。指定されたインデックスがすべてのタブの数を超える場合、例外がスローされます。 入力スプレッドシート ドキュメントに含まれるタブが 1 つだけの場合、このオプションは無視されます。デフォルト値は 0 (最初のタブ). です。
