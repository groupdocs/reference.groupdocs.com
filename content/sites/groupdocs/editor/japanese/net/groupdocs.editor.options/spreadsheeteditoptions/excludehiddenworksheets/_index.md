## SpreadsheetEditOptions.ExcludeHiddenWorksheets property

入力スプレッドシート ドキュメントで非表示のワークシートを除外できるため、それらは完全に無視されます。 デフォルトは false - 非表示のワークシートが利用可能で、通常どおり処理されます。

```csharp
public bool ExcludeHiddenWorksheets { get; set; }
```

### 備考

いくつかのバイナリ スプレッドシート形式 (XLSX など) は、非表示のワークシート (タブ) の概念をサポートしています。 このような形式のドキュメントに複数のワークシートがある場合、追加の非表示のワークシートが含まれる場合があります。 デフォルトでは、このような非表示のワークシートを処理できますが、このオプションを使用するとそれらを無視できます。 のような非表示のワークシートは存在せず、存在しません。このオプションを有効にすると、 で非表示のワークシートを選択できなくなります。[`WorksheetIndex`](../worksheetindex)プロパティ.
