## PresentationEditOptions.SlideNumber property

編集用に開くスライド番号を指定できます

```csharp
public int SlideNumber { get; set; }
```

### 備考

スライド番号は、スライドの 0 から始まるインデックスであり、プレゼンテーションから編集する特定のスライドを 1 つ指定および選択できます。 0 より小さい場合、最初のスライドが選択されます (SlideNumber = 0 と同じ)。プレゼンテーション内のすべてのスライドの量より多い場合、最後のスライドが選択されます。入力プレゼンテーションに 1 つのスライドしか含まれていない場合、このオプションは無視され、この 1 つのスライドが編集されます。 非表示のスライドを編集するために開こうとすると、[`ShowHiddenSlides`](../showhiddenslides)オプションが「false」に設定されている場合、例外がスローされます。
