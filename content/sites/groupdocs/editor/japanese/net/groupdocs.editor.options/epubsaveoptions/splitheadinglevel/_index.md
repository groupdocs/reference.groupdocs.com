## EpubSaveOptions.SplitHeadingLevel property

ePub ファイルを分割する見出しの最大レベルを指定します。デフォルト値は`2` に設定`0`分割を無効にするため、e-Book のすべてのコンテンツが ePub 内の単一のパッケージに組み込まれます。

```csharp
public int SplitHeadingLevel { get; set; }
```

### 備考

このプロパティが 1 ～ 9 の値に設定されている場合、ドキュメントは、 を使用して書式設定された段落で分割されます。**見出し 1** 、**見出し 2** 、**見出し 3**指定された見出しレベルまでスタイルを設定します。

デフォルトでは、**見出し 1**と**見出し 2**段落によりドキュメントが分割されます。 このプロパティを 0 に設定すると、ドキュメントは見出し段落でまったく分割されません。
