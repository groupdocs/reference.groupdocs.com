## FontEmbeddingOptions enumeration

フォント埋め込みオプションは、どのフォント リソースを出力 WordProcessing または PDF ドキュメントに埋め込むかを制御します

```csharp
public enum FontEmbeddingOptions : byte
```

### 値

| 名前 | 価値 | 説明 |
| --- | --- | --- |
| NotEmbed | `0` | EditableDocument からもシステムからもフォント リソースを埋め込まないでください。デフォルト値. |
| EmbedAll | `1` | 入力 EditableDocument からドキュメント コンテンツを分析し、使用されているすべてのフォントを見つけて、それらを出力 WordProcessing または PDF ドキュメントに埋め込みます。 そもそも GroupDocs.Editor は、EditableDocument 内のフォント リソースからフォントを取得します。それらが不十分または欠落している場合、GroupDocs.Editor は OS. からフォントを取得します。 |
| EmbedWithoutSystem | `2` | 正確にEmbedAll、ただし、OS によってシステム fonts として扱われるフォントは除外されます |

### 備考

フォント埋め込みオプションは、ドキュメントの保存中に適用されます (中間の EditableDocument から出力 WordProcessing または PDF 形式へ)。
