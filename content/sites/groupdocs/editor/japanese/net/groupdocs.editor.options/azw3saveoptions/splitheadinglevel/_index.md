## Azw3SaveOptions.SplitHeadingLevel property

AZW3 電子書籍のコンテンツをパッケージに分割するかどうかを指定します。分割する場合は、AZW3 のコンテンツを分割する見出しの最大レベルを指定します。デフォルト値は`2` に設定`0`分割を無効にするため、電子書籍のすべてのコンテンツが AZW3. 内の単一のパッケージに組み込まれます。

```csharp
public int SplitHeadingLevel { get; set; }
```

### 備考

場合によっては、電子書籍のコンテンツをいくつかの小さなパッケージに分割し、出力 AZW3 ファイル内に配置することが望ましい場合があります。このプロパティは、そのような分離を実行する必要があるかどうかを制御し、実行する場合はその方法を制御します。

このプロパティが 1 ～ 9 の値に設定されている場合、ドキュメントは、 を使用して書式設定された段落で分割されます。**見出し 1** 、**見出し 2** 、**見出し 3**指定された見出しレベルまでスタイルを設定します。

デフォルトでは (値`2`）、 それだけ**見出し 1**と**見出し 2**段落によって文書が分割されます。このプロパティを 0 に設定すると、文書は見出し段落でまったく分割されません。
