## WordProcessingEditOptions.ExtractOnlyUsedFont property

ドキュメントのテキスト コンテンツで使用されているフォント リソースのみを抽出するかどうかを示す値を取得または設定します。

```csharp
public bool ExtractOnlyUsedFont { get; set; }
```

### プロパティ値

`真実`ドキュメントのテキスト コンテンツで使用されているフォント リソースのみを抽出する必要がある場合。さもないと、`間違い` .デフォルト値は`間違い`.

### 備考

WordProcessing ドキュメントで使用されるすべてのフォントが 100% 直接使用されるわけではありません (一部のテキストに適用されます)。フォントが文書内で参照され、埋め込まれていても、どのテキストにも適用されていない場合があります。たとえば、一部のフォントが一部のスタイルに関連付けられている場合がありますが、このスタイルはテキストのどの部分にも適用されません。このオプションは、そのようなケースの処理方法を制御します。
