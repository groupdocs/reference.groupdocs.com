## WordProcessingEditOptions.InputControlsClassName property

すべての HTML 要素の「class」属性に配置される、入力 WordProcessing ドキュメント内のフィールドを表すクラス名を指定できます。デフォルトでは NULL です - 'class' 属性は適用されません.

```csharp
public string InputControlsClassName { get; set; }
```

### 備考

WordProcessing フォーマット ファミリーのほとんどすべてのフォーマットには、ユーザーから入力データを取得できる特定のドキュメント エンティティであるフィールドが含まれています。テキストボックス、チェックボックス、コンボボックス、ドロップダウンリスト、ボタン、日付/時刻ピッカーなど、さまざまなフィールドがあります。それらはすべて、入力したユーザーを保持しながら、最も適切な HTML 構造と要素に変換されます。データ (入力ドキュメントに存在する場合)。特定のユース ケースでは、ドキュメント コンテンツ全体を編集するのではなく、入力されたデータをクライアント側で収集するだけで済みます。このような場合、入力コントロールをクライアント側のデータで取得するために、何らかの方法で入力コントロールを識別する必要があります。このプロパティを使用すると、HTML マークアップのすべての入力コントロールに適用されるクラス名を指定できるため、クライアント コードは HTML ドキュメント構造を走査してデータを収集できます。
