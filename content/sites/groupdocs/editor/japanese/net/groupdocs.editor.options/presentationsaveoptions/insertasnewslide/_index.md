## PresentationSaveOptions.InsertAsNewSlide property

Boolean フラグ。編集されたスライドが元のプレゼンテーションの既存のスライドを置き換える必要があるかどうかを指定します。[`SlideNumber`](../slidenumber) デフォルトでは false — 既存のスライドが置き換えられます。の値の場合、このプロパティは無視されます。[`SlideNumber`](../slidenumber)プロパティは「0」に設定されています.

```csharp
public bool InsertAsNewSlide { get; set; }
```

### 備考

デフォルトでは、スライドが置き換えられます。これは、与えられたプレゼンテーションに 5 つのスライドがあり、[`SlideNumber`](../slidenumber) =4 の場合、4 番目のスライドは新しく編集されたスライドに置き換えられますが、プレゼンテーション (5) のスライドの総数は変更されません。 ただし、このプロパティの値が に設定されている場合真実、新しく編集されたスライドが 4 番目のスライドとして挿入され、後続のすべてのスライドが最後にシフトされます。「古い」4 番目のスライドが 5 番目になり、5 番目が 6 番目になり、プレゼンテーション内のスライドの合計量が 1 ずつ増加し、 6に等しい。
