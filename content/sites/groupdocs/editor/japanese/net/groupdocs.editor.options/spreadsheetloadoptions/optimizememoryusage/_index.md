## SpreadsheetLoadOptions.OptimizeMemoryUsage property

入力ドキュメントの処理中にメモリ最適化メカニズムを有効にします。これにより、一部の特殊なケースでパフォーマンスが低下する可能性がありますが、 一方でメモリ使用量が減少します。巨大なドキュメントを処理し、OutOfMemoryException に直面している場合に役立ちます。 デフォルトは false (パフォーマンスを向上させるためにメモリの最適化は無効になっています)。

```csharp
public bool OptimizeMemoryUsage { get; set; }
```
