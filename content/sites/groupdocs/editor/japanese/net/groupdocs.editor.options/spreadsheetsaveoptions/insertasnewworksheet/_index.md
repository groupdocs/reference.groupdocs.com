## SpreadsheetSaveOptions.InsertAsNewWorksheet property

Boolean フラグ。編集されたワークシートが元のスプレッドシートの既存のワークシートを、[`WorksheetNumber`](../worksheetnumber)プロパティ、 または既存のワークシートと前のワークシートの間に挿入する必要があります。の値の場合、このプロパティは無視されます。[`WorksheetNumber`](../worksheetnumber)プロパティは「0」に設定されています.

```csharp
public bool InsertAsNewWorksheet { get; set; }
```

### 備考

デフォルトでは、ワークシートが置き換えられます。これは、指定されたスプレッドシートに 5 つのワークシートがあり、[`WorksheetNumber`](../worksheetnumber) =4 の場合、4 番目のワークシートは新しく編集されたワークシート に置き換えられますが、スプレッドシート (5) のワークシートの総数は変更されません. ただし、このプロパティの値が真実、新しく編集されたワークシートが 4 番目のワークシートとして挿入され、後続のすべてのワークシートが最後に移動されます。「古い」4 番目のワークシートが 5 番目になり、 と 5 番目が 6 番目になり、スプレッドシート内のワークシートの合計量が 1 増加します。そして6に等しい。
