## PresentationLoadOptions.Password property

暗号化されている場合、プレゼンテーション ドキュメントを開くために使用されるパスワードを指定、変更、および取得できます。 パスワードを削除するには、NULL または空の文字列に設定します。

```csharp
public string Password { get; set; }
```

### 備考

デフォルトでは、このプロパティには NULL 値があり、パスワードは設定されていません。入力プレゼンテーション ドキュメントがパスワードで保護されている場合、パスワードは必須であり、パスワードが指定されていないか無効な場合は例外がスローされます。入力されたプレゼンテーション ドキュメントがパスワードで保護されていないが、パスワードが設定されている場合、それは無視されます。
