## XmlFormatOptions.LeafTextNodesOnNewline property

有効にすると、リーフ テキスト ノード (子を持たない XML 要素内のテキスト コンテンツ) は、左インデントが大きい新しい行にレンダリングされます。 デフォルトは false (無効) — リーフ テキスト ノードは、親、新しいインデントなし.
