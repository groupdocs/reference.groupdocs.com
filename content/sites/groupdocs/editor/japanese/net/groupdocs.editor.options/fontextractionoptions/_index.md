## FontExtractionOptions enumeration

フォント抽出オプションは、どのフォントをどこから抽出するかを制御します

```csharp
public enum FontExtractionOptions : byte
```

### 値

| 名前 | 価値 | 説明 |
| --- | --- | --- |
| NotExtract | `0` | ドキュメントからもシステムからもフォント リソースを抽出しません。デフォルト値. |
| ExtractAllEmbedded | `1` | カスタムまたはシステムに関係なく、入力 Word ドキュメントに埋め込まれているすべてのフォント リソースを抽出します。 |
| ExtractEmbeddedWithoutSystem | `2` | カスタム (システムではない) の埋め込みフォント リソースのみを抽出します |
| ExtractAll | `3` | システム フォントを含む、入力 WordProcessing ドキュメントで使用されているすべてのフォントの抽出を試みます。 |
