## ResourceTypeDetector.TryDetectResource method

入力ストリームの分析を試み、それが null でない場合、指定された想定型を考慮して、サポート可能な HTML リソースの 1 つを作成します。

```csharp
public static IHtmlResource TryDetectResource(Stream inputResourceStream, string name, 
    IResourceType assumptiveFormat)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| inputResourceStream | Stream | おそらく HTML リソースを含む入力ストリーム。無効な場合、例外がスローされます。 |
| name | String | 成功時に作成され返されるリソースに使用されるリソース名。 NULL、空、または空白にすることはできません |
| assumptiveFormat | IResourceType | 入力 HTML リソースの想定される形式。最高のパフォーマンスを達成するのに役立ちます。 完全に不明な場合は、NULL 値を使用します。間違っている可能性があります。これはパフォーマンスを悪化させるだけです。 |

### 戻り値

「IHtmlResource」インターフェイスを実装し、成功した場合はサポート可能な HTML リソースの 1 つを表し、失敗した場合は NULL を表すインスタンス
