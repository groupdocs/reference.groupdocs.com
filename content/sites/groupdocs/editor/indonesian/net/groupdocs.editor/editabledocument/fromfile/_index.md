## EditableDocument.FromFile method

Pabrik statis, yang membuat instance EditableDocument dari file HTML, yang ditentukan oleh jalur ke file *.html itu sendiri dan folder dengan sumber daya tertaut

```csharp
public static EditableDocument FromFile(string htmlFilePath, string resourceFolderPath)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| htmlFilePath | String | String, yang berisi path lengkap ke file HTML. Tidak boleh nol, harus berupa path file yang valid, dan file itu sendiri harus ada. |
| resourceFolderPath | String | Jalur opsional ke folder dengan sumber daya HTML. Jika NULL, tidak valid, atau folder semacam itu tidak ada, Editor akan mencoba menemukan folder ini sendiri, menganalisis markup HTML |

### Nilai Pengembalian

Instance EditableDocument bukan nol baru

### Pengecualian

| pengecualian | kondisi |
| --- | --- |
| ArgumentException | Jalur file HTML, dan/atau jalur folder sumber daya tidak valid |
| FileNotFoundException | File HTML yang ditentukan tidak dapat ditemukan |
