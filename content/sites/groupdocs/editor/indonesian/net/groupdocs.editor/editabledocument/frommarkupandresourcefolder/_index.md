## EditableDocument.FromMarkupAndResourceFolder method

Pabrik statis, yang membuat instance EditableDocument dari markup HTML tertentu dan dari sumber daya, terletak di folder, ditentukan oleh path lengkap

```csharp
public static EditableDocument FromMarkupAndResourceFolder(string newHtmlContent, 
    string resourceFolderPath)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| newHtmlContent | String | String, yang berisi markup HTML mentah, yang harus diuraikan. Tidak boleh NULL, kosong, atau tidak valid. |
| resourceFolderPath | String | Jalur wajib ke folder dengan sumber daya. Semua stylesheet, yang terletak di folder ini, akan digunakan. Tidak boleh NULL atau string kosong, dan folder ini harus ada. |

### Nilai Pengembalian

Instance EditableDocument bukan nol baru

### Perkataan

Pabrik statis ini berguna ketika konten dokumen HTML disajikan sebagai string, tetapi semua sumber daya berada di beberapa folder, dan sering kali tautan ke sumber daya ini dalam markup HTML tidak valid dan tidak ada. Saat menjalankan metode ini, ia memindai folder yang ditentukan dan secara otomatis menerapkan semua lembar gaya yang ditemukan ke dokumen. Metode ini sangat berguna saat mendapatkan konten dari editor HTML yang berbeda, yang biasanya memotong metadata dokumen dan sebagainya.
