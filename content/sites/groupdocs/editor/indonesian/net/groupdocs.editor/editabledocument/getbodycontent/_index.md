## GetBodyContent() {#getbodycontent}

Mengembalikan isi dokumen HTML (konten dalam antara membuka dan menutup tag BODY tanpa tag ini) sebagai string.

```csharp
public string GetBodyContent()
```

### Nilai Pengembalian

String, yang berisi badan dokumen HTML (tanpa tag pembuka dan penutup BODY)

### Perkataan

Sebagian besar editor WYSIWYG biasanya beroperasi dengan konten bagian dalam BODY dokumen dan tidak dapat memproses informasi meta dengan benar dari blok HEAD. Metode ini dirancang untuk kasus seperti itu. Kelebihan ini tidak memungkinkan penyesuaian URI untuk permintaan sumber daya eksternal.
