## GetCssContent(string, string) {#getcsscontent_1}

Mengembalikan konten dari semua stylesheet eksternal sebagai daftar string, di mana satu string mewakili satu stylesheet. Awalan yang ditentukan akan diterapkan ke setiap tautan ke sumber daya eksternal di setiap stylesheet yang dihasilkan. Mengembalikan daftar kosong, jika tidak ada CSS untuk ini dokumen.

```csharp
public List<string> GetCssContent(string externalImagesPrefix, string externalFontsPrefix)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| externalImagesPrefix | String | Melalui parameter ini digunakan dapat menentukan awalan, yang akan ditambahkan ke links ke semua gambar eksternal, yang akan hadir dalam deklarasi CSS dalam string CSS yang dihasilkan. Jika NULL atau kosong, awalan tidak akan ditambahkan. |
| externalFontsPrefix | String | Melalui parameter ini digunakan dapat menentukan awalan, yang akan ditambahkan ke links ke semua font eksternal di @font-face at-rules di string CSS yang dihasilkan. Jika NULL atau kosong, awalan tidak akan ditambahkan. |

### Nilai Pengembalian

Daftar string, di mana setiap string berisi konten dari satu dokumen CSS
