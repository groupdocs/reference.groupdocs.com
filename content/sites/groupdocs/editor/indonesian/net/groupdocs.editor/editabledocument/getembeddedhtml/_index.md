## EditableDocument.GetEmbeddedHtml method

Mengembalikan semua konten dokumen HTML ini dengan semua sumber daya terkait dalam bentuk string tunggal, di mana semua sumber daya disematkan di dalam markup HTML dalam bentuk berenkode base64.

```csharp
public string GetEmbeddedHtml()
```

### Nilai Pengembalian

String, yang bukan NULL atau kosong dalam hal apa pun

### Pengecualian

| pengecualian | kondisi |
| --- | --- |
| ObjectDisposedException | Contoh EditableDocument ini sudah dibuang |

### Perkataan

Metode ini mengonversi Dokumen yang Dapat Diedit ini menjadi HTML dan membuat serial menjadi string tunggal, di mana semua sumber daya disematkan ke dalam string bersama dengan markup HTML:

* Semua gambar dari HTML-&gt;BODY dikonversi ke format base64 dan terletak di atribut IMG 'src'
* Semua stylesheet disimpan dalam elemen STYLE di dalam bagian HTML-&gt;HEAD
* Semua gambar dari stylesheet dikonversi ke format base64 dan ditempatkan di deklarasi CSS yang sesuai
* Semua font dari stylesheet dikonversi ke format base64 dan ditempatkan di aturan @font-face yang sesuai
