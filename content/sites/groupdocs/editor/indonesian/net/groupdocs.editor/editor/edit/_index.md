## Edit(IEditOptions) {#edit_1}

Membuka dokumen yang dimuat sebelumnya untuk diedit menggunakan opsi khusus format yang ditentukan dengan membuat dan mengembalikan instance '[`EditableDocument`](../../editabledocument) kelas, yang pada gilirannya berisi metode untuk menghasilkan markup HTML dan sumber daya terkait.

```csharp
public EditableDocument Edit(IEditOptions editOptions)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| editOptions | IEditOptions | Opsi dokumen khusus format, yang memungkinkan proses konversi tune-up. Mungkin NULL — dalam hal ini GroupDocs.Editor mendeteksi format dokumen yang dimuat sebelumnya dan menerapkan opsi, default untuk format ini. Tidak boleh bertentangan dengan opsi pemuatan yang diterapkan sebelumnya. |

### Nilai Pengembalian

Contoh dari '[`EditableDocument`](../../editabledocument) kelas, yang merangkum keseluruhan dokumen input dengan semua sumber dayanya dalam format menengah. Metode ini, jika berhasil diselesaikan, tidak pernah mengembalikan NULL.

### Perkataan

Ketika memasukkan dokumen asli dimuat ke instance 'Editor' melalui konstruktor, metode ini memungkinkan untuk membuka dokumen untuk diedit dengan mengubahnya menjadi format perantara, yang dienkapsulasi dalam instance kelas 'EditableDocument'. '[`EditableDocument`](../../editabledocument) dikembalikan dari metode ini, berisi semua metode dan properti yang diperlukan untuk menghasilkan markup HTML dan sumber daya yang sesuai (seperti gambar, font, dan lembar gaya) dalam semua konfigurasi yang diperlukan untuk meneruskannya ke editor HTML WYSIWYG mana pun. Kelebihan ini mendapatkan opsi edit, yang khusus untuk format keluarga. **Belajarlah lagi**
