## IImage interface

Jenis data CSS gambar merepresentasikan gambar dua dimensi

```csharp
public interface IImage
```

### Perkataan

https://developer.mozilla.org/en-US/docs/Web/CSS/image Referensi CSS menyatakan 4 jenis gambar, yang dapat ditangani: 1. Gambar dengan dimensi intrinsik (ukuran alami) dan diperbaiki rasio aspek, yang mencakup semua gambar raster. 2. Gambar dengan beberapa dimensi intrinsik, ada dalam beberapa versi di dalam satu file, yang menyertakan file ICON (*.ico). Ini mungkin juga TIFF, karena mendukung banyak gambar dalam satu file, namun format TIFF tidak didukung di web. 3. Gambar tanpa dimensi intrinsik tetapi dengan rasio aspek intrinsik antara lebar dan tingginya. Ini mencakup semua gambar vektor, terutama SVG. 4. Gambar tanpa dimensi intrinsik, maupun rasio tinggi lebar intrinsik. Ini mencakup semua gradien.
