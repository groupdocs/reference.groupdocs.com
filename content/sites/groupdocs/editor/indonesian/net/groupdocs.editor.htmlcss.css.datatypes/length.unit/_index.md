## Length.Unit enumeration

Semua satuan panjang yang didukung

```csharp
public enum Unit : byte
```

### Nilai

| Nama | Nilai | Keterangan |
| --- | --- | --- |
| Unitless | `0` | Tanpa unit - tidak ada satuan panjang yang ditentukan. Nilai default. |
| Px | `1` | Piksel. Relatif terhadap perangkat tampilan. Untuk tampilan layar, biasanya satu piksel perangkat (titik) dari tampilan. |
| Em | `2` | Em. Unit ini mewakili ukuran font yang dihitung dari elemen. |
| Ex | `3` | Ex (panjang x). Satuan ini mewakili tinggi-x font elemen. Pada font dengan huruf 'x', biasanya tinggi huruf kecil pada font; 1ex ≈ 0,5em dalam banyak font. |
| Cm | `4` | Cm. Satu sentimeter (10 milimeter). |
| Mm | `5` | Mm. Satu milimeter. |
| In | `6` | Di. Satu inci (2,54 sentimeter). |
| Pt | `7` | Poin. Satu titik adalah 1/72 inci atau 0,353 mm. |
| Pc | `8` | Pc. Satu pica (12 poin). |
| Ch | `9` | Ch. Satuan ini mewakili lebar, atau lebih tepatnya ukuran awal, dari mesin terbang '0' (nol, karakter Unicode U+0030) dalam font elemen. |
| Rem | `10` | Rem. Unit ini mewakili ukuran font dari elemen root (misalnya ukuran font dari elemen &lt;html&gt;). Saat digunakan pada ukuran font pada elemen root ini, ini mewakili nilai awalnya. |
| Vw | `11` | Vw - lebar viewport. 1/100 dari lebar viewport. |
| Vh | `12` | Vh - tinggi viewport. 1/100 dari tinggi viewport. |
| Vmin | `13` | Vmin. 1/100 dari nilai minimum antara tinggi dan lebar viewport. |
| Vmax | `14` | Vmaks. 1/100 dari nilai maksimum antara tinggi dan lebar viewport. |
| Percent | `15` | Nilainya relatif terhadap nilai tetap (eksternal), yang bergantung pada konteks . 1% = 1/100 dari nilai eksternal. |
