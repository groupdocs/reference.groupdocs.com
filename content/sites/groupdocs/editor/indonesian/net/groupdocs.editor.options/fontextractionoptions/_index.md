## FontExtractionOptions enumeration

Opsi ekstraksi font mengontrol font mana yang harus diekstraksi dan dari mana

```csharp
public enum FontExtractionOptions : byte
```

### Nilai

| Nama | Nilai | Keterangan |
| --- | --- | --- |
| NotExtract | `0` | Tidak mengekstrak sumber daya font apa pun baik dari dokumen maupun dari sistem. Nilai default. |
| ExtractAllEmbedded | `1` | Mengekstrak semua sumber daya font, yang disematkan ke dalam dokumen input Word, apa pun itu: kustom atau sistem. |
| ExtractEmbeddedWithoutSystem | `2` | Hanya mengekstrak sumber daya font tersemat, yang dikustomisasi (bukan sistem) |
| ExtractAll | `3` | Mencoba mengekstrak semua font, yang digunakan dalam dokumen input WordProcessing, termasuk font sistem. |
