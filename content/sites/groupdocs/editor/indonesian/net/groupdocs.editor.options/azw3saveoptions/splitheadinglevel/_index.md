## Azw3SaveOptions.SplitHeadingLevel property

Menentukan apakah akan membagi konten e-book AZW3 ke dalam paket dan jika ya, tingkat judul maksimum untuk membagi konten AZW3. Nilai default adalah`2` . Mengaturnya ke`0`akan menonaktifkan pemisahan, sehingga semua konten e-book akan digabungkan menjadi satu paket di dalam AZW3.

```csharp
public int SplitHeadingLevel { get; set; }
```

### Perkataan

Dalam beberapa kasus, sebaiknya membagi konten e-book menjadi beberapa paket yang lebih kecil, yang terletak di dalam file keluaran AZW3. Properti ini mengontrol apakah pemisahan tersebut harus dilakukan dan jika ya, bagaimana caranya.

Saat properti ini disetel ke nilai dari 1 hingga 9, dokumen akan dipisahkan menjadi paragraf yang diformat menggunakan **Tajuk 1** ,**Tajuk 2** ,**Tajuk 3** dll. gaya hingga tingkat tajuk yang ditentukan.

Secara default (nilai`2` ), hanya**Tajuk 1** Dan**Tajuk 2** paragraf menyebabkan dokumen terpecah. Menyetel properti ini ke nol akan menyebabkan dokumen sama sekali tidak terbelah pada paragraf judul.
