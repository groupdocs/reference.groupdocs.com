## EpubSaveOptions.SplitHeadingLevel property

Menentukan tingkat judul maksimum untuk memisahkan file ePub. Nilai default adalah`2` . Mengaturnya ke`0` akan menonaktifkan pemisahan, sehingga semua konten eBuku akan digabungkan menjadi satu paket di dalam ePub.

```csharp
public int SplitHeadingLevel { get; set; }
```

### Perkataan

Saat properti ini disetel ke nilai dari 1 hingga 9, dokumen akan dipisahkan menjadi paragraf yang diformat menggunakan **Tajuk 1** ,**Tajuk 2** ,**Tajuk 3** dll. gaya hingga tingkat tajuk yang ditentukan.

Secara default, hanya**Tajuk 1** Dan**Tajuk 2** paragraf menyebabkan dokumen terbelah. Menyetel properti ini ke nol akan menyebabkan dokumen sama sekali tidak terbelah pada paragraf judul.
