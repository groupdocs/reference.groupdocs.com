## XmlEditOptions.FixIncorrectStructure property

Memungkinkan untuk mengaktifkan atau menonaktifkan mekanisme untuk memperbaiki struktur XML yang rusak. Secara default dinonaktifkan (false).

```csharp
public bool FixIncorrectStructure { get; set; }
```

### Perkataan

Secara default, hanya dokumen XML yang benar dan valid yang dapat diterima. Saat opsi ini diaktifkan, GroupDocs.Editor akan mencoba memperbaiki struktur XML yang rusak jika memungkinkan.
