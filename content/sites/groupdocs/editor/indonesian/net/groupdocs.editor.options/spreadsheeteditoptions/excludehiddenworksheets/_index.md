## SpreadsheetEditOptions.ExcludeHiddenWorksheets property

Memungkinkan untuk mengecualikan lembar kerja tersembunyi di dokumen Lembar Kerja masukan, sehingga akan diabaikan sama sekali. Default salah - lembar kerja tersembunyi tersedia dan diproses seperti biasa.

```csharp
public bool ExcludeHiddenWorksheets { get; set; }
```

### Perkataan

Beberapa format lembar kerja biner (seperti XLSX) mendukung konsep lembar kerja (tab) tersembunyi. Dokumen dengan format seperti itu, jika memiliki lebih dari satu lembar kerja, mungkin berisi lembar kerja tambahan yang tersembunyi. Secara default lembar kerja tersembunyi tersebut tersedia untuk diproses, tetapi dengan opsi ini dapat mengabaikannya, seperti lembar kerja tersembunyi ini tidak ada dan tidak ada. Saat opsi ini diaktifkan, Anda tidak dapat memilih lembar kerja tersembunyi dengan '[`WorksheetIndex`](../worksheetindex) properti.
