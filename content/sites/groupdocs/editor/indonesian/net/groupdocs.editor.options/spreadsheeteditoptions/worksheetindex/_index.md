## SpreadsheetEditOptions.WorksheetIndex property

Memungkinkan untuk menentukan indeks berbasis 0 dari lembar kerja (tab) dari dokumen lembar bentang masukan, yang harus dikonversi ke HTML (lihat keterangan).

```csharp
public int WorksheetIndex { get; set; }
```

### Perkataan

Sebagian besar dokumen Spreadsheet mendukung konsep tab, yaitu bisa multi-tab. Di sisi lain, format HTML tidak mendukung struktur seperti itu. Karena GroupDocs.Editor ini dapat mengonversi ke HTML hanya satu tab tertentu dari dokumen masukan, dan opsi ini memungkinkan untuk menentukannya. Indeks tab berbasis 0 , nilai negatif dilarang. Jika indeks yang ditentukan melebihi jumlah semua tab, pengecualian akan dibuang. Jika dokumen Lembar Bentang masukan hanya berisi satu tab, opsi ini akan diabaikan. Nilai default adalah 0 (tab pertama).
