## PdfSaveOptions.Password property

Kata sandi, yang akan diterapkan ke dokumen PDF yang dihasilkan sebagai kata sandi pengguna, diperlukan untuk membuka. Jika NULL atau kosong, tidak ada kata sandi yang akan diterapkan ke dokumen. Jika tidak, dokumen akan dienkripsi dengan RC4 (panjang kunci 128 bit). Secara default adalah NULL — kata sandi tidak diterapkan.
