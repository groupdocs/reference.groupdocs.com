## WordProcessingEditOptions.InputControlsClassName property

Memungkinkan untuk menentukan nama kelas, yang akan ditempatkan ke atribut 'kelas' di setiap elemen HTML, yang mewakili beberapa bidang dalam dokumen masukan WordProcessing. Secara default adalah NULL - atribut 'kelas' tidak diterapkan.

```csharp
public string InputControlsClassName { get; set; }
```

### Perkataan

Hampir semua format dari keluarga format WordProcessing berisi bidang — entitas dokumen tertentu, yang memungkinkan untuk mendapatkan input data dari pengguna. Ada berbagai macam bidang: kotak teks, kotak centang, kotak kombo, daftar tarik-turun, tombol, pemilih tanggal/waktu, dll. Semuanya diterjemahkan ke dalam struktur dan elemen HTML yang paling sesuai, dengan mempertahankan pengguna yang dimasukkan data, jika ada dalam dokumen masukan. Dalam kasus penggunaan tertentu, hanya perlu mengumpulkan data yang dimasukkan di sisi klien alih-alih mengedit seluruh konten dokumen. Untuk kasus seperti itu, diperlukan untuk mengidentifikasi kontrol input dengan cara tertentu untuk mengambilnya dengan datanya di sisi klien. Properti ini memungkinkan untuk menentukan nama kelas, yang akan diterapkan untuk setiap kontrol masukan dalam markup HTML, sehingga kode klien akan dapat melintasi struktur dokumen HTML dan mengumpulkan data.
