## WordProcessingEditOptions.ExtractOnlyUsedFont property

Mendapat atau menetapkan nilai yang menunjukkan apakah hanya mengekstrak sumber daya font yang digunakan dalam konten tekstual dokumen.

```csharp
public bool ExtractOnlyUsedFont { get; set; }
```

### Nilai properti

`BENAR` jika diperlukan untuk mengekstrak hanya sumber daya font yang digunakan dalam konten teks dokumen; jika tidak,`PALSU` . Nilai default adalah`PALSU` .

### Perkataan

Tidak semua font yang digunakan dalam dokumen WordProcessing 100% digunakan secara langsung (diterapkan ke beberapa teks). Mungkin ada situasi, ketika font direferensikan dalam dokumen dan bahkan mungkin disematkan, tetapi tidak diterapkan pada teks mana pun. Misalnya, beberapa font mungkin dilampirkan ke beberapa gaya, tetapi gaya ini tidak diterapkan ke bagian teks mana pun. Opsi ini mengontrol cara memproses kasus tersebut.
