## FixedLayoutEditOptionsBase.EnablePagination property

Memungkinkan untuk mengaktifkan paginasi (benar) atau menonaktifkan (salah) dalam dokumen HTML yang dihasilkan. Secara default dinonaktifkan (false).

```csharp
public bool EnablePagination { get; set; }
```

### Perkataan

Dokumen dengan format tata letak tetap (khususnya PDF dan XPS) pada dasarnya diberi halaman dengan ketat, kontennya memiliki tata letak tetap dan dibagi menjadi beberapa halaman. Tetapi HTML yang dapat diedit yang dihasilkan dapat direpresentasikan dalam tampilan tanpa halaman atau paginal.
