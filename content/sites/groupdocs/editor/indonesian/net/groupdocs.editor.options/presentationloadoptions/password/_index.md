## PresentationLoadOptions.Password property

Memungkinkan untuk menentukan, memodifikasi, dan mendapatkan kata sandi, yang akan digunakan untuk membuka dokumen Presentasi, jika dikodekan. Atur ke NULL atau string kosong untuk menghapus kata sandi.

```csharp
public string Password { get; set; }
```

### Perkataan

Secara default, properti ini memiliki nilai NULL — kata sandi tidak disetel. Jika dokumen Presentasi masukan dilindungi kata sandi, kata sandi wajib dan pengecualian akan dilemparkan jika kata sandi tidak ditentukan atau tidak valid. Jika dokumen Presentasi input TIDAK dilindungi kata sandi, tetapi kata sandi diatur, itu akan diabaikan.
