## FixedLayoutFormats.Xps field

Le fichier XPS représente les fichiers de mise en page basés sur les spécifications papier XML créées par Microsoft. Il a été développé pour remplacer le format de fichier EMF et est similaire au format de fichier PDF, mais utilise XML dans la mise en page, l'apparence et les informations d'impression d'un document. En savoir plus sur ce format de fichier[ici](https://docs.fileformat.com/page-description-language/xps/) .
