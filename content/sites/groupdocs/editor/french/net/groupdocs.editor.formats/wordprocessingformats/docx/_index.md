## WordProcessingFormats.Docx field

Office Open XML WordProcessingML Macro-Free Document (DOCX) est un format bien connu pour les documents Microsoft Word. Introduit à partir de 2007 avec la sortie de Microsoft Office 2007, la structure de ce nouveau format de document est passée de binaire simple à une combinaison de fichiers XML et binaires.
