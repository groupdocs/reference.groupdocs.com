## WordProcessingSaveOptions.EnablePagination property

Permet d'activer ou de désactiver la pagination qui sera utilisée pour enregistrer le document WordProcessing. Si le document original a été ouvert et modifié en mode pagination, cette option doit également être activée. Par défaut est désactivé.
