## XmlFormatOptions.EachAttributeFromNewline property

Lorsqu'il est activé, chaque paire attribut-valeur dans chaque élément XML sera placée sur une nouvelle ligne. Par défaut, est faux (désactivé) — toutes les paires attribut-valeur sont placées sur une seule ligne.

public bool EachAttributeFromNewline { get; set; }
