## FontEmbeddingOptions enumeration

Les options d'incorporation de polices contrôlent les ressources de police à incorporer dans le document de sortie WordProcessing ou PDF

```csharp
public enum FontEmbeddingOptions : byte
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| NotEmbed | `0` | N'intégrez aucune ressource de police ni de EditableDocument ni du système. Valeur par défaut. |
| EmbedAll | `1` | Analysez le contenu du document à partir de l'entrée EditableDocument, recherchez toutes les polices utilisées et intégrez-les dans le document WordProcessing ou PDF de sortie. En premier lieu, GroupDocs.Editor prend les polices à partir des ressources de polices dans EditableDocument. S'ils sont insuffisants ou manquants, alors GroupDocs.Editor prend les polices du système d'exploitation. |
| EmbedWithoutSystem | `2` | Exactement àEmbedAll , mais excluez ces polices, qui sont traitées par le système d'exploitation comme système fonts |

### Remarques

Les options d'incorporation de polices sont appliquées lors de l'enregistrement du document (du document intermédiaire EditableDocument au format de sortie WordProcessing ou PDF), cette énumération est incluse en tant que propriété dans WordProcessingSaveOptions et PdfSaveOptions, d'où elle doit être utilisée
