## PdfSaveOptions.Password property

Mot de passe, qui sera appliqué au document PDF généré comme mot de passe utilisateur, requis pour l'ouverture. Si NULL ou vide, aucun mot de passe ne sera appliqué au document. Sinon, le document sera crypté avec RC4 (longueur de clé de 128 bits). Par défaut, NULL — le mot de passe n'est pas appliqué.
