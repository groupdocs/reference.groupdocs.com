## Propriétés

| Nom | La description |
| --- | --- |
| [InsertAsNewWorksheet](../../groupdocs.editor.options/spreadsheetsaveoptions/insertasnewworksheet) { get; set; } | Indicateur booléen, qui spécifie si la feuille de calcul modifiée doit remplacer la feuille de calcul existante dans la feuille de calcul d'origine à la position, spécifiée par le[`WorksheetNumber`](./worksheetnumber) propriété, ou elle doit être injectée entre la feuille de calcul existante et la précédente, sans remplacer son contenu. Par défaut, la valeur est fausse - la feuille de calcul existante sera remplacée. Cette propriété est ignorée, si la valeur de[`WorksheetNumber`](./worksheetnumber) propriété est définie sur '0'. |
| [OutputFormat](../../groupdocs.editor.options/spreadsheetsaveoptions/outputformat) { get; set; } | Permet de spécifier un format de feuille de calcul, qui sera utilisé pour enregistrer le document |
| [Password](../../groupdocs.editor.options/spreadsheetsaveoptions/password) { get; set; } | Permet de spécifier, modifier, obtenir ou supprimer un mot de passe, qui sera utilisé pour encoder le document de feuille de calcul généré, si ce format de document prend en charge la protection par mot de passe. Spécifiez NULL ou une chaîne vide pour supprimer (nettoyer) le mot de passe. |
| [WorksheetNumber](../../groupdocs.editor.options/spreadsheetsaveoptions/worksheetnumber) { get; set; } | Permet d'insérer une feuille de calcul modifiée dans la copie d'une feuille de calcul existante au lieu de créer une nouvelle feuille de calcul à une seule feuille de calcul (comportement par défaut). WorksheetNumber est un nombre basé sur 1 d'une feuille de calcul dans la feuille de calcul, chargé dans la classe Editor. S'il est égal à 0 (valeur par défaut), la nouvelle feuille de calcul sera créée avec une seule feuille de calcul modifiée. S'il est supérieur ou inférieur à zéro et qu'il existe une feuille de calcul valide, chargée dans la classe Editor, la feuille de calcul modifiée, représentée par input EditableDocument, sera inséré dans cette feuille de calcul. |
| [WorksheetProtection](../../groupdocs.editor.options/spreadsheetsaveoptions/worksheetprotection) { get; set; } | Permet d'activer une protection de feuille de calcul pour le document de feuille de calcul de sortie. Par défaut est NULL - la protection n'est pas appliquée. Tous les formats ne prennent pas en charge une protection de feuille de calcul. |
