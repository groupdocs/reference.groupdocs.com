## FixedLayoutEditOptionsBase.EnablePagination property

Permet d'activer (true) ou de désactiver (false) la pagination dans le document HTML résultant. Par défaut est désactivé (false).

```csharp
public bool EnablePagination { get; set; }
```

### Remarques

Les documents au format à mise en page fixe (PDF et XPS notamment) sont par essence strictement paginés, leur contenu est à mise en page fixe et découpé en pages. Mais le HTML modifiable résultant peut être représenté en mode sans page ou en mode paginal.
