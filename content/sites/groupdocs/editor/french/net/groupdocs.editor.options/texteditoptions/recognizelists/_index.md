## TextEditOptions.RecognizeLists property

Permet de spécifier comment les éléments de la liste numérotée sont reconnus lorsque le document est importé à partir du format texte brut. La valeur par défaut est true.

```csharp
public bool RecognizeLists { get; set; }
```

### Remarques

Si cette option est définie sur false, l'algorithme de reconnaissance des listes détecte les paragraphes de la liste, lorsque les numéros de liste se terminent par un point, un crochet droit ou des puces (tels que "•", "*", "-" ou "o"). Si cette option est définie sur true, les espaces blancs sont également utilisés comme délimiteurs de numéros de liste : l'algorithme de reconnaissance de liste pour la numérotation de style arabe (1., 1.1.2.) utilise à la fois des espaces blancs et des symboles de point ("".").
