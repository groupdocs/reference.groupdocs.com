## XmlEditOptions.FixIncorrectStructure property

Permet d'activer ou de désactiver le mécanisme de réparation de la structure XML corrompue. Par défaut est désactivé (false).

```csharp
public bool FixIncorrectStructure { get; set; }
```

### Remarques

Par défaut, seuls les documents XML valides et bien formés sont acceptables. Lorsque cette option est activée, GroupDocs.Editor essaiera de réparer la structure XML corrompue si possible.
