## SpreadsheetEditOptions.WorksheetIndex property

Permet de spécifier l'index de base 0 de la feuille de calcul (onglet) du document de feuille de calcul d'entrée, qui doit être converti au format HTML (voir remarques).

```csharp
public int WorksheetIndex { get; set; }
```

### Remarques

La plupart des feuilles de calcul prennent en charge un concept d'onglets, c'est-à-dire qu'ils peuvent être multi-onglets. D'autre part, le format HTML ne prend pas en charge une telle structure. Pour cette raison, GroupDocs.Editor ne peut convertir en HTML qu'un seul onglet spécifique du document d'entrée, et cette option permet de le spécifier. L'index des onglets est basé sur 0 , les valeurs négatives sont interdites. Si l'index spécifié dépasse le nombre de tous les onglets, l'exception sera levée. Si le document de feuille de calcul d'entrée ne contient qu'un seul onglet, cette option sera ignorée. La valeur par défaut est 0 (premier onglet).
