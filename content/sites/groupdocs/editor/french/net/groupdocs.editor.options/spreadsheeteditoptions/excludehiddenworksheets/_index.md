## SpreadsheetEditOptions.ExcludeHiddenWorksheets property

Permet d'exclure les feuilles de calcul masquées dans le document de feuille de calcul d'entrée, de sorte qu'elles seront totalement ignorées. La valeur par défaut est false - les feuilles de calcul masquées sont disponibles et traitées normalement.

```csharp
public bool ExcludeHiddenWorksheets { get; set; }
```

### Remarques

Plusieurs formats de feuilles de calcul binaires (comme XLSX) prennent en charge le concept de feuilles de calcul masquées (onglets). Un document de ce format, s'il contient plus d'une feuille de calcul, peut contenir des feuilles de calcul masquées supplémentaires. Par défaut, ces feuilles de calcul masquées sont disponibles pour le traitement, mais avec cette option, il est possible de les ignorer, comme ces feuilles de calcul masquées sont absentes et n'existent pas. Lorsque cette option est activée, vous ne pouvez pas sélectionner une feuille de calcul masquée avec le '[`WorksheetIndex`](../worksheetindex) propriété.
