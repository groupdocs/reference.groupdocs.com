## WordProcessingEditOptions.ExtractOnlyUsedFont property

Obtient ou définit une valeur indiquant si seules les ressources de police utilisées dans le contenu textuel du document sont extraites.

```csharp
public bool ExtractOnlyUsedFont { get; set; }
```

### Valeur de la propriété

`vrai` s'il est nécessaire d'extraire uniquement les ressources de police utilisées dans le contenu textuel du document ; Par ailleurs,`faux` . La valeur par défaut est`faux` .

### Remarques

Toutes les polices utilisées dans le document WordProcessing ne sont pas utilisées directement à 100 % (appliquées à du texte). Il peut y avoir une situation où la police est référencée dans le document et peut même être incorporée, mais n'est appliquée à aucun morceau de texte. Par exemple, une police peut être associée à un style, mais ce style n'est appliqué à aucune partie du texte. Cette option contrôle la façon de traiter ces cas.
