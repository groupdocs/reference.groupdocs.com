## PresentationSaveOptions.Password property

Permet de spécifier, modifier et obtenir le mot de passe, qui sera utilisé pour encoder le document de présentation résultant. Par défaut est NULL - le mot de passe ne sera pas défini. Définir sur NULL ou chaîne vide afin de supprimer le mot de passe, s'il a été défini précédemment.
