## PresentationSaveOptions class

Permet de spécifier des options personnalisées pour générer et enregistrer des documents de présentation (compatibles PowerPoint)

## Propriétés

| Nom | La description |
| --- | --- |
| [InsertAsNewSlide](../../groupdocs.editor.options/presentationsaveoptions/insertasnewslide) { get; set; } | Indicateur booléen, qui spécifie si la diapositive modifiée doit remplacer la diapositive existante dans la présentation d'origine à la position, spécifiée par le[`SlideNumber`](./slidenumber) , ou elle doit être injectée entre la diapositive existante et la précédente, sans remplacer son contenu. Par défaut est NULL - le mot de passe ne sera pas défini. Définir sur NULL ou chaîne vide afin de supprimer le mot de passe, s'il a été défini précédemment. |
| [SlideNumber](../../groupdocs.editor.options/presentationsaveoptions/slidenumber) { get; set; } | Permet d'insérer une diapositive modifiée dans une présentation existante au lieu de créer une nouvelle présentation à une seule diapositive (comportement par défaut). Le numéro de diapositive est un numéro basé sur 1 d'une diapositive dans la présentation, chargé dans la classe Editor. Si c'est 0 (valeur par défaut), la nouvelle présentation sera créée avec une seule diapositive éditée. S'il est supérieur ou inférieur à zéro et qu'il existe une présentation valide, chargée dans la classe Editor, la diapositive modifiée, stockée dans l'instance d'entrée EditableDocument, sera insérée dans cette présentation. |

### Remarques

L'instance de cette classe doit être transmise au[`Save`](../../groupdocs.editor/editor/save)méthode afin d'enregistrer la présentation modifiée dans le document final d'un format spécifique à la présentation. Son constructeur a un paramètre obligatoire — le format de la présentation de sortie. Tous les autres paramètres sont facultatifs et peuvent être omis.
