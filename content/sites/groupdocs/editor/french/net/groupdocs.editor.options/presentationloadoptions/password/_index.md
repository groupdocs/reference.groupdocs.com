## PresentationLoadOptions.Password property

Permet de spécifier, modifier et obtenir le mot de passe qui sera utilisé pour ouvrir le document Présentation, s'il est encodé. Définir sur NULL ou chaîne vide afin de supprimer le mot de passe.

```csharp
public string Password { get; set; }
```

### Remarques

Par défaut, cette propriété a la valeur NULL — le mot de passe n'est pas défini. Si le document de présentation d'entrée est protégé par un mot de passe, le mot de passe est obligatoire et une exception sera levée si le mot de passe n'est pas spécifié ou n'est pas valide. Si le document de présentation d'entrée n'est PAS protégé par un mot de passe, mais qu'un mot de passe est défini, il sera ignoré.
