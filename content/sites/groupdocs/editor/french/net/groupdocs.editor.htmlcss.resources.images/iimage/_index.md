## IImage interface

Le type de données CSS image représente une image bidimensionnelle

```csharp
public interface IImage
```

### Remarques

https://developer.mozilla.org/en-US/docs/Web/CSS/image La référence CSS déclare 4 types d'images, manipulables : 1. Images à dimensions intrinsèques (une taille naturelle) et donc fixes ratio d'aspect, qui inclut toutes les images raster. 2. Images avec plusieurs dimensions intrinsèques, existant dans plusieurs versions à l'intérieur d'un seul fichier, qui inclut les fichiers ICON (*.ico). Il peut également s'agir de TIFF, car ils prennent en charge plusieurs images dans un seul fichier, mais le format TIFF n'est pas pris en charge sur le Web. 3. Images sans dimensions intrinsèques mais avec un rapport d'aspect intrinsèque entre sa largeur et sa hauteur. Cela inclut toutes les images vectorielles, SVG avant tout. 4. Images sans dimensions intrinsèques, ni rapport d'aspect intrinsèque. Cela inclut tous les dégradés.
