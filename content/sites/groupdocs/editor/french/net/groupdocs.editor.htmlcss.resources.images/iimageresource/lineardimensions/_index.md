## IImageResource.LinearDimensions property

Lors de l'implémentation, le type doit renvoyer les dimensions linéaires de l'image. Pour les images raster, ce sont des dimensions intrinsèques en pixels. Les images vectorielles, en contrepartie, n'ont pas de dimensions fixes, mais leurs métadonnées peuvent contenir certaines dimensions de base dans différentes unités de mesure.
