## Edit(IEditOptions) {#edit_1}

Ouvre un document précédemment chargé pour modification à l'aide d'options spécifiques au format spécifiées en générant et renvoyant une instance de '[`EditableDocument`](../../editabledocument) classe, qui, à son tour, contient des méthodes pour produire le balisage HTML et les ressources associées.

```csharp
public EditableDocument Edit(IEditOptions editOptions)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| editOptions | IEditOptions | Options de document spécifiques au format, qui permettent d'ajuster le processus de conversion. Peut être NULL — dans ce cas, GroupDocs.Editor détecte un format de document précédemment chargé et applique les options, par défaut pour ce format. Ne devrait pas entrer en conflit avec les options de chargement précédemment appliquées. |

### Return_Value

Exemple de '[`EditableDocument`](../../editabledocument) classe, qui encapsule le document d'entrée global avec toutes ses ressources dans un format intermédiaire. Cette méthode, si elle se termine avec succès, ne renvoie jamais NULL.

### Remarques

Lorsque le document original d'entrée est chargé dans l'instance 'Editor' via le constructeur, cette méthode permet d'ouvrir le document pour modification en le convertissant au format intermédiaire, qui est encapsulé dans l'instance de la classe 'EditableDocument'. '[`EditableDocument`](../../editabledocument) renvoyé par cette méthode, contient toutes les méthodes et propriétés nécessaires pour produire le balisage HTML et les ressources correspondantes (comme les images, les polices et les feuilles de style) dans toutes les configurations nécessaires pour les transmettre ensuite dans n'importe quel éditeur HTML WYSIWYG. Cette surcharge obtient des options d'édition spécifiques aux formats de famille. **Apprendre encore plus**

* En savoir plus sur la modification de documents à l'aide de GroupDocs.Editor : [Comment modifier un document à l'aide de GroupDocs.Editor](https://docs.groupdocs.com/display/editornet/Edit+document)
