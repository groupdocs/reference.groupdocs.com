## GetContent(string, string) {#getcontent_1}

Renvoie le contenu global du document HTML sous forme de chaîne, où les liens vers les ressources externes contiennent le préfixe spécifié.

```csharp
public string GetContent(string externalImagesPrefix, string externalCssPrefix)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| externalImagesPrefix | String | Grâce à ce paramètre utilisé peut spécifier un préfixe, qui sera ajouté au links à toutes les images externes dans les éléments IMG, qui seront présents dans la chaîne HTML résultante. Si NULL ou vide, les préfixes ne seront pas ajoutés. |
| externalCssPrefix | String | Grâce à ce paramètre utilisé peut spécifier un préfixe, qui sera ajouté au links à toutes les feuilles de style externes dans les éléments LINK, qui seront présents dans la chaîne HTML résultante. Si NULL ou vide, les préfixes ne seront pas ajoutés. |

### Return_Value

String, qui contient le contenu du document HTML avec des liens, ajusté aux ressources externes
