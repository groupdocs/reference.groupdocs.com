## GetCssContent(string, string) {#getcsscontent_1}

Renvoie le contenu de toutes les feuilles de style externes sous forme de liste de chaînes, où une chaîne représente une feuille de style. Le préfixe spécifié sera appliqué à chaque lien vers la ressource externe dans chaque feuille de style résultante. Renvoie une liste vide, s'il n'y a pas de CSS pour cela document.

```csharp
public List<string> GetCssContent(string externalImagesPrefix, string externalFontsPrefix)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| externalImagesPrefix | String | Grâce à ce paramètre utilisé peut spécifier un préfixe, qui sera ajouté au links à toutes les images externes, qui sera présent dans les déclarations CSS dans les chaînes CSS résultantes. Si NULL ou vide, les préfixes ne seront pas ajoutés. |
| externalFontsPrefix | String | Grâce à ce paramètre utilisé peut spécifier un préfixe, qui sera ajouté au links à toutes les polices externes dans les @font-face at-rules dans les chaînes CSS résultantes. Si NULL ou vide, les préfixes ne seront pas ajoutés. |

### Return_Value

Une liste de chaînes, où chaque chaîne contient le contenu d'un document CSS
