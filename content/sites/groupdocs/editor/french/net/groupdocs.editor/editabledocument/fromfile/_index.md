## EditableDocument.FromFile method

Usine statique, qui crée une instance de EditableDocument à partir d'un fichier HTML, qui est spécifié par un chemin vers le fichier *.html lui-même et un dossier avec des ressources liées

```csharp
public static EditableDocument FromFile(string htmlFilePath, string resourceFolderPath)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| htmlFilePath | String | Chaîne, qui contient un chemin d'accès complet au fichier HTML. Ne peut pas être nul, doit être un chemin de fichier valide et le fichier lui-même doit exister. |
| resourceFolderPath | String | Chemin facultatif vers le dossier contenant les ressources HTML. Si NULL, invalide ou si un tel dossier n'existe pas, Editor essaiera de trouver ce dossier par lui-même, en analysant le balisage HTML |

### Return_Value

Nouvelle instance non nulle de EditableDocument

### Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | Le chemin du fichier HTML et/ou le chemin du dossier de ressources est/sont invalide(s) |
| FileNotFoundException | Le fichier HTML spécifié est introuvable |
