## IImage interface

Il tipo di dati Image CSS rappresenta un'immagine bidimensionale

```csharp
public interface IImage
```

### Osservazioni

https://developer.mozilla.org/en-US/docs/Web/CSS/image Il riferimento CSS dichiara 4 tipi di immagini, che possono essere gestite: 1. Immagini con dimensioni intrinseche (una dimensione naturale) e quindi fisse proporzioni, che include tutte le immagini raster. 2. Immagini con più dimensioni intrinseche, esistenti in più versioni all'interno di un singolo file, che include i file ICON (*.ico). Questo può anche essere TIFF, in quanto supportano più immagini all'interno di un file, tuttavia il formato TIFF non è supportato nel web. 3. Immagini senza dimensioni intrinseche ma con un rapporto di aspetto intrinseco tra larghezza e altezza. Ciò include tutte le immagini vettoriali, SVG prima di tutto. 4. Immagini senza dimensioni intrinseche, né proporzioni intrinseche. Questo include tutti i gradienti.
