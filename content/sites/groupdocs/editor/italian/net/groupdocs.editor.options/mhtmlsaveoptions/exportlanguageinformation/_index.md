## MhtmlSaveOptions.ExportLanguageInformation property

Specifica se le informazioni sulla lingua vengono esportate in MHTML. Il valore predefinito è`falso` .

```csharp
public bool ExportLanguageInformation { get; set; }
```

### Osservazioni

Quando questa proprietà è impostata su`VERO` , GroupDocs.Editor emette il file`lang` Attributo HTML sugli elementi del documento che specificano la lingua. Questo può essere necessario per preservare la semantica relativa alla lingua.
