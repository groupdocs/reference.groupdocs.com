## FixedLayoutEditOptionsBase.EnablePagination property

Consente di abilitare (true) o disabilitare (false) l'impaginazione nel documento HTML risultante. Di default è disabilitato (false).

```csharp
public bool EnablePagination { get; set; }
```

### Osservazioni

I documenti a formato fisso (PDF e XPS in particolare) nella loro essenza sono rigorosamente impaginati, il loro contenuto ha un layout fisso e suddiviso in pagine. Ma l'HTML modificabile risultante può essere rappresentato in una visualizzazione senza pagina o impaginale.
