## XmlFormatOptions.EachAttributeFromNewline property

Se abilitato, ogni singola coppia di valore-attributo in ogni elemento XML verrà posizionata su una nuova riga. Per impostazione predefinita è false (disabilitato): tutte le coppie di valore-attributo vengono posizionate su una singola riga.

public bool EachAttributeFromNewline { get; set; }
