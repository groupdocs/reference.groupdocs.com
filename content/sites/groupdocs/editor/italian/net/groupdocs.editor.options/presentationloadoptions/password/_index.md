## PresentationLoadOptions.Password property

Consente di specificare, modificare e ottenere la password, che verrà utilizzata per l'apertura del documento di presentazione, se codificato. Impostare su NULL o stringa vuota per rimuovere la password.

```csharp
public string Password { get; set; }
```

### Osservazioni

Per impostazione predefinita, questa proprietà ha valore NULL — la password non è impostata. Se il documento di presentazione di input è protetto da password, la password è obbligatoria e verrà generata un'eccezione se la password non è specificata o non è valida. Se il documento di presentazione in ingresso NON è protetto da password, ma la password è impostata, verrà ignorato.
