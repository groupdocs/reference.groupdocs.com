## WorksheetProtection class

Incapsula le opzioni di protezione del foglio di lavoro, che consentono di proteggere un foglio di lavoro nel documento Foglio di calcolo di output dalla modifica del tipo specificato con una password specificata.

### Osservazioni

La maggior parte dei formati di fogli di calcolo come XLSX consente di proteggere un foglio di lavoro dalla modifica con password. Questa classe permette di abilitare tale protezione e specificarne le opzioni.
