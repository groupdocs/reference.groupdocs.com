## DelimitedTextEditOptions.OptimizeMemoryUsage property

Abilita i meccanismi di ottimizzazione della memoria durante l'elaborazione del documento di input, che può ridurre le prestazioni in alcuni casi speciali, ma d'altra parte riduce l'utilizzo della memoria. Utile quando si elaborano documenti di grandi dimensioni e si affronta OutOfMemoryException. L'impostazione predefinita è false (l'ottimizzazione della memoria è disabilitata per migliorare le prestazioni).

public bool OptimizeMemoryUsage { get; set; }
