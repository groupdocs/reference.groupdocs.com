## PresentationEditOptions.SlideNumber property

Consente di specificare i numeri di diapositiva, che devono essere aperti per la modifica

```csharp
public int SlideNumber { get; set; }
```

### Osservazioni

Il numero di diapositiva è un indice in base zero di una diapositiva, che consente di specificare e selezionare una particolare diapositiva da una presentazione da modificare. Se minore di 0, verrà selezionata la prima diapositiva (uguale a SlideNumber = 0). Se è maggiore del numero di tutte le diapositive nella presentazione, verrà selezionata l'ultima diapositiva. Se la presentazione di input contiene solo una singola diapositiva, questa opzione verrà ignorata e questa singola diapositiva verrà modificata. Se si tenta di aprire per modificare una diapositiva nascosta, mentre[`ShowHiddenSlides`](../showhiddenslides) option è impostata su 'false', verrà generata l'eccezione.
