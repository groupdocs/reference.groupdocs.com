## Proprietà

| [InsertAsNewSlide](../../groupdocs.editor.options/presentationsaveoptions/insertasnewslide) { get; set; } | Flag booleano, che specifica se la diapositiva modificata deve sostituire la diapositiva esistente nella presentazione originale nella posizione specificata dal[`SlideNumber`](./slidenumber) proprietà, oppure dovrebbe essere inserita tra la diapositiva esistente e quella precedente, senza sostituirne il contenuto. Di default è NULL - la password non verrà impostata. Impostare su NULL o stringa vuota per rimuovere la password, se è stata impostata in precedenza. |
| [SlideNumber](../../groupdocs.editor.options/presentationsaveoptions/slidenumber) { get; set; } | Consente di inserire la diapositiva modificata nella presentazione esistente invece di creare una nuova presentazione a diapositiva singola (comportamento predefinito). Il numero della diapositiva è un numero in base 1 di una diapositiva nella presentazione, caricata nella classe Editor. Se è 0 (valore predefinito), la nuova presentazione verrà creata con una singola diapositiva modificata. Se è maggiore o minore di zero ed è presente una presentazione valida, caricata nella classe Editor, la diapositiva modificata, memorizzata all'interno dell'istanza di input EditableDocument, verrà inserita in questa presentazione. |

### Osservazioni

L'istanza di questa classe deve essere passata in[`Save`](../../groupdocs.editor/editor/save)metodo per salvare la presentazione modificata nel documento finale di un formato specifico della presentazione. Il suo costruttore ha un parametro obbligatorio: il formato dell'output Presentation. Tutti gli altri parametri sono facoltativi e possono essere omessi.
