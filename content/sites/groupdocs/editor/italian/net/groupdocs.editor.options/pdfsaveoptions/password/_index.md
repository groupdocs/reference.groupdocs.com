## PdfSaveOptions.Password property

Password, che verrà applicata al documento PDF generato come password utente, necessaria per l'apertura. Se NULL o vuoto, nessuna password verrà applicata al documento. In caso contrario, il documento verrà crittografato con RC4 (lunghezza della chiave di 128 bit). Per impostazione predefinita è NULL — la password non viene applicata.
