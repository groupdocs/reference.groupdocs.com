## WordProcessingSaveOptions.LocaleFarEast property

Consente di sovrascrivere la locale (lingua) per il documento WordProcessing per il testo dell'Asia orientale, che verrà applicata durante la sua creazione. Quando non è specificato (valore predefinito), MS Word (o altro programma) rileverà (o sceglierà ) il documento East-Asian locale in base alle proprie impostazioni o ad altri fattori.

```csharp
public CultureInfo LocaleFarEast { get; set; }
```

### Osservazioni

Questa opzione forza l'applicazione forzata delle impostazioni locali specificate a tutto il testo dell'Asia orientale nel documento. Non utilizzarla se il documento contiene diverse parti di testo, che sono scritte in lingue diverse.
