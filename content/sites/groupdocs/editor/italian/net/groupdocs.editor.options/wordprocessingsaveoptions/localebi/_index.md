## WordProcessingSaveOptions.LocaleBi property

Consente di impostare l'override locale (lingua) per il documento WordProcessing per il testo RTL (da destra a sinistra), che verrà applicato durante la sua creazione. Quando non è specificato (valore predefinito), MS Word (o altro programma) rileverà (o sceglierà) il documento RTL locale in base alle proprie impostazioni o ad altri fattori.

```csharp
public CultureInfo LocaleBi { get; set; }
```

### Osservazioni

Questa opzione applica forzatamente la locale specificata al testo RTL complessivo nel documento. Non utilizzarla se il documento contiene diverse parti di testo, che sono scritte in lingue diverse.
