## WordProcessingSaveOptions.Locale property

Consente di impostare l'override della locale (lingua) predefinita per il documento WordProcessing, che verrà applicato durante la sua creazione. Quando non è specificato (valore predefinito), MS Word (o altro programma) rileverà (o sceglierà) la locale del documento in base alle proprie impostazioni o ad altri fattori.

```csharp
public CultureInfo Locale { get; set; }
```

### Osservazioni

Questa opzione applica forzatamente la locale specificata al testo generale nel documento. Non usarla se il documento contiene diverse parti di testo, che sono scritte in lingue diverse.
