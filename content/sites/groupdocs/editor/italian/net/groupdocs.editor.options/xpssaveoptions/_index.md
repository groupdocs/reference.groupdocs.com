### Osservazioni

Un file XPS rappresenta i file di layout di pagina basati su XML Paper Specifications creati da Microsoft. È stato sviluppato per sostituire il formato di file EMF ed è simile al formato di file PDF, ma utilizza XML nel layout, nell'aspetto e nelle informazioni di stampa di un documento.
