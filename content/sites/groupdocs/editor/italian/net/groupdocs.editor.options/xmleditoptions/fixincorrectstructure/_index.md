## XmlEditOptions.FixIncorrectStructure property

Consente di abilitare o disabilitare il meccanismo per correggere la struttura XML danneggiata. Di default è disabilitato (false).

```csharp
public bool FixIncorrectStructure { get; set; }
```

### Osservazioni

Per impostazione predefinita, sono accettabili solo documenti XML validi e ben formati. Quando questa opzione è abilitata, GroupDocs.Editor tenterà di correggere la struttura XML danneggiata, se possibile.
