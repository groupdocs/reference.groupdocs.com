## MarkdownSaveOptions.ExportImagesAsBase64 property

Specifica se le immagini vengono salvate in formato Base64 nel file di output. L'impostazione predefinita è`falso` .

```csharp
public bool ExportImagesAsBase64 { get; set; }
```

### Osservazioni

Quando questa proprietà è impostata su`VERO` i dati delle immagini vengono esportati direttamente negli elementi image ![]() e non vengono creati file separati. Questa proprietà, se impostata su`VERO` , ha la priorità più alta rispetto a[`ImagesFolder`](../imagesfolder) proprietà.
