## MarkdownSaveOptions.OptimizeMemoryUsage property

Abilita i meccanismi di ottimizzazione della memoria durante la generazione del documento da HTML, che riduce le prestazioni come costo della riduzione dell'utilizzo della memoria. Impostando questa opzione su`VERO`può ridurre significativamente il consumo di memoria durante la generazione di documenti di grandi dimensioni al costo di un risparmio di tempo più lento. L'impostazione predefinita è`falso` (l'ottimizzazione della memoria è disabilitata per migliorare le prestazioni).

```csharp
public bool OptimizeMemoryUsage { get; set; }
```
