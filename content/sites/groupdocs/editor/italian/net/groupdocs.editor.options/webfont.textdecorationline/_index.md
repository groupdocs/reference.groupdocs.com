## WebFont.TextDecorationLine enumeration

Tipi di linea decorativa utilizzata sul testo in un elemento, ad esempio una sottolineatura, una sottolineatura o una riga passante. Possono essere combinati insieme o nessuno.

```csharp
[Flags]
public enum TextDecorationLine : byte
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | `0` | Nessuna linea di decorazione del testo applicata |
| Underline | `1` | Sottolineato: sotto ogni riga di testo è presente una linea decorativa |
| Overline | `2` | Sopralinea: ogni riga di testo ha una linea decorativa sopra di essa |
| LineThrough | `4` | Line-through - ogni riga di testo ha una linea decorativa che attraversa il suo centro |
