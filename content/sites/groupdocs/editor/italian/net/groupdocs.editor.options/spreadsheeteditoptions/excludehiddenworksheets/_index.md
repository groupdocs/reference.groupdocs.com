## SpreadsheetEditOptions.ExcludeHiddenWorksheets property

Consente di escludere i fogli di lavoro nascosti nel documento Spreadsheet di input, quindi verranno totalmente ignorati. L'impostazione predefinita è false: i fogli di lavoro nascosti sono disponibili ed elaborati normalmente.

```csharp
public bool ExcludeHiddenWorksheets { get; set; }
```

### Osservazioni

Diversi formati di fogli di calcolo binari (come XLSX) supportano il concetto di fogli di lavoro nascosti (schede). Un documento di tale formato, se ha più di un foglio di lavoro, può contenere ulteriori fogli di lavoro nascosti. Per impostazione predefinita tali fogli di lavoro nascosti sono disponibili per l'elaborazione, ma con questa opzione è in grado di ignorarli, come questi fogli di lavoro nascosti sono assenti e non esistono. Quando questa opzione è abilitata, non puoi selezionare un foglio di lavoro nascosto con '[`WorksheetIndex`](../worksheetindex) proprietà.
