## SpreadsheetEditOptions.WorksheetIndex property

Consente di specificare l'indice in base 0 del foglio di lavoro (scheda) del documento Spreadsheet di input, che deve essere convertito in HTML (vedi commenti).

```csharp
public int WorksheetIndex { get; set; }
```

### Osservazioni

La maggior parte dei documenti Spreadsheet supporta il concetto di schede, ovvero possono essere multischede. D'altra parte, il formato HTML non supporta tale struttura. A causa di ciò GroupDocs.Editor può convertire in HTML solo una scheda specifica del documento di input e questa opzione consente di specificarla. L'indice della scheda è basato su 0 , i valori negativi sono vietati. Se l'indice specificato supera il numero di tutte le schede, verrà generata l'eccezione. Se il documento Spreadsheet di input contiene solo una scheda, questa opzione verrà ignorata. Il valore predefinito è 0 (prima scheda).
