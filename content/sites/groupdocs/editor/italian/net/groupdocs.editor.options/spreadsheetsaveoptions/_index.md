| Nome | Descrizione |

| [InsertAsNewWorksheet](../../groupdocs.editor.options/spreadsheetsaveoptions/insertasnewworksheet) { get; set; } | Flag booleano, che specifica se il foglio di lavoro modificato deve sostituire il foglio di lavoro esistente nel foglio di lavoro originale nella posizione specificata dal[`WorksheetNumber`](./worksheetnumber) property, oppure dovrebbe essere inserito tra il foglio di lavoro esistente e quello precedente, senza sostituirne il contenuto. Specificare NULL o una stringa vuota per rimuovere (pulire) la password. |
| [WorksheetNumber](../../groupdocs.editor.options/spreadsheetsaveoptions/worksheetnumber) { get; set; } | Consente di inserire il foglio di lavoro modificato nella copia del foglio di lavoro esistente invece di creare un nuovo foglio di lavoro a foglio singolo (comportamento predefinito). WorksheetNumber è un numero in base 1 di un foglio di lavoro nel foglio di lavoro, caricato nella classe Editor. Se è 0 (valore predefinito), il nuovo foglio di calcolo verrà creato con un singolo foglio di lavoro modificato. Se è maggiore o minore di zero ed è presente un foglio di calcolo valido, caricato nella classe Editor, il foglio di lavoro modificato, rappresentato da input EditableDocument istanza, verrà inserito in questo foglio di calcolo. |
| [WorksheetProtection](../../groupdocs.editor.options/spreadsheetsaveoptions/worksheetprotection) { get; set; } | Consente di abilitare una protezione del foglio di lavoro per il documento Spreadsheet di output. Per impostazione predefinita è NULL - la protezione non viene applicata. Non tutti i formati supportano una protezione del foglio di lavoro. |
