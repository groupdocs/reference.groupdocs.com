## EditableDocument.GetEmbeddedHtml method

Restituisce tutto il contenuto di questo documento HTML con tutte le risorse correlate sotto forma di una singola stringa, dove tutte le risorse sono incorporate all'interno del markup HTML in un formato con codifica base64.

```csharp
public string GetEmbeddedHtml()
```

### Valore di ritorno

Stringa, che non è NULL o vuota in ogni caso

### Eccezioni

| eccezione | condizione |
| --- | --- |
| ObjectDisposedException | Questa istanza di EditableDocument era già stata eliminata |

### Osservazioni

Questo metodo converte questo EditableDocument in HTML e lo serializza in una singola stringa, dove tutte le risorse sono incorporate nella stringa insieme al markup HTML:

* Tutte le immagini da HTML-&gt;BODY vengono convertite in formato base64 e si trovano nell'attributo IMG 'src'
* Tutti i fogli di stile sono memorizzati negli elementi STYLE all'interno delle sezioni HTML-&gt;HEAD
* Tutte le immagini dei fogli di stile vengono convertite in formato base64 e posizionate nelle appropriate dichiarazioni CSS
* Tutti i font dei fogli di stile vengono convertiti nel formato base64 e posizionati nelle appropriate @font-face at-rules
