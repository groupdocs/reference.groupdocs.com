## GetContent(string, string) {#getcontent_1}

Restituisce il contenuto complessivo del documento HTML come stringa, dove i collegamenti alle risorse esterne contengono il prefisso specificato.

```csharp
public string GetContent(string externalImagesPrefix, string externalCssPrefix)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| externalImagesPrefix | String | Attraverso questo parametro utilizzato è possibile specificare un prefisso, che verrà aggiunto ai link a tutte le immagini esterne negli elementi IMG, che saranno presenti nella stringa HTML risultante. Se NULL o vuoto, i prefissi non verranno aggiunti. |
| externalCssPrefix | String | Attraverso questo parametro utilizzato è possibile specificare un prefisso, che verrà aggiunto ai links a tutti i fogli di stile esterni negli elementi LINK, che saranno presenti nella stringa HTML risultante. Se NULL o vuoto, i prefissi non verranno aggiunti. |

### Valore di ritorno

Stringa, che contiene il contenuto del documento HTML con collegamenti, adattata alle risorse esterne
