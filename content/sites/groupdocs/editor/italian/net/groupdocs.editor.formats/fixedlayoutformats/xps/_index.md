## FixedLayoutFormats.Xps field

Il file XPS rappresenta i file di layout di pagina basati sulle specifiche XML Paper create da Microsoft. È stato sviluppato per sostituire il formato di file EMF ed è simile al formato di file PDF, ma utilizza XML per il layout, l'aspetto e le informazioni di stampa di un documento.
