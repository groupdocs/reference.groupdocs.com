## FontSize.IsRelativeSize property

Indica se questa dimensione del carattere è definita con una dimensione relativa come parola chiave. Il carattere sarà più grande o più piccolo rispetto alla dimensione del carattere dell'elemento genitore, approssimativamente in base al rapporto utilizzato per separare le parole chiave di dimensione assoluta.
