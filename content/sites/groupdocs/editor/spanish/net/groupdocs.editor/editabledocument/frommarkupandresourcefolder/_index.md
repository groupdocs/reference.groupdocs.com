## EditableDocument.FromMarkupAndResourceFolder method

Fábrica estática, que crea una instancia de EditableDocument a partir de un marcado HTML específico y de recursos, ubicados en la carpeta, especificada por la ruta completa

```csharp
public static EditableDocument FromMarkupAndResourceFolder(string newHtmlContent, 
    string resourceFolderPath)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| newHtmlContent | String | Cadena, que contiene marcado HTML sin procesar, que debe analizarse. No puede ser NULL, vacío o inválido. |
| resourceFolderPath | String | Ruta obligatoria a la carpeta con recursos. Se utilizarán todas las hojas de estilo que se encuentran en esta carpeta. No puede ser NULL o una cadena vacía, y esta carpeta debería existir. |

### Valor_devuelto

Nueva instancia no nula de EditableDocument

### Observaciones

Esta fábrica estática es útil cuando el contenido del documento HTML se presenta como una cadena, pero todos los recursos están ubicados en alguna carpeta y, a menudo, los enlaces a estos recursos en el marcado HTML no son válidos y están ausentes. Al invocar este método, escanea la carpeta especificada y aplica automáticamente todas las hojas de estilo encontradas al documento. Este método es muy útil a la hora de obtener contenido de diferentes editores HTML, que suelen cortar los metadatos del documento, etc.
