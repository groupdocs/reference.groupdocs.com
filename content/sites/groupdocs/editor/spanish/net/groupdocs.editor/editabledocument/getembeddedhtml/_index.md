## EditableDocument.GetEmbeddedHtml method

Devuelve todo el contenido de este documento HTML con todos los recursos relacionados en forma de una sola cadena, donde todos los recursos están incrustados dentro del marcado HTML en una forma codificada en base64.

```csharp
public string GetEmbeddedHtml()
```

### Valor_devuelto

Cadena, que no es NULL ni está vacía en ningún caso

### Excepciones

| excepción | condición |
| --- | --- |
| ObjectDisposedException | Esta instancia de EditableDocument ya se eliminó |

### Observaciones

Este método convierte este EditableDocument en HTML y lo serializa en una sola cadena, donde todos los recursos están incrustados en una cadena junto con el marcado HTML:

* Todas las imágenes de HTML-&gt;BODY se convierten al formato base64 y se ubican en el atributo IMG 'src'
* Todas las hojas de estilo se almacenan en los elementos STYLE dentro de las secciones HTML-&gt;HEAD
* Todas las imágenes de las hojas de estilo se convierten al formato base64 y se ubican en las declaraciones CSS correspondientes
* Todas las fuentes de las hojas de estilo se convierten al formato base64 y se ubican en las reglas @font-face at correspondientes
