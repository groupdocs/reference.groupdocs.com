## IImage interface

El tipo de datos CSS de imagen representa una imagen bidimensional

```csharp
public interface IImage
```

### Observaciones

https://developer.mozilla.org/en-US/docs/Web/CSS/image La referencia CSS declara 4 tipos de imágenes que se pueden manejar: 1. Imágenes con dimensiones intrínsecas (un tamaño natural) y, por lo tanto, fijas relación de aspecto, que incluye todas las imágenes raster. 2. Imágenes con múltiples dimensiones intrínsecas, existentes en múltiples versiones dentro de un solo archivo, que incluye archivos ICON (*.ico). Esto también puede ser TIFF, ya que admiten múltiples imágenes dentro de un archivo, sin embargo, el formato TIFF no es compatible con web. 3. Imágenes sin dimensiones intrínsecas pero con una relación de aspecto intrínseca entre su ancho y alto. Esto incluye todas las imágenes vectoriales, SVG en primer lugar. 4. Imágenes sin dimensiones intrínsecas ni relación de aspecto intrínseca. Esto incluye todos los degradados.
