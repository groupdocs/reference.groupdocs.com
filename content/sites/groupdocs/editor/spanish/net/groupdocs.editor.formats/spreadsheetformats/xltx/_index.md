## SpreadsheetFormats.Xltx field

El archivo de plantilla XML abierta de Office sin macros (XLTX) representa la plantilla de Microsoft Excel que se basa en las especificaciones de formato de archivo OpenXML de Office. Se utiliza para crear un archivo de plantilla estándar que se puede utilizar para generar archivos XLSX que muestren la misma configuración que se especifica en el archivo XLTX. Obtenga más información sobre este formato de archivo[aquí](https://wiki.fileformat.com/spreadsheet/xltx) .
