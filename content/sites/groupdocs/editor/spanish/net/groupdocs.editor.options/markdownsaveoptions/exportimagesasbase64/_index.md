## MarkdownSaveOptions.ExportImagesAsBase64 property

Especifica si las imágenes se guardan en formato Base64 en el archivo de salida. El valor predeterminado es`FALSO` .

```csharp
public bool ExportImagesAsBase64 { get; set; }
```

### Observaciones

Cuando esta propiedad se establece en`verdadero` los datos de las imágenes se exportan directamente a los elementos de imagen ![]() y no se crean archivos separados. Esta propiedad, si se establece en`verdadero` , tiene mayor prioridad que el[`ImagesFolder`](../imagesfolder) propiedad.
