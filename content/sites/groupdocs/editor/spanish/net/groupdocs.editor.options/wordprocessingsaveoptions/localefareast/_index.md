## WordProcessingSaveOptions.LocaleFarEast property

Permite anular la configuración regional (idioma) del documento de WordProcessing para el texto de Asia oriental, que se aplicará durante su creación. Cuando no se especifica (valor predeterminado), MS Word (u otro programa) detectará (o elegirá ) el documento East-Asian locale según su propia configuración u otros factores.

```csharp
public CultureInfo LocaleFarEast { get; set; }
```

### Observaciones

Esta opción aplica a la fuerza la configuración regional especificada al texto general de Asia oriental en el documento. No la use si el documento contiene diferentes partes del texto, que están escritas en diferentes idiomas.
