## WordProcessingSaveOptions.EnablePagination property

Permite habilitar o deshabilitar la paginación que se utilizará para guardar el documento de WordProcessing. Si el documento original se abrió y editó en modo paginación, esta opción también debe estar habilitada. Por defecto está deshabilitado.

public bool EnablePagination { get; set; }
