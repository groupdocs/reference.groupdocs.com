## WordProcessingSaveOptions.Locale property

Permite anular la configuración regional predeterminada (idioma) para el documento de WordProcessing, que se aplicará durante su creación. Cuando no se especifica (valor predeterminado), MS Word (u otro programa) detectará (o elegirá) la configuración regional del documento según a su propia configuración o a otros factores.

```csharp
public CultureInfo Locale { get; set; }
```

### Observaciones

Esta opción aplica a la fuerza la configuración regional especificada al texto general del documento. No la use si el documento contiene diferentes partes del texto, que están escritas en diferentes idiomas.
