## MhtmlSaveOptions.ExportLanguageInformation property

Especifica si la información de idioma se exporta a MHTML. El valor predeterminado es`FALSO` .

```csharp
public bool ExportLanguageInformation { get; set; }
```

### Observaciones

Cuando esta propiedad se establece en`verdadero` , GroupDocs.Editor emite el`idioma` Atributo HTML en los elementos del documento que especifican el idioma. Esto puede ser necesario para preservar la semántica relacionada con el lenguaje.
