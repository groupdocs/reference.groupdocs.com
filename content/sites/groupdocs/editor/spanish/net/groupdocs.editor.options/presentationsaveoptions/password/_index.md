## PresentationSaveOptions.Password property

Permite especificar, modificar y obtener la contraseña, que se utilizará para codificar el documento de presentación resultante. Por defecto es NULL: no se establecerá la contraseña. Establézcalo en NULL o cadena vacía para eliminar la contraseña, si se configuró previamente.
