## PdfSaveOptions.Password property

Contraseña, que se aplicará al documento PDF generado como contraseña de usuario, necesaria para abrir. Si es NULL o está vacío, no se aplicará ninguna contraseña al documento. De lo contrario, el documento se cifrará con RC4 (longitud de clave de 128 bits). Por defecto es NULL — no se aplica la contraseña.
