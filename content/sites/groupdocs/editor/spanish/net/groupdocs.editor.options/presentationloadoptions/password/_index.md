## PresentationLoadOptions.Password property

Permite especificar, modificar y obtener la contraseña, que se utilizará para abrir el documento de Presentación, si está codificado. Establecer en NULL o cadena vacía para eliminar la contraseña.

```csharp
public string Password { get; set; }
```

### Observaciones

De forma predeterminada, esta propiedad tiene un valor NULL: la contraseña no está configurada. Si el documento de presentación de entrada está protegido con contraseña, la contraseña es obligatoria y se lanzará una excepción si la contraseña no se especifica o no es válida. Si el documento de presentación de entrada NO está protegido con contraseña, pero la contraseña está configurada, se ignorará.
