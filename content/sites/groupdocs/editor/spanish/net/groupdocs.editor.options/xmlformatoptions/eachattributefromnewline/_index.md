## XmlFormatOptions.EachAttributeFromNewline property

Cuando está habilitado, todos y cada uno de los pares de valor-atributo en cada elemento XML se colocarán en una nueva línea. De manera predeterminada, es falso (deshabilitado): todos los pares de valor-atributo se colocan en una sola línea.

public bool EachAttributeFromNewline { get; set; }
