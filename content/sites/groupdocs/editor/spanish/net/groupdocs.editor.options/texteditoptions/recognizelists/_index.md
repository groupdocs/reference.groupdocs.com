## TextEditOptions.RecognizeLists property

Permite especificar cómo se reconocen los elementos de la lista numerada cuando el documento se importa desde un formato de texto sin formato. El valor predeterminado es verdadero.

```csharp
public bool RecognizeLists { get; set; }
```

### Observaciones

Si esta opción se establece en falso, el algoritmo de reconocimiento de listas detecta los párrafos de la lista, cuando los números de la lista terminan con un punto, corchete derecho o viñetas (como "•", "*", "-" o "o"). Si esta opción se establece en verdadero, los espacios en blanco también se utilizan como delimitadores de números de lista: el algoritmo de reconocimiento de listas para la numeración de estilo árabe (1., 1.1.2.) utiliza espacios en blanco y símbolos de punto (".").
