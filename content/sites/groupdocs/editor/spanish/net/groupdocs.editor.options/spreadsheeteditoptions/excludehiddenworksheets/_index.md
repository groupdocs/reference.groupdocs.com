## SpreadsheetEditOptions.ExcludeHiddenWorksheets property

Permite excluir hojas de cálculo ocultas en el documento de hoja de cálculo de entrada, por lo que se ignorarán por completo. El valor predeterminado es falso: las hojas de trabajo ocultas están disponibles y se procesan normalmente.

```csharp
public bool ExcludeHiddenWorksheets { get; set; }
```

### Observaciones

Varios formatos binarios de hoja de cálculo (como XLSX) admiten el concepto de hojas de trabajo ocultas (pestañas). Documento de tal formato, si tiene más de una hoja de cálculo, puede contener hojas de cálculo adicionales ocultas. Por defecto, dichas hojas de trabajo ocultas están disponibles para su procesamiento, pero con esta opción puede ignorarlas, como si estas hojas de trabajo ocultas estuvieran ausentes y no existieran. Cuando esta opción está habilitada, no puede seleccionar la hoja de trabajo oculta con '[`WorksheetIndex`](../worksheetindex) propiedad.
