## SpreadsheetEditOptions.WorksheetIndex property

Permite especificar el índice basado en 0 de la hoja de trabajo (pestaña) del documento de hoja de cálculo de entrada, que debe convertirse a HTML (ver comentarios).

```csharp
public int WorksheetIndex { get; set; }
```

### Observaciones

La mayoría de los documentos de hojas de cálculo admiten un concepto de pestañas, es decir, pueden tener varias pestañas. Por otro lado, el formato HTML no admite dicha estructura. Debido a esto, GroupDocs.Editor puede convertir a HTML solo una pestaña específica del documento de entrada, y esta opción permite especificarla. El índice de pestaña está basado en 0 , los valores negativos están prohibidos. Si el índice especificado excede el número de todas las pestañas, se lanzará la excepción. Si el documento de hoja de cálculo de entrada contiene solo una pestaña, esta opción se ignorará. El valor predeterminado es 0 (primera pestaña).
