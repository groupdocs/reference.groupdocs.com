## PresentationEditOptions.SlideNumber property

Permite especificar los números de diapositiva, que deben abrirse para editar

```csharp
public int SlideNumber { get; set; }
```

### Observaciones

El número de diapositiva es un índice basado en cero de una diapositiva, que permite especificar y seleccionar una diapositiva en particular de una presentación para editar. Si es menor que 0, se seleccionará la primera diapositiva (igual que SlideNumber = 0). Si es mayor que la cantidad de todas las diapositivas en la presentación, se seleccionará la última diapositiva. Si la presentación de entrada contiene solo una diapositiva, esta opción se ignorará y esta diapositiva única se editará. Si intenta abrir para editar una diapositiva oculta, mientras[`ShowHiddenSlides`](../showhiddenslides) se establece en 'falso', se lanzará la excepción.
