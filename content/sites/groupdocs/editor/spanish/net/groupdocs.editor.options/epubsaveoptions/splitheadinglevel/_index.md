## EpubSaveOptions.SplitHeadingLevel property

Especifica el nivel máximo de encabezados en el que dividir el archivo ePub. El valor predeterminado es`2` Configurándolo en`0` deshabilitará la división, por lo que todo el contenido del libro electrónico se incorporará en un solo paquete dentro del ePub.

```csharp
public int SplitHeadingLevel { get; set; }
```

### Observaciones

Cuando esta propiedad se establece en un valor de 1 a 9, el documento se dividirá en párrafos con formato usando **Título 1** ,**Título 2** ,**Título 3** etc. estilos hasta el nivel de encabezado especificado.

Por defecto, solo**Título 1** y**Título 2** los párrafos hacen que el documento se divida. Establecer esta propiedad en cero hará que el documento no se divida en los párrafos de encabezado.
