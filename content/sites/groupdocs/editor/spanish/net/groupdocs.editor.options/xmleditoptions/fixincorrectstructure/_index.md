## XmlEditOptions.FixIncorrectStructure property

Permite habilitar o deshabilitar el mecanismo para reparar la estructura XML dañada. Por defecto está deshabilitado (falso).

```csharp
public bool FixIncorrectStructure { get; set; }
```

### Observaciones

De forma predeterminada, solo se aceptan documentos XML válidos y bien formados. Cuando esta opción está habilitada, GroupDocs.Editor intentará reparar la estructura XML dañada si es posible.
