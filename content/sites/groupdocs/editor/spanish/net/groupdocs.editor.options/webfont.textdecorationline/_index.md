## WebFont.TextDecorationLine enumeration

Tipos de líneas decorativas que se utilizan en el texto de un elemento, como un subrayado, una superposición o un traspaso. Pueden combinarse entre sí, o ninguno.

```csharp
[Flags]
public enum TextDecorationLine : byte
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | `0` | No se aplica ninguna línea de decoración de texto |
| Underline | `1` | Subrayado: cada línea de texto tiene una línea decorativa debajo |
| Overline | `2` | Overline: cada línea de texto tiene una línea decorativa encima |
| LineThrough | `4` | Line-through: cada línea de texto tiene una línea decorativa que pasa por su medio |
