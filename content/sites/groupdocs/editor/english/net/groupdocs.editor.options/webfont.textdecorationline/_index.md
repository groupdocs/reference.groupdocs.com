## WebFont.TextDecorationLine enumeration

Kinds of decoration line that is used on text in an element, such as an underline, overline or line-through. May be combined together, or no one.

```csharp
[Flags]
public enum TextDecorationLine : byte
```

### Values

| Name | Value | Description |
| --- | --- | --- |
| None | `0` | No text decoration line is applied |
| Underline | `1` | Underline - each line of text has a decorative line beneath it |
| Overline | `2` | Overline - each line of text has a decorative line above it |
| LineThrough | `4` | Line-through - each line of text has a decorative line going through its middle |
