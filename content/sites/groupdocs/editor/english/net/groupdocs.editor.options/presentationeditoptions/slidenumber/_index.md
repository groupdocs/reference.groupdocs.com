## PresentationEditOptions.SlideNumber property

Allows to specify the slide numbers, which should be opened for editing

```csharp
public int SlideNumber { get; set; }
```

### Remarks

Slide number is a zero-based index of a slide, that allows to specify and select one particular slide from a presentation to edit. If lesser then 0, the first slide will be selected (same as SlideNumber = 0). If greater then amount of all slides in presentation, the last slide will be selected. If input presentation contains only single slide, this option will be ignored, and this single slide will be edited. If trying to open for editing a hidden slide, while [`ShowHiddenSlides`](../showhiddenslides) option is set to 'false', the exception will be thrown.
