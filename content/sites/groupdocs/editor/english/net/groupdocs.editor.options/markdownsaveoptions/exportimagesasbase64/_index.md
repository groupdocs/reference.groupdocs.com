## MarkdownSaveOptions.ExportImagesAsBase64 property

Specifies whether images are saved in Base64 format to the output file. Default is `false`.

```csharp
public bool ExportImagesAsBase64 { get; set; }
```

### Remarks

When this property is set to `true`, then images data are exported directly into the image elements ![]() and separate files are not created. This property, if set to `true`, has the higher priority than the [`ImagesFolder`](../imagesfolder) property.
