## SpreadsheetFormats.Xltm field

Office Open XML Template Macro-Enabled (XLTM) represents files that are generated by Microsoft Excel as Macro-enabled template files. XLTM files are similar to XLTX in structure other than that the later doesn't support creating template files with macros. Learn more about this file format [here](https://wiki.fileformat.com/spreadsheet/xltm).
