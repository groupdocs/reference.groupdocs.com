## FixedLayoutFormats.Xps field

XPS file represents page layout files that are based on XML Paper Specifications created by Microsoft. It was developed as a replacement of EMF file format and is similar to PDF file format, but uses XML in layout, appearance, and printing information of a document. Learn more about this file format [here](https://docs.fileformat.com/page-description-language/xps/).
