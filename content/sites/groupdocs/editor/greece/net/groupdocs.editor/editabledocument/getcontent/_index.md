## GetContent(string, string) {#getcontent_1}

Επιστρέφει το συνολικό περιεχόμενο του εγγράφου HTML ως συμβολοσειρά, όπου οι σύνδεσμοι προς τους εξωτερικούς πόρους περιέχουν καθορισμένο πρόθεμα.

```csharp
public string GetContent(string externalImagesPrefix, string externalCssPrefix)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| externalImagesPrefix | String | Μέσω αυτής της παραμέτρου που χρησιμοποιείται μπορεί να καθορίσει ένα πρόθεμα, το οποίο θα προστεθεί στους συνδέσμους σε όλες τις εξωτερικές εικόνες σε στοιχεία IMG, που θα υπάρχουν στη συμβολοσειρά HTML που προκύπτει. Εάν είναι NULL ή κενό, τα προθέματα δεν θα προστεθούν. |
| externalCssPrefix | String | Μέσω αυτής της παραμέτρου που χρησιμοποιείται μπορεί να καθορίσει ένα πρόθεμα, το οποίο θα προστεθεί στα links σε όλα τα εξωτερικά φύλλα στυλ σε στοιχεία LINK, τα οποία θα υπάρχουν στη συμβολοσειρά HTML που προκύπτει. Εάν είναι NULL ή κενό, τα προθέματα δεν θα προστεθούν. |

### Επιστρεφόμενη Αξία

Συμβολοσειρά, η οποία περιέχει το περιεχόμενο του εγγράφου HTML με συνδέσμους, προσαρμοσμένους στους εξωτερικούς πόρους
