## EditableDocument.FromFile method

Στατικό εργοστάσιο, που δημιουργεί μια παρουσία του EditableDocument από ένα αρχείο HTML, που καθορίζεται από μια διαδρομή προς το ίδιο το αρχείο *.html και έναν φάκελο με συνδεδεμένους πόρων

```csharp
public static EditableDocument FromFile(string htmlFilePath, string resourceFolderPath)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| htmlFilePath | String | Συμβολοσειρά, που περιέχει μια πλήρη διαδρομή προς το αρχείο HTML. Δεν μπορεί να είναι null, θα πρέπει να είναι έγκυρη διαδρομή αρχείου και το ίδιο το αρχείο πρέπει να υπάρχει. |
| resourceFolderPath | String | Προαιρετική διαδρομή προς το φάκελο με πόρους HTML. Εάν δεν υπάρχει NULL, μη έγκυρος ή τέτοιος φάκελος, το πρόγραμμα επεξεργασίας θα προσπαθήσει να βρει αυτόν τον φάκελο μόνο του, αναλύοντας τη σήμανση HTML |
