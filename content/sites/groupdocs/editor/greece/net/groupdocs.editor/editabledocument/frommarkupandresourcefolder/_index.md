## EditableDocument.FromMarkupAndResourceFolder method

Στατικό εργοστάσιο, που δημιουργεί μια παρουσία του EditableDocument από μια καθορισμένη σήμανση HTML και από πόρους, που βρίσκονται στο φάκελο, που καθορίζονται από την πλήρη διαδρομή

```csharp
public static EditableDocument FromMarkupAndResourceFolder(string newHtmlContent, 
    string resourceFolderPath)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| newHtmlContent | String | Συμβολοσειρά, που περιέχει ακατέργαστη σήμανση HTML, που πρέπει να αναλυθεί. Δεν μπορεί να είναι NULL, κενό ή μη έγκυρο. |
| resourceFolderPath | String | Υποχρεωτική διαδρομή προς το φάκελο με πόρους. Θα χρησιμοποιηθούν όλα τα φύλλα στυλ που βρίσκονται σε αυτόν τον φάκελο. Δεν μπορεί να είναι NULL ή κενή συμβολοσειρά και αυτός ο φάκελος θα πρέπει να υπάρχει. |

Νέα μη μηδενική παρουσία του EditableDocument

### Παρατηρήσεις

Αυτό το στατικό εργοστάσιο είναι χρήσιμο όταν το περιεχόμενο του εγγράφου HTML παρουσιάζεται ως συμβολοσειρά, αλλά όλοι οι πόροι βρίσκονται σε κάποιο φάκελο και συχνά οι σύνδεσμοι προς αυτούς τους πόρους στη σήμανση HTML είναι άκυροι και απουσιάζουν. Κατά την επίκληση αυτής της μεθόδου, σαρώνει τον καθορισμένο φάκελο και εφαρμόζει αυτόματα όλα τα φύλλα στυλ που βρέθηκαν στο έγγραφο. Αυτή η μέθοδος είναι πολύ χρήσιμη κατά τη λήψη περιεχομένου από διαφορετικούς επεξεργαστές HTML, οι οποίοι συνήθως διακόπτουν τα μεταδεδομένα του εγγράφου και ούτω καθεξής.
