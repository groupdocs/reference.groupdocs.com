public List<string> GetCssContent()

## GetCssContent(string, string) {#getcsscontent_1}

Επιστρέφει το περιεχόμενο όλων των εξωτερικών φύλλων στυλ ως λίστα συμβολοσειρών, όπου μια συμβολοσειρά αντιπροσωπεύει ένα φύλλο στυλ. Το καθορισμένο πρόθεμα θα εφαρμόζεται σε κάθε σύνδεσμο προς τον εξωτερικό πόρο σε κάθε φύλλο στυλ που προκύπτει. Επιστρέφει κενή λίστα, εάν δεν υπάρχει CSS για αυτό έγγραφο.

```csharp
public List<string> GetCssContent(string externalImagesPrefix, string externalFontsPrefix)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| externalImagesPrefix | String | Μέσω αυτής της παραμέτρου που χρησιμοποιείται μπορεί να καθορίσει ένα πρόθεμα, το οποίο θα προστεθεί στα links σε όλες τις εξωτερικές εικόνες, που θα υπάρχουν στις δηλώσεις CSS στις προκύπτουσες συμβολοσειρές CSS. Εάν είναι NULL ή κενό, τα προθέματα δεν θα προστεθούν. |
| externalFontsPrefix | String | Μέσω αυτής της παραμέτρου που χρησιμοποιείται μπορεί να καθορίσει ένα πρόθεμα, το οποίο θα προστεθεί στα links σε όλες τις εξωτερικές γραμματοσειρές στους κανόνες @font-face at-στις προκύπτουσες συμβολοσειρές CSS. Εάν είναι NULL ή κενό, τα προθέματα δεν θα προστεθούν. |
