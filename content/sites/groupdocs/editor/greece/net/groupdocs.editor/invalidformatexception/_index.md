## InvalidFormatException class

Η εξαίρεση που δημιουργείται όταν ο χρήστης προσπαθεί να ανοίξει κάποιο έγγραφο με επιλογές συγκεκριμένης μορφής που δεν είναι συμβατές με την αρχική μορφή εγγράφου.

```csharp
public sealed class InvalidFormatException : Exception
```

### Παρατηρήσεις
