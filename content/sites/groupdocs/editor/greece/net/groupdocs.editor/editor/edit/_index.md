## Edit(IEditOptions) {#edit_1}

Ανοίγει ένα έγγραφο που έχει φορτωθεί προηγουμένως για επεξεργασία χρησιμοποιώντας συγκεκριμένες επιλογές για συγκεκριμένη μορφή, δημιουργώντας και επιστρέφοντας μια παρουσία του '[`EditableDocument`](../../editabledocument) κλάση, η οποία, με τη σειρά της, περιέχει μεθόδους για την παραγωγή σήμανσης HTML και σχετικούς πόρους.

```csharp
public EditableDocument Edit(IEditOptions editOptions)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| editOptions | IEditOptions | Επιλογές εγγράφου για συγκεκριμένη μορφή, που επιτρέπει τη ρύθμιση της διαδικασίας μετατροπής. Μπορεί να είναι NULL — σε αυτήν την περίπτωση το GroupDocs.Editor εντοπίζει μια μορφή εγγράφου που έχει φορτωθεί προηγουμένως και εφαρμόζει επιλογές, προεπιλεγμένες για αυτήν τη μορφή. Δεν πρέπει να έρχεται σε διένεξη με τις επιλογές φόρτωσης που εφαρμόστηκαν προηγουμένως. |

### Επιστρεφόμενη Αξία

παράδειγμα του '[`EditableDocument`](../../editabledocument) class, η οποία ενσωματώνει το συνολικό έγγραφο εισόδου με όλους τους πόρους του σε ενδιάμεση μορφή. Αυτή η μέθοδος, εάν ολοκληρωθεί με επιτυχία, δεν επιστρέφει ποτέ NULL.

### Παρατηρήσεις

Όταν εισάγετε το αρχικό έγγραφο φορτώνεται στην παρουσία 'Editor' μέσω του κατασκευαστή, αυτή η μέθοδος επιτρέπει το άνοιγμα εγγράφου για επεξεργασία μετατρέποντάς το σε ενδιάμεση μορφή, η οποία είναι ενσωματωμένη στην παρουσία της κλάσης 'EditableDocument'. '[`EditableDocument`](../../editabledocument) που επιστράφηκε από αυτήν τη μέθοδο, περιέχει όλες τις απαραίτητες μεθόδους και ιδιότητες για την παραγωγή σήμανσης HTML και αντίστοιχων πόρων (όπως εικόνες, γραμματοσειρές και φύλλα στυλ) σε όλες τις απαραίτητες διαμορφώσεις για τη μετέπειτα μετάβασή τους σε οποιοδήποτε πρόγραμμα επεξεργασίας HTML WYSIWYG. Αυτή η υπερφόρτωση αποκτά επιλογές επεξεργασίας, οι οποίες είναι συγκεκριμένες για οικογενειακές μορφές. **Μάθε περισσότερα**
