## Save(EditableDocument, Stream, ISaveOptions) {#save}

Μετατρέπει το καθορισμένο επεξεργασμένο έγγραφο, που αναπαρίσταται ως παράδειγμα του '[`EditableDocument`](../../editabledocument) , στο έγγραφο που προκύπτει με καθορισμένη μορφή και αποθηκεύει το περιεχόμενό του σε καθορισμένη ροή

```csharp
public void Save(EditableDocument inputDocument, Stream outputDocument, ISaveOptions saveOptions)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| inputDocument | EditableDocument | Έκδοση του εγγράφου εισόδου, που επεξεργάστηκε στο πρόγραμμα επεξεργασίας HTML WYSIWYG και αποθηκεύτηκε ως παράδειγμα του '[`EditableDocument`](../../editabledocument) κλάση, η οποία θα πρέπει να μετατραπεί σε έγγραφο εξόδου κάποιας συγκεκριμένης μορφής. Δεν πρέπει να είναι μηδενικό ή να απορρίπτεται. |
| outputDocument | Stream | Ροή εξόδου, στην οποία θα καταγραφεί το περιεχόμενο του εγγράφου που προκύπτει. Δεν πρέπει να είναι μηδενικό, διατεθειμένο, πρέπει να υποστηρίζει τη γραφή. |
| saveOptions | ISaveOptions | Επιλογές αποθήκευσης εγγράφου, οι οποίες καθορίζουν τη μορφή του εγγράφου που προκύπτει, και επίσης γενικές και συγκεκριμένες επιλογές αποθήκευσης. Δεν πρέπει να είναι μηδενικό. |
