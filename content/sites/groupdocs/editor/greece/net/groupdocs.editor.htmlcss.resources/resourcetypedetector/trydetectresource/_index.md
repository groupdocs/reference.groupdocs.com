## ResourceTypeDetector.TryDetectResource method

Προσπαθεί να αναλύσει μια ροή εισόδου και δημιουργεί έναν από τους υποστηριζόμενους πόρους HTML από αυτήν, λαμβάνοντας υπόψη έναν καθορισμένο υποθετικό τύπο, εάν δεν είναι null

```csharp
public static IHtmlResource TryDetectResource(Stream inputResourceStream, string name, 
    IResourceType assumptiveFormat)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| inputResourceStream | Stream | Ροή εισόδου, η οποία πιθανώς περιέχει έναν πόρο HTML. Εάν δεν είναι έγκυρο, θα γίνει εξαίρεση. |
| name | String | Όνομα πόρου, το οποίο θα χρησιμοποιηθεί για τον πόρο που δημιουργήθηκε και επιστράφηκε με επιτυχία. Δεν μπορεί να είναι NULL, κενό ή κενό |
| assumptiveFormat | IResourceType | Υποτιθέμενη μορφή του πόρου HTML εισόδου, η οποία είναι χρήσιμη για την επίτευξη της καλύτερης απόδοσης. Εάν είναι εντελώς άγνωστο, χρησιμοποιήστε την τιμή NULL. Μπορεί να είναι λανθασμένο, αυτό μόνο θα επιδεινώσει την απόδοση. |

### Επιστρεφόμενη Αξία

Παράδειγμα, το οποίο υλοποιεί τη διεπαφή "IHtmlResource" και αντιπροσωπεύει έναν από τους υποστηριζόμενους πόρους HTML σε περίπτωση επιτυχίας ή NULL σε περίπτωση αποτυχίας
