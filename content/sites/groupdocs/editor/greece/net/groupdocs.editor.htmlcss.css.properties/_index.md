Ο χώρος ονομάτων GroupDocs.Editor.HtmlCss.Css.Properties παρέχει τύπους που αντιπροσωπεύουν διαφορετικές ιδιότητες CSS (συστατικά στοιχεία δηλώσεων CSS).

## Δομές

| Δομή | Περιγραφή |
| --- | --- |
| [FontSize](./fontsize) | Αντιπροσωπεύει ένα μέγεθος γραμματοσειράς ως ειδική μονάδα ή τιμή μήκους, που καθορίζει το μέγεθος της γραμματοσειράς (ιστορικά το πλάτος του κεφαλαίου "M"). |
| [FontStyle](./fontstyle) | Καθορίζει τον τρόπο με τον οποίο θα πρέπει να διαμορφωθεί το στυλ της γραμματοσειράς με: μια κανονική, πλάγια ή πλάγια όψη από την οικογένεια γραμματοσειρών της. |
| [FontWeight](./fontweight) | Η ιδιότητα Font-weight ορίζει το βάρος (ή την έντονη γραφή) της γραμματοσειράς. Τα διαθέσιμα βάρη εξαρτώνται από την οικογένεια γραμματοσειρών που έχει οριστεί αυτήν τη στιγμή. |
