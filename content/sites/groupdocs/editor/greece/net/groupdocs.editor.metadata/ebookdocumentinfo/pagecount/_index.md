## EbookDocumentInfo.PageCount property

Επιστρέφει αριθμό σελίδων σε περίπτωση MOBI ή AZW3 ή αριθμό κεφαλαίων σε περίπτωση ePub.

```csharp
public int PageCount { get; }
```

### Παρατηρήσεις

Τα έγγραφα e-Book συνήθως δεν έχουν σταθερές σελίδες και επομένως πλήθος σελίδων. Στην περίπτωση του ePub είναι δυνατός ο υπολογισμός ενός αριθμού κεφαλαίων. Ωστόσο, ούτε οι μορφές MOBI και AZW3 έχουν κεφάλαια, επομένως αυτός ο αριθμός υπολογίζεται από το τυπικό μέγεθος σελίδας που έχει οριστεί σε A4 σε κατακόρυφο προσανατολισμό.
