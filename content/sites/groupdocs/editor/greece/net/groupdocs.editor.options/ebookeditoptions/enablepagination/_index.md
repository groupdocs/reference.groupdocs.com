## EbookEditOptions.EnablePagination property

Επιτρέπει την ενεργοποίηση ή απενεργοποίηση της σελιδοποίησης στο έγγραφο HTML που προκύπτει. Από προεπιλογή είναι απενεργοποιημένο (`ψευδής`).

```csharp
public bool EnablePagination { get; set; }
```

### Παρατηρήσεις

Στην ουσία οι περισσότερες μορφές ηλεκτρονικών βιβλίων εσωτερικά είναι μια μορφή ροής όπως το Office Open XML, όπου το περιεχόμενο είναι σταθερό και χωρίζεται σε κεφάλαια αλλά όχι σε σελίδες. Ωστόσο, περιέχει κάποιες συγκεκριμένες πληροφορίες για τη σελίδα, όπως τους αριθμούς σελίδων, τις υποσημειώσεις, τις επιγραφές/υποσέλιδα και ούτω καθεξής. Ορισμένοι αναγνώστες ηλεκτρονικών βιβλίων διαχωρίζουν το περιεχόμενο των ηλεκτρονικών βιβλίων σε σελίδες, ενώ άλλοι (ειδικά για κινητά) — όχι. Αυτή η επιλογή επιτρέπει τον έλεγχο του τρόπου με τον οποίο θα πρέπει να αναπαρίσταται το περιεχόμενο των ηλεκτρονικών βιβλίων σε HTML/CSS κατά την επεξεργασία — στο float (`ψευδής`) ή σελιδοποιημένο (`αληθής`) προβολή.
