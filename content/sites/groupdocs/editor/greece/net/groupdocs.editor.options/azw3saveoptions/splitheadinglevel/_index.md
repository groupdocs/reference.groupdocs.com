## Azw3SaveOptions.SplitHeadingLevel property

Καθορίζει εάν θα χωριστεί το περιεχόμενο των ηλεκτρονικών βιβλίων AZW3 σε πακέτα και εάν ναι, το μέγιστο επίπεδο επικεφαλίδων στο οποίο θα χωριστεί το περιεχόμενο του AZW3. Η προεπιλεγμένη τιμή είναι`2` . Ρύθμιση σε`0`θα απενεργοποιήσει τον διαχωρισμό, έτσι όλο το περιεχόμενο του e-book θα ενσωματωθεί σε ένα μόνο πακέτο μέσα στο AZW3.

```csharp
public int SplitHeadingLevel { get; set; }
```

### Παρατηρήσεις

Σε ορισμένες περιπτώσεις, είναι προτιμότερο να χωρίσετε το περιεχόμενο του e-book σε πολλά μικρότερα πακέτα, που βρίσκονται μέσα στο αρχείο εξόδου AZW3. Αυτή η ιδιότητα ελέγχει εάν θα πρέπει να εκτελεστεί αυτός ο διαχωρισμός και εάν ναι, τότε πώς.

Όταν αυτή η ιδιότητα έχει οριστεί σε τιμή από 1 έως 9, το έγγραφο θα χωριστεί σε παραγράφους που έχουν μορφοποιηθεί χρησιμοποιώντας **Επικεφαλίδα 1** ,**Επικεφαλίδα 2** ,**Κεφάλαιο 3** κ.λπ. στυλ μέχρι το καθορισμένο επίπεδο επικεφαλίδας.

Από προεπιλογή (τιμή του`2` ), μόνο**Επικεφαλίδα 1** και**Επικεφαλίδα 2** οι παράγραφοι προκαλούν διαίρεση του εγγράφου. Η ρύθμιση αυτής της ιδιότητας στο μηδέν θα έχει ως αποτέλεσμα να μην διαχωριστεί καθόλου το έγγραφο στις παραγράφους επικεφαλίδων.
