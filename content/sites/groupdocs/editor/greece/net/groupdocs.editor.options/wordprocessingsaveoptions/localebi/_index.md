## WordProcessingSaveOptions.LocaleBi property

Επιτρέπει τον ορισμό τοπικών ρυθμίσεων (γλώσσας) παράκαμψης για το έγγραφο WordProcessing για το κείμενο RTL (από δεξιά προς τα αριστερά), το οποίο θα εφαρμοστεί κατά τη δημιουργία του. Όταν δεν έχει καθοριστεί (προεπιλεγμένη τιμή), MS Word (ή άλλο πρόγραμμα) θα εντοπίσει (ή θα επιλέξει) το έγγραφο RTL locale σύμφωνα με τις δικές του ρυθμίσεις ή άλλους παράγοντες.

```csharp
public CultureInfo LocaleBi { get; set; }
```

### Παρατηρήσεις

Αυτή η επιλογή εφαρμόζει αναγκαστικά τις καθορισμένες τοπικές ρυθμίσεις στο συνολικό κείμενο RTL στο έγγραφο. Μην το χρησιμοποιείτε, εάν το έγγραφο περιέχει διαφορετικά μέρη κειμένου, τα οποία είναι γραμμένα σε διαφορετικές γλώσσες.
