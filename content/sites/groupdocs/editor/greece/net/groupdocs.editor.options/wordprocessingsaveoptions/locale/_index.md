## WordProcessingSaveOptions.Locale property

Επιτρέπει τον ορισμό αντικατάστασης προεπιλεγμένων τοπικών ρυθμίσεων (γλώσσα) για το έγγραφο WordProcessing, το οποίο θα εφαρμοστεί κατά τη δημιουργία του. Όταν δεν καθορίζεται (προεπιλεγμένη τιμή), το MS Word (ή άλλο πρόγραμμα) θα εντοπίσει (ή θα επιλέξει) το έγγραφο locale σύμφωνα με στις δικές του ρυθμίσεις ή άλλους παράγοντες.

```csharp
public CultureInfo Locale { get; set; }
```

### Παρατηρήσεις

Αυτή η επιλογή εφαρμόζει αναγκαστικά τις καθορισμένες τοπικές ρυθμίσεις στο συνολικό κείμενο του εγγράφου. Μην το χρησιμοποιείτε, εάν το έγγραφο περιέχει διαφορετικά μέρη κειμένου, τα οποία είναι γραμμένα σε διαφορετικές γλώσσες.
