## PresentationSaveOptions.InsertAsNewSlide property

Boolean flag, η οποία καθορίζει εάν η επεξεργασμένη διαφάνεια θα πρέπει να αντικαταστήσει την υπάρχουσα διαφάνεια στην αρχική παρουσίαση στη θέση, που καθορίζεται από το[`SlideNumber`](../slidenumber) ιδιότητα ή θα πρέπει να γίνει ένεση μεταξύ της υπάρχουσας και της προηγούμενης, χωρίς να αντικατασταθεί το περιεχόμενό της. Από προεπιλογή είναι ψευδής — η υπάρχουσα διαφάνεια θα αντικατασταθεί. Αυτή η ιδιότητα αγνοείται, εάν η τιμή του[`SlideNumber`](../slidenumber) Η ιδιότητα έχει οριστεί σε '0'.

```csharp
public bool InsertAsNewSlide { get; set; }
```

### Παρατηρήσεις

Από προεπιλογή, η διαφάνεια αντικαθίσταται. Αυτό σημαίνει ότι εάν η παρουσίαση έχει 5 διαφάνειες και[`SlideNumber`](../slidenumber) =4, τότε η 4η διαφάνεια θα αντικατασταθεί με τη νέα επεξεργασμένη διαφάνεια, ενώ ο συνολικός αριθμός διαφανειών στην παρουσίαση (5) θα παραμείνει ανέγγιχτος. Ωστόσο, εάν η τιμή αυτής της ιδιότητας οριστεί σεαληθής, η νέα επεξεργασμένη διαφάνεια θα εισαχθεί ως 4η διαφάνεια και όλες οι επόμενες διαφάνειες θα μετατοπιστούν στο τέλος: η "παλιά" 4η διαφάνεια γίνεται 5η και η 5η γίνεται 6η και ο συνολικός αριθμός των διαφανειών στην παρουσίαση θα αυξηθεί κατά μία και ίσο με 6.
