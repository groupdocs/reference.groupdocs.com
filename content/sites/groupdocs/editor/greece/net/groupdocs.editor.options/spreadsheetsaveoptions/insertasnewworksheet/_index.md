## SpreadsheetSaveOptions.InsertAsNewWorksheet property

Boolean flag, η οποία καθορίζει εάν το επεξεργασμένο φύλλο εργασίας θα πρέπει να αντικαταστήσει το υπάρχον φύλλο εργασίας στο αρχικό υπολογιστικό φύλλο στη θέση, που καθορίζεται από το[`WorksheetNumber`](../worksheetnumber) ιδιότητα, ή θα πρέπει να γίνει ένεση μεταξύ του υπάρχοντος φύλλου εργασίας και του προηγούμενου, χωρίς να αντικατασταθεί το περιεχόμενό του. Από προεπιλογή είναι false — το υπάρχον φύλλο εργασίας θα αντικατασταθεί. Αυτή η ιδιότητα αγνοείται, εάν η τιμή του[`WorksheetNumber`](../worksheetnumber) Η ιδιότητα έχει οριστεί σε '0'.

```csharp
public bool InsertAsNewWorksheet { get; set; }
```

### Παρατηρήσεις

Από προεπιλογή, το φύλλο εργασίας αντικαθίσταται. Αυτό σημαίνει ότι εάν δοθεί το υπολογιστικό φύλλο έχει 5 φύλλα εργασίας και[`WorksheetNumber`](../worksheetnumber) =4, τότε το 4ο φύλλο εργασίας θα αντικατασταθεί με το νέο επεξεργασμένο φύλλο εργασίας, ενώ ο συνολικός αριθμός των φύλλων εργασίας στο υπολογιστικό φύλλο (5) θα παραμείνει ανέγγιχτος. Ωστόσο, εάν η τιμή αυτής της ιδιότητας οριστεί σεαληθήςτο νέο επεξεργασμένο φύλλο εργασίας θα εισαχθεί ως 4ο φύλλο εργασίας και όλα τα επόμενα φύλλα εργασίας θα μεταφερθούν στο τέλος: το "παλιό" 4ο φύλλο εργασίας γίνεται 5ο, και το 5ο γίνεται 6ο και ο συνολικός αριθμός των φύλλων εργασίας στο υπολογιστικό φύλλο θα αυξηθεί κατά ένα και ίσο με 6.
