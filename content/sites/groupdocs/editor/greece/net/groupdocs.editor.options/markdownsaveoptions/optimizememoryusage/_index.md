## MarkdownSaveOptions.OptimizeMemoryUsage property

Ενεργοποιεί μηχανισμούς βελτιστοποίησης μνήμης κατά τη δημιουργία εγγράφων από HTML, γεγονός που υποβαθμίζει την απόδοση ως κόστος μείωσης της χρήσης μνήμης. Ρύθμιση αυτής της επιλογής σε`αληθής`μπορεί να μειώσει σημαντικά την κατανάλωση μνήμης ενώ δημιουργεί μεγάλα έγγραφα με κόστος πιο αργή εξοικονόμηση χρόνου. Η προεπιλογή είναι`ψευδής` (η βελτιστοποίηση μνήμης είναι απενεργοποιημένη για λόγους καλύτερης απόδοσης).

```csharp
public bool OptimizeMemoryUsage { get; set; }
```
