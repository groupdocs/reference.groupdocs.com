## MarkdownSaveOptions.ExportImagesAsBase64 property

Καθορίζει εάν οι εικόνες αποθηκεύονται σε μορφή Base64 στο αρχείο εξόδου. Η προεπιλογή είναι`ψευδής` .

```csharp
public bool ExportImagesAsBase64 { get; set; }
```

### Παρατηρήσεις

Όταν αυτή η ιδιότητα έχει οριστεί σε`αληθής` τότε τα δεδομένα εικόνων εξάγονται απευθείας στα στοιχεία εικόνας ![]() και δεν δημιουργούνται ξεχωριστά αρχεία. Αυτή η ιδιότητα, εάν έχει οριστεί σε`αληθής` , έχει την υψηλότερη προτεραιότητα από το[`ImagesFolder`](../imagesfolder) ιδιοκτησία.
