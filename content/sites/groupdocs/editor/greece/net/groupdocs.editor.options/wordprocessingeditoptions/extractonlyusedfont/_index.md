## WordProcessingEditOptions.ExtractOnlyUsedFont property

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν εξάγει μόνο πόρους γραμματοσειράς που χρησιμοποιούνται στο περιεχόμενο κειμένου του εγγράφου.

```csharp
public bool ExtractOnlyUsedFont { get; set; }
```

### Αξία περιουσίας

`αληθής` εάν απαιτείται η εξαγωγή μόνο εκείνων των πόρων γραμματοσειράς που χρησιμοποιούνται στο περιεχόμενο κειμένου του εγγράφου· σε διαφορετική περίπτωση,`ψευδής` . Η προεπιλεγμένη τιμή είναι`ψευδής` .

### Παρατηρήσεις

Δεν χρησιμοποιούνται απευθείας όλες οι γραμματοσειρές που χρησιμοποιούνται στο έγγραφο WordProcessing 100% (εφαρμόζονται σε κάποιο κείμενο). Μπορεί να υπάρχει μια περίπτωση, όταν η γραμματοσειρά αναφέρεται στο έγγραφο και μάλιστα μπορεί να ενσωματωθεί, αλλά δεν εφαρμόζεται σε οποιοδήποτε κομμάτι κειμένου. Για παράδειγμα, κάποια γραμματοσειρά μπορεί να είναι συνδεδεμένη σε κάποιο στυλ, αλλά αυτό το στυλ δεν εφαρμόζεται σε κανένα μέρος του κειμένου. Αυτή η επιλογή ελέγχει τον τρόπο επεξεργασίας τέτοιων υποθέσεων.
