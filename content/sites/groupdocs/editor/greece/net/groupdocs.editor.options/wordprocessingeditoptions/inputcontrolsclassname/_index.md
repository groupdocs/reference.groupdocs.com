## WordProcessingEditOptions.InputControlsClassName property

Επιτρέπει τον καθορισμό ενός ονόματος κλάσης, το οποίο θα τοποθετηθεί στα χαρακτηριστικά 'class' σε κάθε στοιχείο HTML, που αντιπροσωπεύει κάποιο πεδίο στο έγγραφο εισόδου WordProcessing. Από προεπιλογή είναι NULL - τα χαρακτηριστικά 'class' δεν εφαρμόζονται.

public string InputControlsClassName { get; set; }

### Παρατηρήσεις

Σχεδόν όλες οι μορφές από την οικογένεια μορφών WordProcessing περιέχουν πεδία — συγκεκριμένες οντότητες εγγράφων, που επιτρέπουν τη λήψη δεδομένων εισόδου από τους χρήστες. Υπάρχει μεγάλη ποικιλία πεδίων: πλαίσια κειμένου, πλαίσια ελέγχου, σύνθετα πλαίσια, αναπτυσσόμενες λίστες, κουμπιά, επιλογείς ημερομηνίας/ώρας κ.λπ. Όλα μεταφράζονται στις πιο κατάλληλες δομές και στοιχεία HTML, με διατήρηση του εισαγόμενου χρήστη δεδομένα, εάν υπάρχουν στο έγγραφο εισόδου. Σε συγκεκριμένες περιπτώσεις χρήσης απαιτείται μόνο η συλλογή δεδομένων που έχουν εισαχθεί από την πλευρά του πελάτη αντί της επεξεργασίας ολόκληρου του περιεχομένου του εγγράφου. Για μια τέτοια περίπτωση, απαιτείται η αναγνώριση των στοιχείων ελέγχου εισόδου με κάποιο τρόπο για την ανάκτησή τους με τα δεδομένα τους στην πλευρά του πελάτη. Αυτή η ιδιότητα επιτρέπει τον καθορισμό ενός ονόματος κλάσης, το οποίο θα εφαρμόζεται για κάθε στοιχείο ελέγχου εισόδου στη σήμανση HTML, έτσι ώστε ο κώδικας πελάτη να μπορεί να διασχίζει τη δομή του εγγράφου HTML και να συλλέγει δεδομένα.
