## TextEditOptions.RecognizeLists property

Επιτρέπει τον καθορισμό του τρόπου με τον οποίο αναγνωρίζονται τα αριθμημένα στοιχεία λίστας κατά την εισαγωγή του εγγράφου από μορφή απλού κειμένου. Η προεπιλεγμένη τιμή είναι true.

```csharp
public bool RecognizeLists { get; set; }
```

### Παρατηρήσεις

Εάν αυτή η επιλογή έχει οριστεί σε false, ο αλγόριθμος αναγνώρισης παραθέτει τις παραγράφους της λίστας, όταν οι αριθμοί λίστας τελειώνουν με σύμβολα κουκκίδας, δεξιάς αγκύλης ή κουκκίδας (όπως "•", "*", "-" ή "o"). Εάν αυτή η επιλογή οριστεί σε true, τα κενά διαστήματα χρησιμοποιούνται επίσης ως οριοθέτες αριθμού λίστας: ο αλγόριθμος αναγνώρισης λίστας για αρίθμηση σε αραβικό στυλ (1., 1.1.2.) χρησιμοποιεί σύμβολα και κενά και κουκκίδες (".").
