## PdfSaveOptions.Password property

Κωδικός πρόσβασης, ο οποίος θα εφαρμοστεί στο έγγραφο PDF που δημιουργήθηκε ως κωδικός πρόσβασης χρήστη, που απαιτείται για το άνοιγμα. Εάν είναι NULL ή κενός, δεν θα εφαρμοστεί κωδικός πρόσβασης στο έγγραφο. Διαφορετικά, το έγγραφο θα κρυπτογραφηθεί με RC4 (μήκος κλειδιού 128 bit). Από προεπιλογή είναι NULL — ο κωδικός πρόσβασης δεν εφαρμόζεται.

```csharp
public string Password { get; set; }
```
