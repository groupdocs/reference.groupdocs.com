## XmlEditOptions.FixIncorrectStructure property

Επιτρέπει την ενεργοποίηση ή απενεργοποίηση του μηχανισμού για τη διόρθωση κατεστραμμένης δομής XML. Από προεπιλογή είναι απενεργοποιημένο (false).

```csharp
public bool FixIncorrectStructure { get; set; }
```

### Παρατηρήσεις

Από προεπιλογή είναι αποδεκτά μόνο τα σωστά έγκυρα και καλά διαμορφωμένα έγγραφα XML. Όταν αυτή η επιλογή είναι ενεργοποιημένη, το GroupDocs.Editor θα προσπαθήσει να διορθώσει την κατεστραμμένη δομή XML, εάν είναι δυνατόν.
