## PresentationEditOptions.SlideNumber property

Επιτρέπει τον καθορισμό των αριθμών διαφανειών, οι οποίοι θα πρέπει να ανοίξουν για επεξεργασία

```csharp
public int SlideNumber { get; set; }
```

### Παρατηρήσεις

Ο αριθμός διαφάνειας είναι ένα μηδενικό ευρετήριο μιας διαφάνειας, που επιτρέπει τον καθορισμό και την επιλογή μιας συγκεκριμένης διαφάνειας από μια παρουσίαση προς επεξεργασία. Εάν είναι μικρότερος από 0, θα επιλεγεί η πρώτη διαφάνεια (ίδια με Αριθμός Διαφάνειας = 0). Εάν ο αριθμός όλων των διαφανειών στην παρουσίαση είναι μεγαλύτερος από αυτό, θα επιλεγεί η τελευταία διαφάνεια. Εάν η παρουσίαση εισαγωγής περιέχει μόνο μία διαφάνεια, αυτή η επιλογή θα αγνοηθεί και αυτή η μεμονωμένη διαφάνεια θα επεξεργαστεί. Εάν προσπαθήσετε να ανοίξετε για επεξεργασία μιας κρυφής διαφάνειας, ενώ[`ShowHiddenSlides`](../showhiddenslides) Η επιλογή έχει οριστεί σε "false", η εξαίρεση θα γίνει.
