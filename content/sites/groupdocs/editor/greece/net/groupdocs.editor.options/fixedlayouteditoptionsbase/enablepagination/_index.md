## FixedLayoutEditOptionsBase.EnablePagination property

Επιτρέπει την ενεργοποίηση (true) ή την απενεργοποίηση (false) σελιδοποίησης στο προκύπτον έγγραφο HTML. Από προεπιλογή είναι απενεργοποιημένο (false).

```csharp
public bool EnablePagination { get; set; }
```

### Παρατηρήσεις

Τα έγγραφα σταθερής μορφής (ιδιαίτερα PDF και XPS) στην ουσία τους είναι αυστηρά σελιδοποιημένα, το περιεχόμενό τους έχει σταθερή διάταξη και χωρίζεται σε σελίδες. Αλλά το επεξεργάσιμο HTML που προκύπτει μπορεί να αναπαρασταθεί είτε σε προβολή χωρίς σελίδα είτε σε σελιδική προβολή.
