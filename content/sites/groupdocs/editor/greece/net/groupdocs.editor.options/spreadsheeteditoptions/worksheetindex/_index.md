## SpreadsheetEditOptions.WorksheetIndex property

Επιτρέπει τον καθορισμό του ευρετηρίου με βάση το 0 του φύλλου εργασίας (καρτέλα) του εγγράφου υπολογιστικού φύλλου εισόδου, το οποίο θα πρέπει να μετατραπεί σε HTML (βλ. παρατηρήσεις).

```csharp
public int WorksheetIndex { get; set; }
```

### Παρατηρήσεις

Τα περισσότερα έγγραφα υπολογιστικών φύλλων υποστηρίζουν μια έννοια καρτελών, δηλαδή μπορούν να είναι πολλαπλών καρτελών. Από την άλλη πλευρά, η μορφή HTML δεν υποστηρίζει τέτοια δομή. Λόγω αυτού του GroupDocs.Editor μπορεί να μετατρέψει σε HTML μόνο μία συγκεκριμένη καρτέλα του εγγράφου εισόδου και αυτή η επιλογή επιτρέπει τον καθορισμό της. Ο δείκτης καρτέλας βασίζεται στο 0 , οι αρνητικές τιμές απαγορεύονται. Εάν το καθορισμένο ευρετήριο υπερβαίνει τον αριθμό όλων των καρτελών, θα γίνει η εξαίρεση. Εάν το έγγραφο υπολογιστικού φύλλου εισαγωγής περιέχει μόνο μία καρτέλα, αυτή η επιλογή θα αγνοηθεί. Η προεπιλεγμένη τιμή είναι 0 (πρώτη καρτέλα).
