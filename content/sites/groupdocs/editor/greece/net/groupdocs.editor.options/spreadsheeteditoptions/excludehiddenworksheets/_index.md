## SpreadsheetEditOptions.ExcludeHiddenWorksheets property

Επιτρέπει την εξαίρεση κρυφών φύλλων εργασίας στο έγγραφο του Υπολογιστικού Φύλλου εισαγωγής, ώστε να αγνοηθούν τελείως. Η προεπιλογή είναι ψευδής - τα κρυφά φύλλα εργασίας είναι διαθέσιμα και υποβάλλονται σε επεξεργασία κανονικά.

```csharp
public bool ExcludeHiddenWorksheets { get; set; }
```

### Παρατηρήσεις

Αρκετές μορφές δυαδικών υπολογιστικών φύλλων (όπως το XLSX) υποστηρίζουν την έννοια κρυφών φύλλων εργασίας (καρτέλες). Ένα έγγραφο τέτοιας μορφής, εάν έχει περισσότερα από ένα φύλλα εργασίας, μπορεί να περιέχει επιπλέον κρυφά φύλλα εργασίας. Από προεπιλογή τέτοια κρυφά φύλλα εργασίας είναι διαθέσιμα για επεξεργασία, αλλά με αυτήν την επιλογή μπορεί να τα αγνοήσει, όπως αυτά τα κρυφά φύλλα εργασίας απουσιάζουν και δεν υπάρχουν. Όταν αυτή η επιλογή είναι ενεργοποιημένη, δεν μπορείτε να επιλέξετε κρυφό φύλλο εργασίας με το '[`WorksheetIndex`](../worksheetindex) ιδιοκτησία.
