## EpubSaveOptions.SplitHeadingLevel property

Καθορίζει το μέγιστο επίπεδο επικεφαλίδων στο οποίο θα χωριστεί το αρχείο ePub. Η προεπιλεγμένη τιμή είναι`2` . Ρύθμιση σε`0` θα απενεργοποιήσει τον διαχωρισμό, επομένως όλο το περιεχόμενο του e-Book θα ενσωματωθεί σε ένα μόνο πακέτο μέσα στο ePub.

```csharp
public int SplitHeadingLevel { get; set; }
```

### Παρατηρήσεις

Όταν αυτή η ιδιότητα έχει οριστεί σε τιμή από 1 έως 9, το έγγραφο θα χωριστεί σε παραγράφους που έχουν μορφοποιηθεί χρησιμοποιώντας **Επικεφαλίδα 1** ,**Επικεφαλίδα 2** ,**Κεφάλαιο 3** κ.λπ. στυλ μέχρι το καθορισμένο επίπεδο επικεφαλίδας.

Από προεπιλογή, μόνο**Επικεφαλίδα 1** και**Επικεφαλίδα 2** Οι παράγραφοι προκαλούν τη διαίρεση του εγγράφου. Εάν ορίσετε αυτήν την ιδιότητα στο μηδέν, το έγγραφο δεν θα χωριστεί καθόλου στις επικεφαλίδες παραγράφους.
