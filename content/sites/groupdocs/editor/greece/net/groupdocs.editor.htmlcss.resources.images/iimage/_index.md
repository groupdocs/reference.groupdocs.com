## IImage interface

### Παρατηρήσεις

https://developer.mozilla.org/en-US/docs/Web/CSS/image Η αναφορά CSS δηλώνει 4 τύπους εικόνων που μπορούν να χειριστούν: 1. Εικόνες με εγγενείς διαστάσεις (φυσικό μέγεθος) και επομένως σταθερές αναλογία διαστάσεων, που περιλαμβάνει όλες τις εικόνες ράστερ. 2. Εικόνες με πολλαπλές εγγενείς διαστάσεις, που υπάρχουν σε πολλές εκδόσεις μέσα σε ένα μόνο αρχείο, που περιλαμβάνει αρχεία ICON (*.ico). Αυτό μπορεί να είναι επίσης TIFF, καθώς υποστηρίζουν πολλές εικόνες μέσα σε ένα αρχείο, ωστόσο η μορφή TIFF δεν υποστηρίζεται στο web. 3. Εικόνες χωρίς εγγενείς διαστάσεις αλλά με εγγενή λόγο διαστάσεων μεταξύ του πλάτους και του ύψους του. Αυτό περιλαμβάνει όλες τις διανυσματικές εικόνες, πρώτα από όλα SVG. 4. Εικόνες χωρίς εγγενείς διαστάσεις, ούτε ενδογενή λόγο διαστάσεων. Αυτό περιλαμβάνει όλες τις κλίσεις.
