## EditableDocument.FromMarkupAndResourceFolder method

Statische fabriek, die een exemplaar van EditableDocument maakt van een opgegeven HTML-opmaak en van bronnen in de map, gespecificeerd door het volledige pad

```csharp
public static EditableDocument FromMarkupAndResourceFolder(string newHtmlContent, 
    string resourceFolderPath)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| newHtmlContent | String | Tekenreeks, die onbewerkte HTML-opmaak bevat, die moet worden geparseerd. Mag niet NULL, leeg of ongeldig zijn. |
| resourceFolderPath | String | Verplicht pad naar de map met resources. Alle stylesheets die zich in deze map bevinden, worden gebruikt. Kan niet NULL of lege tekenreeks zijn en deze map zou moeten bestaan. |

### Winstwaarde

Nieuw niet-null exemplaar van EditableDocument

### Opmerkingen

Deze statische fabriek is handig wanneer de inhoud van een HTML-document wordt gepresenteerd als een tekenreeks, maar alle bronnen zich in een bepaalde map bevinden en koppelingen naar deze bronnen in HTML-opmaak vaak ongeldig en afwezig zijn. Bij het aanroepen van deze methode scant het de opgegeven map en past het automatisch alle gevonden stylesheets toe op het document. Deze methode is erg handig bij het verkrijgen van inhoud van verschillende HTML-editors, die meestal de metadata van het document afsnijden, enzovoort.
