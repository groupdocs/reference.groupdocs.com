## GetCssContent(string, string) {#getcsscontent_1}

Retourneert inhoud van alle externe stylesheets als een lijst met strings, waarbij één string één stylesheet vertegenwoordigt. Opgegeven prefix wordt toegepast op elke link naar de externe bron in elke resulterende stylesheet. Retourneert lege lijst, als er geen CSS voor is document.

```csharp
public List<string> GetCssContent(string externalImagesPrefix, string externalFontsPrefix)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| externalImagesPrefix | String | Via deze gebruikte parameter kan een voorvoegsel worden opgegeven, dat zal worden toegevoegd aan de links van alle externe afbeeldingen, die aanwezig zullen zijn in CSS-declaraties in de resulterende CSS-strings. Als NULL of leeg is, worden er geen voorvoegsels toegevoegd. |
| externalFontsPrefix | String | Via deze gebruikte parameter kan een prefix worden opgegeven, die zal worden toegevoegd aan de links naar alle externe lettertypen in de @font-face at-rules in de resulterende CSS-strings. Als NULL of leeg is, worden er geen voorvoegsels toegevoegd. |

### Winstwaarde

Een lijst met tekenreeksen, waarbij elke tekenreeks de inhoud van één CSS-document bevat
