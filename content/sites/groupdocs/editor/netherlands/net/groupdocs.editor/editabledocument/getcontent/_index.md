## GetContent(string, string) {#getcontent_1}

Retourneert de algemene inhoud van het HTML-document als een tekenreeks, waarbij koppelingen naar de externe bronnen een gespecificeerd voorvoegsel bevatten.

```csharp
public string GetContent(string externalImagesPrefix, string externalCssPrefix)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| externalImagesPrefix | String | Via deze gebruikte parameter kan een prefix worden gespecificeerd, die zal worden toegevoegd aan de links voor alle externe afbeeldingen in IMG-elementen, die aanwezig zullen zijn in de resulterende HTML-string. Als NULL of leeg is, worden er geen voorvoegsels toegevoegd. |
| externalCssPrefix | String | Via deze gebruikte parameter kan een prefix worden opgegeven, die zal worden toegevoegd aan de links aan alle externe stylesheets in LINK-elementen, die aanwezig zullen zijn in de resulterende HTML-string. Als NULL of leeg is, worden er geen voorvoegsels toegevoegd. |

### Winstwaarde

String, die de inhoud van het HTML-document met links bevat, aangepast aan de externe bronnen
