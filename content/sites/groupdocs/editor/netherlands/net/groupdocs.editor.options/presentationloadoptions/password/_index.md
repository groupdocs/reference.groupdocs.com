## PresentationLoadOptions.Password property

Hiermee kunt u het wachtwoord specificeren, wijzigen en verkrijgen dat zal worden gebruikt voor het openen van het presentatiedocument, als het gecodeerd is. Stel in op NULL of lege string om het wachtwoord te verwijderen.

```csharp
public string Password { get; set; }
```

### Opmerkingen

Deze eigenschap heeft standaard de NULL-waarde — het wachtwoord is niet ingesteld. Als het invoerpresentatiedocument met een wachtwoord is beveiligd, is het wachtwoord verplicht en wordt er een uitzondering gegenereerd als het wachtwoord niet is opgegeven of ongeldig is. Als het invoerpresentatiedocument NIET met een wachtwoord is beveiligd, maar er wel een wachtwoord is ingesteld, wordt het genegeerd.
