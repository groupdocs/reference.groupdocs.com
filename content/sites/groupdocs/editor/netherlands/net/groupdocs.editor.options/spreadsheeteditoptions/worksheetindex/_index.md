## SpreadsheetEditOptions.WorksheetIndex property

Maakt het mogelijk om de op 0 gebaseerde index van het werkblad (tabblad) van het ingevoerde Spreadsheet-document te specificeren, dat moet worden geconverteerd naar HTML (zie opmerkingen).

```csharp
public int WorksheetIndex { get; set; }
```

### Opmerkingen

De meeste Spreadsheet-documenten ondersteunen een concept van tabbladen, dwz ze kunnen meerdere tabbladen hebben. Aan de andere kant ondersteunt het HTML-formaat een dergelijke structuur niet. Hierdoor kan GroupDocs.Editor slechts één specifiek tabblad van het invoerdocument naar HTML converteren, en met deze optie kunt u het specificeren. Tabbladindex is op 0 gebaseerd , zijn negatieve waarden verboden. Als de opgegeven index het aantal van alle tabbladen overschrijdt, wordt de uitzondering gegenereerd. Als het ingevoerde spreadsheetdocument slechts één tabblad bevat, wordt deze optie genegeerd. Standaardwaarde is 0 (eerste tabblad).
