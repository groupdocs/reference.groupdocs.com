## SpreadsheetLoadOptions.OptimizeMemoryUsage property

Maakt geheugenoptimalisatiemechanismen mogelijk tijdens de verwerking van invoerdocumenten, wat in sommige speciale gevallen de prestaties kan verminderen, maar aan de andere kant het geheugengebruik vermindert. Handig bij het verwerken van grote documenten en geconfronteerd met OutOfMemoryException. Standaard is false (geheugenoptimalisatie is uitgeschakeld voor betere prestaties).

public bool OptimizeMemoryUsage { get; set; }
