## SpreadsheetSaveOptions.WorksheetNumber property

Maakt het mogelijk om een bewerkt werkblad in een kopie van een bestaande spreadsheet in te voegen in plaats van een nieuwe spreadsheet met één werkblad te maken (standaardgedrag). WorksheetNumber is een op 1 gebaseerd nummer van een werkblad in de spreadsheet, geladen in de klasse Editor. Als het 0 is (standaardwaarde), wordt het nieuwe werkblad gemaakt met een enkel bewerkt werkblad. Als het groter of kleiner is dan nul, en er is een geldig werkblad, geladen in de klasse Editor, het bewerkte werkblad, dat wordt weergegeven door input EditableDocument-instantie, wordt ingevoegd in deze spreadsheet.

```csharp
public int WorksheetNumber { get; set; }
```

### Opmerkingen

WerkbladNummerinteger-eigenschap, als deze niet de standaardstatus heeft (gereserveerde waarde '0'), vertegenwoordigt een werkbladnummer, dus het begint bij 1, niet bij nul, en de maximale waarde is het aantal bestaande dia's in een presentatie. Als de opgegeven waarde echter groter is dan het aantal van alle dia's, zal GroupDocs.Editor deze aanpassen om het laatste werkblad te markeren. Negatieve waarden zijn ook toegestaan en tel werkbladen vanaf het einde. "-1" betekent bijvoorbeeld het laatste werkblad in een spreadsheet, "-2" - de voorlaatste, enz. Net als bij positieve waarden, wordt het, wanneer het negatieve werkbladnummer het totale aantal werkbladen in de gegeven spreadsheet overschrijdt, aangepast naar het eerste werkblad. De[`InsertAsNewWorksheet`](../insertasnewworksheet) booleaanse eigenschap is nauw verbonden met deze.

### Voorbeelden

Gegeven spreadsheet heeft 5 werkbladen: WorksheetNumber = 0; — negeer gegeven spreadsheet, maak een nieuwe spreadsheet en plaats het bewerkte werkblad erin. WorksheetNumber = 1; — vervang het eerste werkblad door edit WorksheetNumber = 2; — vervang het tweede werkblad door edit WorksheetNumber = 5; — vervang het laatste (5e) werkblad door edit WorksheetNumber = 6; — vervang het laatste (5e) werkblad door bewerkt, omdat 6 groter is dan 5 en dus aangepast WerkbladNummer = -1; - vervang het laatste (5e) werkblad door bewerkt, omdat "-1" betekent "laatst bestaande" WorksheetNumber = -2; — vervang het 4e werkblad door edit WorksheetNumber = -3; — vervang het 3e werkblad door edit WorksheetNumber = -4; — vervang het 2e werkblad door edit WorksheetNumber = -5; — vervang het eerste werkblad door edit WorksheetNumber = -6; — vervang het eerste werkblad door bewerkt, omdat "-6" groter is dan 5 en dus aangepast
