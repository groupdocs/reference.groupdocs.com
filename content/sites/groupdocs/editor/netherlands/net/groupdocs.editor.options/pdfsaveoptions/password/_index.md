## PdfSaveOptions.Password property

Wachtwoord, dat wordt toegepast op het gegenereerde PDF-document als gebruikerswachtwoord, vereist voor openen. Indien NULL of leeg, wordt geen wachtwoord toegepast op het document. Anders wordt het document gecodeerd met RC4 (sleutellengte van 128 bit). Standaard is NULL — wachtwoord wordt niet toegepast.
