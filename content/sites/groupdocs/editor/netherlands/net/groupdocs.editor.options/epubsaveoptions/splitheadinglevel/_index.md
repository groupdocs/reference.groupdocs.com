## EpubSaveOptions.SplitHeadingLevel property

Specificeert het maximale niveau van koppen waarop het ePub-bestand moet worden gesplitst. Standaardwaarde is`2` . Instellen op`0` schakelt splitsen uit, zodat alle inhoud van het e-book wordt opgenomen in een enkel pakket in de ePub.

```csharp
public int SplitHeadingLevel { get; set; }
```

### Opmerkingen

Wanneer deze eigenschap is ingesteld op een waarde van 1 tot 9, wordt het document gesplitst in alinea's die zijn opgemaakt met **Koptekst 1** ,**Kop 2** ,**Kop 3** etc. stijlen tot het gespecificeerde kopniveau.

Standaard alleen**Koptekst 1** En**Kop 2** alinea's zorgen ervoor dat het document wordt gesplitst. Als u deze eigenschap op nul zet, wordt het document helemaal niet gesplitst bij kopparagrafen.
