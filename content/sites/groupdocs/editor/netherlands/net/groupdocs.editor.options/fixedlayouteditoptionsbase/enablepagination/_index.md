## FixedLayoutEditOptionsBase.EnablePagination property

Maakt het mogelijk om paginering (true) of deactiveren (false) in het resulterende HTML-document in te schakelen. Standaard is uitgeschakeld (false).

```csharp
public bool EnablePagination { get; set; }
```

### Opmerkingen

Documenten met een vaste lay-out (met name PDF en XPS) zijn in wezen strikt gepagineerd, hun inhoud heeft een vaste lay-out en is verdeeld in pagina's. Maar resulterende bewerkbare HTML kan worden weergegeven in paginaloze of paginalistische weergave.
