## WordProcessingSaveOptions.OptimizeMemoryUsage property

Schakelt mechanismen voor geheugenoptimalisatie in tijdens het genereren van documenten vanuit HTML, wat de prestaties verslechtert als kosten van afnemend geheugengebruik. Als u deze optie instelt op true, kan het geheugenverbruik aanzienlijk worden verminderd terwijl grote documenten worden gegenereerd, wat ten koste gaat van een langzamere opslagtijd. Standaard is false (geheugenoptimalisatie is uitgeschakeld omwille van betere prestaties).

public bool OptimizeMemoryUsage { get; set; }
