## MarkdownSaveOptions.OptimizeMemoryUsage property

HTML'den belge oluşturma sırasında bellek optimizasyon mekanizmalarını etkinleştirir ve bu, bellek kullanımını azaltmanın bir maliyeti olarak performansı düşürür. Bu seçeneği şu şekilde ayarlamak`doğru`daha yavaş kaydetme süresi pahasına büyük belgeler oluştururken bellek tüketimini önemli ölçüde azaltabilir. Varsayılan değer:`YANLIŞ` (daha iyi performans için bellek optimizasyonu devre dışı bırakıldı).

```csharp
public bool OptimizeMemoryUsage { get; set; }
```
