## PresentationLoadOptions.Password property

Kodlanmışsa Sunum belgesini açmak için kullanılacak parolanın belirlenmesine, değiştirilmesine ve alınmasına izin verir. Parolayı kaldırmak için NULL veya boş dize olarak ayarlayın.

```csharp
public string Password { get; set; }
```

### Notlar

Varsayılan olarak bu özelliğin NULL değeri vardır — parola ayarlanmamıştır. Giriş Sunum belgesi şifre korumalıysa, şifre zorunludur ve şifre belirtilmezse veya geçersizse bir istisna atılır. Giriş Sunum belgesi parola korumalı DEĞİLSE, ancak parola ayarlanmışsa, dikkate alınmaz.
