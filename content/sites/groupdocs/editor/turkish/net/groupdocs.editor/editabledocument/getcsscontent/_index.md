## GetCssContent(string, string) {#getcsscontent_1}

Tüm harici stil sayfalarının içeriğini, bir dizenin bir stil sayfasını temsil ettiği bir dizi listesi olarak döndürür. Elde edilen her stil sayfasında harici kaynağa giden her bağlantıya belirtilen önek uygulanır. Bunun için CSS yoksa boş liste döndürür belge.

```csharp
public List<string> GetCssContent(string externalImagesPrefix, string externalFontsPrefix)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| externalImagesPrefix | String | Kullanılan bu parametre aracılığıyla, sonuçta ortaya çıkan CSS dizelerinde CSS bildirimlerinde bulunacak olan tüm harici görüntülere links 'ye eklenecek bir önek belirtilebilir. NULL veya boş ise önekler eklenmez. |
| externalFontsPrefix | String | Kullanılan bu parametre aracılığıyla, elde edilen CSS dizelerindeki @font-face at-kurallarındaki tüm harici yazı tiplerine links 'ye eklenecek bir önek belirtilebilir. NULL veya boş ise önekler eklenmez. |
