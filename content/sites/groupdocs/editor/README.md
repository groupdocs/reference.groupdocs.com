# GroupDocs.Editor — API Reference Source Files

This repository holds the Markdown source files used to generate the **GroupDocs.Editor**
API reference, published at <https://reference.groupdocs.com/editor/>.

## About GroupDocs.Editor

[GroupDocs.Editor](https://products.groupdocs.com/editor/) is a developer API for document
processing: load, edit, and save documents through an HTML round-trip. It is available for the following platforms, each with its own
API reference:

- **.NET** — <https://reference.groupdocs.com/editor/net/>
- **Java** — <https://reference.groupdocs.com/editor/java/>
- **Node.js via Java** — <https://reference.groupdocs.com/editor/nodejs-java/>
- **Python via .NET** — <https://reference.groupdocs.com/editor/python-net/>

## Repository purpose

The files in this repository are used to:

- Build and maintain the official **GroupDocs.Editor** API reference.
- Keep the reference in sync with the latest product release.
- Provide a consistent structure for automated Hugo builds and publication.

Content is English-only and compiled into a static site with [Hugo](https://gohugo.io/).
