## FixedLayoutEditOptionsBase.EnablePagination property

Ermöglicht das Aktivieren (true) oder Deaktivieren (false) der Paginierung im resultierenden HTML-Dokument. Standardmäßig ist deaktiviert (false).

```csharp
public bool EnablePagination { get; set; }
```

### Bemerkungen

Dokumente im Fixed-Layout-Format (insbesondere PDF und XPS) sind im Wesentlichen streng paginiert, ihr Inhalt hat ein festes Layout und ist auf Seiten aufgeteilt. Das resultierende bearbeitbare HTML kann jedoch entweder in einer seitenlosen oder paginalen Ansicht dargestellt werden.
