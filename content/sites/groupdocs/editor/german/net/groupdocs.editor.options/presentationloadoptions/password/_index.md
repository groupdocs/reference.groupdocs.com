## PresentationLoadOptions.Password property

Ermöglicht das Festlegen, Ändern und Abrufen des Passworts, das zum Öffnen des Präsentationsdokuments verwendet wird, falls es verschlüsselt ist. Auf NULL oder eine leere Zeichenfolge setzen, um das Passwort zu entfernen.

```csharp
public string Password { get; set; }
```

### Bemerkungen

Standardmäßig hat diese Eigenschaft einen NULL-Wert – das Passwort ist nicht festgelegt. Wenn das eingegebene Präsentationsdokument passwortgeschützt ist, ist das Passwort obligatorisch und es wird eine Ausnahme ausgelöst, wenn das Passwort nicht angegeben oder ungültig ist. Wenn das eingegebene Präsentationsdokument NICHT passwortgeschützt ist, aber ein Passwort gesetzt ist, wird es ignoriert.
