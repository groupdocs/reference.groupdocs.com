## MarkdownSaveOptions.ExportImagesAsBase64 property

Gibt an, ob Bilder im Base64-Format in der Ausgabedatei gespeichert werden. Standard ist`FALSCH` .

```csharp
public bool ExportImagesAsBase64 { get; set; }
```

### Bemerkungen

Wenn diese Eigenschaft auf eingestellt ist`WAHR` dann werden Bilddaten direkt in die Bildelemente exportiert ![]() und es werden keine separaten Dateien erstellt. Diese Eigenschaft, wenn festgelegt auf`WAHR` , hat die höhere Priorität als die[`ImagesFolder`](../imagesfolder) Eigentum.
