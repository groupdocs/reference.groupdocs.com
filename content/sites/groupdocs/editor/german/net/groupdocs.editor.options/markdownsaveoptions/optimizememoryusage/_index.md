## MarkdownSaveOptions.OptimizeMemoryUsage property

Aktiviert Speicheroptimierungsmechanismen während der Dokumentgenerierung aus HTML, was die Leistung auf Kosten der verringerten Speichernutzung beeinträchtigt. Setzen Sie diese Option auf`WAHR`kann den Speicherverbrauch erheblich verringern, während große Dokumente auf Kosten einer langsameren Speicherzeit erstellt werden. Standard ist`FALSCH` (Speicheroptimierung ist zwecks besserer Leistung deaktiviert).

```csharp
public bool OptimizeMemoryUsage { get; set; }
```
