## PresentationSaveOptions.Password property

Ermöglicht das Festlegen, Ändern und Abrufen des Passworts, das zum Kodieren des resultierenden Präsentationsdokuments verwendet wird. Standardmäßig ist NULL - das Passwort wird nicht festgelegt. Auf NULL oder eine leere Zeichenfolge setzen, um das Passwort zu entfernen, falls es zuvor festgelegt wurde.
