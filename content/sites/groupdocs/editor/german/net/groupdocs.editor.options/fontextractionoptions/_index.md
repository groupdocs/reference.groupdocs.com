## FontExtractionOptions enumeration

Optionen zum Extrahieren von Schriftarten steuern, welche Schriftarten extrahiert werden sollen und woher

```csharp
public enum FontExtractionOptions : byte
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NotExtract | `0` | Extrahiert keine Font-Ressourcen, weder aus dem Dokument noch aus dem System. Standardwert. |
| ExtractAllEmbedded | `1` | Extrahiert alle Font-Ressourcen, die in das eingegebene Word-Dokument eingebettet sind, unabhängig davon, was sie sind: Benutzerdefiniert oder System. |
| ExtractEmbeddedWithoutSystem | `2` | Extrahiert nur die eingebetteten Schriftartressourcen, die benutzerdefiniert sind (nicht System) |
| ExtractAll | `3` | Versucht, alle Schriftarten zu extrahieren, die im eingegebenen WordProcessing-Dokument verwendet werden, einschließlich Systemschriftarten. |
