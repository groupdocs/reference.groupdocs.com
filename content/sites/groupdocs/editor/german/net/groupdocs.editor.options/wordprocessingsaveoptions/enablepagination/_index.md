## WordProcessingSaveOptions.EnablePagination property

Ermöglicht das Aktivieren oder Deaktivieren der Paginierung, die zum Speichern des WordProcessing-Dokuments verwendet wird. Wenn das Originaldokument im Paginierungsmodus geöffnet und bearbeitet wurde, sollte diese Option ebenfalls aktiviert werden. Standardmäßig ist deaktiviert.

public bool EnablePagination { get; set; }
