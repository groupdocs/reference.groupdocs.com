## WordProcessingSaveOptions.LocaleBi property

Ermöglicht das Überschreiben des Gebietsschemas (Sprache) für das WordProcessing-Dokument für den RTL-Text (von rechts nach links), der während seiner Erstellung angewendet wird. Wenn nicht angegeben (Standardwert), MS Word (oder ein anderes Programm) erkennt (oder wählt) das Dokument RTL locale gemäß seinen eigenen Einstellungen oder anderen Faktoren.

```csharp
public CultureInfo LocaleBi { get; set; }
```

### Bemerkungen

Diese Option wendet das angegebene Gebietsschema zwangsweise auf den gesamten RTL-Text im Dokument an. Verwenden Sie es nicht, wenn das Dokument verschiedene Textteile enthält, die in verschiedenen Sprachen geschrieben sind.
