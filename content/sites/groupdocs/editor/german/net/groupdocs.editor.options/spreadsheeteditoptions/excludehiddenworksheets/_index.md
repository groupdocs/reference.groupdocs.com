## SpreadsheetEditOptions.ExcludeHiddenWorksheets property

Ermöglicht das Ausschließen versteckter Arbeitsblätter im Eingabe-Tabellendokument, sodass sie vollständig ignoriert werden. Standard ist „false“ – ausgeblendete Arbeitsblätter sind verfügbar und werden normal verarbeitet.

```csharp
public bool ExcludeHiddenWorksheets { get; set; }
```

### Bemerkungen

Mehrere binäre Tabellenkalkulationsformate (wie XLSX) unterstützen das Konzept der versteckten Arbeitsblätter (Tabs). Dokumente dieses Formats können, wenn sie mehr als ein Arbeitsblatt haben, zusätzliche versteckte Arbeitsblätter enthalten. Standardmäßig sind solche versteckten Arbeitsblätter für die Verarbeitung verfügbar, aber mit dieser Option können sie ignoriert werden, wie diese versteckten Arbeitsblätter fehlen und nicht existieren. Wenn diese Option aktiviert ist, können Sie kein ausgeblendetes Arbeitsblatt mit dem auswählen.[`WorksheetIndex`](../worksheetindex) Eigentum.
