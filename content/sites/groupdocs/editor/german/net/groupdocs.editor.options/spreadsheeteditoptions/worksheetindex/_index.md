## SpreadsheetEditOptions.WorksheetIndex property

Ermöglicht die Angabe des 0-basierten Index des Arbeitsblatts (Registerkarte) des eingegebenen Tabellenkalkulationsdokuments, das in HTML konvertiert werden soll (siehe Anmerkungen).

```csharp
public int WorksheetIndex { get; set; }
```

### Bemerkungen

Die meisten Spreadsheet-Dokumente unterstützen ein Konzept von Registerkarten, dh sie können mehrere Registerkarten enthalten. Andererseits unterstützt das HTML-Format eine solche Struktur nicht. Aus diesem Grund kann GroupDocs.Editor nur einen bestimmten Tab des Eingabedokuments in HTML konvertieren, und diese Option ermöglicht es, ihn anzugeben. Der Tab-Index basiert auf 0 , negative Werte sind verboten. Wenn der angegebene Index die Anzahl aller Tabulatoren überschreitet, wird die Ausnahme ausgelöst. Wenn das eingegebene Tabellenkalkulationsdokument nur eine Tabulatortaste enthält, wird diese Option ignoriert. Der Standardwert ist 0 (erste Registerkarte).
