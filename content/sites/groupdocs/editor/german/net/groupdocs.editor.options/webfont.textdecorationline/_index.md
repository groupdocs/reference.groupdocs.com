## WebFont.TextDecorationLine enumeration

Arten von Dekorationslinien, die für Text in einem Element verwendet werden, z. B. eine Unterstreichung, Überstreichung oder Durchstreichung. Kann zusammen kombiniert werden, oder niemand.

```csharp
[Flags]
public enum TextDecorationLine : byte
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Es wird keine Textdekorationslinie angewendet |
| Underline | `1` | Unterstreichen – jede Textzeile hat eine Zierlinie darunter |
| Overline | `2` | Überstrichen – jede Textzeile hat eine Zierlinie darüber |
| LineThrough | `4` | Durchgestrichen – jede Textzeile hat eine dekorative Linie, die durch ihre Mitte geht |
