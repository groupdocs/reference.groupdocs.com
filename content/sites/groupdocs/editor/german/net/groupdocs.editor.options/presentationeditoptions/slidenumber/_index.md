## PresentationEditOptions.SlideNumber property

Ermöglicht die Angabe der Foliennummern, die zur Bearbeitung geöffnet werden sollen

```csharp
public int SlideNumber { get; set; }
```

### Bemerkungen

Die Foliennummer ist ein nullbasierter Index einer Folie, der es ermöglicht, eine bestimmte Folie aus einer Präsentation zum Bearbeiten anzugeben und auszuwählen. Wenn sie kleiner als 0 ist, wird die erste Folie ausgewählt (wie bei SlideNumber = 0). Wenn die Anzahl aller Folien in der Präsentation größer ist, wird die letzte Folie ausgewählt. Wenn die Eingabepräsentation nur eine einzelne Folie enthält, wird diese Option ignoriert, und diese einzelne Folie wird bearbeitet. Wenn versucht wird, eine ausgeblendete Folie zum Bearbeiten zu öffnen, während[`ShowHiddenSlides`](../showhiddenslides) Option auf „false“ gesetzt ist, wird die Ausnahme ausgelöst.
