## Azw3SaveOptions.SplitHeadingLevel property

Gibt an, ob der Inhalt des AZW3-E-Books auf Pakete aufgeteilt werden soll, und wenn ja, die maximale Überschriftenebene, bei der der Inhalt des AZW3 aufgeteilt werden soll. Standardwert ist`2` Einstellung auf`0` wird die Aufteilung deaktivieren, sodass alle Inhalte des E-Books in einem einzigen Paket innerhalb des AZW3 integriert werden.

```csharp
public int SplitHeadingLevel { get; set; }
```

### Bemerkungen

In einigen Fällen ist es vorzuziehen, den Inhalt des E-Books in mehrere kleinere Pakete aufzuteilen, die sich in der AZW3-Ausgabedatei befinden. Diese Eigenschaft steuert, ob eine solche Trennung durchgeführt werden soll und wenn ja, dann wie.

Wenn diese Eigenschaft auf einen Wert zwischen 1 und 9 gesetzt ist, wird das Dokument an Absätzen geteilt, die mit formatiert sind.**Überschrift 1** ,**Überschrift 2** ,**Überschrift 3** usw. Stile bis zur angegebenen Überschriftenebene.

Standardmäßig (Wert von`2` ), nur**Überschrift 1** und**Überschrift 2** Absätze führen dazu, dass das Dokument geteilt wird. Wenn Sie diese Eigenschaft auf Null setzen, wird das Dokument überhaupt nicht an den Überschriftenabsätzen geteilt.
