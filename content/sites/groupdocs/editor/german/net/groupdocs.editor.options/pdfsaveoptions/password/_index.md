## PdfSaveOptions.Password property

Passwort, das als Benutzerpasswort auf das generierte PDF-Dokument angewendet wird, zum Öffnen erforderlich. Wenn NULL oder leer, wird kein Passwort auf das Dokument angewendet. Andernfalls wird das Dokument mit RC4 (Schlüssellänge von 128 Bit) verschlüsselt. Standardmäßig ist NULL – das Passwort wird nicht angewendet.
