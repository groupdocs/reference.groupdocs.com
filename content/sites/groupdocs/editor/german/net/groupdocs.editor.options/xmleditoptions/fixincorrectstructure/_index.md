## XmlEditOptions.FixIncorrectStructure property

Ermöglicht das Aktivieren oder Deaktivieren des Mechanismus zum Reparieren beschädigter XML-Strukturen. Standardmäßig ist deaktiviert (false).

```csharp
public bool FixIncorrectStructure { get; set; }
```

### Bemerkungen

Standardmäßig werden nur gültige, wohlgeformte XML-Dokumente akzeptiert. Wenn diese Option aktiviert ist, versucht GroupDocs.Editor, wenn möglich, beschädigte XML-Strukturen zu reparieren.
