## WordProcessingEditOptions.ExtractOnlyUsedFont property

Ruft einen Wert ab oder legt einen Wert fest, der angibt, ob nur Schriftartressourcen extrahiert werden sollen, die im Textinhalt des Dokuments verwendet werden.

```csharp
public bool ExtractOnlyUsedFont { get; set; }
```

### Eigentumswert

`Stimmt` wenn es erforderlich ist, nur die Font-Ressourcen zu extrahieren, die im Textinhalt des Dokuments verwendet werden; Andernfalls,`FALSCH` . Standardwert ist`FALSCH` .

### Bemerkungen

Nicht alle Schriftarten, die im WordProcessing-Dokument verwendet werden, werden zu 100 % direkt verwendet (auf einen Teil des Textes angewendet). Es kann eine Situation geben, in der die Schriftart im Dokument referenziert und sogar eingebettet ist, aber auf keinen Text angewendet wird. Beispielsweise kann eine Schriftart mit einem Stil verbunden sein, aber dieser Stil wird auf keinen Teil des Textes angewendet. Diese Option steuert, wie solche Fälle verarbeitet werden.
