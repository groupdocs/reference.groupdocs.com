## FontSize.IsRelativeSize property

Gibt an, ob diese Schriftgröße mit einer relativen Größe als Schlüsselwort definiert ist. Die Schriftart wird relativ zur Schriftgröße des übergeordneten Elements größer oder kleiner, ungefähr um das Verhältnis, das verwendet wird, um die Schlüsselwörter mit absoluter Größe zu trennen.
