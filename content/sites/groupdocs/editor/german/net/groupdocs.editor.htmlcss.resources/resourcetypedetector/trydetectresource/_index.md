## ResourceTypeDetector.TryDetectResource method

Versucht, einen Eingabestrom zu analysieren und daraus eine der unterstützten HTML-Ressourcen zu erstellen, wobei ein angegebener angenommener Typ berücksichtigt wird, falls dieser nicht null ist

```csharp
public static IHtmlResource TryDetectResource(Stream inputResourceStream, string name, 
    IResourceType assumptiveFormat)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| inputResourceStream | Stream | Eingabestrom, der vermutlich eine HTML-Ressource enthält. Wenn ungültig, wird eine Ausnahme ausgelöst. |
| name | String | Ressourcenname, der bei Erfolg für die erstellte und zurückgegebene Ressource verwendet wird. Darf nicht NULL, leer oder Leerzeichen sein |
| assumptiveFormat | IResourceType | Angenommenes Format der Eingabe-HTML-Ressource, das nützlich ist, um die beste Leistung zu erzielen. Wenn völlig unbekannt, verwenden Sie den NULL-Wert. Kann falsch sein, dies wird nur die Leistung verschlechtern. |

### Rückgabewert

Instanz, die die Schnittstelle „IHtmlResource“ implementiert und bei Erfolg eine der unterstützbaren HTML-Ressourcen darstellt, oder NULL bei Fehler
