## InvalidFormatException class

Die Ausnahme, die ausgelöst wird, wenn der Benutzer versucht, ein Dokument mit formatspezifischen Optionen zu öffnen, die mit dem ursprünglichen Dokumentformat nicht kompatibel sind.

```csharp
public sealed class InvalidFormatException : Exception
```

### Bemerkungen

Diese Ausnahme wird beispielsweise ausgelöst, wenn Sie versuchen, ein Tabellenkalkulationsdokument mit WordProcessing-Dokumentoptionen zu öffnen.
