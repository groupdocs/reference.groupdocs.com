```csharp
public void Dispose()
```

### Bemerkungen

Nachdem diese Methode aufgerufen wurde, wird beim Aufrufen aller anderen Methoden dieser Instanz eine ausgelöstObjectDisposedException. Es ist sicher, diese Methode mehrmals aufzurufen – alle nachfolgenden Aufrufe werden ignoriert.
