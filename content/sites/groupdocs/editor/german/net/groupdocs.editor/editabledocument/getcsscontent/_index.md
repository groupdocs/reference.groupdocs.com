## GetCssContent(string, string) {#getcsscontent_1}

Gibt den Inhalt aller externen Stylesheets als Liste von Strings zurück, wobei ein String ein Stylesheet darstellt. Das angegebene Präfix wird auf jeden Link auf die externe Ressource in jedem resultierenden Stylesheet angewendet. Gibt eine leere Liste zurück, wenn es kein CSS dafür gibt Dokument.

```csharp
public List<string> GetCssContent(string externalImagesPrefix, string externalFontsPrefix)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| externalImagesPrefix | String | Durch diesen verwendeten Parameter kann ein Präfix angegeben werden, das den links zu allen externen Bildern hinzugefügt wird, die in CSS-Deklarationen in den resultierenden CSS-Strings vorhanden sein werden. Wenn NULL oder leer, werden keine Präfixe hinzugefügt. |
| externalFontsPrefix | String | Über diesen Parameter kann used ein Präfix angeben, das allen externen Fonts in den @font-face at-Regeln in den resultierenden CSS-Strings links hinzugefügt wird. Wenn NULL oder leer, werden keine Präfixe hinzugefügt. |

### Rückgabewert

Eine Liste von Strings, wobei jeder String den Inhalt eines CSS-Dokuments enthält
