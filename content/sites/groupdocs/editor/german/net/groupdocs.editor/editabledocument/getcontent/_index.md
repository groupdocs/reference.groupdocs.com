## GetContent(string, string) {#getcontent_1}

Gibt den gesamten Inhalt des HTML-Dokuments als Zeichenfolge zurück, wobei Links zu externen Ressourcen das angegebene Präfix enthalten.

```csharp
public string GetContent(string externalImagesPrefix, string externalCssPrefix)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| externalImagesPrefix | String | Durch diesen verwendeten Parameter kann ein Präfix angegeben werden, das den links zu allen externen Bildern in IMG-Elementen hinzugefügt wird, die im resultierenden HTML-String vorhanden sein werden. Wenn NULL oder leer, werden keine Präfixe hinzugefügt. |
| externalCssPrefix | String | Durch diesen verwendeten Parameter kann ein Präfix angegeben werden, das den links zu allen externen Stylesheets in LINK-Elementen hinzugefügt wird, die im resultierenden HTML-String vorhanden sein werden. Wenn NULL oder leer, werden keine Präfixe hinzugefügt. |

### Rückgabewert

String, der den Inhalt des HTML-Dokuments mit Links enthält, angepasst an die externen Ressourcen
