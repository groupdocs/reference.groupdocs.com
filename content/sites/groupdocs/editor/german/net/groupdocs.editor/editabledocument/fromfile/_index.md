## EditableDocument.FromFile method

Statische Factory, die eine Instanz von EditableDocument aus einer HTML-Datei erstellt, die durch einen Pfad zur *.html-Datei selbst und einen Ordner mit verknüpften Ressourcen angegeben wird

```csharp
public static EditableDocument FromFile(string htmlFilePath, string resourceFolderPath)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| htmlFilePath | String | Zeichenfolge, die einen vollständigen Pfad zur HTML-Datei enthält. Kann nicht null sein, muss ein gültiger Dateipfad sein und die Datei selbst muss vorhanden sein. |
| resourceFolderPath | String | Optionaler Pfad zum Ordner mit HTML-Ressourcen. Wenn NULL, ungültig oder ein solcher Ordner nicht existiert, versucht der Editor, diesen Ordner selbst zu finden, indem er das HTML-Markup analysiert |

### Rückgabewert

Neue Nicht-Null-Instanz von EditableDocument

### Ausnahmen

| Ausnahme | Bedingung |
| --- | --- |
| ArgumentException | Der HTML-Dateipfad und/oder der Ressourcenordnerpfad ist/sind ungültig |
| FileNotFoundException | Die angegebene HTML-Datei kann nicht gefunden werden |
