## EditableDocument.FromMarkupAndResourceFolder method

Statische Factory, die eine Instanz von EditableDocument aus einem angegebenen HTML-Markup und aus Ressourcen erstellt, die sich in dem Ordner befinden, der durch den vollständigen Pfad angegeben wird

```csharp
public static EditableDocument FromMarkupAndResourceFolder(string newHtmlContent, 
    string resourceFolderPath)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| newHtmlContent | String | Zeichenfolge, die rohes HTML-Markup enthält, das analysiert werden sollte. Darf nicht NULL, leer oder ungültig sein. |
| resourceFolderPath | String | Obligatorischer Pfad zum Ordner mit Ressourcen. Alle Stylesheets, die sich in diesem Ordner befinden, werden verwendet. Kann nicht NULL oder eine leere Zeichenfolge sein, und dieser Ordner sollte vorhanden sein. |

### Rückgabewert

Neue Nicht-Null-Instanz von EditableDocument

### Bemerkungen

Diese statische Factory ist nützlich, wenn der Inhalt eines HTML-Dokuments als Zeichenfolge dargestellt wird, sich jedoch alle Ressourcen in einem Ordner befinden und häufig Links zu diesen Ressourcen im HTML-Markup ungültig sind und fehlen. Beim Aufrufen dieser Methode durchsucht sie den angegebenen Ordner und wendet automatisch alle gefundenen Stylesheets auf das Dokument an. Diese Methode ist sehr nützlich, wenn Sie Inhalte aus verschiedenen HTML-Editoren erhalten, die normalerweise die Dokumentmetadaten usw. abschneiden.
