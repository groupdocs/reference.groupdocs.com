## Edit(IEditOptions) {#edit_1}

Öffnet ein zuvor geladenes Dokument zur Bearbeitung unter Verwendung angegebener formatspezifischer Optionen durch Generieren und Zurückgeben einer Instanz von '[`EditableDocument`](../../editabledocument) -Klasse, die wiederum Methoden zum Erzeugen von HTML-Markup und zugehörigen Ressourcen enthält.

```csharp
public EditableDocument Edit(IEditOptions editOptions)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| editOptions | IEditOptions | Formatspezifische Dokumentoptionen, mit denen der Konvertierungsprozess optimiert werden kann. Kann NULL sein – in diesem Fall erkennt GroupDocs.Editor ein Format eines zuvor geladenen Dokuments und wendet Optionen an, die standardmäßig für dieses Format gelten. Sollte nicht mit zuvor angewendeten Ladeoptionen in Konflikt geraten. |

### Rückgabewert

Instanz des '[`EditableDocument`](../../editabledocument) Klasse, die das gesamte Eingabedokument mit all seinen Ressourcen in einem Zwischenformat kapselt. Wenn diese Methode erfolgreich beendet wurde, gibt sie niemals NULL zurück.

### Bemerkungen

Wenn das eingegebene Originaldokument über den Konstruktor in die 'Editor'-Instanz geladen wird, ermöglicht diese Methode das Öffnen des Dokuments zur Bearbeitung, indem es in ein Zwischenformat konvertiert wird, das in der Instanz der 'EditableDocument'-Klasse gekapselt ist. '[`EditableDocument`](../../editabledocument) das von dieser Methode zurückgegeben wird, enthält alle notwendigen Methoden und Eigenschaften zum Erzeugen von HTML-Markup und entsprechenden Ressourcen (wie Bilder, Schriftarten und Stylesheets) in allen erforderlichen Konfigurationen, um sie anschließend an einen beliebigen WYSIWYG-HTML-Editor weiterzugeben. Diese Überladung erhält Bearbeitungsoptionen, die spezifisch für Familienformate sind. **Erfahren Sie mehr**
