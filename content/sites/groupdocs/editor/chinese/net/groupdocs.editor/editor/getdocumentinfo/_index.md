## Editor.GetDocumentInfo method

返回有关文档的元数据，该元数据已加载到此“编辑器”实例

```csharp
public IDocumentInfo GetDocumentInfo(string password)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| password | String | 用户可以为一个文档指定一个密码，如果这个文档是用密码加密的。 可以是NULL或空字符串，相当于没有密码。对于那些没有密码保护功能的文档格式，该参数将被忽略。如果文档是加密的，并且密码没有在这个参数中指定，但是它之前在创建这个时的加载选项中指定[`Editor`](../../editor)例如，它将被使用。 |

### 返回值

格式特定的继承者[`IDocumentInfo`](../../../groupdocs.editor.metadata/idocumentinfo)接口，表示检测到的带有格式特定元数据的格式，如果文档未被识别为可支持或已损坏，则为 NULL, 。

### 例外

| 例外 | （健康）状况 |
| --- | --- |
| ObjectDisposedException | 在调用“GetDocumentInfo”时已释放 Editor 实例时抛出 |
| [PasswordRequiredException](../../passwordrequiredexception) | 当加载的文档受密码保护时抛出，但参数中未指定密码“*password*" |
| [IncorrectPasswordException](../../incorrectpasswordexception) | 加载的文档受密码保护时抛出，指定了密码，但不正确 |
| InvalidOperationException | 发生未知性质的意外错误时抛出 |

### 评论

当不清楚输入文档是哪种格式、是否受密码保护和/或它包含多少页/工作表/幻灯片时，GetDocumentInfo 方法很有用。根据 GetDocumentInfo 返回的此元数据，可以正确调整主处理管道的加载和编辑选项。

GetDocumentInfo 方法总是返回完整的数据，它不受试用模式的影响，它的使用不会注销消耗的字节或积分。
