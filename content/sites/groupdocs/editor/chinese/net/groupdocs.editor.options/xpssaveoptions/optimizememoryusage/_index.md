## XpsSaveOptions.OptimizeMemoryUsage property

在从 HTML 生成文档期间启用内存优化机制，这会以减少内存使用为代价降低性能。 将此选项设置为 true 可以显着减少内存消耗，同时以较慢的保存时间为代价生成大型文档。 默认值为 false （为了更好的性能禁用了内存优化）。

public bool OptimizeMemoryUsage { get; set; }
