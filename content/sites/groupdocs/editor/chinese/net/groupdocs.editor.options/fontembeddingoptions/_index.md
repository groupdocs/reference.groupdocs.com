## FontEmbeddingOptions enumeration

字体嵌入选项控制应将哪些字体资源嵌入到输出 WordProcessing 或 PDF 文档中

```csharp
public enum FontEmbeddingOptions : byte
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| NotEmbed | `0` | 不要从 EditableDocument 或系统中嵌入任何字体资源。默认值. |
| EmbedAll | `1` | 从输入 EditableDocument 分析文档内容，找到所有使用的字体并将它们嵌入到输出 WordProcessing 或 PDF 文档中。 首先，GroupDocs.Editor 从 EditableDocument 中的字体资源中获取字体。如果它们不足或缺失，则 GroupDocs.Editor 从 OS. 获取字体 |
| EmbedWithoutSystem | `2` | 精确到EmbedAll 但不包括那些被操作系统视为系统字体 的字体 |

### 评论

字体嵌入选项在文档保存期间应用（从中间 EditableDocument 到输出 WordProcessing 或 PDF 格式），此枚举作为属性包含在 WordProcessingSaveOptions 和 PdfSaveOptions 中，应该从那里使用
