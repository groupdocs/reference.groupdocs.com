## PresentationEditOptions.SlideNumber property

允许指定幻灯片编号，应打开以进行编辑

```csharp
public int SlideNumber { get; set; }
```

### 评论

幻灯片编号是从零开始的幻灯片索引，允许从演示文稿中指定和选择一张特定幻灯片进行编辑。 如果小于 0，将选择第一张幻灯片（与 SlideNumber = 0 相同）。如果大于演示文稿中所有幻灯片的数量，将选择最后一张幻灯片。如果输入演示文稿仅包含一张幻灯片，则此选项将被忽略，并且将编辑这张幻灯片。 如果试图打开以编辑隐藏的幻灯片，而[`ShowHiddenSlides`](../showhiddenslides)选项设置为“false”，将抛出异常。
