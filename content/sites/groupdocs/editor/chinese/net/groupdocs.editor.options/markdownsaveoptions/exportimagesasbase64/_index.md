## MarkdownSaveOptions.ExportImagesAsBase64 property

指定图像是否以 Base64 格式保存到输出文件。默认是`错误的`.

```csharp
public bool ExportImagesAsBase64 { get; set; }
```

### 评论

当此属性设置为`真的` 然后图像数据直接导出到 the 图像元素 ![]() 并且不创建单独的文件。此属性，如果设置为`真的` 比[`ImagesFolder`](../imagesfolder)财产。
