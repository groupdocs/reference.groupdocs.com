## WordProcessingEditOptions.InputControlsClassName property

允许指定一个类名，该类名将被放置到每个 HTML 元素中的“类”属性中，表示输入 WordProcessing 文档中的某个字段。默认情况下为 NULL - 不应用“类”属性。

```csharp
public string InputControlsClassName { get; set; }
```

### 评论

几乎所有来自 WordProcessing 格式系列的格式都包含字段——特定的文档实体，允许从用户那里获取输入数据。有各种各样的字段：文本框、复选框、组合框、下拉列表、按钮、日期/时间选择器等。所有这些都被翻译成最合适的 HTML 结构和元素，并保留输入的用户数据，如果它们存在于输入文档中。在特定的用例中，只需要在客户端收集输入的数据，而不是编辑整个文档内容。对于这种情况，需要以某种方式识别输入控件，以便在客户端获取它们及其数据。此属性允许指定一个类名，它将应用于 HTML 标记中的每个输入控件，因此客户端代码将能够遍历 HTML 文档结构并收集数据。
