## WordProcessingEditOptions.ExtractOnlyUsedFont property

获取或设置一个值，该值指示是否仅提取文档文本内容中使用的字体资源。

```csharp
public bool ExtractOnlyUsedFont { get; set; }
```

### 适当的价值

`真的`如果只需要提取文档文本内容中用到的字体资源；否则，`错误的` .默认值为`错误的`.

### 评论

并非所有在 WordProcessing 文档中使用的字体都是 100% 直接使用的（应用于某些文本）。可能有一种情况，当字体在文档中被引用，甚至可能被嵌入，但没有应用于任何一段文本。例如，某些字体可能附加到某些样式，但此样式不适用于文本的任何部分。此选项控制如何处理此类情况。
