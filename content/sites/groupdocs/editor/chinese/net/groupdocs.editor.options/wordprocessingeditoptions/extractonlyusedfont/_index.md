## WordProcessingEditOptions.ExtractOnlyUsedFont property

获取或设置一个值，该值指示是否仅提取文档文本内容中使用的字体资源。

```csharp
public bool ExtractOnlyUsedFont { get; set; }
```

### 适当的价值

`真的`如果只需要提取那些在文档的文本内容中使用的字体资源；否则，`错误的` .默认值为`错误的`.

### 评论

并非 WordProcessing 文档中使用的所有字体都是 100% 直接使用的（应用于某些文本）。可能存在一种情况，当文档中引用了字体，甚至可能嵌入了字体，但并未应用于任何文本片段。例如，某些字体可能附加到某种样式，但这种样式不适用于文本的任何部分。此选项控制如何处理此类情况。
