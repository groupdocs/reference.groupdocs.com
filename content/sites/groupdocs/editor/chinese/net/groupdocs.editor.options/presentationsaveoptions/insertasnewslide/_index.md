## PresentationSaveOptions.InsertAsNewSlide property

布尔标志，它指定编辑的幻灯片是否应该替换现有幻灯片在原始演示文稿中的位置，由[`SlideNumber`](../slidenumber)属性，或者应该在现有幻灯片和上一张幻灯片之间注入，而不替换其内容。 默认为 false - 现有幻灯片将被替换。如果值为[`SlideNumber`](../slidenumber)属性设置为“0”。

```csharp
public bool InsertAsNewSlide { get; set; }
```

### 评论

默认情况下，幻灯片被替换。这意味着如果给定的演示文稿有 5 张幻灯片，并且[`SlideNumber`](../slidenumber)=4，则第 4 张幻灯片将被新编辑的幻灯片替换，而演示文稿 (5) 中的幻灯片总数将保持不变。 但是，如果此属性的值设置为真的新编辑的幻灯片将作为第 4 张幻灯片注入，所有后续幻灯片将移至末尾：“旧”第 4 张幻灯片变为第 5，第 5 变为第 6，演示中的幻灯片总数将增加 1 和等于 6。
