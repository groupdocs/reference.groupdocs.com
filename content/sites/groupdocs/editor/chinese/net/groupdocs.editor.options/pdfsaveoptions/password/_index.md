## PdfSaveOptions.Password property

密码，将作为用户密码应用于生成的 PDF 文档，打开时需要。 如果为 NULL 或为空，则不会将密码应用于文档。否则，文档将使用 RC4（密钥长度为 128 位）加密。 默认为 NULL — 不应用密码。
