## Azw3SaveOptions.SplitHeadingLevel property

指定是否将 AZW3 电子书的内容拆分到包中，如果是，则指定拆分 AZW3 内容的最大标题级别。默认值为`2个` 将其设置为`0`将禁用拆分，因此电子书的所有内容将合并到 AZW3. 内的单个包中

```csharp
public int SplitHeadingLevel { get; set; }
```

### 评论

在某些情况下，最好将电子书的内容分成几个较小的包，位于输出 AZW3 文件中。此属性控制是否应执行此类分离，如果是，则如何执行。

当此属性设置为 1 到 9 之间的值时，文档将按格式为 using 的段落拆分**标题 1**,**标题 2** ,**标题 3**等样式到指定的标题级别。

默认情况下（值`2个`）， 仅有的**标题 1**和**标题 2**段落导致文档被拆分。将此属性设置为零将导致文档根本不会在标题段落处拆分。
