## SpreadsheetEditOptions.WorksheetIndex property

允许指定输入电子表格文档的工作表（选项卡）的从 0 开始的索引，应将其转换为 HTML（见备注）。

```csharp
public int WorksheetIndex { get; set; }
```

### 评论

大多数电子表格文档都支持选项卡的概念，即它们可以是多选项卡。另一方面，HTML 格式不支持这种结构。 因为这个 GroupDocs.Editor 只能将输入文档的一个特定选项卡转换为 HTML，并且此选项允许指定它。 选项卡索引是从 0 开始的, 禁止负值。如果指定的索引超过所有选项卡的数量，则会抛出异常。 如果输入的电子表格文档仅包含一个选项卡，则该选项将被忽略。默认值为 0（第一个选项卡）。
