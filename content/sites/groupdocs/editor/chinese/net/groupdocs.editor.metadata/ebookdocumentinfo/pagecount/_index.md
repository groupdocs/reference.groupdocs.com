## EbookDocumentInfo.PageCount property

在 MOBI 或 AZW3 的情况下返回页数或在 ePub 的情况下返回章节数。

```csharp
public int PageCount { get; }
```

### 评论

电子书文档通常没有固定页数，因此页数不固定。在 ePub 的情况下，可以计算章节数。但是，MOBI 和 AZW3 格式也没有章节，所以这个数字是根据纵向设置为 A4 的标准页面大小计算的。
