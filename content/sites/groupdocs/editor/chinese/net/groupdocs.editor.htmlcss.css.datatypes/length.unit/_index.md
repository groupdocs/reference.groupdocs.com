## Length.Unit enumeration

所有支持的长度单位

```csharp
public enum Unit : byte
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Unitless | `0` | 无单位 - 没有定义的长度单位。默认值。 |
| Px | `1` | 像素。相对于观看设备。对于屏幕显示，通常是显示器的一个设备像素（点）。 |
| Em | `2` | Em。这个单位代表计算出来的元素的字体大小。 |
| Ex | `3` | Ex（x 长度）。该单位表示元素字体的 x 高度。 在带有“x”字母的字体上，这通常是字体中小写字母的高度；许多字体中的 1ex ≈ 0.5em。 |
| Cm | `4` | 厘米。一厘米（10毫米）. |
| Mm | `5` | 嗯。一毫米。 |
| In | `6` | 在。一英寸（2.54 厘米）. |
| Pt | `7` | 铂。一点是 1/72 英寸或 0.353 毫米。 |
| Pc | `8` | 个人计算机。一个异食癖（12 分）. |
| Ch | `9` | 频道。该单位表示元素字体中字形“0” （零，Unicode 字符 U+0030）的宽度，或更准确地说是提前量度。 |
| Rem | `10` | 雷姆。这个单位表示根元素的字体大小（例如 &lt;html&gt; 元素的字体大小）。 当用于这个根元素的字体大小时，它代表它的初始值。 |
| Vw | `11` | Vw - 视口宽度。视口宽度的 1/100。 |
| Vh | `12` | Vh - 视口高度。视口高度的 1/100。 |
| Vmin | `13` | Vmin。视口高度和宽度之间最小值的 1/100。 |
| Vmax | `14` | Vmax。视口高度和宽度之间最大值的 1/100。 |
| Percent | `15` | 该值是相对于固定（外部）值的，即依赖于 context 。 1% = 外部值的 1/100。 |
