## Length.Unit enumeration

所有支持的长度单位

```csharp
public enum Unit : byte
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Unitless | `0` | Unitless - 没有定义的长度单位。默认值. |
| Px | `1` | 像素。相对于观看设备。对于屏幕显示，通常是显示器的一个设备像素（点）。 |
| Em | `2` | Em。这个单位代表计算出的元素的font-size. |
| Ex | `3` | Ex（x 长度）。该单位表示元素字体的 x 高度。 在带有 'x' 字母的字体上，这通常是字体中小写字母的高度；在许多字体中 1ex ≈ 0.5em. |
| Cm | `4` | 厘米。一厘米（10 毫米）. |
| Mm | `5` | 嗯。一毫米. |
| In | `6` | 在。一英寸（2.54 厘米）. |
| Pt | `7` | 铂。一点是 1/72 英寸或 0.353 毫米. |
| Pc | `8` | 个人计算机。一只异食癖（12 分）. |
| Ch | `9` | 通道。该单位表示元素字体中字形“0” （零，Unicode 字符 U+0030）的宽度，或者更准确地说是超前度量。 |
| Rem | `10` | 雷姆。该单位表示根元素的字体大小（例如 &lt;html&gt; 元素的字体大小）。 用在这个根元素上的font-size上时，代表它的初始值。 |
| Vw | `11` | Vw - 视口宽度。视口宽度的 1/100. |
| Vh | `12` | Vh - 视口高度。视口高度的 1/100. |
| Vmin | `13` | Vmin。视口高度和宽度之间最小值的 1/100. |
| Vmax | `14` | Vmax。视口高度和宽度之间最大值的 1/100. |
| Percent | `15` | 该值相对于固定（外部）值，即依赖于 context 。 1% = 外部值的 1/100. |
