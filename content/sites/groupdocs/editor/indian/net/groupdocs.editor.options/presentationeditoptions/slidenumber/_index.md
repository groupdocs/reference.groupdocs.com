## PresentationEditOptions.SlideNumber property

स्लाइड संख्या निर्दिष्ट करने की अनुमति देता है, जिसे संपादन के लिए खोला जाना चाहिए

```csharp
public int SlideNumber { get; set; }
```

### टिप्पणियों

स्लाइड नंबर एक स्लाइड का एक शून्य-आधारित इंडेक्स है, जो एक प्रस्तुति से संपादित करने के लिए एक विशेष स्लाइड को निर्दिष्ट करने और चुनने की अनुमति देता है। यदि 0 से कम है, तो पहली स्लाइड का चयन किया जाएगा (स्लाइडनंबर = 0 के समान)। यदि प्रस्तुतिकरण में सभी स्लाइडों की मात्रा से अधिक है, तो अंतिम स्लाइड का चयन किया जाएगा। यदि इनपुट प्रस्तुति में केवल एक स्लाइड है, तो इस विकल्प को अनदेखा कर दिया जाएगा, और इस एकल स्लाइड को संपादित किया जाएगा। यदि किसी छिपी हुई स्लाइड को संपादित करने के लिए खोलने का प्रयास किया जा रहा है,[`ShowHiddenSlides`](../showhiddenslides) विकल्प 'गलत' पर सेट है, अपवाद फेंक दिया जाएगा।
