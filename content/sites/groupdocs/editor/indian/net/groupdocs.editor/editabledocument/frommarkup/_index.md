## EditableDocument.FromMarkup method

```csharp
public static EditableDocument FromMarkup(string newHtmlContent, 
    IEnumerable<IHtmlResource> resources)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| newHtmlContent | String | स्ट्रिंग, जिसमें कच्चा HTML मार्कअप है, जिसे पार्स किया जाना चाहिए। खाली, खाली या अमान्य नहीं हो सकता। |
| resources | IEnumerable`1 | सभी संसाधनों (छवियों, स्टाइलशीट्स, फोंट) का संग्रह, जो एचटीएमएल-दस्तावेज़ में उपयोग किया जाता है, में निर्दिष्ट है*newHtmlContent* पैरामीटर। अनुपस्थित हो सकता है (शून्य या खाली संग्रह)। |
