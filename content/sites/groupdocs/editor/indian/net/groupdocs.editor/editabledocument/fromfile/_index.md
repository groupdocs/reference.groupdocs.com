## EditableDocument.FromFile method

स्टेटिक फ़ैक्टरी, जो एक HTML फ़ाइल से संपादन योग्य दस्तावेज़ का एक उदाहरण बनाता है, जो कि *.html फ़ाइल के पथ और लिंक किए गए संसाधनों के साथ एक फ़ोल्डर द्वारा निर्दिष्ट किया गया है

```csharp
public static EditableDocument FromFile(string htmlFilePath, string resourceFolderPath)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| htmlFilePath | String | स्ट्रिंग, जिसमें HTML फ़ाइल का पूर्ण पथ शामिल है. रिक्त नहीं हो सकता, मान्य फ़ाइल पथ होना चाहिए, और स्वयं फ़ाइल मौजूद होनी चाहिए. |
| resourceFolderPath | String | HTML संसाधनों वाले फ़ोल्डर का वैकल्पिक पथ. यदि NULL, अमान्य या ऐसा फ़ोल्डर मौजूद नहीं है, तो संपादक HTML मार्कअप का विश्लेषण करते हुए इस फ़ोल्डर को स्वयं खोजने का प्रयास करेगा |

### प्रतिलाभ की मात्रा

संपादन योग्य दस्तावेज़ का नया गैर-शून्य उदाहरण

### अपवाद

| अपवाद | स्थिति |
| --- | --- |
| ArgumentException | HTML फ़ाइल पथ, और/या संसाधन फ़ोल्डर पथ अमान्य है/हैं |
| FileNotFoundException | निर्दिष्ट HTML फ़ाइल नहीं मिली |
