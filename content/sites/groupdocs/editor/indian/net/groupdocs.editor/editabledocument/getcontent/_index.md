## GetContent(string, string) {#getcontent_1}

HTML दस्तावेज़ की समग्र सामग्री को एक स्ट्रिंग के रूप में लौटाता है, जहां बाहरी संसाधनों के लिंक में निर्दिष्ट उपसर्ग होता है।

```csharp
public string GetContent(string externalImagesPrefix, string externalCssPrefix)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| externalImagesPrefix | String | उपयोग किए गए इस पैरामीटर के माध्यम से एक उपसर्ग निर्दिष्ट किया जा सकता है, जो IMG तत्वों में सभी बाहरी छवियों के लिंक में जोड़ा जाएगा, जो परिणामी HTML स्ट्रिंग में मौजूद होगा। यदि NULL या रिक्त है, तो उपसर्ग नहीं जोड़े जाएँगे। |
| externalCssPrefix | String | इस पैरामीटर के माध्यम से उपयोग किया गया एक उपसर्ग निर्दिष्ट कर सकता है, जो लिंक तत्वों में सभी बाहरी स्टाइलशीट के लिंक में जोड़ा जाएगा, जो परिणामी HTML स्ट्रिंग में मौजूद होगा। यदि NULL या रिक्त है, तो उपसर्ग नहीं जोड़े जाएँगे। |
