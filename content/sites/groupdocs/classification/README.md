# GroupDocs.Classification — API Reference Source Files

This repository holds the Markdown source files used to generate the **GroupDocs.Classification**
API reference, published at <https://reference.groupdocs.com/classification/>.

## About GroupDocs.Classification

[GroupDocs.Classification](https://products.groupdocs.com/classification/) is a developer API for document
processing: classify documents and text with IAB-2, Documents, and Sentiment taxonomies. It is available for the following platforms, each with its own
API reference:

- **.NET** — <https://reference.groupdocs.com/classification/net/>
- **Java** — <https://reference.groupdocs.com/classification/java/>

## Repository purpose

The files in this repository are used to:

- Build and maintain the official **GroupDocs.Classification** API reference.
- Keep the reference in sync with the latest product release.
- Provide a consistent structure for automated Hugo builds and publication.

Content is English-only and compiled into a static site with [Hugo](https://gohugo.io/).
