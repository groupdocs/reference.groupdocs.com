## FileType.VTX field

XML-файл шаблона Visio (.vtx) — это шаблон чертежа Microsoft Visio, который сохраняется на диск в формате XML-файла. Шаблон предназначен для предоставления файла с основными настройками, который можно использовать для создания нескольких файлов Visio с одинаковыми настройками.
