## FileType.XLTX field

Los archivos de plantilla de hoja de cálculo XML abierta de Excel (.xltx) se basan en las especificaciones de formato de archivo OpenXML de Office. Se utiliza para crear un archivo de plantilla estándar que se puede utilizar para generar archivos XLSX que muestren la misma configuración que se especifica en el archivo XLTX.
