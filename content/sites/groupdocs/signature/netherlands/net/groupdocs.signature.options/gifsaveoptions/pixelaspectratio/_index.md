## GifSaveOptions.PixelAspectRatio property

Haalt de beeldverhouding van de GIF-pixel op of stelt deze in.

```csharp
public byte PixelAspectRatio { get; set; }
```

### Opmerkingen

Pixel Aspect Ratio - Factor die wordt gebruikt om een benadering te berekenen van de aspect ratio van de pixel in de originele afbeelding. Als de waarde van het veld niet 0 is, wordt deze benadering van de aspectverhouding berekend op basis van de formule: Aspect Ratio = (Pixel Aspect Ratio + 15) / 64 De Pixel Aspect Ratio is gedefinieerd als het quotiënt van de breedte van de pixel over zijn hoogte. De waarde range in dit veld maakt specificatie mogelijk van de breedste pixel van 4:1 tot de hoogste pixel van 1:4 in stappen van 1/64ste. Waarden: 0 - Er wordt geen informatie over de beeldverhouding gegeven. 1..255 - Waarde gebruikt in de berekening.
