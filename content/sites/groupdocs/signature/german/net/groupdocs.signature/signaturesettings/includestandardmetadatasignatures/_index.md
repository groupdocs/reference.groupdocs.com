## SignatureSettings.IncludeStandardMetadataSignatures property

Ruft oder setzt Flag, um die eingebetteten Standard-Metadaten-Signaturen von Dokumenten wie Autor, Besitzer, Erstellungsdatum des Dokuments, Änderungsdatum usw. in die Metadatenliste aufzunehmen. Wenn dieses Flag auf „false“ gesetzt ist (standardmäßig), enthält GetDocumentInfo diese Metadaten nicht Signaturen. Wenn dieses Flag auf „true“ gesetzt ist, enthalten die Dokumentinformationen diese Standard-Metadaten-Signaturen.

```csharp
public bool IncludeStandardMetadataSignatures { get; set; }
```
