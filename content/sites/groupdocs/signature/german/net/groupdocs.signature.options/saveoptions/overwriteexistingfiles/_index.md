## SaveOptions.OverwriteExistingFiles property

Ruft ab oder legt fest, ob eine vorhandene Datei mit einer neuen Ausgabedatei überschrieben werden soll. Andernfalls wird eine neue Datei mit einer Nummer als Suffix erstellt. Standardmäßig ist dieser Wert auf „true“ gesetzt, was bedeutet, dass die Datei überschrieben wird.

public bool OverwriteExistingFiles { get; set; }
