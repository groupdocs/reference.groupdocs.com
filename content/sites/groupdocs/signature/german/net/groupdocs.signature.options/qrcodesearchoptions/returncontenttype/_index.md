## QrCodeSearchOptions.ReturnContentType property

Gibt den Dateityp des zurückgegebenen Bildinhalts der QR-Code-Signatur an, wenn die ReturnContent-Eigenschaft aktiviert ist. Standardmäßig ist sie auf Null gesetzt. Das bedeutet, QR-Code-Bildinhalte im Originalformat zurückzugeben. Dieses Bildformat ist spezifiziert bei[`Format`](../../../groupdocs.signature.domain/qrcodesignature/format) Mögliche unterstützte Werte sind: FileType.JPEG, FileType.PNG, FileType.BMP. Wenn das bereitgestellte Format nicht unterstützt wird, wird der Inhalt des QR-Code-Bildes im ursprünglichen .png-Format zurückgegeben.

public FileType ReturnContentType { get; set; }
