## GifSaveOptions.ColorResolution property

Ruft die GIF-Farbauflösung ab oder legt sie fest.

```csharp
public byte ColorResolution { get; set; }
```

### Bemerkungen

Farbauflösung - Anzahl der Bits pro Primärfarbe, die für das ursprüngliche Bild verfügbar sind, minus 1. Dieser Wert repräsentiert die Größe der gesamten Palette aus , aus der die Farben in der Grafik ausgewählt wurden, nicht die Anzahl der tatsächlich in der Grafik verwendeten Farben. Wenn der Wert in diesem Feld beispielsweise 3 ist, dann hatte die Palette des Originalbildes 4 Bits pro verfügbarer Primärfarbe , um das Bild zu erstellen. Dieser Wert sollte so eingestellt werden, dass er die Reichhaltigkeit von der Originalpalette angibt, auch wenn nicht jede Farbe aus der gesamten Palette auf dem Quellcomputer verfügbar ist.
