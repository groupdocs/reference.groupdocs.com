## LogLevel enumeration

Gibt die verfügbaren Protokollebenentypen an. Diese Enumeration kann als Flag verwendet werden, um mehrere mögliche Werte als aktiviert festzulegen bits Beispiel: LogLevel.Error &#x7C; LogLevel.Warnung oder LogLevel.Fehler &#x7C; LogLevel.Trace

```csharp
[Flags]
public enum LogLevel
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Keine Protokollierungsbeschränkung Alle Informationen werden vom Trace protokolliert, Warnung bis Fehler |
| Error | `1` | Keine Protokollierungsbeschränkung Alle Informationen werden vom Trace protokolliert, Warnung bis Fehler |
| Warning | `2` | Wie alle Level, alle Meldungen einschließlich des Trace-Levels werden protokolliert |
| Trace | `4` | Die Protokollierungsebene zum Einschließen von Meldungen von der Warnungs- bis zur Fehlerebene |
| All | `7` | Alle Ereignisse auf Protokollebene (Fehler, Warnung, Trace) werden in die Protokollierung aufgenommen |
