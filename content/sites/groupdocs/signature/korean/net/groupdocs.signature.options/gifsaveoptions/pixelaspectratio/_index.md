## GifSaveOptions.PixelAspectRatio property

GIF 픽셀 종횡비를 가져오거나 설정합니다.

```csharp
public byte PixelAspectRatio { get; set; }
```

### 비고

픽셀 종횡비 - 원본 이미지에 있는 픽셀의 aspect 비율의 근사치를 계산하는 데 사용되는 요소입니다. 필드 값이 0이 아닌 경우 가로 세로 비율의 근사치는 다음 공식을 기반으로 계산됩니다. 가로 세로 비율 = (픽셀 가로 세로 비율 + 15) / 64 픽셀 가로 세로 비율은 픽셀 너비의 몫으로 정의됩니다. 그것의 높이 이상. 이 필드의 값 range 는 1/64 단위로 4:1의 가장 넓은 픽셀에서 1:4의 가장 큰 픽셀까지 지정할 수 있습니다. 값: 0 - 종횡비 information 가 제공되지 않습니다. 1..255 - 계산에 사용된 값.
