## BarcodeSearchOptions.ReturnContentType property

ReturnContent 속성이 활성화된 경우 바코드 서명의 반환된 이미지 콘텐츠 파일 유형을 지정합니다. 기본적으로 Null로 설정됩니다. 이는 Barcode 이미지 콘텐츠를 원본 형식으로 반환하는 것을 의미합니다. 이 이미지 형식은 다음에 지정됩니다.[`Format`](../../../groupdocs.signature.domain/barcodesignature/format) 가능한 지원 값은 FileType.JPEG, FileType.PNG, FileType.BMP입니다. 제공된 형식이 지원되지 않으면 .png 형식의 바코드 이미지 콘텐츠가 반환됩니다.

public FileType ReturnContentType { get; set; }
