## SignatureSettings.SaveDocumentOnEmptyUpdate property

Update 메서드에 업데이트할 서명이 없을 때 원본 문서를 다시 저장하도록 플래그를 가져오거나 설정합니다. 이 플래그가 true로 설정되면(기본값) 문서는 업데이트하더라도 해당 기록 프로세스 로그(날짜 및 작업 유형)와 함께 저장됩니다. 메서드에 업데이트할 서명이 없습니다. 이 플랫이 false로 설정되면 소스 문서가 전혀 수정되지 않습니다.

```csharp
public bool SaveDocumentOnEmptyUpdate { get; set; }
```
