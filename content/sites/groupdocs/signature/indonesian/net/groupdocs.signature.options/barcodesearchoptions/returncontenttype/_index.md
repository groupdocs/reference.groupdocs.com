## BarcodeSearchOptions.ReturnContentType property

Menentukan jenis file dari konten gambar yang dikembalikan dari tanda tangan Barcode saat properti ReturnContent diaktifkan. Secara default diatur ke Null. Itu artinya mengembalikan konten gambar Barcode dalam format aslinya. Format gambar ini ditentukan di[`Format`](../../../groupdocs.signature.domain/barcodesignature/format) Kemungkinan nilai yang didukung adalah: FileType.JPEG, FileType.PNG, FileType.BMP. Jika format yang diberikan tidak didukung maka konten gambar Barcode dalam format .png akan dikembalikan.
