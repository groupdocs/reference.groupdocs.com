### Nilai

| Nama | Nilai | Keterangan |
| --- | --- | --- |
| Auto | `0` | Secara otomatis mengambil mode penyandian terbaik untuk penyandian DataMatrix |
| ASCII | `1` | Mengkodekan satu karakter alfanumerik atau dua numerik per byte |

| Custom | `7` | Enkode dengan pengkodean yang ditentukan di BarcodeGenerator.Parameters.Barcode.DataMatrix.CodeTextEncoding |

| EDIFACT | `10` | Menggunakan pengkodean EDIFACT. Menggunakan enam bit per karakter, menyandikan angka, huruf besar, dan banyak tanda baca, tetapi tidak mendukung huruf kecil. |
| ANSIX12 | `11` | Menggunakan penyandian ANSI X12. |
