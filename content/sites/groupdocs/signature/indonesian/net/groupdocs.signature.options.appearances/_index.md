Ruang nama menyediakan kelas untuk tampilan tanda tangan tambahan.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [DigitalSignatureAppearance](./digitalsignatureappearance) | Menjelaskan tampilan Baris Tanda Tangan untuk Tanda Tangan Digital. Satu Baris Tanda Tangan hanya dapat diterapkan untuk satu Tanda Tangan Digital. Baris Tanda Tangan selalu ada di halaman pertama. Fitur ini mungkin berguna untuk format file .docx, .doc, .odt dan .xlsx. |
| [ImageAppearance](./imageappearance) | Menjelaskan fitur tampilan yang diperluas untuk Tanda Tangan Gambar. |

| [PdfTextStickerAppearance](./pdftextstickerappearance) | Menjelaskan tampilan objek stiker anotasi teks PDF dan jendela pop-up stiker. |
