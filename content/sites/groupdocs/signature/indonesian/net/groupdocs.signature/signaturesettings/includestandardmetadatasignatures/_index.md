## SignatureSettings.IncludeStandardMetadataSignatures property

Mendapat atau menyetel bendera untuk disertakan ke dalam Daftar Metadata tanda tangan metadata dokumen standar tersemat seperti Penulis, Pemilik, tanggal pembuatan dokumen, tanggal modifikasi, dll. Jika bendera ini disetel ke salah (secara default), GetDocumentInfo tidak akan menyertakan metadata ini signatures. Ketika flag ini disetel ke true, informasi dokumen akan menyertakan tanda tangan metadata standar ini.

```csharp
public bool IncludeStandardMetadataSignatures { get; set; }
```
