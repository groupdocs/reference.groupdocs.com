## SignatureSettings.SaveDocumentOnEmptyUpdate property

Mendapat atau menyetel bendera untuk menyimpan kembali dokumen sumber saat metode Pembaruan tidak memiliki tanda tangan untuk diperbarui. Jika bendera ini disetel ke true (secara default), dokumen akan disimpan dengan log proses riwayat yang sesuai (tanggal dan jenis operasi) meskipun Pembaruan metode tidak memiliki tanda tangan untuk diperbarui. Ketika flat ini disetel ke dokumen sumber palsu tidak akan dimodifikasi sama sekali.

public bool SaveDocumentOnEmptyUpdate { get; set; }
