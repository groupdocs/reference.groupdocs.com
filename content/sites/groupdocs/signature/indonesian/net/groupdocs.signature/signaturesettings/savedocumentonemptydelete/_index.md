## SignatureSettings.SaveDocumentOnEmptyDelete property

Mendapat atau menyetel bendera untuk menyimpan kembali dokumen sumber saat metode Hapus tidak memiliki tanda tangan yang terpengaruh untuk dihapus. Jika bendera ini disetel ke true (secara default), dokumen akan disimpan dengan log proses riwayat yang sesuai (tanggal dan jenis operasi) bahkan jika Metode hapus tidak memiliki tanda tangan untuk dihapus. Ketika flat ini disetel ke dokumen sumber palsu tidak akan diubah sama sekali.

public bool SaveDocumentOnEmptyDelete { get; set; }
