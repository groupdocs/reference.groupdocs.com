## LogLevel enumeration

Menentukan jenis Level Log yang tersedia. Pencacahan ini dapat digunakan sebagai flag untuk menetapkan beberapa kemungkinan nilai sebagai bit yang diaktifkan Contoh: LogLevel.Error &#x7C; LogLevel.Peringatan atau LogLevel.Error &#x7C; LogLevel.Trace

```csharp
[Flags]
public enum LogLevel
```

### Nilai

| Nama | Nilai | Keterangan |
| --- | --- | --- |
| None | `0` | Tidak ada batasan pencatatan semua informasi akan dicatat dari jejak, peringatan hingga kesalahan |
| Error | `1` | Tidak ada batasan pencatatan semua informasi akan dicatat dari jejak, peringatan hingga kesalahan |
| Warning | `2` | Sama seperti Semua level, semua pesan termasuk level Trace akan dicatat |
| Trace | `4` | Level logging untuk memasukkan pesan dari level Peringatan ke Error |
| All | `7` | Semua peristiwa level Log (Kesalahan, Peringatan, Pelacakan) akan dimasukkan ke dalam logging |
