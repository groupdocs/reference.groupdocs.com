## BarcodeSearchOptions.ReturnContent property

Получает или устанавливает флаг для получения содержимого изображения штрих-кода подписи на странице документа. Если для этого флага установлено значение true, содержимое изображения подписи штрих-кода будет хранить необработанные данные изображения в требуемом формате.[`ReturnContentType`](../returncontenttype) . По умолчанию эта опция отключена.

public bool ReturnContent { get; set; }
