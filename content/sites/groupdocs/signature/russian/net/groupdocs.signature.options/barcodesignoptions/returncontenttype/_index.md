## BarcodeSignOptions.ReturnContentType property

Указывает тип файла возвращаемого содержимого изображения подписи штрих-кода, когда свойство ReturnContent включено. По умолчанию установлено значение Null. Это означает возврат содержимого изображения штрих-кода в исходном формате. Этот формат изображения указан в[`Format`](../../../groupdocs.signature.domain/barcodesignature/format) Возможные поддерживаемые значения: FileType.JPEG, FileType.PNG, FileType.BMP. Если предоставленный формат не поддерживается, будет возвращено содержимое изображения штрих-кода в формате .png.

public FileType ReturnContentType { get; set; }
