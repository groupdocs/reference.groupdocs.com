## SaveOptions.OverwriteExistingFiles property

Получает или задает, перезаписывать ли существующий файл новым выходным файлом. В противном случае будет создан новый файл с номером в качестве суффикса. По умолчанию для этого значения установлено значение true, что означает, что файл будет перезаписан.

public bool OverwriteExistingFiles { get; set; }
