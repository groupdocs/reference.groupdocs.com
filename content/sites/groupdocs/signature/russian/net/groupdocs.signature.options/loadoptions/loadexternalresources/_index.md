## LoadOptions.LoadExternalResources property

Получает или задает параметры, определяющие, следует ли загружать внешние ресурсы документа. Этот параметр с отключенным значением (false) позволяет сократить время загрузки документов с большим количеством внешних ссылок на ресурсы. По умолчанию значение включено (true).

public bool LoadExternalResources { get; set; }
