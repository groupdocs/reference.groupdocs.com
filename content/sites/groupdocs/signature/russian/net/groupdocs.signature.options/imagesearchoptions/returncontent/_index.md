## ImageSearchOptions.ReturnContent property

Получает или устанавливает флаг для захвата содержимого изображения подписи на странице документа. Если этот флаг установлен в значение true, содержимое подписи изображения будет хранить необработанные данные изображения в требуемом формате.[`ReturnContentType`](../returncontenttype) . По умолчанию эта опция отключена.

public bool ReturnContent { get; set; }
