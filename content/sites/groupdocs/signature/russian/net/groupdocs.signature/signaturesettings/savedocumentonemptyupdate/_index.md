## SignatureSettings.SaveDocumentOnEmptyUpdate property

Получает или устанавливает флаг для повторного сохранения исходного документа, когда метод обновления не имеет сигнатур для обновления. нет подписей для обновления. Если для этой квартиры задано значение false, исходный документ вообще не будет изменен.
