## SignatureSettings.SaveDocumentOnEmptyDelete property

Получает или устанавливает флаг для повторного сохранения исходного документа, когда метод Delete не имеет затронутых подписей для удаления. У метода удаления нет подписей для удаления. Если для этой квартиры установлено значение false, исходный документ вообще не будет изменен.

```csharp
public bool SaveDocumentOnEmptyDelete { get; set; }
```
