## SignatureSettings.SaveDocumentOnEmptyDelete property

Получает или устанавливает флаг для повторного сохранения исходного документа, когда метод Удалить не имеет затронутых подписей для удаления. Если для этого флага установлено значение true (по умолчанию), документ будет сохраняться с соответствующим журналом обработки истории (дата и тип операции), даже если метод Удалить не имеет подписей для удаления. Если для этой квартиры задано значение false, исходный документ вообще не будет изменен.

```csharp
public bool SaveDocumentOnEmptyDelete { get; set; }
```
