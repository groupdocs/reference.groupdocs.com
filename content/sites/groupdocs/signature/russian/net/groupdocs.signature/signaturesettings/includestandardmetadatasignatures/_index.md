## SignatureSettings.IncludeStandardMetadataSignatures property

Получает или устанавливает флаг для включения в список метаданных встроенных стандартных подписей метаданных документа, таких как автор, владелец, дата создания документа, дата изменения и т. д. Если для этого флага установлено значение false (по умолчанию), GetDocumentInfo не будет включать эти метаданные подписи. Если для этого флага установлено значение true, информация о документе будет включать эти стандартные подписи метаданных.

```csharp
public bool IncludeStandardMetadataSignatures { get; set; }
```
