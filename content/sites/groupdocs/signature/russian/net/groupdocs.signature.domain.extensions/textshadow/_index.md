## TextShadow class

Представляет свойства тени текста для текстовых подписей. Результат может различаться в зависимости от типа подписи и формата документа. TextShadow рекомендуется для использования с подписью TextAsImage для всех поддерживаемых типов документов, также с простой TextSignature и TextSignature в качестве водяного знака для электронных таблиц ( .xslx) и Презентации (.pptx). Простая текстовая подпись для Word (.docx) также рекомендуется, но имеет ограниченную функциональность.

public class TextShadow : SignatureExtension
