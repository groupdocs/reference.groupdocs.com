## LogLevel enumeration

Указывает доступные типы уровней журнала. Это перечисление можно использовать в качестве флагов для установки нескольких возможных значений в качестве разрешенных битов Пример: LogLevel.Error &#x7C; LogLevel.Warning или LogLevel.Error &#x7C; LogLevel.Trace

```csharp
[Flags]
public enum LogLevel
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| None | `0` | Нет ограничений на журналирование, вся информация будет записываться из трассировки, предупреждение об ошибках |
| Error | `1` | Нет ограничений на журналирование, вся информация будет записываться из трассировки, предупреждение об ошибках |
| Warning | `2` | То же, что и уровень All, все сообщения, включая уровень Trace, будут регистрироваться |
| Trace | `4` | Уровень ведения журнала для включения сообщений с уровня "Предупреждение" до "Ошибка" |
| All | `7` | Все события уровня журнала (ошибка, предупреждение, трассировка) будут включены в logging |
