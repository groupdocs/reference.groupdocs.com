# GroupDocs.Signature — API Reference Source Files

This repository holds the Markdown source files used to generate the **GroupDocs.Signature**
API reference, published at <https://reference.groupdocs.com/signature/>.

## About GroupDocs.Signature

[GroupDocs.Signature](https://products.groupdocs.com/signature/) is a developer API for document
processing: sign and verify with digital, barcode, QR-code, text, image, and stamp signatures. It is available for the following platforms, each with its own
API reference:

- **.NET** — <https://reference.groupdocs.com/signature/net/>
- **Java** — <https://reference.groupdocs.com/signature/java/>
- **Node.js via Java** — <https://reference.groupdocs.com/signature/nodejs-java/>
- **Python via .NET** — <https://reference.groupdocs.com/signature/python-net/>

## Repository purpose

The files in this repository are used to:

- Build and maintain the official **GroupDocs.Signature** API reference.
- Keep the reference in sync with the latest product release.
- Provide a consistent structure for automated Hugo builds and publication.

Content is English-only and compiled into a static site with [Hugo](https://gohugo.io/).
