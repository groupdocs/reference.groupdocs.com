## DigitalSignatureAppearance class

Décrit l'apparence de la ligne de signature pour la signature numérique. Une ligne de signature peut être appliquée pour une seule signature numérique. La ligne de signature est toujours sur la première page. Cette fonctionnalité peut être utile pour les formats de fichiers .docx, .doc, .odt et .xlsx.
