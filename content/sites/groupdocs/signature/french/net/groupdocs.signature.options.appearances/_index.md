L'espace de noms fournit des classes pour des apparences de signature supplémentaires.

## Des classes

| Classer | La description |
| --- | --- |
| [DigitalSignatureAppearance](./digitalsignatureappearance) | Décrit l'apparence de la ligne de signature pour la signature numérique. Une ligne de signature peut être appliquée pour une seule signature numérique. La ligne de signature est toujours sur la première page. Cette fonctionnalité peut être utile pour les formats de fichiers .docx, .doc, .odt et .xlsx. |
| [ImageAppearance](./imageappearance) | Décrit les fonctionnalités d'apparence étendues pour Image Signature. |
| [PdfDigitalSignatureAppearance](./pdfdigitalsignatureappearance) | Décrit l'apparence de la signature numérique sur les documents PDF. |
| [PdfTextAnnotationAppearance](./pdftextannotationappearance) | Décrit l'apparence de l'objet d'annotation de texte PDF (Titre, Objet, Contenu). |
| [PdfTextStickerAppearance](./pdftextstickerappearance) | Décrit l'apparence de l'objet autocollant d'annotation de texte PDF et de la fenêtre contextuelle de l'autocollant. |
| [SignatureAppearance](./signatureappearance) | Représente l'apparence de la signature - options supplémentaires pour des implémentations alternatives de signature sur la page du document. |
