## LogLevel enumeration

Spécifie les types de niveau de journalisation disponibles. Cette énumération peut être utilisée comme indicateur pour définir plusieurs valeurs possibles comme bits activés Exemple : LogLevel.Error &#x7C; LogLevel.Warning ou LogLevel.Error &#x7C; LogLevel.Trace

```csharp
[Flags]
public enum LogLevel
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| None | `0` | Aucune limitation de journalisation toutes les informations seront enregistrées à partir de la trace, avertissement aux erreurs |
| Error | `1` | Aucune limitation de journalisation toutes les informations seront enregistrées à partir de la trace, avertissement aux erreurs |
| Warning | `2` | Identique au niveau Tous, tous les messages, y compris le niveau Trace, seront consignés |
| Trace | `4` | Le niveau de journalisation pour inclure les messages du niveau d'avertissement au niveau d'erreur |
| All | `7` | Tous les événements de niveau journal (Erreur, Avertissement, Trace) seront inclus dans la journalisation |
