### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| None | `0` | Le filtre nul signifie qu'il n'y a pas de filtrage pour les lignes de données d'image. |
| Sub | `1` | Le sous-filtre signifie qu'un filtrage soustractif sera appliqué aux données d'image. |

| Paeth | `4` | Le filtre prédicteur de chemin. |
| Adaptive | `5` | Filtrage adaptatif, signifie que le processus de sauvegarde choisira le filtre le plus approprié pour chaque ligne de données. Meilleure compression, temps d'exécution le plus lent. |
