## LoadOptions.LoadExternalResources property

Obtient ou définit les options qui spécifient si les ressources de documents externes doivent être chargées. Cette option avec une valeur désactivée (false) permet de gagner du temps de chargement pour les documents avec de nombreux ou grands liens de ressources externes. Par défaut, la valeur est activée (true).
