## FileType.CDR field

CorelDraw Vector Graphic Drawing (.cdr) è un file immagine di disegno vettoriale creato nativamente con CorelDRAW per la memorizzazione di immagini digitali codificate e compresse. Tale file di disegno contiene testo, linee, forme, immagini, colori ed effetti per la rappresentazione vettoriale del contenuto dell'immagine.
