## DigitalSignatureAppearance class

Descrive l'aspetto della riga della firma per la firma digitale. È possibile applicare una riga della firma per una sola firma digitale. La riga della firma è sempre sulla prima pagina. Questa funzione può essere utile per i formati di file .docx, .doc, .odt e .xlsx.
