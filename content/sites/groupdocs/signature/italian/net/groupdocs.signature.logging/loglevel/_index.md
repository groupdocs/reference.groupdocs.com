## LogLevel enumeration

Specifica i tipi di livello di registro disponibili. Questa enumerazione può essere utilizzata come flag per impostare diversi valori possibili come bit abilitati Esempio: LogLevel.Error &#x7C; LogLevel.Warning o LogLevel.Error &#x7C; LogLevel.Trace

```csharp
[Flags]
public enum LogLevel
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | `0` | Nessuna limitazione di registrazione tutte le informazioni verranno registrate dalla traccia, dall'avviso agli errori |
| Error | `1` | Nessuna limitazione di registrazione tutte le informazioni verranno registrate dalla traccia, dall'avviso agli errori |
| Warning | `2` | Uguale al livello Tutti, verranno registrati tutti i messaggi incluso il livello Trace |
| Trace | `4` | Il livello di registrazione per includere i messaggi dal livello Avviso a quello Errore |
| All | `7` | Tutti gli eventi a livello di registro (errore, avviso, traccia) verranno inclusi nella registrazione |
