## SignatureSettings.SaveDocumentOnEmptyDelete property

Delete メソッドが削除する影響を受ける署名を持たない場合に、ソース ドキュメントを再保存するためのフラグを取得または設定します。 Delete メソッドには削除する署名がありません。 このフラットが false に設定されている場合、ソース ドキュメントはまったく変更されません。

public bool SaveDocumentOnEmptyDelete { get; set; }
