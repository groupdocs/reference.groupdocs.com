## SignatureSettings.SaveDocumentOnEmptyUpdate property

Update メソッドに更新する署名がない場合に、ソース ドキュメントを再保存するためのフラグを取得または設定します。 このフラグが true (デフォルト) に設定されている場合、Updateメソッドには更新する署名がありません. このフラットが false に設定されている場合、ソース ドキュメントはまったく変更されません.

```csharp
public bool SaveDocumentOnEmptyUpdate { get; set; }
```
