## BarcodeSignOptions.ReturnContentType property

ReturnContent プロパティが有効な場合に返されるバーコード署名のイメージ コンテンツのファイル タイプを指定します。 デフォルトでは Null に設定されています。これは、バーコード イメージ コンテンツを元の形式で返すことを意味します。 この画像フォーマットは[`Format`](../../../groupdocs.signature.domain/barcodesignature/format) サポートされている可能性のある値は、FileType.JPEG、FileType.PNG、FileType.BMP です。 提供された形式がサポートされていない場合、.png 形式のバーコード イメージ コンテンツが返されます。

public FileType ReturnContentType { get; set; }
