### 備考

ピクセル アスペクト比 - 元の画像のピクセルのアスペクト 比の近似値を計算するために使用される係数。フィールドの値が 0 でない場合、このアスペクト比の近似値は次の式に基づいて計算されます: アスペクト比 = (ピクセル アスペクト比 + 15) / 64 ピクセル アスペクト比は ピクセルの幅の商として定義されますその高さを超えています。このフィールドの値 range により、4:1 の最も広いピクセルから 1:4 の最も高い ピクセルまでを 1/64 単位で指定できます。値 : 0 - アスペクト比情報 は指定されません。 1..255 - 計算で使用される値.
