## GifSaveOptions.ColorResolution property

GIF の色解像度を取得または設定します。

```csharp
public byte ColorResolution { get; set; }
```

### 備考

色解像度 - 元の 画像で使用できる原色あたりのビット数から 1 を引いた値。この値は、グラフィックで実際に 使用されている色の数ではなく、グラフィックの色が選択された からのパレット全体のサイズを表します。たとえば、このフィールドの値が 3 の場合、 元の画像のパレットには、画像を作成するために使用可能な原色あたり 4 ビット がありました。この値は、ソース マシンで パレット全体のすべての色が使用可能ではない 場合でも、元のパレットの豊富さを示すように設定する必要があります.
