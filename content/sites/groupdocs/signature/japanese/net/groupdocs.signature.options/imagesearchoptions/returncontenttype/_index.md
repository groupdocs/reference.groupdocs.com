## ImageSearchOptions.ReturnContentType property

ReturnContent プロパティが有効な場合に返される画像署名の内容のファイル タイプを指定します。 デフォルトでは Null に設定されています。これは、画像コンテンツを元の形式で返すことを意味します。 この画像フォーマットは[`Format`](../../../groupdocs.signature.domain/imagesignature/format) サポートされている可能性のある値は、FileType.JPEG、FileType.PNG、FileType.BMP です。 提供された形式がサポートされていない場合、元の形式の画像コンテンツが返されます.

public FileType ReturnContentType { get; set; }
