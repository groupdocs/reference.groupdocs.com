## LogLevel enumeration

利用可能なログ レベル タイプを指定します。 この列挙をフラグとして使用して、いくつかの可能な値を有効に設定できます。 bits 例: LogLevel.Error &#x7C; LogLevel.Warning または LogLevel.Error &#x7C; LogLevel.Trace

```csharp
[Flags]
public enum LogLevel
```

### 値

| 名前 | 価値 | 説明 |
| --- | --- | --- |
| None | `0` | ログの制限なし すべての情報がトレースからログに記録され、エラーへの警告が表示されます |
| Error | `1` | ログの制限なし すべての情報がトレースからログに記録され、エラーへの警告が表示されます |
| Warning | `2` | All レベルと同じ、Trace レベルを含むすべてのメッセージがログに記録されます |
| Trace | `4` | Warning から Error level までのメッセージを含むログ レベル |
| All | `7` | すべてのログ レベル イベント (エラー、警告、トレース) が logging に含まれます。 |
