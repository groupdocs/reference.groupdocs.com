## TextShadow class

テキスト署名のテキスト シャドウ プロパティを表します。 結果は、署名の種類とドキュメント形式によって異なる場合があります。 TextShadow は、サポートされているすべてのドキュメントの種類で TextAsImage 署名と共に使用することをお勧めします。 単純な TextSignature と TextSignature をスプレッドシートの透かしとして使用することもお勧めします ( .xslx) およびプレゼンテーション (.pptx). Word 用の単純な TextSignature (.docx) も推奨されますが、機能が制限されています.

```csharp
public class TextShadow : SignatureExtension
```
