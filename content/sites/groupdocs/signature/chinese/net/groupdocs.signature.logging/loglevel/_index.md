## LogLevel enumeration

指定可用的日志级别类型。 此枚举可用作标志以将几个可能的值设置为启用位 示例：LogLevel.Error &#x7C; LogLevel.Warning 或 LogLevel.Error &#x7C;日志级别.Trace

```csharp
[Flags]
public enum LogLevel
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| None | `0` | 没有日志记录限制所有信息将从跟踪记录，警告错误 |
| Error | `1` | 没有日志记录限制所有信息将从跟踪记录，警告错误 |
| Warning | `2` | 与所有级别相同，将记录包括跟踪级别在内的所有消息 |
| Trace | `4` | 包含从警告到错误级别的消息的日志记录级别 |
| All | `7` | 所有日志级别事件（错误、警告、跟踪）都将包含在 logging |
