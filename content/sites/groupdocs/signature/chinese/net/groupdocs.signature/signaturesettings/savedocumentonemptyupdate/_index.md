## SignatureSettings.SaveDocumentOnEmptyUpdate property

获取或设置标志以在 Update 方法没有要更新的签名时重新保存源文档。 如果此标志设置为 true（默认情况下）文档将与相应的历史进程日志（日期和操作类型）一起保存，即使 Update method has no signatures to update. 当这个 flat 设置为 false source document 根本不会被修改.

public bool SaveDocumentOnEmptyUpdate { get; set; }
