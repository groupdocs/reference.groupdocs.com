## SignatureSettings.IncludeStandardMetadataSignatures property

获取或设置标志以将嵌入的标准文档元数据签名（如作者、所有者、文档创建日期、修改日期等）包含到元数据列表中。 如果此标志设置为 false（默认情况下），GetDocumentInfo 将不包含这些元数据signatures. 当此标志设置为 true 时，文档信息将包括这些标准元数据 signatures.

```csharp
public bool IncludeStandardMetadataSignatures { get; set; }
```
