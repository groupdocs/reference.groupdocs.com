## GifSaveOptions.ColorResolution property

获取或设置 GIF 颜色分辨率。

```csharp
public byte ColorResolution { get; set; }
```

### 评论

颜色分辨率 - original 图像可用的每种原色的位数，减 1。此值表示从 中选择图形中的颜色的整个调色板的大小，而不是图形中实际使用的 颜色的数量。例如，如果此字段中的值为 3，则 原始图像的调色板有 4 位可用的原色 来创建图像。此值应设置为指示 原始调色板的丰富性，即使并非整个调色板中的每种颜色在源计算机上都可用 。
