## LoadOptions.LoadExternalResources property

Obtiene o establece opciones que especifican si se deben cargar recursos de documentos externos. Esta opción con valor deshabilitado (falso) permite ahorrar tiempo de carga para los documentos con muchos o grandes enlaces de recursos externos. Por defecto, el valor está habilitado (verdadero).

public bool LoadExternalResources { get; set; }
