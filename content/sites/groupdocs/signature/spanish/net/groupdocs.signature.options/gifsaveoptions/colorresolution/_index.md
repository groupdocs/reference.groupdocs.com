## GifSaveOptions.ColorResolution property

Obtiene o establece la resolución de color del GIF.

```csharp
public byte ColorResolution { get; set; }
```

### Observaciones

Resolución de color : número de bits por color primario disponibles para la imagen original, menos 1. Este valor representa el tamaño de toda la paleta desde en la que se seleccionaron los colores del gráfico, no el número de colores realmente utilizados en el gráfico. Por ejemplo, si el valor de este campo es 3, entonces la paleta de la imagen original tenía disponibles 4 bits por color primario para crear la imagen. Este valor debe configurarse para indicar la riqueza de la paleta original, incluso si no todos los colores de la paleta completa están disponibles en la máquina de origen.
