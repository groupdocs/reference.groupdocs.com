## GifSaveOptions.DoPaletteCorrection property

Obtiene o establece un valor que indica si se aplica la corrección de paleta.

```csharp
public bool DoPaletteCorrection { get; set; }
```

### Observaciones

La corrección de la paleta significa que cada vez que la imagen se exporta a GIF, los colores de la imagen de origen se analizarán para crear la paleta que mejor se adapte (en caso de que la Paleta de imagen no exista o no esté especificada en las opciones). El proceso de análisis lleva algo de tiempo, sin embargo, la imagen de salida tendrá la paleta de colores que mejor coincida con y el resultado será visualmente mejor.
