## SaveOptions.OverwriteExistingFiles property

Obtiene o establece si se sobrescribe el archivo existente con un nuevo archivo de salida. De lo contrario, se creará un nuevo archivo con el número como sufijo. De forma predeterminada, este valor se establece en verdadero, lo que significa que se sobrescribirá el archivo.

public bool OverwriteExistingFiles { get; set; }
