## BarcodeSignOptions.ReturnContent property

Obtiene o establece el indicador para obtener el contenido de la imagen del código de barras de una firma que se colocó en la página del documento. Si este indicador se establece como verdadero, el contenido de la imagen de la firma del código de barras mantendrá los datos de imagen sin procesar en el formato requerido[`ReturnContentType`](../returncontenttype) . Por defecto esta opción está deshabilitada.

public bool ReturnContent { get; set; }
