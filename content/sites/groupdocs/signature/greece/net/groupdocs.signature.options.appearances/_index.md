Ο χώρος ονομάτων παρέχει κλάσεις για επιπλέον εμφανίσεις υπογραφών.

## Τάξεις

| Τάξη | Περιγραφή |
| --- | --- |
| [DigitalSignatureAppearance](./digitalsignatureappearance) | Περιγράφει την εμφάνιση της γραμμής υπογραφής για ψηφιακή υπογραφή. Μία γραμμή υπογραφής θα μπορούσε να εφαρμοστεί μόνο για μία ψηφιακή υπογραφή. Η γραμμή υπογραφής βρίσκεται πάντα στην πρώτη σελίδα. Αυτή η δυνατότητα μπορεί να είναι χρήσιμη για μορφές αρχείων .docx, .doc, .odt και .xlsx. |
| [ImageAppearance](./imageappearance) | Περιγράφει εκτεταμένες δυνατότητες εμφάνισης για το Image Signature. |
| [PdfDigitalSignatureAppearance](./pdfdigitalsignatureappearance) | Περιγράφει την εμφάνιση της ψηφιακής υπογραφής σε έγγραφα PDF. |
| [PdfTextAnnotationAppearance](./pdftextannotationappearance) | Περιγράφει την εμφάνιση του αντικειμένου σχολιασμού κειμένου PDF (Τίτλος, Θέμα, Περιεχόμενο). |
| [PdfTextStickerAppearance](./pdftextstickerappearance) | Περιγράφει την εμφάνιση του αντικειμένου αυτοκόλλητου σχολιασμού κειμένου PDF και του αναδυόμενου παραθύρου του αυτοκόλλητου. |
| [SignatureAppearance](./signatureappearance) | Αντιπροσωπεύει την εμφάνιση της υπογραφής - πρόσθετες επιλογές για εναλλακτικές υλοποιήσεις σήματος στη σελίδα εγγράφου. |
