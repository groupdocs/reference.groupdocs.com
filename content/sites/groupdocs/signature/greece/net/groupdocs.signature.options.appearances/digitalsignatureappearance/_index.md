## DigitalSignatureAppearance class

Περιγράφει την εμφάνιση της γραμμής υπογραφής για ψηφιακή υπογραφή. Μία γραμμή υπογραφής θα μπορούσε να εφαρμοστεί μόνο για μία ψηφιακή υπογραφή. Η γραμμή υπογραφής βρίσκεται πάντα στην πρώτη σελίδα. Αυτή η δυνατότητα μπορεί να είναι χρήσιμη για μορφές αρχείων .docx, .doc, .odt και .xlsx.

public sealed class DigitalSignatureAppearance : SignatureAppearance
