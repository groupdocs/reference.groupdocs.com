## DataMatrixEncodeMode enumeration

Λειτουργία κωδικοποίησης κωδικοποιητή DataMatrix, προεπιλογή σε Auto

```csharp
public enum DataMatrixEncodeMode
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| Auto | `0` | Αυτόματη λήψη της καλύτερης λειτουργίας κωδικοποίησης για DataMatrix encoding |
| ASCII | `1` | Κωδικοποιεί έναν αλφαριθμητικό ή δύο αριθμητικούς χαρακτήρες ανά byte |
| Full | `6` | Κωδικοποίηση τιμών 8 bit |
| Custom | `7` | Κωδικοποιήστε με την κωδικοποίηση που καθορίζεται στο BarcodeGenerator.Parameters.Barcode.DataMatrix.CodeTextEncoding |
| C40 | `8` | Χρησιμοποιεί κωδικοποίηση C40. Κωδικοποιεί κεφαλαία αλφαριθμητικά, πεζά και ειδικούς χαρακτήρες |
| Text | `9` | Χρησιμοποιεί κωδικοποίηση κειμένου. Κωδικοποιεί πεζά αλφαριθμητικά, κεφαλαία και ειδικούς χαρακτήρες. |
| EDIFACT | `10` | Χρησιμοποιεί την κωδικοποίηση EDIFACT. Χρησιμοποιεί έξι bit ανά χαρακτήρα, κωδικοποιεί ψηφία, κεφαλαία γράμματα και πολλά σημεία στίξης, αλλά δεν υποστηρίζει πεζά γράμματα. |
| ANSIX12 | `11` | Χρησιμοποιεί κωδικοποίηση ANSI X12. |
| ExtendedCodetext | `12` | Η λειτουργία ExtendedCodetext επιτρέπει τη μη αυτόματη εναλλαγή σχημάτων κωδικοποίησης σε κείμενο κώδικα. Μορφή : "\Encodation_scheme_name:text\Encodation_scheme_name:text". Επιτρεπόμενα σχήματα κωδικοποίησης είναι: EDIFACT, ANS_0, EXT. παράδειγμα κειμένου: @"\ansix12:ANSIX12TEXT\ascii:backslash πρέπει να είναι \\ doubled\edifact:EdifactEncodedText" Όλες οι ανάστροφες κάθετες (\) πρέπει να διπλασιαστούν στο κείμενο. |
