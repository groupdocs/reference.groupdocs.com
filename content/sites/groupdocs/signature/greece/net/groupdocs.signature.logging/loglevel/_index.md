## LogLevel enumeration

Καθορίζει τους διαθέσιμους τύπους επιπέδου καταγραφής. Αυτή η απαρίθμηση μπορεί να χρησιμοποιηθεί ως σημαίες για να ορίσετε πολλές πιθανές τιμές ως ενεργοποιημένα bits Παράδειγμα: LogLevel.Error &#x7C; LogLevel.Warning ή LogLevel.Error &#x7C; LogLevel.Trace

```csharp
[Flags]
public enum LogLevel
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| None | `0` | Χωρίς περιορισμό καταγραφής, όλες οι πληροφορίες θα καταγράφονται από το ίχνος, προειδοποίηση για errors |
| Error | `1` | Χωρίς περιορισμό καταγραφής, όλες οι πληροφορίες θα καταγράφονται από το ίχνος, προειδοποίηση για errors |
| Warning | `2` | Ίδιο με το Όλο το επίπεδο, όλα τα μηνύματα συμπεριλαμβανομένου του επιπέδου παρακολούθησης θα καταγραφούν |
| Trace | `4` | Το επίπεδο καταγραφής που περιλαμβάνει μηνύματα από το επίπεδο προειδοποίησης έως σφάλματος |
| All | `7` | Όλα τα συμβάντα επιπέδου καταγραφής (Σφάλμα, Προειδοποίηση, Ιχνηλάτηση) θα συμπεριληφθούν στο logging |
