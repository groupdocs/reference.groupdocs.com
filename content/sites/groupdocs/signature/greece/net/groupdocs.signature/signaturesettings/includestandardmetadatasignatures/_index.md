## SignatureSettings.IncludeStandardMetadataSignatures property

Λαμβάνει ή ορίζει τη σημαία για συμπερίληψη στη Λίστα Μεταδεδομένων των ενσωματωμένων τυπικών υπογραφών μεταδεδομένων εγγράφων όπως Συγγραφέας, Κάτοχος, ημερομηνία δημιουργίας εγγράφου, ημερομηνία τροποποίησης κ.λπ. Εάν αυτή η σημαία έχει οριστεί ως ψευδής (από προεπιλογή), το GetDocumentInfo δεν θα περιλαμβάνει αυτά τα μεταδεδομένα signatures. Όταν αυτή η σημαία οριστεί σε true, οι πληροφορίες του εγγράφου θα περιλαμβάνουν αυτές τις τυπικές υπογραφές μεταδεδομένων.

```csharp
public bool IncludeStandardMetadataSignatures { get; set; }
```
