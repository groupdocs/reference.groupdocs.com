## SaveOptions.OverwriteExistingFiles property

Λαμβάνει ή ορίζει εάν θα αντικατασταθεί το υπάρχον αρχείο με νέο αρχείο εξόδου. Διαφορετικά θα δημιουργηθεί νέο αρχείο με τον αριθμό ως επίθημα. Από προεπιλογή αυτή η τιμή έχει οριστεί σε true που σημαίνει ότι το αρχείο θα αντικατασταθεί.

public bool OverwriteExistingFiles { get; set; }
