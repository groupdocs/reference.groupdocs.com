## QrCodeSignOptions.ReturnContentType property

Καθορίζει τον τύπο αρχείου του επιστρεφόμενου περιεχομένου εικόνας της υπογραφής QR-Code όταν είναι ενεργοποιημένη η ιδιότητα ReturnContent. Από προεπιλογή ορίζεται σε Null. Αυτό σημαίνει να επιστρέψετε το περιεχόμενο εικόνας με κωδικό QR στην αρχική μορφή. Αυτή η μορφή εικόνας καθορίζεται στο[`Format`](../../../groupdocs.signature.domain/qrcodesignature/format) Οι πιθανές υποστηριζόμενες τιμές είναι: FileType.JPEG, FileType.PNG, FileType.BMP. Εάν δεν υποστηρίζεται η παρεχόμενη μορφή, θα επιστραφεί περιεχόμενο εικόνας με κωδικό QR σε μορφή .png.

```csharp
public FileType ReturnContentType { get; set; }
```
