## LoadOptions.LoadExternalResources property

Λαμβάνει ή ορίζει επιλογές που καθορίζουν εάν πρέπει να φορτωθούν εξωτερικοί πόροι εγγράφων. Αυτή η επιλογή με απενεργοποιημένη τιμή (false) επιτρέπει την εξοικονόμηση χρόνου φόρτωσης για τα έγγραφα με πολλούς ή μεγάλους εξωτερικούς συνδέσμους πόρων. Η προεπιλεγμένη τιμή είναι ενεργοποιημένη (true).

public bool LoadExternalResources { get; set; }
