## GifSaveOptions.PixelAspectRatio property

Λαμβάνει ή ορίζει την αναλογία διάστασης εικονοστοιχείων GIF.

```csharp
public byte PixelAspectRatio { get; set; }
```

### Παρατηρήσεις

Αναλογία διαστάσεων εικονοστοιχείου - Συντελεστής που χρησιμοποιείται για τον υπολογισμό μιας προσέγγισης του λόγου διαστάσεων του εικονοστοιχείου στην αρχική εικόνα. Εάν η τιμή του πεδίου δεν είναι 0, αυτή η προσέγγιση του λόγου διαστάσεων υπολογίζεται με βάση τον τύπο: Αναλογία διαστάσεων = (Pixel Aspect Ratio + 15) / 64 Ο λόγος διαστάσεων pixel ορίζεται ως το πηλίκο του πλάτους των pixel πάνω από το ύψος του. Η τιμή range σε αυτό το πεδίο επιτρέπει την προδιαγραφή του ευρύτερου εικονοστοιχείου από 4:1 έως το υψηλότερο pixel του 1:4 σε βήματα του 1/64ου. Τιμές : 0 - Δεν δίνονται πληροφορίες αναλογίας διαστάσεων . 1..255 - Τιμή που χρησιμοποιείται στον υπολογισμό.
