## GifSaveOptions.DoPaletteCorrection property

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν εφαρμόζεται διόρθωση παλέτας.

```csharp
public bool DoPaletteCorrection { get; set; }
```

### Παρατηρήσεις

Διόρθωση παλέτας σημαίνει ότι κάθε φορά που η εικόνα εξάγεται σε GIF, τα χρώματα της εικόνας source θα αναλύονται προκειμένου να δημιουργηθεί η καλύτερη αντίστοιχη παλέτα (σε περίπτωση που η Παλέτα εικόνων δεν υπάρχει ή δεν καθορίζεται στις επιλογές). Η διαδικασία ανάλυσης διαρκεί λίγο χρόνο, ωστόσο η εικόνα εξόδου θα έχει την που ταιριάζει καλύτερα στην παλέτα χρωμάτων και το αποτέλεσμα είναι οπτικά καλύτερο.
