## GifSaveOptions.ColorResolution property

Λαμβάνει ή ορίζει την ανάλυση χρώματος GIF.

```csharp
public byte ColorResolution { get; set; }
```

### Παρατηρήσεις

Ανάλυση χρώματος - Αριθμός bit ανά κύριο χρώμα διαθέσιμα στην αρχική εικόνα, μείον 1. Αυτή η τιμή αντιπροσωπεύει το μέγεθος ολόκληρης της παλέτας από που επιλέχθηκαν τα χρώματα στο γραφικό, όχι τον αριθμό των χρωμάτων που χρησιμοποιούνται στην πραγματικότητα στο γραφικό. Για παράδειγμα, εάν η τιμή σε αυτό το πεδίο είναι 3, τότε η παλέτα της αρχικής εικόνας είχε 4 bit ανά κύριο χρώμα διαθέσιμα για τη δημιουργία της εικόνας. Αυτή η τιμή πρέπει να οριστεί για να υποδεικνύει τον πλούτο της αρχικής παλέτας, ακόμα κι αν δεν είναι διαθέσιμα όλα τα χρώματα από ολόκληρη την παλέτα στο μηχάνημα προέλευσης.
