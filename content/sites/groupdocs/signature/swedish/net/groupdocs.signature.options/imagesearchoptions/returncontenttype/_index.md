## ImageSearchOptions.ReturnContentType property

Anger filtyp för returnerat innehåll i bildsignaturen när ReturnContent-egenskapen är aktiverad. Som standard är den inställd på Null. Det betyder att returnera bildinnehåll i originalformat. Detta bildformat anges på[`Format`](../../../groupdocs.signature.domain/imagesignature/format) Möjliga värden som stöds är: FileType.JPEG, FileType.PNG, FileType.BMP. Om det angivna formatet inte stöds kommer bildinnehåll i originalformat att returneras.

public FileType ReturnContentType { get; set; }
