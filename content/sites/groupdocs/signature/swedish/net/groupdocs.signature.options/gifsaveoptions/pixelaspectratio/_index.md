## GifSaveOptions.PixelAspectRatio property

Hämtar eller ställer in GIF-pixelns bildförhållande.

```csharp
public byte PixelAspectRatio { get; set; }
```

### Anmärkningar

Pixel Aspect Ratio - Faktor som används för att beräkna en approximation av aspekt förhållandet för pixeln i originalbilden. Om fältets värde inte är 0, beräknas denna approximation av bildförhållandet baserat på formeln: Bildförhållande = (Pixelbildförhållande + 15) / 64 Pixelbildförhållandet är definierat att vara kvoten av pixelns bredd över sin höjd. Värdet range i det här fältet tillåter specificering av den bredaste pixeln på 4:1 till den högsta pixeln på 1:4 i steg om 1/64:e. Värden: 0 - Ingen information om bildförhållande ges. 1..255 - Värde som används i beräkningen.
