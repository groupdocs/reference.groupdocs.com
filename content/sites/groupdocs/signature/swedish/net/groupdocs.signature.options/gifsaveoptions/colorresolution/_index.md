## GifSaveOptions.ColorResolution property

Hämtar eller ställer in GIF-färgupplösningen.

```csharp
public byte ColorResolution { get; set; }
```

### Anmärkningar

Färgupplösning - Antal bitar per primärfärg tillgängligt för original -bilden, minus 1. Detta värde representerar storleken på hela paletten from där färgerna i grafiken valdes, inte antalet färger som faktiskt används i grafiken. Till exempel, om värdet i det här fältet är 3, då hade paletten för originalbilden 4 bitar per primärfärg tillgänglig för att skapa bilden. Detta värde bör ställas in för att indikera rikedomen av den ursprungliga paletten, även om inte alla färger från hela paletten är tillgängliga på källmaskinen.
