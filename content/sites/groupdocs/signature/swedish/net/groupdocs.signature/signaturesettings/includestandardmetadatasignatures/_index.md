## SignatureSettings.IncludeStandardMetadataSignatures property

Hämtar eller ställer in flaggan för att inkludera i metadatalistan de inbäddade standarddokumentmetadatasignaturerna som författare, ägare, dokumentets skapandedatum, ändringsdatum, etc. Om denna flagga är inställd på false (som standard) kommer GetDocumentInfo inte att inkludera dessa metadata signaturer. När denna flagga är inställd på sant kommer dokumentinformationen att inkludera dessa standardmetadatasignaturer.

```csharp
public bool IncludeStandardMetadataSignatures { get; set; }
```
