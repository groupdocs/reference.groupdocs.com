## QrCodeSignOptions.ReturnContent property

Gets or sets flag to get QR-Code image content of a signature which was put on document page. If this flag is set true, QR-Code signature image content will keep raw image data by required format [`ReturnContentType`](../returncontenttype). By default this option is disabled.
