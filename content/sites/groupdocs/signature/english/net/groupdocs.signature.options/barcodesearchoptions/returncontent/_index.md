## BarcodeSearchOptions.ReturnContent property

Gets or sets flag to grab Barcode image content of signature on document page. If this flag is set true, Barcode signature image content will keep raw image data by required format [`ReturnContentType`](../returncontenttype). By default this option is disabled.
